## Chapter 1: ノルム空間
**テーマ:** ノルムの公理、距離構造、基本的な例

### §1.1 ノルムの定義と例
**内容:** ノルムの3公理と代表的なノルム空間

- **1. ノルムの定義**
  - Definition 1.1-1 (ノルム：ベクトル空間 $V$ 上の関数 $\|\cdot\| : V \to [0, \infty)$ が (i) $\|x\| = 0 \iff x = 0$（正定値性）；(ii) $\|\alpha x\| = |\alpha|\|x\|$（斉次性）；(iii) $\|x + y\| \leq \|x\| + \|y\|$（三角不等式）；を満たすこと)
  - Definition 1.1-2 (ノルム空間：ノルムが定められたベクトル空間の対 $(V, \|\cdot\|)$)
  - Remark (ノルムは「大きさ」の抽象化であり、線形代数学II（2-1）で扱ったベクトルのノルムを一般化したもの)

- **2. 代表的なノルム空間**
  - Example 1.1-1 ($\mathbb{R}^n$ 上の $p$-ノルム：$\|x\|_p = (\sum |x_i|^p)^{1/p}$（$1 \leq p < \infty$）および $\|x\|_\infty = \max |x_i|$ の計算；$x = (3, -4)$ に対する $\|x\|_1 = 7$, $\|x\|_2 = 5$, $\|x\|_\infty = 4$ の確認)
  - Example 1.1-2 ($C[a,b]$ 上のノルム：$\|f\|_\infty = \sup_{x \in [a,b]} |f(x)|$ が3公理を満たすことの確認；$f(x) = x^2$ に対する $\|f\|_\infty$ の計算)
  - Example 1.1-3 ($\ell^p$ 空間：数列 $x = (x_n)$ に対する $\|x\|_p = (\sum |x_n|^p)^{1/p}$ の定義；$x = (1, 1/2, 1/4, \ldots)$ の $\ell^2$ ノルムの計算)
  - Example 1.1-4 ($L^p$ 空間：測度論（3-3）で構成したルベーグ積分を用いた $\|f\|_p = (\int |f|^p \, d\mu)^{1/p}$ の定義；a.e. 等しい関数の同一視の必要性)
  - Remark ($L^p$ のノルムでは $\|f\|_p = 0$ は $f = 0$ a.e. を意味し、関数そのものではなく同値類がノルム空間の元となること)

### §1.2 ノルムから誘導される距離
**内容:** 距離空間としての構造と三角不等式

- **1. 誘導距離の定義**
  - Definition 1.2-1 (誘導距離：$d(x, y) = \|x - y\|$ により定まる距離関数)
  - Proposition 1.2-1 (誘導距離が距離の公理を満たすこと)
  - Proof (距離の公理の確認：非退化性はノルムの正定値性から、対称性は $\|-z\| = \|z\|$ から、三角不等式はノルムの三角不等式から従う)
  - Example 1.2-1 ($\mathbb{R}^2$ 上の $\|\cdot\|_1$ と $\|\cdot\|_2$ が定める距離：同じ2点 $(0,0)$, $(1,1)$ に対する距離がそれぞれ $2$ と $\sqrt{2}$ になることの計算)

- **2. 距離空間としてのノルム空間**
  - Remark (ノルム空間は距離空間の特別な場合であり、位相空間論II（3-12）で扱った距離空間の理論がそのまま適用できること)
  - Proposition 1.2-2 (ノルムの連続性：写像 $x \mapsto \|x\|$ がノルム位相に関して連続であること)
  - Proof (連続性の証明：逆三角不等式 $|\|x\| - \|y\|| \leq \|x - y\|$ を用いる)
  - Example 1.2-2 (逆三角不等式の適用：$\|x\| = 5$, $\|y\| = 3$ のとき $\|x - y\| \geq 2$ であることの確認)

### §1.3 収束・コーシー列・開集合
**内容:** ノルム空間における位相的概念の整理

- **1. 収束とコーシー列**
  - Definition 1.3-1 (ノルム収束：$\|x_n - x\| \to 0$ のとき $x_n \to x$ と書くこと)
  - Definition 1.3-2 (コーシー列：任意の $\varepsilon > 0$ に対して十分大きな $N$ 以降 $\|x_m - x_n\| < \varepsilon$ となる列)
  - Example 1.3-1 ($C[0,1]$ の上限ノルムでの収束：$f_n(x) = x^n$ が $f(x) = 0$（$0 \leq x < 1$）に各点収束するが、$\|f_n - 0\|_\infty = 1$ より一様ノルムでは収束しないことの確認)
  - Remark (各点収束とノルム収束の違い：ノルム収束は「すべての点で一様に」近づくことを要求する)

- **2. 開集合と閉集合**
  - Definition 1.3-3 (開球：$B(x, r) = \{y \in V \mid \|x - y\| < r\}$)
  - Definition 1.3-4 (開集合・閉集合：距離空間としての定義を適用)
  - Example 1.3-2 ($\mathbb{R}^2$ 上の $\|\cdot\|_1$, $\|\cdot\|_2$, $\|\cdot\|_\infty$ が定める単位開球の形状の違い：菱形、円板、正方形)
  - Remark (位相空間論II（3-12）の位相の概念がノルム空間でも成立し、収束の概念を統一的に扱えること)

### §1.4 ノルムの同値性
**内容:** 有限次元でのノルムの同値性定理とその意味

- **1. ノルムの同値性の定義**
  - Definition 1.4-1 (ノルムの同値性：定数 $c, C > 0$ が存在して $c\|x\|_a \leq \|x\|_b \leq C\|x\|_a$ がすべての $x$ に対して成り立つこと)
  - Example 1.4-1 ($\mathbb{R}^n$ 上の $\|\cdot\|_1$ と $\|\cdot\|_2$ の同値性：$\|x\|_2 \leq \|x\|_1 \leq \sqrt{n}\|x\|_2$ の確認)
  - Remark (同値なノルムは同じ位相を定め、収束やコーシー列の概念が一致すること)

- **2. 有限次元の同値性定理**
  - Theorem 1.4-1 (有限次元ノルム空間上の任意の2つのノルムは同値である)
  - Proof (証明の概略：$\|\cdot\|_2$ と任意のノルム $\|\cdot\|$ を比較する；コンパクト集合（単位球面）上の連続関数の最大値・最小値の存在を用いる)
  - Example 1.4-2 (無限次元での反例：$C[0,1]$ 上の $\|f\|_\infty$ と $\|f\|_1 = \int_0^1 |f(x)|\,dx$ が同値でないこと；$f_n(x) = x^n$ で $\|f_n\|_\infty = 1$ だが $\|f_n\|_1 = 1/(n+1) \to 0$)
  - Remark (無限次元の空間では位相の違いが本質的に現れる；これが関数解析の難しさと豊かさの源泉である)

## Chapter 2: バナッハ空間
**テーマ:** 完備性、主要な例、シャウダー基底、縮小写像定理

### §2.1 完備性とバナッハ空間の定義
**内容:** コーシー列による完備性の定義と直感的意味

- **1. バナッハ空間の定義**
  - Definition 2.1-1 (バナッハ空間：すべてのコーシー列が収束するノルム空間（完備ノルム空間）)
  - Example 2.1-1 ($\mathbb{R}^n$ がバナッハ空間であること：有限次元の場合に実解析（2-3）のボルツァーノ-ワイエルシュトラスの定理が利用できることの確認)
  - Remark (完備性は「穴がない」ことを意味し、極限操作が空間の中で閉じることを保証する)

- **2. 完備でないノルム空間の例**
  - Example 2.1-2 ($C[0,1]$ の $L^1$ ノルムでの非完備性：連続関数の列が $L^1$ ノルムではコーシー列をなすが、極限関数が不連続となり $C[0,1]$ に含まれない例の構成)
  - Remark (空間の完備性はノルムの選び方にも依存する；同じ関数の集合でもノルムが変われば完備性が変わりうる)

### §2.2 バナッハ空間の例
**内容:** $C[a,b]$, $\ell^p$, $L^p$ の完備性の確認

- **1. $C[a,b]$ と $\ell^p$ の完備性**
  - Theorem 2.2-1 ($C[a,b]$ が上限ノルム $\|\cdot\|_\infty$ のもとでバナッハ空間であること)
  - Proof (完備性の証明：コーシー列の一様収束極限が連続関数であることを示す；各点での収束を確認し、一様収束性と連続性の保存を議論する)
  - Example 2.2-1 ($C[0,1]$ のコーシー列の具体例：$f_n(x) = \sum_{k=0}^n x^k/k!$ が $e^x$ に一様収束する列)
  - Theorem 2.2-2 ($1 \leq p \leq \infty$ に対し $\ell^p$ がバナッハ空間であること)
  - Proof (完備性の証明：成分ごとの収束を確認し、極限の $\ell^p$ への所属をファトゥの補題（3-3）を用いて示す)

- **2. $L^p$ 空間の完備性**
  - Theorem 2.2-3 (リース-フィッシャーの定理：$1 \leq p \leq \infty$ に対し $L^p(\mu)$ がバナッハ空間であること)
  - Proof (リース-フィッシャーの定理の証明：急速コーシー部分列を取り、単調収束定理（3-3）を用いて a.e. 収束する極限を構成し、$L^p$ への所属を確認する)
  - Example 2.2-2 ($L^2[0,1]$ のコーシー列の収束：具体的な関数列を挙げ、$L^2$ ノルムでの収束と各点 a.e. 収束の関係を確認)
  - Remark ($L^p$ 空間は測度論（3-3）のルベーグ積分と密接に結びついており、関数解析の最も重要な舞台を提供する)

### §2.3 完備化
**内容:** 任意のノルム空間の完備化の存在と一意性

- **1. 完備化の存在**
  - Theorem 2.3-1 (完備化定理：任意のノルム空間 $V$ に対して、$V$ を稠密部分空間として含むバナッハ空間 $\hat{V}$ が同型を除いて一意に存在すること)
  - Proof (完備化の証明の概略：コーシー列の同値類を元とする空間を構成し、自然なノルムを定義して完備性を確認する)
  - Example 2.3-1 ($\mathbb{Q}$ のユークリッドノルムの完備化が $\mathbb{R}$ であること：実数の構成との対比)

- **2. 完備化の一意性と意味**
  - Remark (完備化は「最小の」完備空間であり、元の空間の構造をすべて保存したまま穴を埋める操作である)
  - Example 2.3-2 ($C[0,1]$ の $L^1$ ノルムでの完備化が $L^1[0,1]$ であること：不連続な可積分関数が「穴を埋める」元として現れる直感的説明)

### §2.4 シャウダー基底
**内容:** シャウダー基底の定義、$\ell^p$ の標準基底、基底を持たないバナッハ空間の存在

- **1. シャウダー基底の定義**
  - Definition 2.4-1 (シャウダー基底：バナッハ空間 $X$ の列 $\{e_n\}$ で、任意の $x \in X$ が $x = \sum_{n=1}^\infty \alpha_n e_n$（ノルム収束）と一意に表されること)
  - Remark (有限次元のハメル基底（線形代数学II（2-1））では有限和であったが、シャウダー基底では無限級数の収束が問われる点が本質的に異なる)
  - Example 2.4-1 ($\ell^p$（$1 \leq p < \infty$）の標準基底 $\{e_n\}$ がシャウダー基底であること：$x = (x_1, x_2, \ldots)$ に対して $\|x - \sum_{k=1}^N x_k e_k\|_p = (\sum_{k>N} |x_k|^p)^{1/p} \to 0$ の確認)
  - Example 2.4-2 ($C[0,1]$ のシャウダー基底の例：シャウダー系（折れ線関数の系）の構成の概略)

- **2. 基底の存在と非存在**
  - Proposition 2.4-1 (シャウダー基底を持つバナッハ空間は可分であること)
  - Proof (有理数係数の有限線形結合が可算稠密集合を与えること)
  - Example 2.4-3 ($\ell^\infty$ が非可分であるためシャウダー基底を持たないこと：非可算個の互いに距離 $1$ 以上離れた元の存在)
  - Theorem 2.4-1 (エンフロの定理：シャウダー基底を持たない可分バナッハ空間が存在すること)
  - Proof (証明の概略の指摘：近似性質を持たない可分バナッハ空間の構成による；証明は高度であり主張の紹介にとどめる)
  - Remark (エンフロの反例はバナッハの近似問題に否定的解答を与えたものであり、無限次元バナッハ空間の構造の複雑さを象徴する結果である)

### §2.5 バナッハの不動点定理
**内容:** 縮小写像定理の証明と微分方程式への応用

- **1. 縮小写像定理**
  - Definition 2.5-1 (縮小写像：定数 $0 \leq q < 1$ が存在して $\|T(x) - T(y)\| \leq q\|x - y\|$ を満たす写像)
  - Theorem 2.5-1 (バナッハの不動点定理：完備距離空間上の縮小写像は一意の不動点を持つこと)
  - Proof (不動点定理の証明：初期点 $x_0$ から反復列 $x_{n+1} = T(x_n)$ を構成し、等比級数を用いてコーシー列であることを示し、完備性から極限の存在を得る；一意性は縮小条件から直ちに従う)
  - Example 2.5-1 ($T(x) = x/2 + 1$ が $\mathbb{R}$ 上の縮小写像であること：$q = 1/2$ で不動点 $x = 2$ を反復で確認)

- **2. 微分方程式への応用**
  - Theorem 2.5-2 (ピカール-リンデレフの定理の証明への応用：初期値問題 $y' = f(t,y)$, $y(t_0) = y_0$ のリプシッツ条件下での局所解の存在と一意性)
  - Proof (ピカール反復の収束性：$C[t_0-\delta, t_0+\delta]$ の適切な閉部分集合上で積分作用素が縮小写像となることを確認する)
  - Example 2.5-2 ($y' = y$, $y(0) = 1$ に対するピカール反復：$y_0(t) = 1$, $y_1(t) = 1 + t$, $y_2(t) = 1 + t + t^2/2$ が $e^t$ に収束することの計算)
  - Remark (バナッハの不動点定理は関数解析の最初の成功例であり、具体的な解を構成的に与える点で強力である)

## Chapter 3: 有界線形作用素
**テーマ:** 有界性と連続性の同値、作用素ノルム、$B(X,Y)$ の構造

### §3.1 線形作用素の有界性と連続性
**内容:** 有界性と連続性の同値定理

- **1. 有界線形作用素の定義**
  - Definition 3.1-1 (有界線形作用素：ノルム空間 $X$ から $Y$ への線形写像 $T$ で、$\|Tx\| \leq M\|x\|$ を満たす定数 $M$ が存在するもの)
  - Example 3.1-1 (有界な例：$T : \mathbb{R}^n \to \mathbb{R}^m$ を行列 $A$ で定める線形写像；$\|Tx\| \leq \|A\|\|x\|$ の確認)
  - Example 3.1-2 (非有界な例：$C^1[0,1]$ 上の微分作用素 $Tf = f'$；$f_n(x) = \sin(nx)$ で $\|f_n\|_\infty = 1$ だが $\|f_n'\|_\infty = n \to \infty$ となることの計算)

- **2. 有界性と連続性の同値**
  - Theorem 3.1-1 (線形作用素について、有界性、連続性、原点での連続性の3条件が同値であること)
  - Proof (同値性の証明：有界ならば連続は直接的；連続ならば有界は背理法で示す；$\|x_n\| \to 0$ かつ $\|Tx_n\| \geq n$ と仮定して矛盾を導く)
  - Example 3.1-3 (連続性の崩壊：上の微分作用素が $\|f_n/n\|_\infty \to 0$ だが $\|T(f_n/n)\|_\infty = 1$ により原点で連続でないことの確認)
  - Remark (有限次元空間では線形写像は自動的に有界だが、無限次元ではそうとは限らない；これが有界性の仮定の重要性の源泉である)

### §3.2 作用素ノルムの定義と性質
**内容:** sup による定義と $B(X,Y)$ のバナッハ空間としての構造

- **1. 作用素ノルムの定義**
  - Definition 3.2-1 (作用素ノルム：$\|T\| = \sup_{\|x\| \leq 1} \|Tx\| = \sup_{\|x\|=1} \|Tx\| = \sup_{x \neq 0} \|Tx\|/\|x\|$)
  - Proposition 3.2-1 (作用素ノルムがノルムの3公理を満たすこと)
  - Proof (三角不等式の確認：$\|(T+S)x\| \leq \|Tx\| + \|Sx\|$ から上限を取る)
  - Example 3.2-1 ($2 \times 2$ 行列 $A = \begin{pmatrix} 1 & 0 \\ 0 & 2 \end{pmatrix}$ の作用素ノルム（$\ell^2 \to \ell^2$）が $\|A\| = 2$ であることの計算)

- **2. $B(X,Y)$ の完備性**
  - Definition 3.2-2 ($B(X,Y)$：$X$ から $Y$ への有界線形作用素全体のノルム空間)
  - Theorem 3.2-1 ($Y$ がバナッハ空間ならば $B(X,Y)$ もバナッハ空間であること)
  - Proof (完備性の証明：コーシー列 $\{T_n\}$ に対して各点極限 $Tx = \lim T_n x$ を構成し、$T$ の線形性・有界性・$T_n \to T$ の一様収束を確認する)
  - Example 3.2-2 ($Y$ がバナッハ空間でない場合に $B(X,Y)$ が完備とならない例の存在)
  - Remark (特に $Y = \mathbb{K}$（スカラー体）のとき $B(X, \mathbb{K}) = X^*$（双対空間）が常にバナッハ空間であること)

### §3.3 有界線形作用素の例
**内容:** 行列作用素、積分作用素、微分作用素の有界性

- **1. 行列作用素と積分作用素**
  - Example 3.3-1 (行列作用素：$A : \ell^2 \to \ell^2$ を無限行列で定める場合；対角行列 $\mathrm{diag}(\lambda_n)$ の作用素ノルムが $\sup_n |\lambda_n|$ であることの計算)
  - Example 3.3-2 (積分作用素：$Tf(x) = \int_a^b K(x,t)f(t)\,dt$（核 $K$ が連続）が $C[a,b] \to C[a,b]$ の有界作用素であること；$\|T\| \leq (b-a)\sup|K|$ の評価)

- **2. 有界性の判定例**
  - Example 3.3-3 (左シフト作用素：$\ell^2$ 上の $L(x_1, x_2, x_3, \ldots) = (x_2, x_3, \ldots)$ が有界で $\|L\| = 1$ であることの計算)
  - Example 3.3-4 (右シフト作用素：$R(x_1, x_2, \ldots) = (0, x_1, x_2, \ldots)$ が等長作用素であること：$\|Rx\| = \|x\|$ の確認)
  - Remark (シフト作用素は無限次元空間に特有の現象を示す基本的な作用素であり、有限次元には対応物がない)

### §3.4 作用素の収束
**内容:** 一様収束・強収束・弱収束の定義と関係性

- **1. 3種の収束の定義**
  - Definition 3.4-1 (一様収束（ノルム収束）：$\|T_n - T\| \to 0$)
  - Definition 3.4-2 (強収束：すべての $x \in X$ に対して $\|T_n x - Tx\| \to 0$)
  - Definition 3.4-3 (弱作用素収束：すべての $x \in X$ と $f \in Y^*$ に対して $f(T_n x) \to f(Tx)$)
  - Proposition 3.4-1 (一様収束 $\implies$ 強収束 $\implies$ 弱作用素収束)
  - Proof (含意の証明：一様から強は $\|T_n x - Tx\| \leq \|T_n - T\|\|x\|$ から直接；強から弱は $|f(T_n x - Tx)| \leq \|f\|\|T_n x - Tx\|$ から従う)

- **2. 逆が成り立たない例**
  - Example 3.4-1 (強収束するが一様収束しない例：$\ell^2$ 上の射影 $P_n(x_1, x_2, \ldots) = (x_1, \ldots, x_n, 0, 0, \ldots)$ が恒等作用素に強収束するが $\|P_n - I\| = 1$ で一様収束しないことの確認)
  - Example 3.4-2 (弱収束するが強収束しない例の存在の指摘)
  - Remark (一様有界性定理（§4.3）により、強収束する作用素列のノルムは一様有界であることが保証される)

## Chapter 4: 関数解析の基本定理
**テーマ:** ハーン-バナッハ定理・一様有界性原理・開写像定理・閉グラフ定理

### §4.1 ハーン-バナッハ定理（拡張定理）
**内容:** 劣線形汎関数による線形汎関数の拡張

- **1. 劣線形汎関数と拡張**
  - Definition 4.1-1 (劣線形汎関数：$p(\alpha x) = \alpha p(x)$（$\alpha \geq 0$）かつ $p(x+y) \leq p(x) + p(y)$ を満たす関数)
  - Theorem 4.1-1 (ハーン-バナッハの拡張定理：部分空間上の線形汎関数 $f$ が $f(x) \leq p(x)$ を満たすとき、全空間への拡張 $\tilde{f}$ で $\tilde{f}(x) \leq p(x)$ を保つものが存在すること)
  - Proof (拡張定理の証明の概略：1次元拡張の繰り返しを行う；1次元拡張では $\tilde{f}(x_0)$ の取りうる範囲を劣線形条件から決定する；一般の場合はツォルンの補題を用いる)
  - Example 4.1-1 ($\mathbb{R}^2$ の $x$ 軸上の線形汎関数 $f(a,0) = a$ を $p(x) = \|x\|_\infty$ 以下で $\mathbb{R}^2$ 全体に拡張する例；$\tilde{f}(a,b) = a$ が一つの拡張であることの確認)

- **2. ノルム保存拡張**
  - Corollary 4.1-1 (ノルム空間上の部分空間で定義された有界線形汎関数は、ノルムを保存して全空間に拡張できること)
  - Proof (ノルム保存拡張の証明：$p(x) = \|f\|\|x\|$ と置いてハーン-バナッハの拡張定理を適用する)
  - Example 4.1-2 ($c_0$（零に収束する数列の空間）の部分空間上の汎関数を $\ell^\infty$ 全体に拡張する例)
  - Remark (ハーン-バナッハ定理は双対空間が「十分大きい」ことを保証する；この豊富さが分離定理の基礎となる)

### §4.2 ハーン-バナッハ定理の系（分離定理）
**内容:** 点と閉部分空間の分離、双対空間の豊富さ

- **1. 双対空間の豊富さ**
  - Corollary 4.2-1 ($x \neq 0$ ならば $f(x) = \|x\|$ かつ $\|f\| = 1$ を満たす $f \in X^*$ が存在すること)
  - Proof (存在の証明：$\text{span}\{x\}$ 上で $f(\alpha x) = \alpha\|x\|$ と定め、ハーン-バナッハ定理で拡張する)
  - Example 4.2-1 ($\ell^1$ の元 $x = (1, 0, 0, \ldots)$ に対する分離汎関数の具体的構成)
  - Corollary 4.2-2 ($x \neq y$ ならば $f(x) \neq f(y)$ を満たす $f \in X^*$ が存在すること)
  - Proof (分離の証明：$x - y$ に対する Corollary 4.2-1 を適用する)

- **2. 閉部分空間と点の分離**
  - Theorem 4.2-1 ($M$ が $X$ の閉部分空間で $x_0 \notin M$ ならば、$M$ 上で零かつ $f(x_0) \neq 0$ となる $f \in X^*$ が存在すること)
  - Proof (分離の証明：商空間 $X/M$ を用いるか、$M + \text{span}\{x_0\}$ 上で汎関数を構成して拡張する)
  - Example 4.2-2 ($\ell^2$ において $M = \{(x_n) \mid x_1 = 0\}$ と $x_0 = (1, 0, 0, \ldots)$ に対する分離汎関数の計算)
  - Remark (分離定理は凸解析（4-4）における凸集合の分離定理の線形版であり、最適化理論への橋渡しとなる)

### §4.3 一様有界性定理（バナッハ-スタインハウスの定理）
**内容:** ベールのカテゴリー定理に基づく証明

- **1. ベールのカテゴリー定理の準備**
  - Theorem 4.3-1 (ベールのカテゴリー定理：完備距離空間は稠密開集合の可算共通部分が稠密であること（第一類集合の可算和では尽くせないこと）)
  - Proof (ベールの定理の証明：入れ子の閉球列を構成し、完備性から共通点の存在を得る)
  - Example 4.3-1 ($\mathbb{R}$ が有理数の集合の可算個の「ずらし」で覆えることと、$\mathbb{Q}$ が $G_\delta$ 集合でないことの対比)

- **2. 一様有界性定理**
  - Theorem 4.3-2 (一様有界性定理：バナッハ空間 $X$ からノルム空間 $Y$ への有界線形作用素の族 $\{T_\alpha\}$ が各点で有界ならば、作用素ノルムで一様に有界であること)
  - Proof (一様有界性定理の証明：$A_n = \{x \mid \sup_\alpha \|T_\alpha x\| \leq n\}$ が閉集合であること、ベールの定理からある $A_n$ が内点を持つことを示し、線形性でノルムの一様評価を得る)
  - Example 4.3-2 (フーリエ級数の発散への応用：$L^1[0,2\pi]$ 上のフーリエ部分和作用素 $S_n$ のノルムが非有界であること；ある連続関数に対してフーリエ級数が発散することの証明への接続)
  - Remark (「各点で有界なのに一様に有界でない」ことが無限次元で起こりうるため、この定理は完備性の仮定が不可欠である)

### §4.4 開写像定理
**内容:** 全射な有界線形作用素の開写像性

- **1. 開写像定理の主張と証明**
  - Definition 4.4-1 (開写像：開集合の像が開集合であること)
  - Theorem 4.4-1 (開写像定理：バナッハ空間の間の全射な有界線形作用素は開写像であること)
  - Proof (開写像定理の証明：$T(B_X(0,1))$ が $B_Y(0, r)$ を含むことを示す；ベールのカテゴリー定理を $Y = \bigcup_n \overline{T(B_X(0,n))}$ に適用し、完備性から近似的な開性を正確な開性に引き上げる)
  - Example 4.4-1 ($T : \ell^1 \to c_0$ が全射でないため開写像にならない例と、$T : \ell^1 \to \ell^1$ の恒等写像が開写像である自明な例の対比)

- **2. 逆作用素の有界性**
  - Corollary 4.4-1 (バナッハ空間の間の全単射な有界線形作用素の逆作用素は有界であること)
  - Proof (逆作用素の有界性の証明：$T$ が開写像であることから $T^{-1}$ の連続性が直ちに従う)
  - Example 4.4-2 (同値ノルムの判定：$V$ 上の2つのバナッハ空間ノルムに対して恒等写像が両方向とも有界であれば同値であること)
  - Remark (完備性の仮定を外すと開写像定理は成立しない；バナッハ空間の理論における完備性の中心的役割が再確認される)

### §4.5 閉グラフ定理と逆作用素定理
**内容:** グラフの閉性と有界性の同値関係、全単射の逆作用素

- **1. 閉グラフ定理**
  - Definition 4.5-1 (作用素のグラフ：$\mathrm{Gr}(T) = \{(x, Tx) \mid x \in X\} \subset X \times Y$)
  - Theorem 4.5-1 (閉グラフ定理：バナッハ空間の間の線形作用素が閉グラフを持てば有界であること)
  - Proof (閉グラフ定理の証明：$X$ 上の新しいノルム $\|x\|_T = \|x\| + \|Tx\|$ がバナッハ空間を定めること（グラフの閉性より）を示し、恒等写像 $(X, \|\cdot\|_T) \to (X, \|\cdot\|)$ に開写像定理を適用する)
  - Example 4.5-1 (閉グラフの確認：積分作用素 $Tf(x) = \int_0^x f(t)\,dt$ が $C[0,1] \to C[0,1]$ で閉グラフを持つことの検証)

- **2. 閉グラフ定理の応用**
  - Example 4.5-2 (非有界作用素の閉グラフ性：微分作用素 $T = d/dx$ が $C^1[0,1] \subset C[0,1]$ 上で閉グラフを持つが、$C[0,1]$ 全体では定義されないこと)
  - Remark (閉グラフ定理は「有界性を直接確認しにくい」作用素の有界性を証明するのに便利な道具であり、特にスペクトル理論（第10章）で活用される)

## Chapter 5: 双対空間と弱位相
**テーマ:** 連続線形汎関数、双対ノルム、弱収束、反射空間

### §5.1 双対空間 $X^*$ と双対ノルム
**内容:** 連続線形汎関数全体の空間と双対ノルム

- **1. 双対空間の定義**
  - Definition 5.1-1 (双対空間：$X^* = B(X, \mathbb{K})$、すなわち $X$ 上の連続線形汎関数全体)
  - Definition 5.1-2 (双対ノルム：$\|f\| = \sup_{\|x\| \leq 1} |f(x)|$)
  - Proposition 5.1-1 ($X^*$ は常にバナッハ空間であること)
  - Proof ($\mathbb{K}$ が完備であることと §3.2 の結果から従う)

- **2. 第二双対空間と標準埋め込み**
  - Definition 5.1-3 (第二双対空間 $X^{**}$ と標準埋め込み $J : X \to X^{**}$；$J(x)(f) = f(x)$)
  - Proposition 5.1-2 ($J$ は等長埋め込みであること)
  - Proof (等長性の証明：$\|J(x)\| = \sup_{\|f\| \leq 1} |f(x)| = \|x\|$ をハーン-バナッハ定理を用いて確認する)
  - Example 5.1-1 ($\ell^1$ の標準埋め込み $J : \ell^1 \to (\ell^\infty)^*$ が全射でないことの指摘)

### §5.2 双対空間の具体例
**内容:** $\ell^p$ の双対 $\ell^q$ と $C[a,b]^*$ のリース-マルコフ表現

- **1. $\ell^p$ の双対**
  - Theorem 5.2-1 ($1 \leq p < \infty$ のとき $(\ell^p)^* \cong \ell^q$（$1/p + 1/q = 1$）；対応は $f(x) = \sum x_n y_n$（$y \in \ell^q$）)
  - Proof (同型の証明の概略：ヘルダーの不等式で $\|f\| \leq \|y\|_q$ を示し、逆評価を $x_n = |y_n|^{q-1}\text{sgn}(y_n)$ の列で達成する)
  - Example 5.2-1 ($(\ell^2)^* \cong \ell^2$ の場合：$f(x) = x_1 + x_2/2$ に対応する $y = (1, 1/2, 0, 0, \ldots) \in \ell^2$ の確認)

- **2. $C[a,b]$ の双対**
  - Theorem 5.2-2 (リース-マルコフの表現定理：$C[a,b]^*$ は $[a,b]$ 上の有界変動な符号付き正則ボレル測度の空間と等長同型であること)
  - Proof (証明の概略：ダニエルの積分拡張を用いて連続線形汎関数から測度を構成する手順)
  - Example 5.2-2 (ディラック測度 $\delta_{x_0}$ が $C[a,b]$ 上の連続線形汎関数 $f \mapsto f(x_0)$ に対応することの確認)
  - Remark ($L^p$ の双対については $L^q$（$1 < p < \infty$）、$L^1$ の双対は $L^\infty$ であるが、$L^\infty$ の双対は $L^1$ よりも大きいこと)

### §5.3 弱収束と強収束の比較
**内容:** 弱収束の定義と強収束との違い

- **1. 弱収束の定義**
  - Definition 5.3-1 (弱収束：$x_n \xrightarrow{w} x$ とは、すべての $f \in X^*$ に対して $f(x_n) \to f(x)$ が成り立つこと)
  - Proposition 5.3-1 (強収束は弱収束を含意すること)
  - Proof ($|f(x_n) - f(x)| \leq \|f\|\|x_n - x\| \to 0$ から直ちに従う)
  - Example 5.3-1 ($\ell^2$ の標準基底 $e_n$ が弱収束する例：すべての $y \in \ell^2$ に対して $\langle e_n, y \rangle = y_n \to 0$ より $e_n \xrightarrow{w} 0$ だが、$\|e_n\| = 1$ より強収束しないことの確認)

- **2. 弱収束の基本性質**
  - Proposition 5.3-2 (弱収束列のノルムの一様有界性：一様有界性定理の応用)
  - Proof (一様有界性定理を標準埋め込み $J(x_n) \in X^{**}$ に適用する)
  - Example 5.3-2 ($L^2[0,1]$ の直交関数列 $\sqrt{2}\sin(n\pi x)$ が $0$ に弱収束するがノルムが $1$ であることの確認)
  - Remark (弱収束は「すべての観測者から見て収束する」という意味を持ち、物理学における可観測量の収束と関連する)

### §5.4 弱位相と反射的空間
**内容:** 弱位相、アラオグルの定理、$L^p$ の反射性

- **1. 弱位相と弱*位相**
  - Definition 5.4-1 (弱位相：$X$ 上で $X^*$ の元が連続であるような最弱の位相)
  - Definition 5.4-2 (弱*位相：$X^*$ 上で標準埋め込み $J(X)$ の元が連続であるような最弱の位相)
  - Remark (弱*位相は弱位相より粗い位相であり、$X$ が反射的でない限り異なる)
  - Example 5.4-1 ($\ell^1 = (c_0)^*$ 上の弱*収束：$e_n \xrightarrow{w^*} 0$ の確認)

- **2. バナッハ-アラオグルの定理と反射性**
  - Theorem 5.4-1 (バナッハ-アラオグルの定理：$X^*$ の閉単位球は弱*位相でコンパクトであること)
  - Proof (証明の概略：$X^*$ の閉単位球をティコノフの定理（3-12）を用いて積空間のコンパクト部分集合として実現する)
  - Definition 5.4-3 (反射的空間：標準埋め込み $J : X \to X^{**}$ が全射であること)
  - Example 5.4-2 ($\ell^p$（$1 < p < \infty$）と $L^p$（$1 < p < \infty$）が反射的であること；$\ell^1$ と $\ell^\infty$ が反射的でないことの確認)
  - Remark (反射的空間では弱位相の閉単位球がコンパクトとなり、最適化問題において解の存在が保証されやすくなる；凸解析（4-4）への接続)

## Chapter 6: 内積空間とヒルベルト空間
**テーマ:** 内積の公理、基本不等式、完備性

### §6.1 内積空間の定義と例
**内容:** 内積の公理と $\ell^2$, $L^2$ 空間などの例

- **1. 内積の定義**
  - Definition 6.1-1 (内積：$\langle \cdot, \cdot \rangle : V \times V \to \mathbb{K}$ が (i) $\langle x, x \rangle \geq 0$, $\langle x,x \rangle = 0 \iff x = 0$（正定値性）；(ii) $\langle \alpha x + \beta y, z \rangle = \alpha\langle x,z \rangle + \beta\langle y,z \rangle$（第一引数の線形性）；(iii) $\langle x, y \rangle = \overline{\langle y, x \rangle}$（共役対称性）；を満たすこと)
  - Definition 6.1-2 (内積空間：内積が定められたベクトル空間)
  - Remark (内積は $\|x\| = \sqrt{\langle x, x \rangle}$ によりノルムを誘導するため、内積空間はノルム空間の特別な場合である)

- **2. 内積空間の例**
  - Example 6.1-1 ($\mathbb{R}^n$ の標準内積：$\langle x, y \rangle = \sum x_i y_i$；$x = (1,2,3)$, $y = (4,-1,2)$ に対する内積の計算)
  - Example 6.1-2 ($\ell^2$ の内積：$\langle x, y \rangle = \sum x_n \overline{y_n}$ の収束がコーシー-シュワルツの不等式で保証されること)
  - Example 6.1-3 ($L^2[a,b]$ の内積：$\langle f, g \rangle = \int_a^b f(x)\overline{g(x)}\,dx$；$f(x) = 1$, $g(x) = x$ の $[0,1]$ での内積値 $1/2$ の計算)

### §6.2 コーシー-シュワルツの不等式と平行四辺形法則
**内容:** 証明とノルムが内積に由来する条件

- **1. コーシー-シュワルツの不等式**
  - Theorem 6.2-1 (コーシー-シュワルツの不等式：$|\langle x, y \rangle| \leq \|x\|\|y\|$；等号条件は $x$ と $y$ が線形従属であること)
  - Proof (証明：$t \in \mathbb{R}$ に対して $\|x - t y\|^2 \geq 0$ を展開し、判別式条件から不等式を導く)
  - Example 6.2-1 ($\ell^2$ での具体的確認：$x = (1, 1/2, 0, \ldots)$, $y = (1, 0, 0, \ldots)$ に対して $|\langle x, y \rangle| = 1 \leq \sqrt{5/4} \cdot 1$ の計算)
  - Corollary 6.2-1 (内積から誘導されるノルムが三角不等式を満たすこと)
  - Proof ($\|x+y\|^2$ を展開してコーシー-シュワルツの不等式を適用する)

- **2. 平行四辺形法則**
  - Theorem 6.2-2 (平行四辺形法則：$\|x+y\|^2 + \|x-y\|^2 = 2\|x\|^2 + 2\|y\|^2$)
  - Proof (直接展開：内積を展開すると交差項が打ち消し合うことの確認)
  - Example 6.2-2 ($\mathbb{R}^2$ の標準内積での平行四辺形法則の幾何学的意味：$x = (1,0)$, $y = (0,1)$ に対する確認)
  - Theorem 6.2-3 (ヨルダン-フォン・ノイマンの定理：ノルムが内積に由来する必要十分条件は平行四辺形法則が成り立つことである)
  - Proof (証明の概略：平行四辺形法則から偏極等式を用いて内積を構成し、線形性を確認する)
  - Example 6.2-3 ($\ell^p$（$p \neq 2$）のノルムが内積由来でないこと：$x = e_1$, $y = e_2$ で平行四辺形法則の不成立を $p = 1$ で確認)

### §6.3 ヒルベルト空間の定義と例
**内容:** 完備内積空間としての $\ell^2$, $L^2$ の完備性

- **1. ヒルベルト空間の定義**
  - Definition 6.3-1 (ヒルベルト空間：完備な内積空間)
  - Example 6.3-1 ($\ell^2$ がヒルベルト空間であること：§2.2 の結果の内積構造との整合性)
  - Example 6.3-2 ($L^2[a,b]$ がヒルベルト空間であること：リース-フィッシャーの定理（§2.2）の特殊ケース)

- **2. ヒルベルト空間でないノルム空間**
  - Example 6.3-3 ($L^1[0,1]$ はバナッハ空間だがヒルベルト空間でないこと：平行四辺形法則が成立しない反例)
  - Remark (ヒルベルト空間はバナッハ空間の中で最も「幾何学的に良い」空間であり、直交性の概念が自然に定まる点で特別である)

### §6.4 直交性と直交補空間
**内容:** 直交の定義と直交補空間の性質

- **1. 直交の定義**
  - Definition 6.4-1 (直交：$\langle x, y \rangle = 0$ のとき $x \perp y$ と書く)
  - Definition 6.4-2 (直交補空間：$M^\perp = \{x \in H \mid \langle x, m \rangle = 0, \, \forall m \in M\}$)
  - Example 6.4-1 ($\ell^2$ で $M = \{x \mid x_1 = 0\}$ のとき $M^\perp = \text{span}\{e_1\}$ であることの確認)

- **2. 直交補空間の性質**
  - Proposition 6.4-1 ($M^\perp$ は常に $H$ の閉部分空間であること)
  - Proof ($M^\perp$ が部分空間であることは内積の線形性から、閉であることは内積の連続性から従う)
  - Proposition 6.4-2 ($M \subset (M^\perp)^\perp$ であること；$M$ が閉部分空間ならば等号が成立すること)
  - Proof (包含関係は定義から直接；等号は射影定理（§7.1）を用いて示す)
  - Example 6.4-2 ($L^2[-\pi, \pi]$ で偶関数全体 $M_{\text{even}}$ と奇関数全体 $M_{\text{odd}}$ が互いの直交補空間であることの確認)

## Chapter 7: ヒルベルト空間の幾何学
**テーマ:** 射影定理、正規直交系、リースの表現定理

### §7.1 射影定理と最近傍点
**内容:** 凸閉部分集合への射影の存在と一意性

- **1. 凸集合への最近傍点**
  - Theorem 7.1-1 (最近傍点の定理：ヒルベルト空間の空でない閉凸部分集合 $C$ に対して、各 $x \in H$ から $C$ への最近傍点が一意に存在すること)
  - Proof (存在の証明：下限列を取り平行四辺形法則からコーシー列であることを示し、完備性から極限を得る；一意性も平行四辺形法則から)
  - Example 7.1-1 ($\mathbb{R}^3$ の平面 $\{(x,y,0)\}$ 上への点 $(1,2,3)$ の射影が $(1,2,0)$ であることの確認)

- **2. 閉部分空間への射影**
  - Corollary 7.1-1 ($M$ が閉部分空間のとき、最近傍点 $m_0$ は $x - m_0 \perp M$ を満たす唯一の元であること)
  - Proof (直交条件の証明：$m \in M$ に対して $\|x - m_0 - tm\|^2$ を $t$ で微分して $t = 0$ で最小となる条件を導く)
  - Example 7.1-2 ($L^2[0,1]$ における $f(x) = x$ の $\text{span}\{1\}$ への射影が $1/2$ であることの計算)
  - Remark (射影定理はヒルベルト空間理論の中核であり、バナッハ空間では一般に最近傍点の一意性が保証されない)

### §7.2 直交射影と直和分解
**内容:** 閉部分空間による直和分解 $H = M \oplus M^\perp$

- **1. 直交直和分解**
  - Theorem 7.2-1 (直交分解定理：$M$ が閉部分空間のとき $H = M \oplus M^\perp$ が成り立ち、任意の $x \in H$ が $x = m + m'$（$m \in M$, $m' \in M^\perp$）と一意に分解されること)
  - Proof (存在は射影定理から；一意性は $m_1 + m_1' = m_2 + m_2'$ のとき $m_1 - m_2 \in M \cap M^\perp = \{0\}$ から従う)
  - Example 7.2-1 ($\ell^2$ で $M = \{x \mid x_n = 0 \, (n \geq 3)\}$ のとき、$x = (1,2,3,4,\ldots)$ の分解を計算)

- **2. 直交射影の性質**
  - Definition 7.2-1 (直交射影 $P_M : H \to M$：$x = m + m'$ に対して $P_M x = m$)
  - Proposition 7.2-1 ($P_M$ は有界線形作用素で $\|P_M\| \leq 1$、$P_M^2 = P_M$、$P_M^* = P_M$ を満たすこと)
  - Proof (べき等性 $P_M^2 = P_M$ は $P_M m = m$ から；$\|P_M\| \leq 1$ はピタゴラスの定理 $\|x\|^2 = \|m\|^2 + \|m'\|^2$ から従う)
  - Example 7.2-2 ($P_M x$ が $x$ の $M$ への「影」であるという幾何学的直感と、ピタゴラスの定理 $\|x\|^2 = \|P_M x\|^2 + \|(I-P_M)x\|^2$ の確認)

### §7.3 正規直交系とベッセルの不等式
**内容:** 正規直交系の定義とベッセルの不等式

- **1. 正規直交系の定義**
  - Definition 7.3-1 (正規直交系：$\langle e_i, e_j \rangle = \delta_{ij}$ を満たす集合 $\{e_i\}$)
  - Example 7.3-1 ($\ell^2$ の標準基底 $\{e_n\}$ が正規直交系であることの確認)
  - Example 7.3-2 ($L^2[-\pi, \pi]$ のフーリエ基底 $\{e^{inx}/\sqrt{2\pi}\}_{n \in \mathbb{Z}}$ が正規直交系であることの計算)

- **2. ベッセルの不等式とフーリエ係数**
  - Definition 7.3-2 (フーリエ係数：$\hat{x}_n = \langle x, e_n \rangle$)
  - Theorem 7.3-1 (ベッセルの不等式：$\sum_n |\langle x, e_n \rangle|^2 \leq \|x\|^2$)
  - Proof (ベッセルの不等式の証明：$\|x - \sum_{k=1}^N \hat{x}_k e_k\|^2 \geq 0$ を展開して直交性を利用する)
  - Example 7.3-3 ($L^2[-\pi, \pi]$ で $f(x) = x$ のフーリエ係数 $\hat{f}_n = (-1)^{n+1}i\sqrt{2\pi}/n$（$n \neq 0$）を計算し、ベッセルの不等式 $\sum 2\pi/n^2 \leq \pi^2/3$ を確認)
  - Remark (ベッセルの不等式はフーリエ展開の「エネルギー保存」の不完全版であり、等号が成立する条件がパーセバルの等式（§7.4）である)

### §7.4 完全正規直交系とパーセバルの等式
**内容:** 完全性、パーセバルの等式と $L^2$ フーリエ展開

- **1. 完全正規直交系**
  - Definition 7.4-1 (完全正規直交系（正規直交基底）：$\langle x, e_n \rangle = 0$（すべての $n$）ならば $x = 0$ であること)
  - Theorem 7.4-1 (パーセバルの等式：正規直交系が完全であることと、すべての $x$ に対して $\|x\|^2 = \sum |\langle x, e_n \rangle|^2$ が成り立つことが同値であること)
  - Proof (パーセバルの等式の証明：完全性からフーリエ展開 $x = \sum \hat{x}_n e_n$ の収束を示し、連続性とベッセルの不等式から等号を得る)
  - Example 7.4-1 ($L^2[-\pi, \pi]$ のフーリエ基底の完全性とパーセバルの等式 $\|f\|^2 = \sum |\hat{f}_n|^2$；$f(x) = x$ で $\sum 1/n^2 = \pi^2/6$ を導出)

- **2. 可分ヒルベルト空間の構造**
  - Definition 7.4-2 (可分空間：可算稠密部分集合を持つ位相空間)
  - Theorem 7.4-2 (可分なヒルベルト空間は可算正規直交基底を持ち、$\ell^2$ と等長同型であること)
  - Proof (証明の概略：グラム-シュミットの直交化法で可算稠密集合から正規直交系を構成し、フーリエ係数による写像が等長同型を与える)
  - Example 7.4-2 ($L^2[0,1]$ が $\ell^2$ と同型であること；すべての可分無限次元ヒルベルト空間は「本質的に同じ」であるという意味)
  - Remark (この同型定理はヒルベルト空間理論の美しさの源であり、フーリエ解析（3-4）の基礎を抽象的に支える)

### §7.5 リースの表現定理
**内容:** $H^*$ と $H$ の同型とディラック記法との対応

- **1. リースの表現定理**
  - Theorem 7.5-1 (リースの表現定理：ヒルベルト空間 $H$ 上の任意の連続線形汎関数 $f \in H^*$ に対して、$f(x) = \langle x, y \rangle$ を満たす一意の $y \in H$ が存在し、$\|f\| = \|y\|$ であること)
  - Proof (存在の証明：$\ker f$ が閉部分空間であること、$\ker f \neq H$ のとき $\ker f^\perp$ が1次元であることを用い、$y$ を構成する；一意性はコーシー-シュワルツの不等式から)
  - Example 7.5-1 ($\ell^2$ 上の汎関数 $f(x) = x_1 + 2x_2$ に対する代表元 $y = (1, 2, 0, 0, \ldots)$ の計算)

- **2. リースの表現定理の意味と応用**
  - Corollary 7.5-1 (ヒルベルト空間は反射的であること：$J : H \to H^{**}$ が全射)
  - Proof (リースの表現定理を2回適用して $H \cong H^* \cong H^{**}$ の同型を構成する)
  - Remark (物理学ではリースの表現定理がディラックのブラ-ケット記法の数学的正当化を与える；$|x\rangle$ がケットベクトル、$\langle y|$ がブラベクトルに対応する；量子力学への接続は数理物理学（A-11）で扱う)

## Chapter 8: ヒルベルト空間上の作用素
**テーマ:** 随伴作用素、自己共役・ユニタリ・正規・射影作用素

### §8.1 随伴作用素の定義と存在
**内容:** リースの表現定理による存在証明

- **1. 随伴作用素の定義**
  - Definition 8.1-1 (随伴作用素：$T \in B(H)$ に対して $\langle Tx, y \rangle = \langle x, T^*y \rangle$ をすべての $x, y$ に対して満たす作用素 $T^*$)
  - Theorem 8.1-1 (随伴作用素の存在と一意性：リースの表現定理を用いて $T^*$ が一意に定まり、$T^* \in B(H)$ かつ $\|T^*\| = \|T\|$ であること)
  - Proof (存在の証明：各 $y$ に対して $f_y(x) = \langle Tx, y \rangle$ が連続線形汎関数であることからリースの表現定理で $T^*y$ を構成する)
  - Example 8.1-1 ($\ell^2$ 上の左シフト作用素 $L$ の随伴が右シフト作用素 $R$ であることの確認：$\langle L x, y \rangle = \langle x, Ry \rangle$ の計算)

- **2. 随伴の基本性質**
  - Proposition 8.1-1 (随伴の性質：$(S+T)^* = S^* + T^*$; $(\alpha T)^* = \bar{\alpha}T^*$; $(ST)^* = T^*S^*$; $T^{**} = T$)
  - Proof (各性質を内積の定義に遡って確認する)
  - Example 8.1-2 (積分作用素 $Tf(x) = \int K(x,t)f(t)\,dt$ の随伴が核 $\overline{K(t,x)}$ による積分作用素であることの確認)

### §8.2 自己共役・歪自己共役・ユニタリ作用素
**内容:** 定義とスペクトルに関する基本性質

- **1. 自己共役作用素**
  - Definition 8.2-1 (自己共役作用素：$T^* = T$ を満たす有界線形作用素)
  - Example 8.2-1 ($\ell^2$ 上の対角作用素 $T(x_n) = (\lambda_n x_n)$（$\lambda_n \in \mathbb{R}$, $\sup|\lambda_n| < \infty$）が自己共役であることの確認)
  - Proposition 8.2-1 (自己共役作用素の固有値は実数であり、異なる固有値の固有ベクトルは直交すること)
  - Proof (固有値の実数性の証明：$\lambda\langle x, x \rangle = \langle Tx, x \rangle = \langle x, Tx \rangle = \bar{\lambda}\langle x, x \rangle$ から $\lambda = \bar{\lambda}$)
  - Example 8.2-2 (有限次元での対応物：実対称行列の固有値がすべて実数であること（線形代数学II（2-1）の結果）との対比)

- **2. 歪自己共役作用素とユニタリ作用素**
  - Definition 8.2-2 (歪自己共役作用素：$T^* = -T$ を満たすこと)
  - Definition 8.2-3 (ユニタリ作用素：$T^*T = TT^* = I$ を満たすこと；すなわち $T$ が等長全射であること)
  - Example 8.2-3 ($\ell^2$ 上の座標の並べ替え作用素がユニタリであることの確認)
  - Proposition 8.2-2 (ユニタリ作用素の固有値は絶対値1の複素数であること)
  - Proof ($\|Tx\| = \|x\|$ と $Tx = \lambda x$ から $|\lambda| = 1$)
  - Remark (量子力学における時間発展はユニタリ群で記述される；作用素環論（5-8）への接続)

### §8.3 正規作用素
**内容:** $A^*A = AA^*$ とスペクトル定理の準備

- **1. 正規作用素の定義と性質**
  - Definition 8.3-1 (正規作用素：$T^*T = TT^*$ を満たす作用素)
  - Remark (自己共役作用素とユニタリ作用素はどちらも正規作用素の特別な場合である)
  - Proposition 8.3-1 (正規作用素の性質：$\|Tx\| = \|T^*x\|$ がすべての $x$ に対して成り立つこと)
  - Proof ($\|Tx\|^2 = \langle T^*Tx, x \rangle = \langle TT^*x, x \rangle = \|T^*x\|^2$)
  - Example 8.3-1 (正規だが自己共役でもユニタリでもない作用素：$\ell^2$ 上の対角作用素 $T(x_n) = (\lambda_n x_n)$（$\lambda_n \in \mathbb{C}$, $\sup|\lambda_n| < \infty$）)

- **2. 正規作用素の固有空間**
  - Proposition 8.3-2 (正規作用素の異なる固有値に属する固有ベクトルは直交すること)
  - Proof (直交性の証明：$\langle Tx - \lambda x, y \rangle$ と正規性から $T^*y = \bar{\mu}y$ を用いる)
  - Example 8.3-2 (複素対角行列の固有ベクトルの直交性が一般次元に拡張されることの確認)
  - Remark (正規作用素に対するスペクトル定理（第10章）は、有限次元におけるユニタリ対角化の無限次元版である)

### §8.4 直交射影作用素
**内容:** $P^2 = P = P^*$ による特徴づけと部分空間との対応

- **1. 射影作用素の代数的特徴づけ**
  - Theorem 8.4-1 ($P \in B(H)$ が閉部分空間 $M$ への直交射影であることと、$P^2 = P$ かつ $P^* = P$ が同値であること)
  - Proof (直交射影ならば §7.2 の結果から $P^2 = P$, $P^* = P$；逆に $P^2 = P = P^*$ ならば $\text{Im}(P)$ が閉部分空間で $\ker P = (\text{Im} P)^\perp$ を示す)
  - Example 8.4-1 ($\ell^2$ で最初の $n$ 成分への射影 $P_n$ が $P_n^2 = P_n = P_n^*$ を満たすことの計算)

- **2. 射影の順序構造**
  - Proposition 8.4-1 (2つの直交射影 $P, Q$ に対して、$\text{Im}(P) \subset \text{Im}(Q)$ と $QP = P$ が同値であること)
  - Proof ($QP = P$ ならば $\text{Im}(P) \subset \text{Im}(Q)$ は直ちに；逆は $P x \in \text{Im}(Q)$ より $QP x = P x$)
  - Example 8.4-2 ($\ell^2$ で $P_2$ と $P_5$ に対する $P_5 P_2 = P_2$ の確認)
  - Remark (射影の格子構造は量子力学における命題論理と対応する；詳細は作用素環論（5-8）で扱う)

## Chapter 9: コンパクト作用素

### §9.1 コンパクト作用素の定義と例
**内容:** 有界集合の相対コンパクトな像、有限ランク・積分作用素

- **1. コンパクト作用素の定義**
  - Definition 9.1-1 (コンパクト作用素：$X$ の有界部分集合を $Y$ の相対コンパクト集合（閉包がコンパクト）に写す有界線形作用素)
  - Remark (有限次元空間では有界作用素はすべてコンパクトだが、無限次元では特別な性質を持つ作用素のクラスとなる)
  - Example 9.1-1 ($\ell^2$ 上の恒等作用素はコンパクトでないこと：標準基底 $\{e_n\}$ が有界だがコンパクト部分列を持たないことの確認)

- **2. コンパクト作用素の例**
  - Definition 9.1-2 (有限ランク作用素：像が有限次元である有界線形作用素)
  - Proposition 9.1-1 (有限ランク作用素はコンパクトであること)
  - Proof (有限次元空間の有界閉集合はコンパクトであることから従う)
  - Example 9.1-2 (積分作用素のコンパクト性：連続な核 $K$ を持つ積分作用素 $Tf(x) = \int_a^b K(x,t)f(t)\,dt$ が $C[a,b] \to C[a,b]$ のコンパクト作用素であることの確認；アスコリ-アルツェラの定理を用いる)
  - Example 9.1-3 ($\ell^2$ 上の対角作用素 $T(x_n) = (x_n/n)$ がコンパクトであることの確認；有限ランク作用素 $T_N(x_n) = (x_1, x_2/2, \ldots, x_N/N, 0, \ldots)$ の極限として表せること)

### §9.2 コンパクト作用素の基本性質
**内容:** 合成・随伴のコンパクト性と有限ランクの極限

- **1. コンパクト性の保存**
  - Proposition 9.2-1 (コンパクト作用素と有界作用素の合成はコンパクトであること)
  - Proof (合成のコンパクト性：有界作用素が有界集合を有界集合に写し、コンパクト作用素がそれを相対コンパクト集合に写すことから)
  - Proposition 9.2-2 (コンパクト作用素の随伴はコンパクトであること（シャウダーの定理）)
  - Proof (証明の概略：弱収束列に対する作用素の振る舞いを用いる)

- **2. 有限ランク近似**
  - Theorem 9.2-1 (ヒルベルト空間上のコンパクト作用素は有限ランク作用素の作用素ノルム極限であること)
  - Proof (有限ランク近似の証明：正規直交基底を用いた射影による近似列の構成)
  - Example 9.2-1 (§9.1 の対角作用素 $T(x_n) = (x_n/n)$ を $T_N$ で近似したときの誤差 $\|T - T_N\| = 1/(N+1)$ の計算)
  - Remark (コンパクト作用素のクラスは $B(H)$ の中で閉両側イデアルをなす)

### §9.3 コンパクト自己共役作用素のスペクトル
**内容:** 固有値の性質と正規直交基底の構成

- **1. スペクトルの構造**
  - Theorem 9.3-1 (コンパクト自己共役作用素のスペクトル定理：$T$ の0でない固有値は有限個または $0$ に収束する列をなし、各固有空間は有限次元であること)
  - Proof (証明の概略：$\|T\|$ または $-\|T\|$ が固有値であることを変分法で示し、帰納的に固有値列を構成する)
  - Example 9.3-1 ($\ell^2$ 上の $T(x_n) = (x_n/n)$ の固有値が $\{1/n\}$（$n = 1, 2, \ldots$）であり、$0$ に収束することの確認)

- **2. 正規直交基底の構成**
  - Theorem 9.3-2 (正規直交基底による表示：コンパクト自己共役作用素 $T$ に対して $Tx = \sum_n \lambda_n \langle x, e_n \rangle e_n$ と表示できること)
  - Proof (§9.3 Theorem 9.3-1 と直交分解を組み合わせて帰納的に構成する)
  - Example 9.3-2 (有限次元の対称行列の対角化（線形代数学II（2-1））が無限次元に一般化されたものとしての解釈)
  - Remark (この定理は有限次元のスペクトル定理の直接的な拡張であり、第10章の一般的なスペクトル理論への出発点となる)

### §9.4 リース-フレドホルム理論
**内容:** $(I - K)u = f$ の可解条件と交代定理

- **1. フレドホルム方程式**
  - Definition 9.4-1 (第二種フレドホルム方程式：$u - Ku = f$（$K$ はコンパクト作用素）)
  - Theorem 9.4-1 (リース-フレドホルムの交代定理：$\ker(I - K) = \{0\}$ と $(I - K)$ の全射性が同値であること)
  - Proof (交代定理の証明の概略：$\ker(I-K)$ の有限次元性（$K$ のコンパクト性から）を示し、像の閉性と余次元の一致を導く)
  - Example 9.4-1 ($K = \lambda I$（$|\lambda| < 1$）に対して $(I - K)^{-1} = \sum_{n=0}^\infty \lambda^n I$ がノイマン級数で与えられることの計算)

- **2. フレドホルム方程式の可解条件**
  - Theorem 9.4-2 (フレドホルムの可解条件：$u - Ku = f$ が解を持つための必要十分条件は $f$ が $\ker(I - K^*)$ と直交すること)
  - Proof (可解条件の証明：$(I - K)$ の像が $\ker(I - K^*)^\perp$ と一致することを示す)
  - Example 9.4-2 (積分方程式 $u(x) - \int_0^1 K(x,t)u(t)\,dt = f(x)$ への応用；同次方程式の解の個数が可解条件を決定すること)
  - Remark (リース-フレドホルム理論は偏微分方程式（5-2）における楕円型方程式の可解性の基礎となる)

### §9.5 積分方程式と境界値問題への応用
**内容:** グリーン作用素のコンパクト性と定式化

- **1. 積分方程式への定式化**
  - Example 9.5-1 (ヴォルテラ積分方程式：$u(x) - \int_0^x K(x,t)u(t)\,dt = f(x)$ の逐次近似による解法；ノイマン級数の収束がコンパクト性なしに $\|K^n\| \to 0$ で保証されることの確認)
  - Example 9.5-2 (フレドホルム積分方程式：核が対称のとき $K^* = K$ であり、コンパクト自己共役作用素のスペクトル分解による固有関数展開を利用した解の構成)

- **2. 境界値問題への応用**
  - Example 9.5-3 (シュトゥルム-リウヴィル問題のグリーン関数：$-u'' + q(x)u = \lambda u$, $u(0) = u(1) = 0$ をグリーン作用素を用いた積分方程式に変換し、コンパクト自己共役作用素のスペクトル定理を適用して固有値の離散性を導く)
  - Remark (偏微分方程式（5-2）では楕円型作用素の逆がコンパクト作用素となることが中心的な事実であり、本章の理論が直接的に応用される)

## Chapter 10: スペクトル理論
**テーマ:** スペクトルとレゾルベント、有界自己共役・正規作用素のスペクトル定理、非有界作用素の基礎

### §10.1 スペクトルとレゾルベント
**内容:** レゾルベント集合とスペクトルの分類

- **1. スペクトルの定義**
  - Definition 10.1-1 (レゾルベント集合：$\rho(T) = \{\lambda \in \mathbb{C} \mid (T - \lambda I)^{-1} \in B(X)\}$；スペクトル：$\sigma(T) = \mathbb{C} \setminus \rho(T)$)
  - Definition 10.1-2 (レゾルベント作用素：$R(\lambda, T) = (T - \lambda I)^{-1}$)
  - Example 10.1-1 ($\ell^2$ 上の対角作用素 $T(x_n) = (\lambda_n x_n)$ のスペクトルが $\overline{\{\lambda_n\}}$（値の閉包）であることの計算)
  - Remark (有限次元ではスペクトルは固有値全体と一致するが、無限次元では固有値でないスペクトルが存在しうる)

- **2. スペクトルの分類**
  - Definition 10.1-3 (点スペクトル $\sigma_p(T)$：$T - \lambda I$ が単射でない（固有値の全体）)
  - Definition 10.1-4 (連続スペクトル $\sigma_c(T)$：$T - \lambda I$ が単射、像が稠密だが全射でないもの)
  - Definition 10.1-5 (残余スペクトル $\sigma_r(T)$：$T - \lambda I$ が単射だが像が稠密でないもの)
  - Example 10.1-2 ($\ell^2$ 上の右シフト作用素 $R$ のスペクトル：$\sigma_p(R) = \emptyset$, $\sigma(R) = \{\lambda \mid |\lambda| \leq 1\}$ の計算)
  - Example 10.1-3 ($\ell^2$ 上の左シフト作用素 $L$ のスペクトル：$\sigma_p(L) = \{\lambda \mid |\lambda| < 1\}$ であり、$L$ と $R = L^*$ のスペクトルの関係)

### §10.2 スペクトルの基本性質
**内容:** コンパクト性、実数性とスペクトル半径

- **1. スペクトルの基本性質**
  - Theorem 10.2-1 ($T \in B(X)$ のスペクトルはコンパクトで空でないこと)
  - Proof (コンパクト性の証明：$|\lambda| > \|T\|$ のときノイマン級数 $\sum T^n/\lambda^{n+1}$ が収束して $(T - \lambda I)^{-1}$ を与えること；空でないことはリューヴィルの定理を用いる)
  - Theorem 10.2-2 (スペクトル半径の公式：$r(T) = \lim_{n \to \infty} \|T^n\|^{1/n} = \inf_n \|T^n\|^{1/n}$)
  - Proof (スペクトル半径公式の証明の概略：レゾルベント関数のローラン展開と収束半径の議論)
  - Example 10.2-1 (冪零作用素 $N^k = 0$ のスペクトル半径が $0$ であることの確認)

- **2. 自己共役作用素のスペクトルの実数性**
  - Theorem 10.2-3 (自己共役作用素のスペクトルは実数直線の有界閉部分集合であること)
  - Proof (実数性の証明：$\text{Im}(\lambda) \neq 0$ のとき $\|(T - \lambda I)x\| \geq |\text{Im}(\lambda)|\|x\|$ から $(T - \lambda I)$ の逆の有界性を得る)
  - Example 10.2-2 ($L^2[0,1]$ 上の掛け算作用素 $M_g f = gf$（$g$ は実数値有界可測関数）のスペクトルが $g$ の本質的値域であることの計算)
  - Remark (正規作用素では残余スペクトルが空であるという付加的性質が成り立つ)

### §10.3 スペクトル測度と連続関数計算
**内容:** スペクトル測度と $f(A)$ の性質

- **1. 連続関数計算**
  - Theorem 10.3-1 (連続関数計算：有界自己共役作用素 $T$ と連続関数 $f : \sigma(T) \to \mathbb{C}$ に対して $f(T)$ を一意に定義でき、$\|f(T)\| = \sup_{\lambda \in \sigma(T)} |f(\lambda)|$ が成り立つこと)
  - Proof (連続関数計算の証明の概略：多項式に対しては自然な定義を与え、ストーン-ワイエルシュトラスの近似定理を用いて連続関数に拡張する)
  - Example 10.3-1 ($T$ が正の自己共役作用素（$\sigma(T) \subset [0, \infty)$）のとき $\sqrt{T}$ を $f(\lambda) = \sqrt{\lambda}$ で定義できること)

- **2. スペクトル測度**
  - Definition 10.3-1 (スペクトル測度（射影値測度）：ボレル集合 $B \subset \mathbb{R}$ に直交射影 $E(B)$ を対応させる $\sigma$-加法的な写像)
  - Theorem 10.3-2 (スペクトル測度の存在：有界自己共役作用素 $T$ に対してスペクトル測度 $E$ が一意に存在し、$T = \int_{\sigma(T)} \lambda \, dE(\lambda)$ と表されること)
  - Proof (証明の概略：リースの表現定理を $\langle f(T)x, x \rangle$ の正の線形汎関数としての性質に適用してスカラー測度を構成し、偏極等式で一般化する)
  - Example 10.3-2 ($\ell^2$ 上の対角作用素 $T(x_n) = (\lambda_n x_n)$ のスペクトル測度：$E(B)$ が $\{n \mid \lambda_n \in B\}$ に対応する座標への射影であることの確認)
  - Remark (スペクトル測度は測度論（3-3）の概念を作用素論に持ち込んだものであり、「作用素の値の分布」を記述する)

### §10.4 有界自己共役作用素のスペクトル定理
**内容:** スペクトル分解定理の主張と応用

- **1. スペクトル定理の統合**
  - Theorem 10.4-1 (有界自己共役作用素のスペクトル定理の統合形：$T$ に対してスペクトル測度 $E$ が一意に存在し、(i) $T = \int \lambda\, dE(\lambda)$；(ii) 任意のボレル関数 $f$ に対して $f(T) = \int f(\lambda)\,dE(\lambda)$；(iii) $\sigma(T) = \text{supp}(E)$；が成り立つこと)
  - Proof (統合形の証明の概略：§10.3 の結果を組み合わせ、ボレル関数への拡張を単調族定理で行う)
  - Example 10.4-1 (コンパクト自己共役作用素への適用：$T = \sum_n \lambda_n P_n$（$P_n$ は固有空間への射影）の形が §9.3 の結果と一致することの確認)

- **2. スペクトル定理の応用**
  - Example 10.4-2 (掛け算作用素のスペクトル分解：$L^2$ 上の $M_g$ に対してスペクトル測度が $E(B)f = \mathbf{1}_{g^{-1}(B)}f$ で与えられることの確認)
  - Example 10.4-3 (コンパクト自己共役作用素の場合：$T = \sum_n \lambda_n P_n$ がスペクトル定理の離散版であることと §9.3 の結果の整合性)
  - Remark (スペクトル定理は関数解析の中心的成果であり、量子力学における物理量の測定値の分布を数学的に記述する基礎となる)

### §10.5 有界正規作用素のスペクトル定理
**内容:** 実部・虚部の同時分解と正規作用素のスペクトル分解

- **1. 正規作用素の実部・虚部分解**
  - Definition 10.5-1 (正規作用素の実部と虚部：$A = (T + T^*)/2$, $B = (T - T^*)/(2i)$；$A$, $B$ は自己共役で $T = A + iB$)
  - Proposition 10.5-1 ($T$ が正規であることと $AB = BA$ が同値であること)
  - Proof ($T^*T = TT^*$ を $A$, $B$ で展開して可換条件を導く)
  - Example 10.5-1 ($\ell^2$ 上の対角作用素 $T(x_n) = (\lambda_n x_n)$（$\lambda_n \in \mathbb{C}$）に対する $A(x_n) = (\text{Re}(\lambda_n) x_n)$, $B(x_n) = (\text{Im}(\lambda_n) x_n)$ の確認)

- **2. 正規作用素のスペクトル定理**
  - Theorem 10.5-1 (有界正規作用素のスペクトル定理：$T = \int_{\sigma(T)} \lambda \, dE(\lambda)$ を満たすスペクトル測度 $E$（$\mathbb{C}$ のボレル集合上）が一意に存在すること)
  - Proof (証明の概略：可換な自己共役作用素 $A$, $B$ のスペクトル測度を同時構成する；$A$ のスペクトル測度 $E_A$ と $B$ のスペクトル測度 $E_B$ の可換性から積測度 $E$ を構成し、$\sigma(T) \subset \mathbb{C}$ 上のスペクトル分解を得る)
  - Example 10.5-2 (ユニタリ作用素への適用：$\sigma(U) \subset \{|\lambda| = 1\}$ に対するスペクトル分解；$U = e^{iA}$（$A$ は自己共役）との関係)
  - Example 10.5-3 (コンパクト正規作用素のスペクトル分解：$T = \sum_n \lambda_n P_n$（$\lambda_n \in \mathbb{C}$）の形が対角化の無限次元版であることの確認)
  - Remark (正規作用素のスペクトル定理は C*-代数のゲルファント-ナイマルクの定理と密接に関連する；この方向の発展は作用素環論（5-8）で扱う)

### §10.6 非有界作用素への展望
**内容:** 閉作用素、本質的自己共役性、フリードリヒス拡大と量子力学への接続

- **1. 閉作用素と非有界作用素の基本概念**
  - Definition 10.6-1 (非有界作用素：稠密な定義域 $D(T) \subset H$ 上で定義された線形作用素で、有界とは限らないもの)
  - Definition 10.6-2 (閉作用素：グラフ $\mathrm{Gr}(T) = \{(x, Tx)\}$ が $H \times H$ の閉部分集合であること)
  - Example 10.6-1 ($L^2(\mathbb{R})$ 上の位置作用素 $Qf(x) = xf(x)$ と運動量作用素 $Pf(x) = -if'(x)$ が非有界であること；$D(Q) = \{f \in L^2 \mid xf \in L^2\}$ の確認)
  - Remark (量子力学の基本的な観測量（位置・運動量・ハミルトニアン）はいずれも非有界作用素であり、有界作用素の理論だけでは不十分である；閉グラフ定理（§4.5）との整合性に注意：非有界閉作用素は全空間で定義されていない)

- **2. 自己共役性とフリードリヒス拡大**
  - Definition 10.6-3 (非有界作用素の随伴：$D(T^*) = \{y \mid x \mapsto \langle Tx, y \rangle$ が $D(T)$ 上有界\}$ と $\langle Tx, y \rangle = \langle x, T^*y \rangle$ による定義)
  - Definition 10.6-4 (対称作用素：$T \subset T^*$（$D(T) \subset D(T^*)$ かつ $Tx = T^*x$ for $x \in D(T)$）；自己共役：$T = T^*$（定義域も含めて一致）)
  - Remark (対称性と自己共役性の違い：$D(T) \subsetneq D(T^*)$ のとき対称だが自己共役でない；この区別は定義域の精密な制御が不可欠であり、自己共役性の判定が非有界作用素の理論の核心である)
  - Definition 10.6-5 (本質的自己共役：$T$ の閉包 $\bar{T}$ が自己共役であること)
  - Example 10.6-2 ($-d^2/dx^2$ が $C_c^\infty(\mathbb{R})$ 上で本質的自己共役であること；$[0,1]$ 上のディリクレ境界条件とノイマン境界条件が異なる自己共役拡大を与えることの指摘)
  - Theorem 10.6-1 (フリードリヒス拡大：下に有界な対称作用素は自己共役拡大を持つこと)
  - Proof (フリードリヒス拡大の証明の概略：$T$ から定まるエネルギー内積でヒルベルト空間を完備化し、リースの表現定理を用いて自己共役拡大を構成する)
  - Example 10.6-3 ($-\Delta$（ラプラシアン）の $C_c^\infty(\Omega)$（$\Omega$ は有界領域）上でのフリードリヒス拡大がディリクレ・ラプラシアンを与えること；偏微分方程式（5-2）で本格的に用いる結果の予告)
  - Remark (非有界自己共役作用素に対してもスペクトル定理が成立する；その理論は数理物理学（A-11）における量子力学の基礎を支える；詳細な展開は作用素環論（5-8）で扱う)
