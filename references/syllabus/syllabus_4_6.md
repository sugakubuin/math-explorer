## Chapter 1: リーマン計量と等長変換
**テーマ:** 計量構造・距離・等長変換・キリング場

### §1.1 リーマン計量の定義と例
**内容:** 接空間上の内積族としてのリーマン計量、および基本的な例

- **1. リーマン計量の定義**
  - Definition 1.1-1 (リーマン計量：$n$次元滑らか多様体 $M$ 上のリーマン計量とは、各点 $p \in M$ で接空間 $T_p M$ 上に内積 $g_p$ を与え、そして各点 $p$ の周りで局所座標 $(x^1, \ldots, x^n)$ に関して、成分 $g_{ij}(p) = g_p\left(\frac{\partial}{\partial x^i}\bigg|_p, \frac{\partial}{\partial x^j}\bigg|_p\right)$ が $p$ に関して滑らかに変化する対称 (2,0)-型テンソル場)
  - Example 1.1-1 (ユークリッド空間：$\mathbb{R}^n$ に標準計量 $g = \sum_{i=1}^n (dx^i)^2$ を与える；成分は $g_{ij} = \delta_{ij}$（クロネッカー記号）で常数；任意の点における内積は標準内積 $\langle v, w \rangle = \sum_i v^i w^i$ に一致することの確認)

- **2. 球面上の計量**
  - Example 1.1-2 (単位球面 $\mathbb{S}^n \subset \mathbb{R}^{n+1}$：外部ユークリッド空間から誘導される計量は、接ベクトルを $\mathbb{R}^{n+1}$ のベクトルと同一視して内積を取ったもの；$\mathbb{S}^1$ 上では局所座標 $\theta \in [0, 2\pi)$ に関して $g = d\theta^2$；$\mathbb{S}^2$ では標準座標 $(\theta, \phi)$ に関して $g = d\theta^2 + \sin^2\theta \, d\phi^2$ であることの確認)
  - Remark ($\mathbb{S}^n$ は正の曲率を持つコンパクト多様体として3章以降で重要な例となる)

- **3. 双曲空間の計量**
  - Example 1.1-3 (ポアンカレ上半平面：$\mathbb{H} = \{(x, y) : y > 0\}$ 上の計量 $g = \frac{1}{y^2}(dx^2 + dy^2)$；この計量は定負曲率 $-1$ を持つことが後に確認される)
  - Example 1.1-4 (双曲空間 $\mathbb{H}^n$：上半空間モデル $\{(x^1, \ldots, x^n) : x^n > 0\}$ 上の計量 $g = \frac{1}{(x^n)^2}\sum_{i=1}^n (dx^i)^2$；ポアンカレ円板 $|z| < 1$ 上の計量 $g = \frac{4}{(1-|z|^2)^2}|dz|^2$ との関係は共形变換によって与えられることの記述)

### §1.2 計量から誘導される距離と体積形式
**内容:** 曲線の長さ、リーマン距離関数、体積要素

- **1. 曲線の長さと距離**
  - Definition 1.2-1 (曲線の長さ：リーマン計量 $g$ をもつ多様体 $M$ 上の区分的滑らか曲線 $c : [a,b] \to M$ に対し、長さを $L(c) = \int_a^b \sqrt{g_{c(t)}(c'(t), c'(t))} \, dt = \int_a^b |c'(t)| \, dt$（ここで $|\cdot|$ は計量に関する大きさ）)
  - Example 1.2-1 (ユークリッド平面での長さ：$\mathbb{R}^2$ に標準計量を持たせて、直線 $c(t) = (at, bt)$, $t \in [0,1]$ の長さは $\int_0^1 \sqrt{a^2+b^2} \, dt = \sqrt{a^2+b^2}$ となること；これが通常の距離概念に一致することの確認)
  - Example 1.2-2 (球面上の大円：$\mathbb{S}^2$ 上の大円（赤道）を $(t, 0)$ で $t \in [0, 2\pi]$ と媒介変数化すると、長さは $\int_0^{2\pi} 1 \, dt = 2\pi$；これが単位球の赤道の周囲長であることの確認)

- **2. リーマン距離関数**
  - Definition 1.2-2 (リーマン距離：計量多様体 $(M, g)$ 上の相異なる2点 $p, q$ に対して、$d_g(p, q) = \inf\{L(c) : c \text{は} p \text{から} q \text{への区分的滑らか曲線}\}$；これは多様体上の距離関数となる（三角不等式など）；この距離から生じる位相を計量位相という)
  - Remark (この距離位相が元々の多様体位相と一致するかどうかは、多様体の完備性に関連する（3章参照）)

- **3. 体積形式**
  - Definition 1.2-3 (リーマン体積形式：$n$次元リーマン多様体 $(M, g)$ に対し、局所座標 $(x^1, \ldots, x^n)$ での体積要素を $dV = \sqrt{\det(g_{ij})} \, dx^1 \wedge \cdots \wedge dx^n$ と定義；これは座標変換に対して不変な $n$-形式を与える)
  - Example 1.2-3 (ユークリッド空間：$g_{ij} = \delta_{ij}$ より $\det(g_{ij}) = 1$ で、体積形式は $dV = dx^1 \cdots dx^n$（通常のルベーグ測度）)
  - Example 1.2-4 (単位球面 $\mathbb{S}^2$：標準座標 $(\theta, \phi)$ での計量は $g = d\theta^2 + \sin^2\theta \, d\phi^2$ より $\det(g_{ij}) = \sin^2\theta$；体積形式は $dV = \sin\theta \, d\theta \, d\phi$；全体積は $\int_0^{2\pi} \int_0^\pi \sin\theta \, d\theta \, d\phi = 4\pi$（半径1の球面）であることの確認)

### §1.3 計量の引き戻しと等長写像
**内容:** 引き戻し計量、等長写像、局所等長写像

- **1. 計量の引き戻し**
  - Definition 1.3-1 (計量の引き戻し：$f : M \to N$ が滑らか写像で $(N, g^N)$ がリーマン多様体のとき、$M$ 上の引き戻し計量を $(f^*g^N)_p(v, w) = g^N_{f(p)}(df_p(v), df_p(w))$（$v, w \in T_p M$）で定義；局所座標では $(f^*g^N)_{ij} = g^N_{kl}(f) \frac{\partial f^k}{\partial x^i} \frac{\partial f^l}{\partial x^j}$ で与えられる)
  - Example 1.3-1 (円への巻き付け：$f : \mathbb{R} \to \mathbb{S}^1$, $f(t) = e^{2\pi i t}$ で定義し、$\mathbb{S}^1$ に標準計量 $g = d\theta^2$ を持たせると、引き戻し計量は $f^*g = (2\pi)^2 dt^2$；この計量は $\mathbb{R}$ を無限倍スケールした円に同型にする)

- **2. 等長写像と局所等長写像**
  - Definition 1.3-2 (等長写像：リーマン多様体 $(M, g^M)$ から $(N, g^N)$ への微分同相写像 $f : M \to N$ が等長写像であるとは、$g^M = f^*g^N$；つまり全ての $p \in M$ と接ベクトル $v, w \in T_p M$ に対して $g^M_p(v, w) = g^N_{f(p)}(df_p(v), df_p(w))$ が成立)
  - Definition 1.3-3 (局所等長写像：$f : M \to N$ が局所等長写像であるとは、各点 $p \in M$ の周りで $f$ を制限したものが等長写像となる開部分集合が存在すること)
  - Example 1.3-2 (円筒と平面の局所等長性：円筒 $\mathbb{S}^1 \times \mathbb{R} \subset \mathbb{R}^3$ に誘導計量を、平面 $\mathbb{R}^2$ に標準計量を与えたとき、写像 $f : (0, 2\pi) \times \mathbb{R} \to \mathbb{R}^2$, $f(\theta, z) = (\theta, z)$ は局所等長写像であること；この写像を球面に拡張できないのは球面が正曲率を持つことに関連する)

- **3. 等長変換群**
  - Definition 1.3-4 (等長変換群：リーマン多様体 $(M, g)$ の自身への等長写像全体の集合を $\text{Isom}(M, g)$ で表す；これは合成と逆写像で群をなし、デカルト積と矛盾しない位相構造を備えたリー群となる)
  - Example 1.3-3 (ユークリッド空間：$\text{Isom}(\mathbb{R}^n)$ は直交行列と並進変換の合成で表される群；これは直交群 $O(n)$ と $\mathbb{R}^n$ の半直積で $O(n) \ltimes \mathbb{R}^n$)
  - Example 1.3-4 (球面：$\text{Isom}(\mathbb{S}^n)$ は回転群 $SO(n+1)$；直感的には球面を固定する剛体回転全体)

### §1.4 キリング場と等長変換群
**内容:** キリング場の方程式、等長変換群のリー代数としての特徴づけ

- **1. キリング場の定義と方程式**
  - Definition 1.4-1 (キリング場：リーマン多様体 $(M, g)$ 上のベクトル場 $X$ がキリング場（あるいはキリングベクトル場）であるとは、$X$ の流 $\phi_t$ がすべての $t$ で局所等長写像となること；同値的に、リー微分が消える条件 $\mathcal{L}_X g = 0$ で表現される)
  - Proposition 1.4-1 (キリングの方程式：ベクトル場 $X$ がキリング場であることと、局所座標で $X$ の成分 $X^k$ が偏微分方程式 $\nabla_i X_j + \nabla_j X_i = 0$（ここで $X_i = g_{ij}X^j$）を満たすことは同値；つまり $X$ の共変微分（後の2章で定義）が完全反対称テンソル)
  - Proof (条件 $\mathcal{L}_X g = 0$ はリー微分の定義から $d(i_X g) + i_X dg$ の形で表される；$g$ が計量なので $dg = 0$ 局所的に成り立ち、内積縮約を座標で展開するとキリング方程式を得る)

- **2. キリング場の例と計算**
  - Example 1.4-1 (ユークリッド空間のキリング場：$\mathbb{R}^n$ で $X = \sum_i a^i \frac{\partial}{\partial x^i}$ は並進に対応し、常にキリング場（$\nabla_i X_j = 0$ より自動的に満たされる）；また $X = \sum_{i<j} c_{ij}(x^j \frac{\partial}{\partial x^i} - x^i \frac{\partial}{\partial x^j})$ は無限小回転で、これもキリング場であることの計算確認)
  - Example 1.4-2 (球面上のキリング場：$\mathbb{S}^n \subset \mathbb{R}^{n+1}$ に対して $X = x \times v$（ここで $x$ は位置ベクトル、$v$ は定ベクトル、$\times$ は外積）は回転に対応するキリング場；$\mathbb{S}^2$ では赤道周りの回転 $X = \frac{\partial}{\partial \phi}$ がキリング場であることの確認)
  - Example 1.4-3 (双曲平面のキリング場：ポアンカレ上半平面 $\mathbb{H}$ で計量 $g = \frac{1}{y^2}(dx^2 + dy^2)$ のもと、$X = \frac{\partial}{\partial x}$ と $X = x\frac{\partial}{\partial x} + y\frac{\partial}{\partial y}$ および $X = -\frac{\partial}{\partial y}$ の組み合わせがキリング場族を生成すること)

- **3. キリング場の幾何的意味**
  - Proposition 1.4-2 (キリング場と等長性：キリング場 $X$ の完全な流 $\phi_t : M \to M$ は全時間 $t$ に対して等長変換 $\phi_t \in \text{Isom}(M, g)$ を定める；逆に $\text{Isom}(M, g)$ が連結1パラメータ部分群を含むとき、その無限小生成元はキリング場である)
  - Proof (キリング方程式 $\mathcal{L}_X g = 0$ が流 $\phi_t$ に対して $\frac{d}{dt}\phi_t^* g = \phi_t^*(\mathcal{L}_X g) = 0$ を意味することから、$\phi_t^* g = g$（等長性）が従う；逆方向は1パラメータ群の生成元のリー微分がゼロであることから直接得られる)
  - Definition 1.4-2 (キリング場のリー代数：リーマン多様体 $(M, g)$ のすべてのキリング場の集合 $\mathfrak{isom}(M, g)$ はベクトル場のブラケット $[X, Y] = XY - YX$ の下でリー代数をなす；この代数は等長変換群 $\text{Isom}(M, g)$ のリー代数である)
  - Remark (有限次元空間での結果：コンパクト多様体上のキリング場のリー代数は有限次元で、リーマン多様体の曲率に強く制約される；例えば定曲率空間のみが最大次元のキリング場族を持つ（コンパクト場合）)

- **4. 定曲率空間のキリング群**
  - Example 1.4-4 (ユークリッド空間：$\mathbb{R}^n$ の等長変換群 $\text{Isom}(\mathbb{R}^n) \cong O(n) \ltimes \mathbb{R}^n$ は半直積で、次元は $\frac{n(n-1)}{2} + n = \frac{n(n+1)}{2}$；これはユークリッド空間の対称性が最大（定正曲率0）であることを反映)
  - Example 1.4-5 (単位球面：$\mathbb{S}^n$ の等長変換群は $\text{Isom}(\mathbb{S}^n) = SO(n+1)$ で次元 $\frac{(n+1)n}{2}$；キリング場は無限小回転で、$\mathbb{S}^2$ では3次元族 $SO(3)$、$\mathbb{S}^3$ では6次元族 $SO(4)$ を生成することの記述)
  - Example 1.4-6 (双曲空間：$\mathbb{H}^n$ の等長変換群は $SO(n,1)$ で次元 $\frac{n(n+1)}{2}$；上半平面では無限小双曲回転、並進、鏡映がキリング場を生成することの記述)

## Chapter 2: レビ-チビタ接続と共変微分
**テーマ:** アフィン接続・ねじれテンソル・計量的条件・クリストッフェル記号・共変微分

### §2.1 アフィン接続の定義
**内容:** 接空間の平行移動、接続の公理的定義

- **1. アフィン接続の公理**
  - Definition 2.1-1 (アフィン接続：$n$次元滑らか多様体 $M$ 上のアフィン接続 $\nabla$ とは、滑らかベクトル場 $X, Y$ に対して新たなベクトル場 $\nabla_X Y$ を対応させる演算で、以下を満たすもの：(i) $\nabla_{fX+gY} Z = f\nabla_X Z + g\nabla_Y Z$（$X$ 方向について $\mathbb{R}$-線形かつ関数係数線形）、(ii) $\nabla_X(fY) = X(f) Y + f\nabla_X Y$（積則）、(iii) $\nabla_X Y = \nabla_Y X + [X,Y]$ は接続の一般化ではなく、ねじれをコントロール；この $\nabla_X Y$ を $X$ 方向の $Y$ の共変微分という)
  - Remark (定義(iii)は後の定義とは異なり、ここはアフィン接続の最小限の公理に限定する)

- **2. 接続の公理系**
  - Definition 2.1-2 (正確なアフィン接続の定義：$n$次元滑らか多様体 $M$ 上のアフィン接続 $\nabla$ は、各点 $p$ に対して $T_p M \times \Gamma(TM) \to T_p M$ を、$(X, Y) \mapsto (\nabla_X Y)|_p$ で与える対応で、以下の条件を満たす：(i) 各 $p$ で $(\nabla_X Y)|_p$ は $X|_p$ のみに依存（局所性）、(ii) 上記の3つの公理；あるいは局所座標 $(x^i)$ で $\nabla_{\frac{\partial}{\partial x^j}} \frac{\partial}{\partial x^k} = \Gamma^l_{jk} \frac{\partial}{\partial x^l}$（クリストッフェル記号 $\Gamma^l_{jk}$）で指定される)

- **3. 接続の例**
  - Example 2.1-1 (ユークリッド空間の標準接続：$\mathbb{R}^n$ で $\nabla_X Y = DY(X)$（方向微分；座標では常に $\frac{\partial}{\partial x^k}$ に沿う微分）；クリストッフェル記号は全て0で $\Gamma^l_{jk} = 0$；この接続は自明接続)
  - Example 2.1-2 (部分多様体の誘導接続：$M$ が $\mathbb{R}^m$ 内の部分多様体であるとき、外部ユークリッド接続を $M$ に投影することで接続を定義；一般には非零のクリストッフェル記号を持つ)

### §2.2 ねじれテンソルと計量的条件
**内容:** ねじれの定義、計量適合性、レビ-チビタ接続の一意性

- **1. ねじれテンソル**
  - Definition 2.2-1 (ねじれテンソル：アフィン接続 $\nabla$ に対し、ねじれテンソルを $T(X,Y) = \nabla_X Y - \nabla_Y X - [X,Y]$ で定義；これはスムーズなベクトル場に対して（1,2)-型テンソル場をなす；ねじれなし（torsion-free）であるとは $T(X,Y) = 0$ が全ての $X, Y$ に対して成立することを意味する)
  - Example 2.2-1 (ユークリッド空間の標準接続はねじれなし：$\nabla_X Y - \nabla_Y X = DY(X) - DX(Y) = [X,Y]$ より $T(X,Y) = 0$ であることの確認)

- **2. 計量適合性**
  - Definition 2.2-2 (計量的条件：リーマン計量 $g$ を持つ多様体上のアフィン接続 $\nabla$ が計量的であるとは、全ての滑らかベクトル場 $X, Y, Z$ に対して $X(g(Y,Z)) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)$；つまり平行移動によって内積が保存される)
  - Proposition 2.2-1 (計量的条件の局所的表示：接続が計量的であることと、クリストッフェル記号が $g_{kl}\Gamma^l_{ij} + g_{il}\Gamma^l_{jk} + \frac{\partial g_{ij}}{\partial x^k} = 0$ を満たすことは同値；すなわち $\nabla g = 0$)
  - Proof (計量条件を座標で展開し、$\frac{\partial g_{jk}}{\partial x^i} = \Gamma^l_{ij}g_{lk} + \Gamma^l_{ik}g_{jl}$ という対称性を用いて整理する)

- **3. レビ-チビタ接続の存在と一意性**
  - Theorem 2.2-1 (リーマン多様体の基本定理：リーマン計量 $g$ をもつ多様体 $M$ に対し、ねじれなし・計量的なアフィン接続が一意に存在する；これをレビ-チビタ接続という)
  - Proof (一意性：2つのそのような接続 $\nabla$ と $\nabla'$ の差を $D = \nabla - \nabla'$ とすると、$D$ は (1,2)-テンソルで、ねじれなし条件から $D(X,Y) = D(Y,X)$、計量性から $g(D(X,Y), Z) + g(Y, D(X,Z)) = 0$ を得て、これらから $D = 0$ が従う；存在性は $\Gamma^l_{jk}$ をクレプシュの公式で陰的に定義することで与えられる)

- **4. クレプシュの公式への導出**
  - Proposition 2.2-2 (クレプシュの公式の形式：レビ-チビタ接続のクリストッフェル記号は $\Gamma^l_{ij} = \frac{1}{2}g^{lk}\left(\frac{\partial g_{kj}}{\partial x^i} + \frac{\partial g_{ki}}{\partial x^j} - \frac{\partial g_{ij}}{\partial x^k}\right)$；これは計量とその1階偏導関数のみから決定される)
  - Remark (この公式は一見複雑であるが、ねじれなし・計量的という2つの条件から純粋に代数的に導出できることが本質的)

### §2.3 クリストッフェル記号
**内容:** クリストッフェル記号の計算、変換規則、特別な座標系

- **1. クリストッフェル記号の計算**
  - Example 2.3-1 (ユークリッド空間の極座標：$\mathbb{R}^2$ を極座標 $(r, \theta)$ で表すと計量は $g = dr^2 + r^2 d\theta^2$；非零クリストッフェル記号は $\Gamma^1_{22} = -r$, $\Gamma^2_{12} = \Gamma^2_{21} = \frac{1}{r}$ のみ；クレプシュの公式での計算例として具体的導出)
  - Example 2.3-2 (球面の標準計量：$\mathbb{S}^2$ 上で $(\theta, \phi)$ 座標での計量 $g = d\theta^2 + \sin^2\theta \, d\phi^2$ のクリストッフェル記号：$\Gamma^1_{22} = -\sin\theta\cos\theta$, $\Gamma^2_{12} = \Gamma^2_{21} = \cot\theta$；これらは大円測地線の方程式に直結する)

- **2. 座標変換と記号の変換規則**
  - Proposition 2.3-1 (クリストッフェル記号の変換則：座標 $(x^i)$ から $(y^j)$ への座標変換 $x^i = x^i(y)$ のもと、クリストッフェル記号は $\Gamma'^{\alpha}_{\beta\gamma} = \frac{\partial x^i}{\partial y^\beta} \frac{\partial x^j}{\partial y^\gamma} \frac{\partial y^\alpha}{\partial x^k} \Gamma^k_{ij} + \frac{\partial y^\alpha}{\partial x^k} \frac{\partial^2 x^k}{\partial y^\beta \partial y^\gamma}$；この複雑さから、クリストッフェル記号はテンソル場ではなく接続の成分に過ぎないことが明らかになる)
  - Example 2.3-3 (単位円の計量の記述：$\mathbb{S}^1$ に角度 $\theta \in [0, 2\pi)$ で計量 $g = d\theta^2$；別の座標 $u = \cos\theta$, $v = \sin\theta$ での記述は複雑だが、本来の計量の本質は保存される)

- **3. 通常座標と法座標**
  - Definition 2.3-1 (通常座標系（normal coordinates）：点 $p \in M$ の周りで、指数写像（3章で定義）を用いて局所座標を引き出したもの；このとき $p$ での $\Gamma^l_{jk}|_p = 0$ が成立し、クリストッフェル記号の1階導関数のみが現れる特別な座標系)
  - Remark (通常座標では計算が大幅に簡潔化され、特に曲率テンソル（4章）の定義がシンプルになる)

### §2.4 テンソル場の共変微分
**内容:** 一般テンソル場への共変微分の拡張、成分表示、交換関係

- **1. 共変微分の拡張と規則**
  - Definition 2.4-1 (テンソル場の共変微分：接続 $\nabla$ をテンソル場に拡張する；$(r,s)$-型テンソル場 $T$ に対して $\nabla_X T$ を，ベクトル場 $X$ 方向の $T$ の共変微分として定義；これはライプニッツ則 $\nabla_X(S \otimes T) = (\nabla_X S) \otimes T + S \otimes (\nabla_X T)$ と縮約に関する可換性を満たすよう拡張される)
  - Example 2.4-1 (計量テンソルの共変微分：レビ-チビタ接続では $\nabla g = 0$（計量的条件）；成分で $\nabla_k g_{ij} = 0$；これはレビ-チビタ接続の定義特性)

- **2. 成分による計算**
  - Definition 2.4-2 (共変微分の成分表示：$(r,s)$-型テンソル $T^{i_1\cdots i_r}_{j_1\cdots j_s}$ に対して、共変微分 $(\nabla_k T)^{i_1\cdots i_r}_{j_1\cdots j_s}$ は各上付き添字に対して $+\Gamma^{i_a}_{kl}$ 項を，各下付き添字に対して $-\Gamma^l_{kj_b}$ 項を追加した形式で与えられる；例えば $(1,1)$-テンソル $T^i_j$ では $\nabla_k T^i_j = \frac{\partial T^i_j}{\partial x^k} + \Gamma^i_{kl}T^l_j - \Gamma^l_{kj}T^i_l$)
  - Example 2.4-2 (ベクトル場の共変微分：$X^i$ の共変微分は $\nabla_j X^i = \frac{\partial X^i}{\partial x^j} + \Gamma^i_{jk}X^k$ で与えられ、これは2章冒頭のベクトル場への作用と一致することの確認)
  - Example 2.4-3 (1-形式の共変微分：$\omega_i$（1-形式）に対して $\nabla_j\omega_i = \frac{\partial\omega_i}{\partial x^j} - \Gamma^k_{ji}\omega_k$；特に計量 $g_{ij}$ では $\nabla_k g_{ij} = 0$ がすべての $i,j,k$ で成立する)

- **3. 共変微分のブラケット交換と曲率への萌芽**
  - Proposition 2.4-1 (共変微分の非可換性：一般にベクトル場 $X, Y$ に対して $\nabla_X \nabla_Y Z \neq \nabla_Y \nabla_X Z$ であり、その差は曲率テンソル（4章）に関連する；具体的には $[\nabla_X, \nabla_Y]Z = \nabla_{[X,Y]}Z + R(X,Y)Z$ という形で、$R$ はリーマン曲率テンソル)
  - Proof (成分計算で左辺から右辺を引き、クリストッフェル記号の偏導関数項が現れることを示す；この差がテンソルをなすことが曲率テンソルの存在を正当化)
  - Remark (この非可換性は多様体が曲がっていることの本質的な指標であり、平坦空間（ユークリッド）では共変微分は可換)

- **4. 平行移動と接続の幾何的解釈**
  - Definition 2.4-3 (ベクトル場の平行移動：曲線 $c(t)$ 沿いのベクトル場 $Y(t)$ が平行であるとは、$\frac{D Y}{dt} := \nabla_{c'(t)} Y = 0$ が成立することを意味する；この常微分方程式の解は平行移動ベクトルである)
  - Example 2.4-4 (ユークリッド平面での平行移動：直線に沿ったベクトルを"平行移動"すると、無限小変位 $dx$ で新座標での成分が変わらない（自明な接続）；これ対比して球面での平行移動はジオデシック方程式を解く過程で自動的に定まり、一般には経路に依存する)

## Chapter 3: 測地線
**テーマ:** 測地線方程式・変分特徴づけ・指数写像・ホップ-リノウの定理

### §3.1 測地線の定義と測地線方程式
**内容:** 自己平行曲線としての測地線、基本方程式

- **1. 測地線の定義**
  - Definition 3.1-1 (測地線：リーマン多様体 $(M, g)$ 上の曲線 $c(t)$ が測地線であるとは、$\frac{Dc'}{dt} = 0$（レビ-チビタ接続に関する平行移動条件）を満たすこと；あるいは測地線方程式 $\ddot{c}^k(t) + \Gamma^k_{ij}(c(t)) \dot{c}^i(t) \dot{c}^j(t) = 0$ で表現される；ここで $\dot{c}^i = \frac{dc^i}{dt}$, $\ddot{c}^k = \frac{d^2 c^k}{dt^2}$)
  - Remark (測地線は曲線上の接ベクトルが"平行移動"で進化する曲線；局所的には最短経路の候補である)

- **2. 測地線方程式の解の存在**
  - Proposition 3.1-1 (常微分方程式としての性質：測地線方程式は2階自律常微分方程式系で、初期条件 $c(0) = p$, $c'(0) = v \in T_p M$ に対して一意な局所解が存在する；これは定値の時間パラメータに対して定常性を持たない（パラメータが弧長パラメータでない場合）)
  - Proof (常微分方程式理論の標準的存在一意性定理；複数の初期条件に対して解がグローバルに定義されるかどうかは多様体の完備性に関わる（定義3.3-1参照）)

- **3. 重要な特殊ケース**
  - Example 3.1-1 (ユークリッド空間の測地線：$\mathbb{R}^n$ ではすべてのクリストッフェル記号が0なので、測地線方程式は $\ddot{c}^k = 0$；解は直線 $c(t) = p + tv$；これは通常の直線が最短経路であることに対応)
  - Example 3.1-2 (球面の測地線：$\mathbb{S}^2$ 上で、赤道は測地線（大円）；極座標 $(\theta, \phi)$ での方程式を立てると、赤道 $\theta = \pi/2$ が確かに方程式を満たすことの確認)
  - Example 3.1-3 (双曲平面の測地線：ポアンカレ上半平面 $\mathbb{H}$ ではクリストッフェル記号の複雑性から、測地線方程式は非自明；解は鉛直線と円弧で与えられることが知られている)

### §3.2 測地線の変分的特徴づけ
**内容:** 弧長汎関数とエネルギー汎関数、第一変分公式

- **1. 弧長汎関数と弧長パラメータ**
  - Definition 3.2-1 (弧長汎関数：区分的滑らか曲線 $c : [a, b] \to M$ に対して、弧長汎関数を $L(c) = \int_a^b |c'(t)| \, dt$（$|c'(t)| = \sqrt{g_{c(t)}(c'(t), c'(t))}$）で定義；弧長パラメータ化 (arc length parametrization) は $|c'(t)| = 1$ を常に満たす特殊な媒介変数化)
  - Remark (弧長パラメータを持つ測地線は特に扱いやすく、加速度ベクトルが完全に曲率に依存する)

- **2. 第一変分公式**
  - Theorem 3.2-1 (測地線と第一変分：$c(t)$, $t \in [a,b]$ を固定端点 $c(a) = p$, $c(b) = q$ を持つ区分的滑らか曲線とする；$c$ を含む変分 $c_s(t)$ に対して（s-パラメータで $c_0 = c$, $\frac{\partial c_s}{\partial s}\big|_{s=0} = V$），弧長汎関数の第一変分は $\frac{dL}{ds}\big|_{s=0} = \int_a^b g\left(\frac{Dc'}{dt}, V\right) dt + $ 境界項で与えられる；固定端点条件 $V(a) = V(b) = 0$ のもと、臨界点 $\frac{dL}{ds} = 0$ となる必要条件は $\frac{Dc'}{dt} = 0$（測地線方程式）)
  - Proof (変分場 $V$ に関する計算；部分積分とライプニッツ則を用いて臨界性から測地線方程式を導く)

- **3. エネルギー汎関数と等価性**
  - Definition 3.2-2 (エネルギー汎関数：曲線 $c : [a, b] \to M$ のエネルギーを $E(c) = \frac{1}{2}\int_a^b |c'(t)|^2 \, dt$ で定義；これは弧長汎関数 $L$ と異なるが、弧長パラメータ化に対しては同一の臨界点を持つ)
  - Proposition 3.2-1 (エネルギー最小化と測地線：定速度曲線の中で、エネルギーを最小化するものは測地線である；逆に測地線でない曲線の変分はエネルギーを常に増加させる)
  - Proof (コーシー-シュワルツの不等式 $L(c)^2 \leq 2(b-a)E(c)$ から、等号は定速度曲線のときに限ること；エネルギーの第一変分がゼロとなる条件が測地線方程式と一致することを直接計算で確認する)

### §3.3 指数写像
**内容:** 指数写像の定義、局所微分同相、法座標系

- **1. 指数写像の定義**
  - Definition 3.3-1 (指数写像：点 $p \in M$ に対して、接空間 $T_p M$ の原点を始点とするベクトル $v$ に対応する測地線 $c_v(t)$ を $c_v(0) = p$, $c_v'(0) = v$ で定義し、$\exp_p(v) := c_v(1)$ と定義する；この写像 $\exp_p : T_p M \to M$ を指数写像という；一般に $v = tv_0$ の短い $t$ に対してのみ定義可能（最大時間を $t_{\max}$ とする）で、$\exp_p$ の定義域を convex ball $B_\epsilon = \{v \in T_p M : |v| < \epsilon\}$ に制限)
  - Remark (指数写像の詳細な収束半径 $\epsilon$ は多様体の曲率に依存；小さい曲率では $\epsilon$ は大きい傾向)

- **2. 指数写像の基本性質**
  - Proposition 3.3-1 (指数写像は局所微分同相：$p$ を中心とする十分に小さい凸球 $B_\epsilon \subset T_p M$ に対して、$\exp_p|_{B_\epsilon} : B_\epsilon \to \exp_p(B_\epsilon) \subset M$ は微分同相；特に $d(\exp_p)_0 : T_p M \to T_p M$ は恒等写像)
  - Proof (逆関数定理；微分 $d(\exp_p)_0$ を計算して恒等性を確認し、十分に小さい球で局所微分同相を得る)

- **3. 法座標と正規座標系**
  - Definition 3.3-2 (法座標系（normal coordinates）：点 $p$ に対して、$T_p M$ の正規直交基 $\{e_1, \ldots, e_n\}$ を選び、$v = \sum_i t^i e_i$ で表される $v \in T_p M$ に対して，$(t^1, \ldots, t^n)$ を $p$ 周辺の法座標とする；つまり $(t^1, \ldots, t^n) \leftrightarrow \exp_p(\sum_i t^i e_i)$；この座標では $p$ での計量は $g_{ij}(p) = \delta_{ij}$ であり、クリストッフェル記号は $\Gamma^k_{ij}|_p = 0$)
  - Example 3.3-1 (ユークリッド平面の法座標：$\mathbb{R}^2$ の原点で標準座標を法座標とすると、指数写像は恒等写像；成分 $(t^1, t^2)$ はそのまま直線座標)
  - Example 3.3-2 (球面での法座標：$\mathbb{S}^2$ の北極 $N$ での法座標は、赤道面内の角座標で与えられ、$\exp_N(r\mathbf{u}) = $ 北極から角距離 $r$ の点；すべての測地線（大円）は指数写像の像となる)

### §3.4 ホップ-リノウの定理
**内容:** 完備性の同値条件、直径の有限性

- **1. リーマン多様体の完備性**
  - Definition 3.4-1 (測地線完備性・計量完備性：多様体 $M$ が測地線完備であるとは、すべての測地線が時間パラメータ $t \in \mathbb{R}$ で無限に延びること；計量完備であるとは、リーマン距離函数 $(d_g)$ から誘導される距離位相が多様体の通常位相と一致し、かつこの距離に関して $(M, d_g)$ が位相群としての距離空間であること)
  - Proposition 3.4-1 (完備性の初等的性質：(i) コンパクト多様体は常に測地線完備；(ii) ユークリッド空間 $\mathbb{R}^n$ は測地線完備；(iii) 開球 $B(p, r) = \{q \in M : d_g(p,q) < r\}$ は完備でない一般的例)

- **2. ホップ-リノウの定理**
  - Theorem 3.4-1 (ホップ-リノウの定理：$n$次元リーマン多様体 $M$ に対して、以下は同値：(i) $M$ が測地線完備；(ii) $M$ が計量完備；(iii) 任意の $p, q \in M$ に対して、最短経路を実現する測地線が存在する（つまり $d_g(p,q) = L(c_{pq})$ を最小化する曲線 $c_{pq}$ が存在して測地線）；(iv) 指数写像 $\exp_p$ がすべての $p$ で全接空間 $T_p M$ で定義される；(v) 任意の有界閉集合が全有界かつ相対コンパクト)
  - Proof (主要なステップ：(i)$\Rightarrow$(iv)は測地線の拡張定理；(iv)$\Rightarrow$(ii)は距離の連続性；(ii)$\Rightarrow$(iii)はArzelà-Ascoliの定理と測地線方程式の連続依存性；(v)は距離空間の完備性とコンパクト性の特性化)

- **3. 指数写像の単射半径**
  - Definition 3.4-2 (単射半径・共役点の距離：点 $p \in M$ での単射半径 $r_p = \sup\{t > 0 : \exp_p|_{B_t(0)} \text{は単射}\}$ を定義；$\exp_p$ が単射性を失う点を共役点という（4章で詳述）；多様体全体での単射半径 $\text{inj}(M) = \inf_p r_p$ は幾何的に最も小さい特異性スケール)
  - Example 3.4-1 (球面の単射半径：$\mathbb{S}^n$ では単射半径は $\pi$（対点までの距離）；$\exp_p : B_\pi(0) \to \mathbb{S}^n \setminus \{-p\}$ は微分同相)
  - Example 3.4-2 (トーラスの単射半径：$\mathbb{T}^n = \mathbb{R}^n / \mathbb{Z}^n$（標準計量）では単射半径は $\frac{1}{2}$；異なる測地線が同じ像に収束し始める距離)

- **4. 距離関数と勾配**
  - Definition 3.4-3 (距離関数 $d_q : M \to \mathbb{R}$, $d_q(x) = d_g(x, q)$ の微分可能性と勾配：点 $q$ が固定されているとき、$d_q$ は $q$ 以外で $C^1$ であり、$\nabla d_q$ は $q$ から $x$ への最短測地線方向の単位ベクトル；ただし測地線分岐地点では微分不可能)
  - Remark (距離関数の非滑らか点（共役点や分岐点）の研究は幾何測度論（後続科目）につながる)

## Chapter 4: 曲率テンソル
**テーマ:** リーマン曲率テンソル・断面曲率・リッチテンソル・ヤコビ場・共役点

### §4.1 リーマン曲率テンソルの定義
**内容:** 共変微分の非可換性、曲率テンソルの定義と対称性

- **1. 曲率テンソルの定義**
  - Definition 4.1-1 (リーマン曲率テンソル：レビ-チビタ接続 $\nabla$ に対して、リーマン曲率テンソル $R$ を $R(X,Y)Z = [\nabla_X, \nabla_Y]Z - \nabla_{[X,Y]}Z$ で定義；これは $(1,3)$-型テンソルで，（下付き化して） $(0,4)$-型にすると $R(X,Y,Z,W) = g(R(X,Y)Z, W)$ で書ける；成分では $R^i_{jkl}$ またはより対称な記法 $R_{ijkl}$ で表現)
  - Example 4.1-1 (ユークリッド空間の曲率：$\mathbb{R}^n$ ではすべてのクリストッフェル記号が定数なので、$\nabla$ の高階微分は全て0；したがって $R \equiv 0$；平坦性の特性化)

- **2. リーマン曲率の対称性と ビアンキ恒等式**
  - Proposition 4.1-1 (曲率テンソルの対称性：リーマン曲率テンソル $R_{ijkl}$ は以下を満たす：(i) 反対称性 $R_{ijkl} = -R_{jikl}$ （最初の2添字）；(ii) 反対称性 $R_{ijkl} = -R_{ijlk}$ （後の2添字）；(iii) ブロック対称性 $R_{ijkl} = R_{klij}$；（計量テンソルに関する特性）)
  - Proposition 4.1-2 (ビアンキ恒等式：第一ビアンキ恒等式 $R(X,Y)Z + R(Y,Z)X + R(Z,X)Y = 0$；第二ビアンキ恒等式（微分形式）$\nabla R = 0$ をクライン記号で表現，あるいは成分 $\nabla_l R_{ijkm} + \nabla_k R_{ijml} + \nabla_m R_{ijlk} = 0$)
  - Proof (第一恒等式は定義から直接導出；第二恒等式は共変微分の交換関係とテンソルの対称性から従う)

- **3. 曲率テンソルの計算**
  - Example 4.1-2 (球面 $\mathbb{S}^2$ の曲率：標準計量 $g = d\theta^2 + \sin^2\theta \, d\phi^2$ のもと、非零成分は $R_{1212} = \sin^2\theta$；これから断面曲率（§4.2）が1であることが分かる)
  - Example 4.1-3 (双曲平面の曲率：ポアンカレ上半平面の計量 $g = \frac{1}{y^2}(dx^2 + dy^2)$ では、対応する曲率成分 $R_{1212} = -1$；これは定負曲率を持つ面の特性)

### §4.2 断面曲率
**内容:** 2平面張る曲率、定曲率空間

- **1. 断面曲率の定義**
  - Definition 4.2-1 (断面曲率：点 $p \in M$ で、2次元線形部分空間 $\sigma = \text{span}\{X, Y\} \subset T_p M$（$X, Y$ 線形独立）が与えられたとき、この平面での断面曲率を $K(\sigma) = \frac{R(X,Y,X,Y)}{|X \wedge Y|^2}$ で定義；ここで $|X \wedge Y|^2 = g(X,X)g(Y,Y) - g(X,Y)^2$（グラミアン）は2形式の大きさ)
  - Remark (この定義は張る基の選び方に依らず，2平面 $\sigma$ にのみ依存することが対称性から保証される)

- **2. 定曲率空間**
  - Definition 4.2-2 (定曲率多様体：リーマン多様体が定曲率 $K$ を持つとは，すべての点で，すべての2平面での断面曲率が同じ定数 $K$ であること；定曲率空間ではリーマン曲率テンソルが $R(X,Y,Z,W) = K[g(X,Z)g(Y,W) - g(X,W)g(Y,Z)]$ で完全に決定される)
  - Proposition 4.2-1 (定曲率空間の特性化：$n$ 次元単連結完備リーマン多様体が定曲率 $K$ を持つなら，以下のいずれかに大域的に等長同型：(i) $K > 0$ のとき $\mathbb{S}^n(1/\sqrt{K})$；(ii) $K = 0$ のとき $\mathbb{R}^n$；(iii) $K < 0$ のとき $\mathbb{H}^n(1/\sqrt{|K|})$)
  - Proof (指数写像が大域的な等長写像となることを示す；定曲率条件からヤコビ場が比較空間と同一の挙動を示し、指数写像が被覆写像であること、単連結性から同型であることを導く)
  - Example 4.2-1 (球面：半径 $R$ の球面は定曲率 $K = 1/R^2$ を持つ；単位球面 $\mathbb{S}^n$ では $K = 1$；極限 $R \to \infty$ でユークリッド空間へ遷移)
  - Example 4.2-2 (双曲空間：ポアンカレ上半平面は定曲率 $K = -1$；双曲幾何での角の和が $\pi$ より小さいことの関連付け)
  - Example 4.2-3 (トーラス：$\mathbb{T}^2 = \mathbb{S}^1 \times \mathbb{R}$ に積計量を入れると定曲率 $K = 0$；異なる2平面での曲率を計算して確認)

### §4.3 リッチテンソルとスカラー曲率
**内容:** リッチテンソルの定義、スカラー曲率、アインシュタイン方程式

- **1. リッチテンソルの定義**
  - Definition 4.3-1 (リッチテンソル：リーマン曲率テンソル $R^i_{jkl}$ に対して、リッチテンソルを最初と3番目の指標の縮約 $\text{Ric}_{jl} = R^i_{jil}$ で定義；これは $(0,2)$-型対称テンソル；成分形式では $\text{Ric}_{ij} = g^{kl} R_{ikjl}$)
  - Example 4.3-1 (球面のリッチテンソル：$\mathbb{S}^n$ では定曲率から $\text{Ric}_{ij} = (n-1)g_{ij}$；特に $\mathbb{S}^2$ では $\text{Ric} = g$（リッチテンソルが計量に比例）)

- **2. スカラー曲率**
  - Definition 4.3-2 (スカラー曲率：リッチテンソルのトレース $R = g^{ij}\text{Ric}_{ij}$ をスカラー曲率という；これは各点での平均的な曲率を測る関数；$R > 0$ は正曲率，$R < 0$ は負曲率，$R = 0$ は一部が平坦)
  - Example 4.3-2 (定曲率空間のスカラー曲率：定曲率 $K$ を持つ $n$-次元多様体では $R = n(n-1)K$；球面 $\mathbb{S}^n$ では $R = n(n-1)$；$\mathbb{R}^n$ では $R = 0$)

- **3. アインシュタイン方程式への示唆**
  - Definition 4.3-3 (アインシュタインテンソル：アインシュタインテンソル $G_{ij} = \text{Ric}_{ij} - \frac{1}{2}R g_{ij}$ を定義；一般相対性理論ではアインシュタイン方程式 $G_{ij} = 8\pi T_{ij}$（$T$ は応力-エネルギーテンソル）で物質の分布が時空の曲率を決める；定曲率空間では $G_{ij} = \left(K - \frac{R}{2}\right)g_{ij}$ となり、カタストロフ的に簡潔)
  - Remark (本コース内では一般相対性理論への深い接続は扱わないが，幾何学的構造の力強さが見える)

### §4.4 ヤコビ場と共役点
**内容:** ヤコビ方程式、第二変分公式、共役点の存在

- **1. ヤコビ方程式の導出**
  - Definition 4.4-1 (ヤコビ場：測地線 $\gamma(t)$ に沿うベクトル場 $J(t)$ がヤコビ場であるとは，ヤコビ方程式 $\frac{D^2 J}{dt^2} + R(J, \gamma'))\gamma' = 0$ を満たすこと；これは $R$ に依存する2階線形常微分方程式；初期条件 $J(0) = 0, \frac{DJ}{dt}\big|_{t=0} = v$ で解が一意に決まる)
  - Proof (測地線の変分における2階変分を計算すると、変分場 $V(s,t)$ の最初の成分がこの方程式を満たすことが導出される）

- **2. 定曲率空間でのヤコビ場**
  - Example 4.4-1 (ユークリッド空間のヤコビ場：$\mathbb{R}^n$ では $R \equiv 0$ から，ヤコビ方程式は $\frac{d^2 J}{dt^2} = 0$；解は $J(t) = a + bt$（線形）；初期条件 $J(0) = 0$ から $a = 0$ なので $J(t) = bt$ で，接ベクトル方向に沿う平行変位)
  - Example 4.4-2 (球面のヤコビ場：$\mathbb{S}^n$ 上の測地線（大円）に沿ったヤコビ方程式は $\frac{d^2 J}{dt^2} + J = 0$ に簡約（定曲率 $K=1$）；解は $J(t) = A\cos t + B\sin t$；初期条件 $J(0) = 0$ から $A = 0$ となり $J(t) = B\sin t$）
  - Example 4.4-3 (双曲平面のヤコビ場：定曲率 $K = -1$ からヤコビ方程式は $\frac{d^2 J}{dt^2} - J = 0$；解は $J(t) = Ae^t + Be^{-t}$ で指数的増大；初期条件で $A = 0$ なら $J(t) = Be^{-t}$ となり減衰)

- **3. 共役点と単射半径**
  - Definition 4.4-2 (共役点：点 $p$ から出た測地線 $\gamma(t)$ に沿う非零ヤコビ場 $J(t)$ で $J(0) = 0$ かつ $J(t_0) = 0$ となる時刻 $t_0 > 0$ が存在するとき，$q = \gamma(t_0)$ を $p$ の共役点という；最小の正のそのような時刻を共役距離という)
  - Proposition 4.4-1 (共役点と指数写像の特異性：$q = \exp_p(tv_0)$ で $|v_0| = 1$ のとき，$q$ が $p$ の共役点 $\Leftrightarrow$ $d(\exp_p)_{tv_0}$ が特異（核が非零）；これは§3.3の単射半径と共役点距離の関連を明確化する)
  - Proof (指数写像の導関数計算；$d(\exp_p)$ の核はヤコビ場方程式の解と一対一対応)

- **4. 第二変分公式**
  - Theorem 4.4-1 (弧長汎関数の第二変分：測地線 $\gamma$ に沿う変分場 $V$ に対して，$\frac{d^2 L}{ds^2}\big|_{s=0} = \int_a^b [g(DV/dt, DV/dt) + g(R(V,\gamma')\gamma', V)] \, dt$；これが正（共役点がない場合）なら測地線は局所的に最短)
  - Proof (弧長汎関数の変分を2回計算し、端点固定条件 $V(a) = V(b) = 0$ と部分積分を用いて曲率テンソルを含む表現に整理する；測地線方程式 $D\gamma'/dt = 0$ を使って第一変分の消失と第二変分の導出を得る)
  - Corollary 4.4-1 (共役点がない場合の最短性：測地線 $\gamma : [a,b] \to M$ で $[a,b]$ 上に $p = \gamma(a)$ の共役点がなければ，$\gamma$ は端点を固定する近傍内で唯一の最短曲線)
  - Proof (共役点がなければ全てのヤコビ場が非退化であり、第二変分の指数形式が正定値であることが従う；従って測地線は弧長汎関数の厳密な極小点を与える)
  - Remark (この系は §3.4 のホップ-リノウの定理と密接に関連し、共役点の幾何的意味を明確にする)

## Chapter 5: 比較幾何学と大域的結果
**テーマ:** 曲率比較・体積比較・直径有限性・基本群

### §5.1 ラウフの比較定理
**内容:** ヤコビ場の比較、曲率制約下での解の挙動

- **1. ヤコビ場の基準方程式**
  - Definition 5.1-1 (断面曲率の下界と比較方程式：リーマン多様体 $(M, g)$ の全断面曲率が $K(\sigma) \geq K_0$（定数）を満たすとき，対応する比較多様体（定曲率 $K_0$ の空間）でのヤコビ場との比較を考える；ユークリッド（$K_0 = 0$）では $\frac{d^2 J}{dt^2} = 0$，正曲率（$K_0 > 0$）では $\frac{d^2 j}{dt^2} + K_0 j = 0$，負曲率（$K_0 < 0$）では $\frac{d^2 j}{dt^2} - |K_0| j = 0$ となり，本来の方程式 $\frac{d^2 J}{dt^2} + K J = 0$（で $K \geq K_0$）と比較)
  - Remark (これらの定曲率モデルでのヤコビ場の大きさ（ノルム）比較が，複雑な多様体での挙動を制御する本質)

- **2. ラウフの比較定理**
  - Theorem 5.1-1 (Rauch 比較定理：リーマン多様体 $M$ で測地線 $\gamma(t)$, $t \in [0, T]$ に沿う2つのヤコビ場 $J, j$ を，初期条件 $J(0) = j(0) = 0$, $|J'(0)| = |j'(0)|$ で考え，$M$ の断面曲率が $K \geq K_0$ を満たし，$j$ が定曲率 $K_0$ 空間でのヤコビ場ならば，$|J(t)| \geq |j(t)|$ が $t \in [0, T]$ で成立（共役点がない範囲内））
  - Proof (微分不等式の論証；$f(t) = |J(t)|$ と $g(t) = |j(t)|$ に対して $f'' \geq g''$ を示し，初期条件で等しいことから $f \geq g$ を導く)

- **3. 応用例**
  - Example 5.1-1 (正曲率多様体での共役点：$K(\sigma) \geq K_0 > 0$ ならば，測地線は長さ $\leq \frac{\pi}{\sqrt{K_0}}$ の時点で共役点を持つ；$\mathbb{S}^2$ では $\pi$ が赤道の長さで，共役点は対点）
  - Example 5.1-2 (負曲率多様体でのヤコビ場の増大：$K(\sigma) \leq -K_0 < 0$ ならば，ヤコビ場は指数的に増大；これは負曲率空間の"拡散的"性質を反映)

### §5.2 トポノゴフの比較定理
**内容:** 三角形の辺の長さ比較、グローバルな幾何

- **1. トポノゴフの定理の主張と証明**
  - Theorem 5.2-1 (Toponogov 三角形比較定理：完備リーマン多様体 $M$ の断面曲率が $K \geq K_0$ を満たすとする；$M$ 内の測地三角形 $\triangle pqr$ と定曲率 $K_0$ の比較空間 $M_{K_0}$ 内の対応する比較三角形 $\triangle p'q'r'$（辺の長さが等しい）について，$M$ の三角形の各頂点での角度は比較三角形の対応する角度以上；すなわち $\angle(p,q,r) \geq \angle(p',q',r')$)
  - Proof (ラウフの比較定理を用いて辺の中点での距離を評価し，測地三角形の角度が比較三角形より小さくならないことを帰納的に示す)
  - Remark (これはユークリッド幾何との対比で，正曲率空間では三角形の内角の和が $\pi$ より大きくなることを一般化)

- **2. 定曲率空間での検証**
  - Example 5.2-1 (ユークリッド平面 $\mathbb{R}^2$：$K_0 = 0$ で三角形の内角の和は $\pi$；比較三角形も同じ内角を持つ)
  - Example 5.2-2 (球面での三角形：$\mathbb{S}^2$ では正曲率 $K = 1$；測地三角形（大円でつないだ3点）の内角の和 $> \pi$；極三角形が例)
  - Example 5.2-3 (双曲平面での三角形：$\mathbb{H}^2$ では負曲率 $K = -1$；測地三角形の内角の和 $< \pi$；これはロバチェフスキー幾何の特徴)

- **3. 幾何的帰結**
  - Proposition 5.2-1 (トポノゴフ定理からの帰結：断面曲率 $K \geq K_0 > 0$ を持つ完備リーマン多様体は有限な直径 $\text{diam}(M) \leq \pi/\sqrt{K_0}$ を持つ；曲率 $K \leq 0$ ならば直径は無限大となりうる)
  - Proof (トポノゴフの比較定理を直径の長さの測地線に適用し，比較空間 $\mathbb{S}^n(1/\sqrt{K_0})$ での対径点間の距離 $\pi/\sqrt{K_0}$ を上界として得る)

### §5.3 ビショップ-グロモフの体積比較定理
**内容:** 体積要素の比較、体積増大の制約

- **1. 体積要素と体積形式**
  - Definition 5.3-1 (リーマン体積形式と測地球の体積：点 $p$ を中心とする測地球 $B(p, r) = \{q \in M : d(p,q) < r\}$ の体積を $V(p, r) = \int_{B(p,r)} dV$；定曲率空間 $M_{K_0}$ での対応する体積を $V_{K_0}(r)$ と記す；例えば $K_0 > 0$ のとき $V_{K_0}(r) = \omega_n \int_0^r \sin^{n-1}(\sqrt{K_0} t) / K_0^{(n-1)/2} \, dt$，$K_0 = 0$ のとき $V_0(r) = \omega_n r^n$，$K_0 < 0$ のとき $V_{K_0}(r) = \omega_n \int_0^r \sinh^{n-1}(\sqrt{|K_0|} t) / |K_0|^{(n-1)/2} \, dt$，ここで $\omega_n = \text{vol}(\mathbb{S}^{n-1})$)
  - Remark （これらは積分で陽に表現でき，曲率がゼロから離れるにつれて体積増大が制約される様子が見える）

- **2. ビショップ-グロモフ定理**
  - Theorem 5.3-1 (Bishop-Gromov 体積比較定理：リーマン多様体 $M$ の Ricci 曲率が $\text{Ric}(M) \geq (n-1)K_0 g$ を満たすとき（ここで $n = \dim M$），体積の比率 $\frac{V(p, r)}{V_{K_0}(r)}$ は $r$ に関して非増加；つまり曲率が正に向かうほど，測地球の体積増大が遅くなる）
  - Proof (ラウフの比較定理とヤコビ方程式の体積形式への適用；測地球の境界での平均曲率（extrinsic mean curvature）の比較から導く)

- **3. 応用：体積の有限性と増大率**
  - Corollary 5.3-1 (体積増大の制御：$\text{Ric} \geq (n-1)K_0$ で $K_0 > 0$ ならば全体積が有限；$K_0 = 0$ ならば $V(p, r) \leq \omega_n r^n$（ユークリッド体積以下）)
  - Proof (ビショップ-グロモフの体積比の単調性から，$V(p,r)/V_{K_0}(r) \leq \lim_{r \to 0} V(p,r)/V_{K_0}(r) = 1$ を適用する)
  - Example 5.3-1 (球面の体積：$\mathbb{S}^n$ では $\text{Ric} = (n-1)g$ で $K_0 = 1$；比較体積が $V_{K_0}(r)$ で与えられ，$r \to \pi$ で有限（全体積）に到達することの確認)
  - Example 5.3-2 (ユークリッド空間の体積：$\mathbb{R}^n$ では $\text{Ric} = 0$，$V(p,r) = \omega_n r^n$；多項式増大の限界的ケース)

### §5.4 マイヤースの定理とシンジの定理
**内容:** リッチ曲率の下界から基本群の構造へ

- **1. マイヤースの定理**
  - Theorem 5.4-1 (Myers の定理：完備リーマン多様体 $M$ の Ricci 曲率が $\text{Ric}(M) \geq (n-1)K_0 g$（$n = \dim M$, $K_0 > 0$）を満たすとき，$M$ はコンパクト，直径 $\text{diam}(M) \leq \frac{\pi}{\sqrt{K_0}}$ を持ち，基本群 $\pi_1(M)$ は有限)
  - Proof (ホップ-リノウ定理で完備性から测地球の列が有限被覆を持つことを示し，ビショップ-グロモフ定理の体積単調性を用いてコンパクト性を導く；基本群の有限性は基本領域の体積有限性から従う)

- **2. 定曲率と基本群**
  - Example 5.4-1 (球面：$\mathbb{S}^n$ は $n \geq 2$ で単連結で基本群 $\pi_1(\mathbb{S}^n) = \{e\}$；$\mathbb{S}^1$ は $\pi_1(\mathbb{S}^1) = \mathbb{Z}$）
  - Example 5.4-2 (射影空間：$\mathbb{RP}^n$ は球面の2-葉被覆で，$\pi_1(\mathbb{RP}^n) = \mathbb{Z}/2\mathbb{Z}$；正曲率を持つコンパクト多様体の例)
  - Example 5.4-3 (トーラス：$\mathbb{T}^n$ は$\text{Ric} = 0$ の平坦多様体で $\pi_1(\mathbb{T}^n) = \mathbb{Z}^n$；リッチ曲率0に対してマイヤースの定理は非コンパクト性を許す)

- **3. シンジの定理**
  - Theorem 5.4-2 (Synge の定理：偶数次元コンパクト向き付け可能リーマン多様体 $M$ が正の断面曲率を持つならば，$M$ は単連結である；奇数次元の場合は，正の断面曲率を持つコンパクト多様体は向き付け可能である)
  - Proof (非自明な閉測地線が存在すると仮定し，その最短代表元に沿ってヤコビ場を構成する；偶数次元かつ向き付け可能という条件から，法ベクトルの平行移動が元に戻ることを利用して，長さを短くする変分を構成し矛盾を導く)
  - Example 5.4-4 ($\mathbb{CP}^n$（$n \geq 1$）：偶数実次元 $2n$ の向き付け可能多様体でフビニ-スタディ計量により正の断面曲率を持ち，単連結；シンジの定理の結論と整合)
  - Remark (シンジの定理はマイヤースの定理と相補的であり，マイヤースはリッチ曲率から基本群の有限性を，シンジは断面曲率と次元から単連結性を導く)

## Chapter 6: 部分多様体とガウス-ボンネの定理
**テーマ:** 等長はめ込み・第二基本形式・ガウス方程式・ガウス-ボンネの定理

### §6.1 リーマン部分多様体と誘導計量
**内容:** 等長はめ込みの定義、誘導計量と法束の構成、接空間の直交分解

- **1. 等長はめ込みと部分多様体**
  - Definition 6.1-1 (等長はめ込み：滑らかなはめ込み $f : (M, g) \to (\bar{M}, \bar{g})$ が等長はめ込みであるとは、各点 $p \in M$ で $g_p = (f^*\bar{g})_p$ を満たすこと；すなわち接写像 $df_p$ が内積を保存し、$g_p(v, w) = \bar{g}_{f(p)}(df_p(v), df_p(w))$ が全ての $v, w \in T_p M$ で成立すること)
  - Example 6.1-1 (球面 $S^n \hookrightarrow \mathbb{R}^{n+1}$：包含写像による引き戻し計量が $S^n$ の丸い計量に一致すること；$f : S^2 \to \mathbb{R}^3$, $f(x,y,z) = (x,y,z)$ のとき、誘導計量は $ds^2 = dx^2 + dy^2$（接空間で）であり、外部ユークリッド計量の制限として確認される)
  - Definition 6.1-2 (余次元：等長はめ込み $f : M^m \to \bar{M}^n$ の余次元を $k = n - m$ と定義；$k = 1$ の場合を超曲面と呼ぶ)
  - Remark (アファイン部分多様体（線形部分空間）や曲面は余次元を持つ重要な例；余次元が大きいほど法方向の構造が複雑になる)

- **2. 法束と直交分解**
  - Definition 6.1-3 (法束：等長はめ込み $f : M \to \bar{M}$ に対し、各点 $p \in M$ で法空間 $N_p M := \{v \in T_{f(p)}\bar{M} : \bar{g}(v, w) = 0 \text{ for all } w \in df_p(T_p M)\}$ を定義；法束 $NM = \bigcup_{p \in M} N_p M$ は $M$ 上のベクトル束をなす；次元は余次元 $k$ に等しい)
  - Definition 6.1-4 (直交分解：外部多様体の接空間は誘導計量に関して接空間と法空間に直交分解される；$T_{f(p)}\bar{M} = df_p(T_p M) \oplus N_p M$；全ての外部ベクトル場は接成分と法成分に一意に分解できる)
  - Example 6.1-2 ($S^n \subset \mathbb{R}^{n+1}$ の法束：点 $p \in S^n$ での法空間は位置ベクトル $p$ で張られる1次元空間；外向き単位法ベクトル $\nu_p = p$ とすると、全法束は直線束を形成；逆に双曲面では法空間の計量符号が異なる)

- **3. 誘導計量の性質**
  - Proposition 6.1-1 (誘導計量の非退化性：等長はめ込み $f : (M, g) \to (\bar{M}, \bar{g})$ の引き戻し計量 $f^*\bar{g}$ は $M$ 上の非退化対称2-形式であり、リーマン計量をなす；すなわち $(M, f^*\bar{g})$ はリーマン多様体の構造を持つ；この計量に関して $f$ は局所等長となる)
  - Proof (非退化性は $df_p$ がはめ込みで単射であること、および $\bar{g}$ が非退化であることから従う；内積の正定値性も $\bar{g}$ の正定値性と $df_p$ の単射性から保証される)
  - Example 6.1-3 (円筒 $S^1 \times \mathbb{R}$：$\mathbb{R}^3$ への標準的なはめ込み $f(\theta, t) = (\cos\theta, \sin\theta, t)$ のもと、誘導計量は $d\theta^2 + dt^2$ であること；クリストッフェル記号を計算すると、接続はユークリッド的で曲率はゼロであることが確認される)

### §6.2 第二基本形式とガウス・コダッチ方程式
**内容:** 第二基本形式の定義、ワインガルテン写像、ガウス方程式、コダッチ・マイナルディ方程式

- **1. 第二基本形式の定義**
  - Definition 6.2-1 (第二基本形式：等長はめ込み $f : M \to \bar{M}$ に対し、外部多様体の共変微分 $\bar{\nabla}$ を用いて、外部接空間内で $X \in T_p M$ 方向の $Y \in T_p M$ の微分を分解するとき、法成分を第二基本形式という；$\bar{\nabla}_{df(X)} df(Y) = df(\nabla_X Y) + h(X, Y)$；ここで $h(X, Y) \in N_p M$ は接線性と対称性を持つ2-形式で、$h(X, Y) = h(Y, X)$ が成立）
  - Remark (第二基本形式は誘導計量によってのみ決まらず、外部空間への埋め込み方に依存する；この依存性が外的曲率を表現する)
  - Example 6.2-1 (球面 $S^n(\rho) \subset \mathbb{R}^{n+1}$：半径 $\rho$ の球面では外向き単位法ベクトル $\nu_p = p/\rho$ に対し、第二基本形式は $h(X, Y) = \frac{1}{\rho}\bar{g}(X, Y)\nu_p$ で与えられること；接ベクトル $(1, 0)$ と $(1, 0)$ に対して $h = \frac{1}{\rho}$ となることが直接計算で確認される)

- **2. ワインガルテン写像（形演算子）**
  - Definition 6.2-2 (ワインガルテン写像：超曲面の場合（$k=1$）、各点で唯一の単位法ベクトル $\nu$ を選ぶと、第二基本形式 $h$ を用いて形演算子（Weingarten map）$S : T_p M \to T_p M$ を $\bar{g}(S(X), Y) = h(X, Y)$ で定義；局所座標では成分 $S^i_j = g^{ik} h_{kj}$）
  - Definition 6.2-3 (主曲率と主方向：形演算子 $S$ の固有値を主曲率 $\kappa_1, \ldots, \kappa_m$ といい、対応する固有ベクトルを主方向という；形演算子は自己随伴で対角化可能）
  - Example 6.2-2 (円筒 $S^1 \times \mathbb{R} \subset \mathbb{R}^3$ での主曲率：点 $(\theta, 0)$ での形演算子を計算すると、周方向 $\partial_\theta$ での主曲率は $\kappa_1 = 1$、軸方向 $\partial_z$ での主曲率は $\kappa_2 = 0$；これは円筒が一方向には曲がり、他方向では直線であることを反映する)

- **3. ガウス方程式とコダッチ・マイナルディ方程式**
  - Theorem 6.2-1 (ガウス方程式：等長はめ込み $f : M^m \to \bar{M}^n$ に対し、リーマン曲率テンソルは $R^M(X, Y, Z, W) = R^{\bar{M}}(df(X), df(Y), df(Z), df(W)) - \bar{g}(h(X, Z), h(Y, W)) + \bar{g}(h(X, W), h(Y, Z))$ を満たす；すなわち外部曲率は第二基本形式を通じて内部曲率に現れる)
  - Proof (外部共変微分の分解式 $\bar{\nabla}_{df(X)} df(Y) = df(\nabla_X Y) + h(X, Y)$ を反復適用して、曲率テンソルの定義式 $[\bar{\nabla}_{df(X)}, \bar{\nabla}_{df(Y)}]$ を計算すれば、右辺に第二基本形式による補正項が現れることが確認される)
  - Theorem 6.2-2 (コダッチ・マイナルディ方程式：第二基本形式 $h$ と外部多様体の共変微分は $(\bar{\nabla}_{df(X)} h)(Y, Z) = (\bar{\nabla}_{df(Y)} h)(X, Z)$ を満たす；座標で書くと $\frac{\partial h_{ij}}{\partial x^k} - \Gamma^l_{ki} h_{lj} - \Gamma^l_{kj} h_{il} = \frac{\partial h_{ij}}{\partial x^l}$ といったテンソル条件）
  - Proof (第二基本形式を法ベクトル場の値として見直し、外部接続のねじれゼロ条件と compatible性から直接導出される)

### §6.3 超曲面のガウス曲率とガウスの驚くべき定理
**内容:** 主曲率によるガウス曲率と平均曲率の定義、ガウスの驚くべき定理、極小曲面

- **1. ガウス曲率と平均曲率**
  - Definition 6.3-1 (ガウス曲率：超曲面 $M^n \subset \bar{M}^{n+1}$ に対し、形演算子 $S$ の行列式をガウス曲率 $K = \det(S) = \kappa_1 \cdots \kappa_n$ と定義；ここで $\kappa_1, \ldots, \kappa_n$ は主曲率)
  - Definition 6.3-2 (平均曲率：形演算子のトレースの $1/n$ 倍を平均曲率 $H = \frac{1}{n}\text{tr}(S) = \frac{1}{n}(\kappa_1 + \cdots + \kappa_n)$ と定義；$H \equiv 0$ の超曲面を極小曲面という)
  - Example 6.3-1 (球面 $S^n(\rho) \subset \mathbb{R}^{n+1}$：外向き単位法ベクトルに対し全ての主曲率が $\kappa_i = 1/\rho$；ガウス曲率は $K = 1/\rho^n$、平均曲率は $H = 1/\rho$；2次元球面 $S^2(\rho)$ では $K = 1/\rho^2$)
  - Example 6.3-2 (回転面の主曲率：$\mathbb{R}^3$ 内で曲線 $(f(u), 0, g(u))$ を $z$ 軸周りに回転させた曲面は、子午線方向の主曲率が $\kappa_1 = (f'g'' - f''g')/(f'^2 + g'^2)^{3/2}$、緯線方向の主曲率が $\kappa_2 = g'/(f\sqrt{f'^2 + g'^2})$；円筒では $\kappa_1 = 0$, $\kappa_2 = 1/r$ でガウス曲率 $K = 0$ となる)

- **2. ガウスの驚くべき定理（Theorema Egregium）**
  - Theorem 6.3-1 (Theorema Egregium：$\mathbb{R}^3$ 内の曲面のガウス曲率 $K$ は第一基本形式（誘導計量）のみから決定される内在的不変量である；すなわち、$K$ は外部空間への埋め込み方には依存せず、曲面上の計量のみで計算できる)
  - Proof (ガウス方程式において外部空間を $\mathbb{R}^3$（曲率ゼロ）とすると、$K = R_{1212}/\det(g_{ij})$ が導かれ、右辺はクリストッフェル記号とその導関数のみで表現される；主曲率を介さずにガウス曲率が計量から決定されることが確認される)
  - Example 6.3-3 (平面と円筒：$\mathbb{R}^2$ はガウス曲率 $K = 0$；円筒 $S^1 \times \mathbb{R}$ も $K = 0$（主曲率 $1/r$ と $0$ の積）；両者は局所等長であり、展開可能曲面の例；一方、球面 $S^2$ は $K = 1/R^2 > 0$ で局所的にも平面と等長にならないことを Theorema Egregium が保証)
  - Remark (Theorema Egregium は「曲がったシートを伸縮せずに変形してもガウス曲率は不変」という日常的直感の数学的定式化；ピザを半分に折ると主曲率の一方がゼロでなくなるがガウス曲率は保たれる)

- **3. 極小曲面と曲率計算の実例**
  - Example 6.3-4 (カテノイド：$\mathbb{R}^3$ 内で $r = c\cosh(z/c)$ で定まる回転面は極小曲面で、主曲率が $\kappa_1 = -\kappa_2$ となり $H = 0$；カテノイドは石鹸膜の形状としても実現される)
  - Example 6.3-5 (トーラス $T^2 \subset \mathbb{R}^3$：$(R + r\cos u)\cos v, (R + r\cos u)\sin v, r\sin u)$ のパラメータ化に対し、主曲率は $\kappa_1 = 1/r$, $\kappa_2 = \cos u/(R + r\cos u)$；外側（$u = 0$）では $K > 0$、内側（$u = \pi$）では $K < 0$、$u = \pi/2$ では $K = 0$；§6.4 でガウス-ボンネの定理によりトーラスの全ガウス曲率の積分が $0$ になることを確認する)

### §6.4 ガウス-ボンネの定理
**内容:** 2次元曲面のガウス-ボンネ定理、オイラー標数、グローバルジオメトリーとトポロジーの結合

- **1. ガウス-ボンネ定理の主張**
  - Theorem 6.4-1 (ガウス-ボンネの定理：コンパクト向き付け可能な2次元リーマン多様体 $(M, g)$（境界なし）に対し、$\int_M K \, dA = 2\pi \chi(M)$ が成立；ここで $K$ はガウス曲率、$dA$ は面積要素、$\chi(M)$ はオイラー標数)
  - Proof (曲面を測地三角形に三角形分割し、各三角形でのガウス-ボンネ公式（角の和と面積の関係）を適用する；三角形分割の辺で隣接三角形の境界項がキャンセルされ、オイラー標数 $\chi = V - E + F$（頂点数、辺数、面数）が残ることを示す)
  - Remark (オイラー標数は位相的不変量で、種数 $g$ の向き付け可能曲面では $\chi = 2 - 2g$；定理は曲率の積分というグローバルな幾何的量がトポロジーで決まることを示す)

- **2. 証明の概要と測地線**
  - Definition 6.4-1 (測地線曲率：曲線 $\gamma(t)$ が曲面上にあるとき、その曲率ベクトルの法成分の大きさを測地線曲率 $\kappa_g$ という；$\gamma$ が測地線（平行移動ベクトルの法成分がゼロ）のとき $\kappa_g = 0$；境界近くでは測地線曲率が拘束条件として現れる)
  - Theorem 6.4-2 (ガウス-ボンネ（境界あり）：コンパクト領域 $D \subset M$ で境界が区分的滑らかな場合、$\int_D K \, dA + \int_{\partial D} \kappa_g \, ds + \sum_i \theta_i = 2\pi\chi(D)$；ここで $\theta_i$ は境界の頂点での外角、$\kappa_g$ は境界曲線の測地線曲率；これは定理の一般化形）
  - Proof (外微分と発散定理を用いて、Gauss-Bonnet形式 $d\omega = K \, dA$（ここで $\omega$ は接続1-形式）を領域で積分し、境界項を計算すれば得られる；詳細は微分幾何の標準テキスト参照）

- **3. ガウス-ボンネ定理の応用**
  - Example 6.4-1 (球面：$S^2$ はガウス曲率 $K = 1/R^2$ の定曲率球で、$\int_{S^2} K \, dA = (1/R^2) \cdot 4\pi R^2 = 4\pi = 2\pi \cdot 2 = 2\pi\chi(S^2)$ となり、定理を確認；オイラー標数は $\chi(S^2) = 2$）
  - Example 6.4-2 (トーラス：種数1のトーラス $T^2$ ではオイラー標数 $\chi(T^2) = 0$；従って全ガウス曲率の積分は0；トーラスは外側で正曲率、内側で負曲率を持つため、その和がキャンセルされることを反映)
  - Example 6.4-3 (種数 $g$ の曲面：種数 $g$ の向き付け可能曲面では $\chi = 2 - 2g$；ガウス-ボンネにより $\int_M K \, dA = 2\pi(2 - 2g)$；特に $g \geq 1$ では必ず負曲率点が存在することを示唆する；$g = 0$（球面）のみ全体が非負曲率可能）

- **4. 高次元への拡張の展望**
  - Remark (第8章で高次元のガウス-ボンネ-チャーン定理を扱うことになり、それは特性類（チャーン類など）を用いて、より高い次元の多様体に対する積分公式に拡張される)
  - Remark (ガウス-ボンネは指数定理（第9章）の特別な場合として再解釈でき、より広い文脈でトポロジーと解析の深い結合を示す)



## Chapter 7: ファイバー束と接続
**テーマ:** ベクトル束・主束・構造群・接続形式・曲率形式・ホロノミー

### §7.1 ベクトル束の定義と例
**内容:** ベクトル束の公理的定義、遷移関数、標準的な例（接束、余接束、テンソル束）、セクション

- **1. ベクトル束の定義**
  - Definition 7.1-1 (ベクトル束：$n$次元滑らか多様体 $M$ 上の階数 $k$ のベクトル束 $E$ とは、次を満たす関数 $\pi : E \to M$ の対 $(E, \pi)$：(i) 全空間 $E$ は滑らか多様体、(ii) 各 $p \in M$ に対してファイバー $\pi^{-1}(p) = E_p$ は $k$-次元ベクトル空間、(iii) 局所自明性：各点 $p$ の周りに開近傍 $U$ が存在し、微分同相 $\phi_U : \pi^{-1}(U) \to U \times \mathbb{R}^k$ で $\pi_1 \circ \phi_U = \pi$ を満たす、(iv) 遷移関数の滑らか性）
  - Remark (局所自明性はベクトル束を局所的には自明な積に見えるが、グローバルには非自明な幾何構造を持たせる)

- **2. 遷移関数と bundle structure**
  - Definition 7.1-2 (遷移関数：開被覆 $\{U_\alpha\}$ に対し、自明化 $\phi_\alpha : \pi^{-1}(U_\alpha) \to U_\alpha \times \mathbb{R}^k$ が与えられるとき、重複領域 $U_\alpha \cap U_\beta$ 上の遷移関数を $g_{\alpha\beta} : U_\alpha \cap U_\beta \to GL(k, \mathbb{R})$ で $\phi_\beta \circ \phi_\alpha^{-1}(p, v) = (p, g_{\alpha\beta}(p) v)$ と定義；遷移関数は cocycle条件 $g_{\alpha\beta} g_{\beta\gamma} = g_{\alpha\gamma}$ を満たす）
  - Example 7.1-1 (自明束：$E = M \times \mathbb{R}^k$ は任意の開被覆に対して $g_{\alpha\beta} = \text{id}$ となる単純なベクトル束；多くの多様体では全体で自明化でき、例えば平行化可能多様体）
  - Example 7.1-2 (非自明束：$\mathbb{RP}^1$ 上の階数1 Möbius束は全体で自明化できず、任意の点の近傍から全体への拡張が不可能；遷移関数が $g = -1$ となる）

- **3. 接束と基本的なベクトル束**
  - Definition 7.1-3 (接束：$n$次元滑らか多様体 $M$ の接束 $TM = \bigcup_{p \in M} T_p M$ は階数 $n$ のベクト束で、局所座標 $(x^i)$ による自明化 $\phi_U : TU \to U \times \mathbb{R}^n$ を $(p, \sum_i v^i \frac{\partial}{\partial x^i}|_p) \mapsto (p, (v^1, \ldots, v^n))$ で与える；遷移関数は $g_{\alpha\beta} = J(\text{座標変換のヤコビアン})$）
  - Definition 7.1-4 (余接束：$T^*M = \bigcup_{p \in M} T_p^* M$ も同様に階数 $n$ のベクトル束で、双対ペアリング下の遷移関数は逆転置 $g_{\alpha\beta} = (J^{-1})^T$）
  - Definition 7.1-5 (テンソル束：$T^{(r,s)}M = TM^{\otimes r} \otimes (T^*M)^{\otimes s}$ は階数 $n^{r+s}$ のベクトル束で、テンソル場は本束のセクション)

- **4. セクションとフレーム**
  - Definition 7.1-6 (セクション：ベクトル束 $E \to M$ のセクション $\sigma$ は滑らか写像 $\sigma : M \to E$ で $\pi \circ \sigma = \text{id}_M$ を満たすもの；セクションの空間 $\Gamma(E)$ は $\mathbb{R}$-加群（あるいは $C^\infty(M)$-加群）をなす；接束のセクションはベクトル場 $\Gamma(TM)$、余接束のセクションは1-形式 $\Gamma(T^*M)$）
  - Definition 7.1-7 (局所フレーム：ベクトル束 $E \to M$ の局所フレームは、開集合 $U \subset M$ 上で定義された linearly independent な $k$ 個のセクション $(e_1, \ldots, e_k)$ で、各点 $p \in U$ でファイバー $E_p$ の基を成すもの；局所フレームを用いてセクション $\sigma$ を $\sigma = \sigma^i e_i$ と展開できる)
  - Example 7.1-3 (座標フレーム：接束 $TM$ では局所座標 $(x^i)$ から誘導される座標ベクトル場 $(\partial_1, \ldots, \partial_n)$ が標準的な局所フレーム；全てのセクションはこのフレームの係数関数で表現される)

### §7.2 主ファイバー束
**内容:** 主束の定義、構造群、フレーム束、随伴表現、関連ベクトル束

- **1. 主ファイバー束の定義と構造群**
  - Definition 7.2-1 (主ファイバー束：リー群 $G$ に関する主ファイバー束 $P(M, G)$ とは、以下を満たすベクトル束の拡張：(i) 全空間 $P$ は滑らか多様体で射影 $\pi : P \to M$、(ii) ファイバー $P_p = \pi^{-1}(p)$ は $G$ と微分同相、(iii) 右作用 $R_g : P \to P$ がファイバー保存で滑らか、(iv) 局所自明性 $\phi_\alpha : \pi^{-1}(U_\alpha) \to U_\alpha \times G$ は右作用と compatible；すなわち $\phi_\alpha(p \cdot g) = (\pi(p), \phi_\alpha^{(2)}(p) \cdot g)$ ここで $\phi_\alpha^{(2)} : \pi^{-1}(U_\alpha) \to G$）
  - Remark (主束は構造群 $G$ を本質的な対称性として持つ；遷移関数 $g_{\alpha\beta} : U_\alpha \cap U_\beta \to G$ で記述される)

- **2. フレーム束と正規化フレーム**
  - Definition 7.2-2 (線形フレーム束：$n$次元多様体 $M$ の線形フレーム束 $FM$ は、各点 $p \in M$ でのフレーム（基底） $\{e_1, \ldots, e_n\}$ の全体の集合；FM は $GL(n, \mathbb{R})$ に関する主束で、ファイバー $(FM)_p$ は $p$ での全順序基の空間で $GL(n, \mathbb{R})$ に同型；構造群は自然に $GL(n, \mathbb{R})$）
  - Definition 7.2-3 (正規直交フレーム束：リーマン多様体 $(M, g)$ の場合、フレーム束を正規直交フレーム（計量に関して）に制限したものを $O(M)$ で表す；これは $O(n)$ に関する主束で、従来の線形フレーム束の subbundle；構造群は $O(n)$）
  - Example 7.2-1 (球面のフレーム束：$S^2$ のフレーム束 $FS^2$ は6次元の主束で、各点での可動フレームは位置とその上での orthonormal pair で記述；$O(2)$ 主束として、大域セクション（グローバルフレーム）が存在しないことは4色定理に関連する)

- **3. 随伴表現と関連ベクトル束**
  - Definition 7.2-4 (随伴表現：リー群 $G$ と主束 $P(M, G)$ に対し、随伴表現 $\text{Ad} : G \to \text{GL}(\mathfrak{g})$（$\mathfrak{g}$ は $G$ のリー代数）を用いて、関連ベクトル束 $P \times_G \mathfrak{g}$ を定義；これは $M$ 上の階数 $\dim G$ のベクトル束）
  - Definition 7.2-5 (ベクトル表現：より一般に、$G$ の表現 $\rho : G \to GL(V)$（$V$ はベクトル空間）に対し、関連ベクトル束 $P \times_\rho V$ を定義；遷移関数は $g_{\alpha\beta} = \rho(\tilde{g}_{\alpha\beta})$ で、$\tilde{g}_{\alpha\beta}$ は主束の遷移関数)
  - Example 7.2-2 (接束と関連ベクトル束：リーマン多様体の正規直交フレーム束 $O(M)$ から、基本表現 $\rho : O(n) \to GL(n, \mathbb{R})$ を用いると、関連束 $O(M) \times_\rho \mathbb{R}^n$ は接束 $TM$ に同型；これは主束の観点から接束を再解釈する)

### §7.3 主束上の接続
**内容:** 接続の定義、水平分布、接続1-形式、曲率2-形式、構造方程式、レビ-チビタ接続との関係

- **1. 接続としての水平分布**
  - Definition 7.3-1 (水平分布：主束 $P(M, G)$ 上の接続は、各 $u \in P$ での接空間 $T_u P$ を垂直部分（ファイバー接空間） $V_u P$ と水平部分（水平分布） $H_u P$ に分解すること；$T_u P = V_u P \oplus H_u P$；水平分布は G-同変性 $(R_g)_* H_u P = H_{u \cdot g}P$ を満たす）
  - Definition 7.3-2 (垂直ベクトル：垂直分布 $VP$ は基本ベクトル場（基本的な無限小生成元） $X^*_u = \frac{d}{dt}|_{t=0}(u \exp(tX))$（$X \in \mathfrak{g}$）で張られる；$\text{dim}(VP) = \dim G$）
  - Remark (水平分布の選択は非自明で、多くの異なる接続が存在；最小コストは測地線の概念を導入する)

- **2. 接続1-形式と曲率2-形式**
  - Definition 7.3-3 (接続1-形式：主束 $P$ 上の接続1-形式 $\omega$ は $\Omega^1(P, \mathfrak{g})$ に属する形式で、$\omega(X^*) = X$（基本ベクトルの逆像）および $R_g^*\omega = \text{Ad}(g^{-1})\omega$（同変性）を満たす；局所自明化 $U$ での成分 $\omega = \omega_i dx^i + \omega_\alpha dg_\alpha$）
  - Definition 7.3-4 (曲率2-形式：接続1-形式 $\omega$ の曲率を $\Omega = d\omega + \frac{1}{2}[\omega, \omega]$ で定義；これは $\Omega^2(P, \mathfrak{g})$ に属し、基本ベクトルに対しゼロで $R_g^*\Omega = \text{Ad}(g^{-1})\Omega$（同変）；局所的には $\Omega = d\omega + \omega \wedge \omega$）
  - Example 7.3-1 (自明接続：$P = M \times G$ の自明束上で接続1-形式を $\omega = g^{-1}dg$ と取ると、曲率は $\Omega = 0$ となり、平坦接続；これはユークリッド空間の標準接続に対応)

- **3. 構造方程式と曲率テンソル**
  - Theorem 7.3-1 (Cartan構造方程式：接続1-形式 $\omega$ と曲率2-形式 $\Omega$ に対し、$\Omega = d\omega + \frac{1}{2}[\omega, \omega]$ の関係が成立；これをCartan構造方程式と呼ぶ；高次元では $\Omega \wedge \Omega = 0$ などの恒等式も従う)
  - Proof (外微分の定義と Lie括弧の関係式 $[\omega_A, \omega_B] = [A, B]\omega \wedge \omega$ から直接確認)
  - Definition 7.3-5 (関連ベクトル束上の接続：主束の接続から、関連ベクトル束 $E = P \times_\rho V$ 上に自然な接続が誘導される；セクション $\sigma : M \to E$ の共変微分は $\nabla_X \sigma = \rho_*(\omega(X^\text{lift}))\sigma + X(\sigma)$ で与えられ、$X$ の lift は水平分布で定義)

- **4. レビ-チビタ接続との関係**
  - Theorem 7.3-2 (正規直交フレーム束の接続：リーマン多様体 $(M, g)$ の正規直交フレーム束 $O(M)$ 上の接続1-形式 $\omega$ が計量compatible（つまり $\nabla g = 0$）およびねじれゼロ条件を満たすとき、この接続は唯一に決まり、レビ-チビタ接続に対応する；フレーム束の言語ではクリストッフェル記号が1-形式 $\omega$ の成分として表現される)
  - Remark (この視点からレビ-チビタ接続は主束の言語で再定義され、より抽象的で functorial な理解が可能になる；6章と2章の接続論を一般化)

### §7.4 ホロノミー群
**内容:** 平行移動、ホロノミー、ホロノミー群、Ambrose-Singer定理、制限ホロノミー

- **1. 平行移動とホロノミー**
  - Definition 7.4-1 (平行移動：主束 $P(M, G)$ 上の接続が与えられたとき、曲線 $\gamma : [0,1] \to M$ に沿った平行移動は、$\gamma$ の lift $\tilde{\gamma} : [0,1] \to P$ で、$\tilde{\gamma}'(t)$ が水平であるもの；つまり $\omega(\tilde{\gamma}'(t)) = 0$ を満たす；初期点 $p$ での平行移動は終点での元素 $\text{Hol}_\gamma(p) \in G$ を定義)
  - Definition 7.4-2 (ホロノミー群：点 $p_0 \in M$ を固定し、$p_0$ を始終点とするすべての閉曲線に沿った平行移動の合成で得られる $G$ の元素全体が成す部分群を $\text{Hol}(P, \omega, p_0) \subset G$ で表す；これが点 $p_0$ でのホロノミー群）
  - Remark (ホロノミー群は接続に依存し、異なる接続は異なるホロノミーを与える；平坦接続のホロノミーは基本群に関連する)

- **2. ホロノミーの例と計算**
  - Example 7.4-1 (ユークリッド空間のホロノミー：$\mathbb{R}^n$ の標準接続は平坦で、任意の閉曲線に沿った平行移動は恒等変換；ホロノミー群は $\{e\}$（自明）；これは空間に曲率がないことを反映)
  - Example 7.4-2 (球面のホロノミー：$S^2$ のレビ-チビタ接続では、赤道周りの一周に沿った平行移動が非自明で、例えば北極点での方向が角度だけ回転（北極での赤道の「面積」に比例）；ホロノミー群は $SO(2)$（回転群）に同型）
  - Example 7.4-3 (一般曲面のホロノミー：$M$ が2次元リーマン多様体のとき、$S^1$ 上のホロノミー群は $SO(2)$ の部分群；曲率が0なら自明、定正曲率なら全体 $SO(2)$）

- **3. Ambrose-Singer定理**
  - Theorem 7.4-1 (Ambrose-Singer：リー代数レベルで、ホロノミー群の Lie代数はすべての曲率2-形式 $\Omega(X, Y)$ で生成される；つまり $\text{Lie}(\text{Hol}) = \text{span}\{\Omega_p(X, Y) : p \in M, X, Y \in T_p M\}$；この定理は曲率とホロノミーの本質的関係を示す)
  - Proof (曲率の定義から無限小平行移動を計算し、Hol群のLie代数がこれらの元素で張られることを確認；詳細は9-5参照)
  - Example 7.4-4 (一般ケーラー多様体のホロノミー：複素構造を持つリーマン多様体のホロノミーは $U(n)$ に含まれ、特別な場合（Calabi-Yau）では $SU(n)$ になることが知られており、これは現代幾何学・物理学で重要)

- **4. 制限ホロノミー**
  - Definition 7.4-3 (制限ホロノミー群：点 $p_0$ を始終点とする縮約可能な閉曲線（基本群の単位元に属する）に沿った平行移動で得られる群を制限ホロノミー群 $\text{Hol}_0(P, \omega, p_0)$ という；これはホロノミー群の連結成分）
  - Proposition 7.4-1 (de Rham分解定理：コンパクト単純連結リーマン多様体が制限ホロノミー群として $G_1 \times \cdots \times G_k$ を持つなら、多様体そのものが幾何学的に対応する成分に分解される；例えば $G = SO(2) \times SO(3)$ なら局所積構造；詳細は後の章で)



## Chapter 8: 特性類
**テーマ:** チャーン類・ポントリャーギン類・オイラー類・チャーン-ウェイユ理論

### §8.1 特性類の概念と動機
**内容:** ベクトル束の位相的不変量、非自明束の存在、特性類の役割

- **1. ベクトル束の分類問題**
  - Definition 8.1-1 (ベクトル束の同型：$M$ 上の2つのベクトル束 $E$, $F$ が同型であるとは、微分同相写像 $\phi : E \to F$ で $\pi_F \circ \phi = \pi_E$ かつ各ファイバーで線形同型を誘導するものが存在すること；同型束は幾何学的に同じ構造）
  - Definition 8.1-2 (束の分類空間：階数 $k$ の実ベクトル束の同型類全体は、Grassmann多様体 $\text{Gr}_k(\mathbb{R}^\infty)$ で分類される；すなわち $M$ 上の束 $E$ は分類写像 $\phi : M \to \text{Gr}_k$ で本質的に一意に対応；ここで $\text{Gr}_k$ は無限次元Grassmann多様体)
  - Remark (分類空間はトポロジー的に複雑で、異なる $M$ 上の束を識別するには不変量が必要)

- **2. 特性類の定義と存在**
  - Definition 8.1-3 (特性類：ベクトル束 $E$ の特性類とは、$E$ の同型類を分類するコホモロジー不変量の総称；より正確には、分類写像 $\phi : M \to \text{Gr}_k$ から引き戻された $\text{Gr}_k$ 上の universal cohomology classes；異なる自明化では同じ特性類が得られることで bundle-independent)
  - Remark (特性類は bundle-independent で、計量や接続の選択に依存しない；これが特性類の強力さ）

- **3. 非自明束の存在と Möbius束**
  - Example 8.1-1 (Möbius束：$\mathbb{RP}^1$ 上の階数1 Möbius束 $M$ は全体で自明化できない；最初の Chern-Whitney class $c_1(M) \neq 0$ で検出され、この class が0でないことが非自明性を証明；任意の開被覆に対して、遷移関数が $\pm 1$ で、接合図式が本質的に非自明)
  - Example 8.1-2 (接束の非自明性：偶数次元球面 $S^{2n}$ の接束 $TS^{2n}$ は全体で自明化できないこと（毛玉定理 hairy ball theorem に関連）；第一 Chern class が非ゼロであり、球上の non-vanishing tangent vector field が存在しないことを示す)

### §8.2 チャーン-ウェイユ理論
**内容:** 不変多項式、Chern-Weil同準同型、接続からコホモロジー類へ

- **1. 不変多項式と特性多項式**
  - Definition 8.2-1 (不変多項式：Lie環 $\mathfrak{g}$ 上の多項式 $f$ が不変多項式であるとは、随伴表現 $\text{Ad}_g$ に対して $f(\text{Ad}_g(X)) = f(X)$ がすべての $g \in G$, $X \in \mathfrak{g}$ で成立すること；例えば $f(A) = \text{tr}(A^k)$ は不変多項式)
  - Definition 8.2-2 (特性多項式：$k \times k$ 行列 $A$ に対し、特性多項式 $\det(\lambda I - A)$ の係数 $e_j(A)$（基本対称多項式）はMatrixアルジェブラの基本的な不変多項式；Chern多項式 $\det(I + \frac{i}{2\pi}A) = \sum_j c_j(A)$ も不変多項式)
  - Example 8.2-1 (トレースと行列式：$\text{tr}(A)$ と $\det(A)$ は2つの最も基本的な不変多項式；より高次では $\text{tr}(A^k)$ が Newton多項式をなす)

- **2. Chern-Weil同準同型**
  - Theorem 8.2-1 (Chern-Weil同準同型：$M$ 上のベクトル束 $E$ に接続 $\omega$ を固定したとき、不変多項式 $f \in I(\mathfrak{gl}(k))$ に対し、$f(\Omega)$（曲率形式を代入）は $M$ の上の閉微分形式 $\Omega^*(M)$ を与え、その de Rham cohomology class は接続の選択に依存しない（独立性定理）；すなわち $[f(\Omega)] \in H^*(M, \mathbb{R})$ はコホモロジー類として well-defined)
  - Proof (2つの接続 $\omega_0$, $\omega_1$ のhomotopy $\omega_t = \omega_0 + t(\omega_1 - \omega_0)$ を考え、対応する曲率2-形式の族 $\Omega_t$ を計算する；$f(\Omega_t)$ の外微分を調べると exact form となることが確認される；従って $[f(\Omega_0)] = [f(\Omega_1)]$）
  - Definition 8.2-3 (特性コホモロジー類：$f \in I(\mathfrak{gl}(k))$ に対し、$c_f(E) := [f(\Omega)] \in H^*(M, \mathbb{R})$ をベクトル束 $E$ の特性コホモロジー類と呼ぶ；特に $f = \det(I + x)$ のとき生じる類がChern class)

- **3. Pontryagin類とEuler類**
  - Definition 8.2-4 (Pontryagin多項式：実ベクトル束の Pontryagin多項式を $\det(I + \frac{1}{2\pi}A^2) = \sum_j p_j(A)$ で定義；ここで $A^2$ は行列の平方；係数 $p_j$ は even次数の不変多項式で、Pontryagin class $p_j(E)$ を生成)
  - Definition 8.2-5 (Euler多項式：oriented 実ベクトル束の Euler多項式を Pfaffian $\text{Pf}(\frac{1}{2\pi}\Omega)$ で定義；偶数階数 $2m$ の場合のみ非ゼロで、最高次数の Chern-Weil form を与える)
  - Example 8.2-2 (2次元リーマン多様体の Euler class：$S^2$ の接束の曲率2-形式 $\Omega$ に対し、$\int_{S^2} \text{Pf}(\Omega) = \chi(S^2) \cdot 2\pi$ となり、ガウス-ボンネとの対応を示す)

### §8.3 チャーン類とポントリャーギン類
**内容:** 計算例、トポロジカル意味付け、複素多様体上での応用

- **1. 複素ベクトル束とChern類**
  - Definition 8.3-1 (複素ベクトル束：$\mathbb{C}$ 上のベクトル空間をファイバーとする束で、遷移関数が $GL(k, \mathbb{C})$ に値を取るもの；複素構造を持つリーマン多様体や複素多様体の接束）
  - Definition 8.3-2 (Chern類：複素ベクトル束 $E$ のChern多項式を $\det(I + \frac{i}{2\pi}A) = \sum_{j=0}^k c_j(A)$ で定義；係数 $c_j$ が第 $j$ Chern多項式；$c_j(E) \in H^{2j}(M, \mathbb{Z})$ は整数係数コホモロジー類として実現（Atiyah-Hirzebruch）)
  - Example 8.3-1 ($\mathbb{CP}^1$ の接束：$\mathbb{CP}^1 = S^2$ の接束は $\mathcal{O}(2)$ で、第一Chern class $c_1(\mathcal{O}(2)) = 2H$（ここで $H$ はhyperplane class）；Chern-Weil積分により $\int_{\mathbb{CP}^1} c_1 = 2$（オイラー標数）)
  - Example 8.3-2 ($\mathbb{CP}^n$ の接束：接束は hyperplane束 $\mathcal{O}(1)$ と関連；Chern多項式は $(1 + c_1(\mathcal{O}(1)))^{n+1}$ で与えられ、各 Chern class が大幾何学的な意味を持つ)

- **2. Pontryagin類の計算と应用**
  - Example 8.3-3 (球面の接束：$S^4$ の接束に対して、第一Pontryagin class $p_1$ は $H^4(S^4) \cong \mathbb{Z}$ に非ゼロで、この class が微分構造を制約する；実際にはexotic differentiable structure が存在するかについて Pontryagin class が情報を与える)
  - Example 8.3-4 (3次元多様体のPontryagin class：3次元では接束は階数3で、Pontryagin多項式は $p_1$ (4次元)が最初の項；$p_1 = 0$ が automatic となり、トポロジーの制約が弱くなる)

- **3. Whitney sum と multiplicativity**
  - Theorem 8.3-1 (Whitney sum formula：$E = E_1 \oplus E_2$ のとき、Chern多項式は multiplicative；$c(E) = c(E_1) \cdot c(E_2)$（ring multiplication）；特に $c_j(E)$ は lower Chern class で表現される）
  - Proof (Whitney sum の接続を2つの接続の直和として定義し、曲率の直和形式がChern多項式の乗法性を導く)
  - Example 8.3-5 (接束とcotangent束：任意の多様体で $TM \oplus T^*M$ は自明束（秩 $2n$）；Whitney formula により $c(TM) \cdot c(T^*M) = 1$ となり、cotangent束のChern class が接束のそれで決定される)

### §8.4 オイラー類とガウス-ボンネ-チャーンの定理
**内容:** 偶数次元の Gauss-Bonnet-Chern 定理、Euler class、Chern-Simons form

- **1. Euler類と Pfaffian**
  - Definition 8.4-1 (Euler類：oriented 実ベクトル束 $E$ の rank が偶数 $2m$ のとき、接続の曲率2-形式 $\Omega$ に対して Pfaffian $\text{Pf}(\frac{1}{2\pi}A)$ で生成される $(2m)$-形式を Euler形式と呼ぶ；その de Rham class を Euler class $e(E) \in H^{2m}(M, \mathbb{R})$ という；Chern-Weil theory により接続非依存)
  - Proposition 8.4-1 (Euler class の性質：Euler class は cohomology ring において multiplicative；$e(E_1 \oplus E_2) = e(E_1) \cup e(E_2)$；また oriented sphere bundle の Euler class はその Euler characteristic を記述する)
  - Example 8.4-1 (2次元表面の Euler class：$S^2$ の接束の Euler class は標準生成元で、$\int_{S^2} e(TS^2) = \chi(S^2) = 2$；一般に種数 $g$ の surface では $\int e = 2 - 2g$)

- **2. Gauss-Bonnet-Chern定理**
  - Theorem 8.4-1 (Gauss-Bonnet-Chern（偶数次元）：コンパクト向き付け可能な $2m$ 次元リーマン多様体 $(M, g)$ に対し、$\int_M e(TM) = \chi(M)$ が成立；ここで $e(TM)$ は接束の Euler形式で、Levi-Civita接続の曲率から計算される形式)
  - Proof (Chern-Weil theory で Euler形式が接続非依存のコホモロジー類を与えることから、de Rham定理で積分がコホモロジー双対と計算される；詳細は示さないが、6.4の2次元定理の高次元化)
  - Example 8.4-2 (偶数球面：$S^{2m}$ は正定曲率を持つコンパクト多様体で、$\chi(S^{2m}) = 2$（$m$ が任意）；従って $\int_{S^{2m}} e(TS^{2m}) = 2$ が成立；奇数次元球面では $\chi = 0$ で定理は自動的)

- **3. Hirzebruch署名定理と Riemann-Roch**
  - Theorem 8.4-2 (Hirzebruch signature theorem：$4k$ 次元のコンパクト向き付け可能多様体で、intersection form の署名 $\sigma(M)$ は Pontryagin class で記述される；具体的には $\sigma(M) = \int_M L(M)$ ここで $L(M)$ は Pontryagin class から定義される Hirzebruch $L$-多項式)
  - Example 8.4-3 (複素曲面のカテゴリー：複素曲面（4次元実多様体）では Hirzebruch署名定理が K3 surface などの topological type を分類；Chern class と Pontryagin class の関係から signatures が確定される)
  - Definition 8.4-2 (Riemann-Roch：複素多様体の場合、ベクトル束 $E$ の holomorphic セクション数（Euler characteristic）は Chern class で与えられる； Riemann-Roch-Hirzebruch定理により $\chi(E) = \int_M \text{ch}(E) \cdot \text{td}(M)$ ここで $\text{ch}$ は Chern character、$\text{td}$ は Todd class)

- **4. 特性類による幾何学の統一**
  - Remark (Gauss-Bonnet-Chern定理により、local curvature の積分がglobal topology（Euler characteristic）を与える統一的視点が確立される；これが Chapter 9 の指数定理へ向かう重要なステップ)
  - Remark (Chern-Simons form や secondary characteristic class も存在し、Gromov-Witten 理論など現代幾何学で活躍；特性類はモダン微分幾何の中核的ツール)



## Chapter 9: 指数定理への展望
**テーマ:** 楕円型作用素の概念・解析的指数と位相的指数・Atiyah-Singer指数定理・応用の概観

### §9.1 楕円型微分作用素の概念
**内容:** 微分作用素の主記号、楕円性条件、基本的な例、Fredholm性

- **1. 微分作用素と主記号**
  - Definition 9.1-1 (微分作用素：滑らか多様体 $M$ 上の線形微分作用素 $D : \Gamma(E) \to \Gamma(F)$（$E, F$ はベクトル束）を、局所座標で $Du = \sum_{|\alpha| \leq k} A_\alpha(x) \partial^\alpha u + \text{lower order}$ で表されるもの；$k$ を order という；積則を満たす）
  - Definition 9.1-2 (主記号：order $k$ の微分作用素 $D$ の principal symbol $\sigma_k(D)(x, \xi)$（$\xi \in T^*_x M$）は、最高階数項から得られるベクトル束準同型 $\sigma_k(D) : \pi^*E \to \pi^*F$；$\sigma_k(D)(x, \lambda \xi) = \lambda^k \sigma_k(D)(x, \xi)$ で定数でスケール；局所座標では $\sigma_k(D)(x, \xi) = \sum_{|\alpha|=k} A_\alpha(x) \xi^\alpha$)
  - Example 9.1-1 (Laplacian：$\Delta = \sum_i \partial_i^2$ の主記号は $\sigma_2(\Delta)(x, \xi) = -|\xi|^2$；metric に関して負定値で、楕円型の典型例；曲線幾何では第2章で接続を通じて共変Laplacian に拡張)
  - Example 9.1-2 (外微分複合：$d : \Omega^k(M) \to \Omega^{k+1}(M)$ の主記号は $\sigma_1(d)(x, \xi) = i\xi \wedge \cdot$（外積）；楕円性の定義に近いが異なる)

- **2. 楕円性条件**
  - Definition 9.1-3 (楕円型作用素：微分作用素 $D : \Gamma(E) \to \Gamma(F)$ が楕円型であるとは、全ての $(x, \xi) \in T^*M \setminus 0$ で主記号 $\sigma_k(D)(x, \xi)$ が同型であること；すなわち $\sigma_k : E_x \to F_x$ が線形同型（rank $E =$ rank $F$ かつ），あるいは一般に Fredholm operator をなす）
  - Remark (楕円性は fiber-wise condition で、グローバルな位相の影響と interplay する；実際には global space of sections で Fredholm property を示す必要があり、functional analysis（4-1参照）が必要)
  - Example 9.1-3 (Dirac作用素：Clifford加群構造を持つスピン構造付き多様体上の Dirac 作用素 $D : \Gamma(S) \to \Gamma(S)$（$S$ はスピノル束）は1階楕円型；主記号 $\sigma_1(D)(x, \xi) = \xi$ Clifford積；これは素粒子物理の Dirac 方程式の幾何学的抽象化)

- **3. Fredholm性と指数**
  - Definition 9.1-4 (Fredholm 作用素：Banach空間の有界線形作用素 $T$ が Fredholm であるとは、$\ker T$ と $\text{coker} T = F / \text{im} T$ が共に有限次元であること；このとき指数 $\text{ind}(T) := \dim \ker T - \dim \text{coker} T$ を定義）
  - Theorem 9.1-1 (楕円型作用素の Fredholm性：コンパクト多様体 $M$ 上のorder $k$ 楕円型微分作用素 $D : \Gamma(E) \to \Gamma(F)$ を適切なSobolev空間で考えると Fredholm operator になる；すなわち kernel と cokernel が有限次元；従って解析的指数 $\text{ind}(D) = \dim \ker D - \dim \text{coker} D$ が well-defined（証明は functional analysis 4-1）
  - Proof (楕円型性から parametrix（近似逆元）が構成でき、Fredholm property に帰着；詳細は楕円型理論の標準教科書参照)

### §9.2 解析的指数と位相的指数
**内容:** 指数の定義、位相的不変性、Chern-Weil特性類による計算

- **1. 解析的指数の定義と例**
  - Definition 9.2-1 (解析的指数：楕円型作用素 $D : \Gamma(E) \to \Gamma(F)$ に対し、解析的指数（analytic index）を $\text{ind}_a(D) := \dim \ker D - \dim \text{coker} D \in \mathbb{Z}$ で定義；kernel の解（調和解）と cokernel の余剰自由度が相殺した最終的なスカラー値）
  - Example 9.2-1 (Dolbeault複合体：複素多様体 $X$ の Dolbeault 작용소 $\bar{\partial} : \Omega^{p,q} \to \Omega^{p,q+1}$ のコホモロジー $H^{p,q} = \ker \bar{\partial} / \text{im} \bar{\partial}$ の次元がベクトル束の holomorphic セクション数を記述；Riemann-Roch は $\text{ind}(\bar{\partial}) = \dim H^{p,0} - \dim H^{p,1} + \cdots$ を計算)
  - Example 9.2-2 (署名複合体：4k次元向き付け可能多様体の署名複合体は接束の exterior algebra で定義された複合体で、その指数が intersection form の署名 $\sigma(M)$ に等しい)

- **2. 位相的指数と特性類**
  - Definition 9.2-2 (位相的指数：楕円型作用素 $D$ に対し、位相的指数（topological index）を主記号の特性類（Chern character など）とベースの特性類の cup product で計算した値；$\text{ind}_t(D) := \int_M \text{some characteristic class}$；これは純粋に微分形式の integrate として決定）
  - Theorem 9.2-1 (指数定理の主張：任意の楕円型微分作用素 $D$ に対し、$\text{ind}_a(D) = \text{ind}_t(D)$ が成立；すなわち analytic index（微分方程式の解の個数）が topological index（幾何学的特性類の積分）と一致する；この一致が Atiyah-Singer 指数定理の内容)

- **3. 特性類による指数計算**
  - Theorem 9.2-2 (Atiyah-Singer 指数公式：楕円型作用素 $D : \Gamma(E) \to \Gamma(F)$ に対し、$\text{ind}(D) = \int_M \text{ch}(\sigma(D)) \cup \text{td}(TM)$ あるいは場合に応じて別の Chern-Weil form；ここで $\sigma(D)$ は principal symbol のvector bundle 層面，$\text{td}$ は Todd class；この公式は次元が一致し integral で整数を与える）
  - Proof (Chern-Weil理論と Atiyah-Segal K-理論の duality から，主張は non-trivial で，元々の Atiyah-Singer 論文（1960年代）を参照；ここでは完全な証明は省略)
  - Example 9.2-3 ($\bar{\partial}$-作用素の指数：Dolbeault作用素 $\bar{\partial} : \Gamma(\mathcal{O}(L)) \to \Gamma(\mathcal{O}(L) \otimes \Omega^{0,1})$ の指数は Riemann-Roch formula $h^0 - h^1 = \deg L + 1 - g$（複素曲線の場合）で Chern character積分で回復される)

### §9.3 アティヤ-シンガーの指数定理の主張
**内容:** 定理の正確な statement、証明への展望、数学物理への影響

- **1. 一般的な statement**
  - Theorem 9.3-1 (Atiyah-Singer指数定理（一般形））：コンパクト多様体 $M$ 上の楕円型微分作用素 $D : \Gamma(E) \to \Gamma(F)$ に対し、解析的指数 $\text{ind}(D)$ が、$M$ と symbol の特性類から定まる位相的指数に等しい；$\text{ind}(D) = \left\langle \text{ch}(\sigma(D)) \cup \text{td}(TM), [M] \right\rangle$（Poincaré双対）;ここで $[\sigma(D)] \in K(T^*M)$ は relative K-theory class で，双対cycleを定義）
  - Remark (statement の正確さには K-理論と Chern-Weil cohomology の同型性が重要；K-homology と cohomology の pairing を通じて指数は integer として computed)

- **2. 証明への展望**
  - Remark (Atiyah-Singer 指数定理の証明は複数の方法が存在：(1) 熱核（heat kernel）法，(2) cobordism と characteristic class，(3) K-理論と Chern character，(4) Clifford algebra と spectral flow；各方法が異なる幾何学的insightを提供；完全な証明はこのコース外であるが，4-7（functional analysis）と並列して学習することが推奨される）
  - Remark (証明の鍵は principal symbol がベクトル束のK-class を定め，その over sphere reduction（sphere bundle）で cohomology計算に帰着させること；この reduction技法が微分幾何学的に最も elegant)

- **3. 指数定理の数学的意義**
  - Remark (Atiyah-Singer定理は20世紀微分幾何の最高峰の成果で，解析（楕円型PDE理論）と位相（characteristic classes）と代数（K-理論）を統一；これにより differential geometry が他の数学分野と深く結びついた)
  - Remark (定理の系として，Gauss-Bonnet（Chapter 6），Hirzebruch signature，Riemann-Roch（Chapter 8）が全て指数定理の特別な場合として統一的に理解される；mathematical unification の美しい例)

### §9.4 特別な場合への応用
**内容:** de Rham複合体、署名定理、Riemann-Roch、指数定理による統一的視点

- **1. de Rham複合体と Gauss-Bonnet**
  - Theorem 9.4-1 (de Rham複合体の指数：Riemannian多様体 $M$ に対し，外微分複合体 $d : \Omega^* \to \Omega^{*+1}$ は楕円型複合体で，その指数は $\text{ind}(d) = \sum_{i=0}^n (-1)^i \dim H^i_{dR}(M)$ で定義；Atiyah-Singer 定理により，$\text{ind}(d) = \chi(M)$（オイラー標数）が成立；これが Gauss-Bonnet-Chern 定理の一般化)
  - Example 9.4-1 (2次元曲面：曲面 $M$ での de Rham cohomology $H^* = H^0 \oplus H^1 \oplus H^2$ に対し，$\chi(M) = \dim H^0 - \dim H^1 + \dim H^2$；Hodge decomposition により $\dim H^1 = \dim H^0(1-形式)$；指数定理から $\int K = 2\pi\chi$ が導出される)

- **2. Hirzebruch署名定理**
  - Theorem 9.4-2 (署名複合体の指数定理解釈：4k次元向き付け可能多様体 $M$ の署名 $\sigma(M)$（intersection form の固有値の差）は，指数定理から $\sigma(M) = \int_M L(M)$（Hirzebruch L-多項式）；L-多項式は Pontryagin classes $p_i$ で定義された Chern-Weil form；これにより signature がglobal topological quantity であることが確認される)
  - Example 9.4-2 (K3曲面：複素曲面（4次元）K3の signature $\sigma = 16$ が定数；Hirzebruch formula により Pontryagin class $p_1$ の値が constraint；$p_1(K3) = 24$）

- **3. Riemann-Roch定理と複素幾何**
  - Theorem 9.4-3 (Riemann-Roch-Hirzebruch：複素多様体 $X$ と vector bundle $E$ に対し，holomorphic セクション数の Euler characteristic $\chi(X, E) = \int_X \text{ch}(E) \cup \text{td}(X)$；ここで $\text{ch}(E) = \sum_j e^{c_j(E)/(2\pi i)}$ は Chern character，$\text{td}(X) = \prod_i \frac{c_i(TX)/(2\pi i)}{1 - e^{-c_i(TX)/(2\pi i)}}$ は Todd class)
  - Example 9.4-3 (複素曲線（Riemann面）：曲線上の line bundle $L$ に対し，$h^0(L) - h^1(L) = \deg L + 1 - g$（古典的 Riemann-Roch）；指数定理解釈では Chern character $c_1(L) + \dim X / 2 = \deg L + 1 - g$；複素Laplacian の kernel/cokernel dimensionが 古典的formulation を recover)
  - Example 9.4-4 (複素曲面：複素曲面 $X$ の接束 $TX$ に対し，自己双対微分形式（署名複合体のセクション）の個数が signature theorem で記述；Chern class $c_1, c_2$ から Hirzebruch polynomial を経由して index が computed）

- **4. 指数定理の統一的視点**
  - Remark (Atiyah-Singer定理により，微分幾何学（Riemann計量，接続，曲率），位相幾何学（characteristic classes，cohomology），解析学（楕円型PDE，kernel/cokernel），および代数幾何学（holomorphic sections，Chern classes）が統一的に理解される frame workが確立された)
  - Remark (応用として，string theory や gauge theory の数学的foundation が index theorem に基盤を置く；現代理論物理と数学の深い結びつきの起点)
  - Remark (Chapter 6-9 全体を通じて，古典的微分幾何（接続，曲率）から現代的位相幾何（特性類，指数定理）への発展の道筋が示された；より高度な学習への門戸が開かれる)
