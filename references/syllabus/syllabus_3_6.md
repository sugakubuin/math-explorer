## Chapter 1: 汎関数と変分問題
**テーマ:** 汎関数の基本概念と変分問題の定式化

### §1.1 汎関数の定義と例
**内容:** 関数空間上の写像としての汎関数と変分問題の設定

- **1. 汎関数の定義**
  - Definition 1.1-1 (汎関数：関数空間 $\mathcal{A}$ から実数 $\mathbb{R}$ への写像 $J : \mathcal{A} \to \mathbb{R}$；変数が「関数」である点で通常の関数と異なる)
  - Example 1.1-1 (弧長汎関数：平面曲線 $y = u(x)$（$a \leq x \leq b$）の弧長 $J[u] = \int_a^b \sqrt{1 + u'^2}\,dx$；$u(x) = x$ のとき $J = \sqrt{2}(b-a)$)
  - Example 1.1-2 (ディリクレ積分：$J[u] = \int_\Omega |\nabla u|^2\,dx$；静電場のエネルギーや弾性膜の歪みエネルギーに対応)
  - Remark (変分問題の一般形：$J[u] = \int_a^b F(x, u, u')\,dx$ の形に多くの問題が帰着する；$F$ を被積分関数（ラグランジアン）という)

- **2. 変分問題の設定**
  - Definition 1.1-2 (変分問題：汎関数 $J[u]$ を極小（または極大）にする関数 $u^* \in \mathcal{A}$ を求める問題)
  - Example 1.1-3 (最短経路問題：2 点 $(a, \alpha)$、$(b, \beta)$ を結ぶ曲線のうち弧長を最小にするものは直線；弧長汎関数の最小化問題として定式化できる)
  - Example 1.1-4 (最小曲面問題：与えられた閉曲線を境界に持つ曲面のうち面積を最小にするもの（極小曲面）；石鹸膜がこの問題の物理的実現)
  - Remark (最適化との違い：変分問題は有限次元の最適化問題の無限次元版；極値の必要条件が「勾配ゼロ」から「オイラー＝ラグランジュ方程式」に置き換わる)

### §1.2 許容関数のクラスと境界条件
**内容:** 変分問題の解を探す関数空間と端点条件の設定

- **1. 許容関数クラスの定義**
  - Definition 1.2-1 (許容関数クラス：変分問題で解を探す関数の集合 $\mathcal{A}$；典型的には $\mathcal{A} = \{u \in C^2[a,b] \mid u(a) = \alpha,\, u(b) = \beta\}$（固定端条件）)
  - Example 1.2-1 (固定端条件の設定：弦の振動問題で両端が固定されている場合 $u(0,t) = u(L,t) = 0$；変分問題の文脈では $\mathcal{A} = \{u \in C^2[0,L] \mid u(0) = u(L) = 0\}$)

- **2. 変分と第一変分の準備**
  - Definition 1.2-2 (変分（admissible variation）：許容関数 $u \in \mathcal{A}$ に対して $u + \varepsilon\eta$（$\varepsilon \in \mathbb{R}$、$\eta \in C^2[a,b]$、$\eta(a) = \eta(b) = 0$）の形の摂動；$\eta$ を変分方向という)
  - Example 1.2-2 (変分方向の確認：$u(x) = x^2$ への変分 $\eta(x) = \sin(\pi x)$（$[0,1]$ 上）；$\eta(0) = \eta(1) = 0$ を満たすので許容；$\varepsilon = 0.1$ のとき $u + \varepsilon\eta = x^2 + 0.1\sin(\pi x)$ を確認)
  - Remark (無限次元微積分のアナロジー：有限次元で $f(x + \varepsilon d) = f(x) + \varepsilon \nabla f \cdot d + O(\varepsilon^2)$ となるように、汎関数でも $J[u + \varepsilon\eta] = J[u] + \varepsilon \delta J[u;\eta] + O(\varepsilon^2)$ と展開する；$\delta J$ が第一変分)

### §1.3 変分と第一変分
**内容:** 第一変分の定義と極値の必要条件

- **1. 第一変分の定義**
  - Definition 1.3-1 (第一変分：$\delta J[u;\eta] = \frac{d}{d\varepsilon}J[u+\varepsilon\eta]\Big|_{\varepsilon=0}$；汎関数の「方向微分」に相当する)
  - Proposition 1.3-1 ($J[u] = \int_a^b F(x,u,u')\,dx$ の第一変分：$\delta J[u;\eta] = \int_a^b (F_u \eta + F_{u'}\eta')\,dx$；$F$ の全微分を $\varepsilon$ で微分してから $\varepsilon = 0$ を代入することで得られる)
  - Proof (命題の証明：$\frac{d}{d\varepsilon}F(x, u+\varepsilon\eta, u'+\varepsilon\eta') = F_u\eta + F_{u'}\eta'$；積分と微分の順序を交換して積分する)
  - Example 1.3-1 (弧長汎関数の第一変分：$F = \sqrt{1+u'^2}$ に対して $F_u = 0$、$F_{u'} = u'/\sqrt{1+u'^2}$；$\delta J[u;\eta] = \int_a^b \frac{u'\eta'}{\sqrt{1+u'^2}}\,dx$)

- **2. 極値の必要条件**
  - Theorem 1.3-2 (極値の必要条件：$u^* \in \mathcal{A}$ が $J$ の極値を与えるならば任意の変分方向 $\eta$ に対して $\delta J[u^*;\eta] = 0$)
  - Proof (必要条件の証明：$\phi(\varepsilon) = J[u^* + \varepsilon\eta]$ は $\varepsilon = 0$ で極値を持つので $\phi'(0) = \delta J[u^*;\eta] = 0$；有限次元の「$\nabla f = 0$」と同じ論理)
  - Remark (必要条件と十分条件：$\delta J = 0$ は極値の必要条件だが十分条件ではない；真に極小かどうかの判定には第二変分（Chapter 4）が必要)

## Chapter 2: オイラー＝ラグランジュ方程式
**テーマ:** 極値を与える関数が満たす微分方程式の導出と応用

### §2.1 方程式の導出
**内容:** 変分の基本補題を用いたオイラー＝ラグランジュ方程式の導出

- **1. 変分の基本補題**
  - Lemma 2.1-1 (変分の基本補題：$g \in C[a,b]$ が任意の $\eta \in C^2[a,b]$（$\eta(a)=\eta(b)=0$）に対して $\int_a^b g(x)\eta(x)\,dx = 0$ を満たすならば $g \equiv 0$ on $[a,b]$)
  - Proof (基本補題の証明：$g$ がある点 $x_0$ でゼロでないと仮定する；連続性から $g > 0$（または $<0$）となる小区間 $I$ が存在する；$I$ のみサポートを持つ $\eta \geq 0$ を構成すると積分が非ゼロになり矛盾)
  - Example 2.1-1 (基本補題の必要性：$g(x) = x$ に対して $\eta(x) = x(1-x)$ を取ると $\int_0^1 x \cdot x(1-x)\,dx = 1/12 \neq 0$；つまり $g \not\equiv 0$ のとき積分がゼロでない $\eta$ が取れることの確認)

- **2. オイラー＝ラグランジュ方程式**
  - Theorem 2.1-2 (オイラー＝ラグランジュ方程式：$J[u] = \int_a^b F(x,u,u')\,dx$ の極値関数 $u^*$ は $F_u - \frac{d}{dx}F_{u'} = 0$ を満たす)
  - Proof (E-L 方程式の導出：$\delta J = 0$ の式 $\int_a^b (F_u\eta + F_{u'}\eta')\,dx = 0$ で第 2 項を部分積分すると $\int_a^b (F_u - \frac{d}{dx}F_{u'})\eta\,dx + [F_{u'}\eta]_a^b = 0$；固定端条件から境界項はゼロ；基本補題より括弧内がゼロ)
  - Example 2.1-2 (弧長汎関数への適用：$F = \sqrt{1+u'^2}$ の E-L 方程式は $\frac{d}{dx}\frac{u'}{\sqrt{1+u'^2}} = 0$；積分すると $u'/\sqrt{1+u'^2} = c$（定数）；$u' = c/\sqrt{1-c^2}$（定数）より解は直線 $u = mx + n$)
  - Remark (E-L 方程式の地位：変分問題の「必要条件の方程式」であり ODE になる；解が E-L 方程式を満たすことは極値の候補であることを意味し、「停留関数（stationary function）」という)

### §2.2 具体例：最速降下線と測地線
**内容:** 重要な変分問題への E-L 方程式の適用と解の導出

- **1. 最速降下線（ブラキストクロン問題）**
  - Example 2.2-1 (問題の設定：原点 $(0,0)$ から点 $(a,b)$（$b < 0$）へ重力下で最短時間で滑る曲線を求める；下向きを正とすると所要時間 $T[y] = \int_0^a \frac{\sqrt{1+y'^2}}{\sqrt{2gy}}\,dx$；$F = \sqrt{(1+y'^2)/y}$ として E-L 方程式を立てる)
  - Example 2.2-2 (最速降下線の解（サイクロイド）：$F$ が $x$ を陽に含まないので保存則 $F - y'F_{y'} = c$ が成立；整理すると $y(1+y'^2) = 1/(2gc^2)$；媒介変数 $t$ を用いた置換 $y = r(1-\cos t)$、$x = r(t - \sin t)$（サイクロイド）が得られることの確認)
  - Remark (ベルヌーイ兄弟の問題：1696 年のヨハン・ベルヌーイの挑戦状に端を発した問題；ニュートン・ライプニッツ・ヤコブ・ベルヌーイらが解を提出した変分法の原点の一つ)

- **2. 測地線**
  - Example 2.2-3 (平面の測地線：$\mathbb{R}^2$ での弧長最小化問題の E-L 方程式は $y'' = 0$；解は直線；2 点間の最短曲線が直線であることの変分法による確認)
  - Example 2.2-4 (球面の測地線：半径 $R$ の球面をパラメータ表示し弧長汎関数の E-L 方程式を解くと大円（球の中心を通る平面との断面）が得られることの概略；航路計算への応用)

### §2.3 高階導関数を含む場合
**内容:** オイラー＝ポアソン方程式の導出と梁の曲げ問題への応用

- **1. オイラー＝ポアソン方程式**
  - Theorem 2.3-1 (オイラー＝ポアソン方程式：$J[u] = \int_a^b F(x, u, u', u'')\,dx$ の極値関数は $F_u - \frac{d}{dx}F_{u'} + \frac{d^2}{dx^2}F_{u''} = 0$ を満たす)
  - Proof (導出：$\delta J = 0$ を部分積分 2 回適用；$u''$ の変分 $\eta''$ の項を 2 回部分積分して $\eta$ にまとめ、基本補題を適用)
  - Example 2.3-1 (弾性梁の曲げ問題：ひずみエネルギー $J[u] = \frac{1}{2}\int_0^L (EI)(u'')^2\,dx$ に分布荷重 $-\int_0^L qu\,dx$ を加えた汎関数；$F = \frac{EI}{2}(u'')^2 - qu$ から E-P 方程式は $EI u'''' = q$（梁の方程式）)
  - Remark (高階微分の出現と境界条件の増加：2 階導関数を含む場合、E-P 方程式は 4 階 ODE になり、境界条件が 4 つ（各端点で 2 つ）必要になる；自然境界条件（§3.1）がここで重要)

- **2. 変数の多い場合**
  - Proposition 2.3-2 (多変数の E-L 方程式：$J[u_1,\ldots,u_n] = \int_a^b F(x, u_i, u_i')\,dx$ の極値は各 $u_k$ について独立に $F_{u_k} - \frac{d}{dx}F_{u_k'} = 0$（$k=1,\ldots,n$）を満たす)
  - Proof (各変分方向 $\eta_k$ を独立に取ることで $n$ 本の E-L 方程式が得られる；基本補題を各成分に適用)
  - Example 2.3-2 (3 次元空間での最短経路：曲線 $(x(t), y(t), z(t))$ の弧長 $J = \int \sqrt{\dot{x}^2+\dot{y}^2+\dot{z}^2}\,dt$ の E-L 方程式は $\ddot{x} = \ddot{y} = \ddot{z} = 0$；各成分が等速直線運動（弧長パラメータで自然パラメータ付けした直線）になることの確認)

## Chapter 3: 境界条件とトランザーバリティ
**テーマ:** 端点が固定されていない場合の境界条件の導出

### §3.1 固定端・自由端の自然境界条件
**内容:** 部分積分の境界項から自然に導かれる境界条件

- **1. 自然境界条件の導出**
  - Definition 3.1-1 (自由端条件：端点での $u$ の値を指定しない場合；許容クラスは $\mathcal{A} = \{u \in C^2[a,b]\}$（端点条件なし）となり変分方向 $\eta$ も端点で自由)
  - Theorem 3.1-1 (自然境界条件：自由端での E-L 方程式の導出過程で現れる境界項 $[F_{u'}\eta]_a^b$ がゼロになるためには $F_{u'}|_{x=a} = 0$ および $F_{u'}|_{x=b} = 0$ が成立しなければならない；これを自然境界条件という)
  - Proof (導出：$\eta(a), \eta(b)$ が任意の実数なので $[F_{u'}\eta]_a^b = F_{u'}(b)\eta(b) - F_{u'}(a)\eta(a) = 0$ は $\eta(a), \eta(b)$ が独立に任意であることから $F_{u'}(a) = F_{u'}(b) = 0$ を強制する)
  - Example 3.1-1 (弾性梁の自由端条件：ひずみエネルギー汎関数 $J[u] = \int_0^L \frac{EI}{2}(u'')^2\,dx$ の自由端では $EI u''|_{\text{端点}} = 0$（曲げモーメントゼロ）かつ $EI u'''|_{\text{端点}} = 0$（せん断力ゼロ）が自然境界条件となる)

- **2. 固定端と自由端の混合**
  - Example 3.1-2 (片持ち梁の条件：$x = 0$ が固定端（$u(0) = 0, u'(0) = 0$）で $x = L$ が自由端（自然境界条件）の場合；$x = L$ での条件が自然に $u''(L) = u'''(L) = 0$ となることの確認)
  - Remark (自然境界条件の名前の意味：「問題が自然に要求する」条件という意味；強制（essential）境界条件（値を直接指定するもの）と区別される；有限要素法では自然境界条件は弱定式化に自動的に組み込まれる)

### §3.2 トランザーバリティ条件
**内容:** 端点が曲線上を動く場合の横断条件

- **1. 端点移動の変分**
  - Definition 3.2-1 (端点移動を伴う変分問題：端点 $(b, u(b))$ が曲線 $\Gamma: x = \psi(y)$（または $y = \phi(x)$）上を自由に動く場合；許容クラスと変分の定義を端点の移動を込めて拡張する)
  - Theorem 3.2-1 (トランザーバリティ条件：端点が曲線 $y = \phi(x)$ 上を動く場合、E-L 方程式に加えて端点での横断条件 $(F + (\phi' - u')F_{u'})\big|_{x=b} = 0$ が成立しなければならない)
  - Proof (導出：変分 $\delta J$ に端点の移動分 $\delta b$ と $\delta u(b) = \phi'(b)\delta b$ を加えた完全な第一変分 $\delta J_{\text{total}} = 0$ から条件を読み取る)

- **2. 横断条件の幾何的意味**
  - Example 3.2-1 (直交横断条件：$F = \sqrt{1+u'^2}$（弧長）のときトランザーバリティ条件は $1/\sqrt{1+u'^2} + u'(\phi'-u')/\sqrt{1+u'^2} = 0$；整理すると $1 + u'\phi' = 0$（$u' = -1/\phi'$）；すなわち最短経路の端点での接線が境界曲線 $\Gamma$ と直交することを意味する)
  - Remark (反射・屈折の変分原理：光が鏡で反射する際の反射角＝入射角や、スネルの屈折則もトランザーバリティ条件として導出できる；変分原理が物理の対称性・境界挙動を統一的に記述する)

## Chapter 4: 第二変分と極値条件
**テーマ:** 停留関数が真に極小（または極大）になるための条件

### §4.1 第二変分の定義と計算
**内容:** 汎関数の 2 次の変化量と極値判定

- **1. 第二変分の定義**
  - Definition 4.1-1 (第二変分：$\delta^2 J[u;\eta] = \frac{d^2}{d\varepsilon^2}J[u+\varepsilon\eta]\Big|_{\varepsilon=0}$；汎関数の「ヘッセ行列」に相当する 2 次形式)
  - Proposition 4.1-1 ($J[u] = \int_a^b F(x,u,u')\,dx$ の第二変分：$\delta^2 J[u;\eta] = \int_a^b (F_{uu}\eta^2 + 2F_{uu'}\eta\eta' + F_{u'u'}\eta'^2)\,dx$)
  - Proof ($J[u+\varepsilon\eta]$ を $\varepsilon$ で 2 回微分し $\varepsilon=0$ を代入；$F$ の引数が $\varepsilon$ に依存することに注意して連鎖律を適用)

- **2. 極値判定への応用**
  - Theorem 4.1-2 (第二変分による極値判定：停留関数 $u^*$ で $\delta^2 J[u^*;\eta] > 0$（すべての $\eta \neq 0$）ならば $u^*$ は弱極小；$< 0$ ならば弱極大)
  - Example 4.1-1 (弧長汎関数の第二変分：$F_{u'u'} = (1+u'^2)^{-3/2} > 0$、$F_{uu} = F_{uu'} = 0$；直線 $u^*$ での第二変分は $\int_a^b \eta'^2/(1+u^{*\prime 2})^{3/2}\,dx > 0$（$\eta \neq 0$）；直線が弱極小であることの確認)
  - Remark (弱極小と強極小の違い：第二変分による極値判定は「$\eta$ と $\eta'$ が小さい」という弱い位相での極小（弱極小）しか保証しない；$\eta'$ だけが大きい摂動に対する安定性（強極小）は別の条件（ヤコビの条件、§4.3）が必要)

### §4.2 ルジャンドル条件
**内容:** 弱極小の必要条件

- **1. ルジャンドル条件の導出**
  - Theorem 4.2-1 (ルジャンドル条件：$u^*$ が弱極小ならば $F_{u'u'}(x, u^*, u^{*\prime}) \geq 0$（$a \leq x \leq b$ 全体で）；厳格な弱極小では $> 0$ が必要)
  - Proof (証明：$\eta$ として $[x_0 - \varepsilon, x_0 + \varepsilon]$ 上のみ振動する「細い三角波」をとり $\varepsilon \to 0$ の極限を取る；第二変分の主要項が $F_{u'u'}\|\eta'\|^2$ に帰着し符号が $F_{u'u'}$ で決まる)
  - Example 4.2-1 (ルジャンドル条件の確認：最速降下線の $F = \sqrt{(1+y'^2)/y}$ に対して $F_{y'y'} = 1/(y(1+y'^2)^{3/2})$；$y > 0$（曲線が $x$ 軸より下）の領域で $> 0$；ルジャンドル条件が満たされることの確認)
  - Example 4.2-2 (ルジャンドル条件が破れる例：$J[u] = \int_{-1}^1 u'^3\,dx$ の停留関数 $u^* = 0$（直線）では $F_{u'u'} = 6u' = 0$；退化してルジャンドル条件が判定不能；$u^* = 0$ が極値でないことも確認できる)

- **2. 必要条件の意味**
  - Remark (ルジャンドル条件と「下に凸」：$F_{u'u'} \geq 0$ は $F$ が $u'$ の変数として「下に凸」であることを意味する；凸汎関数（$F$ が凸）なら弱極小と強極小が一致する（後述の補足事項）；詳細は凸解析（4-3）で扱う)

### §4.3 ヤコビ方程式と共役点
**内容:** 強極小の必要条件と共役点の概念

- **1. ヤコビ方程式**
  - Definition 4.3-1 (ヤコビ方程式：停留関数 $u^*$ に沿った第二変分 $\delta^2 J$ の Euler-Lagrange 方程式 $\frac{d}{dx}(F_{u'u'}h') - (F_{uu} - \frac{d}{dx}F_{uu'})h = 0$；$h = du^*/d\alpha$（$\alpha$ はパラメータ族 $u^*_\alpha$ の微分）と解釈できる)
  - Example 4.3-1 (弧長汎関数のヤコビ方程式：直線 $u^* = mx + n$ での計算；$F_{uu} = F_{uu'} = 0$、$F_{u'u'} = (1+m^2)^{-3/2}$；ヤコビ方程式は $h'' = 0$（$h$ は直線）；ヤコビ方程式の解が端点以外でゼロを持たないことの確認)

- **2. 共役点とヤコビの条件**
  - Definition 4.3-2 (共役点：停留関数 $u^*$ の左端点 $a$ に共役な点とは、ヤコビ方程式の解 $h$（$h(a) = 0$、$h'(a) \neq 0$）がゼロになる最初の点 $\bar{a} > a$)
  - Theorem 4.3-1 (ヤコビの条件：停留関数 $u^*$ が開区間 $(a, b)$ 内に共役点を持たないことが強極小の必要条件；共役点が $(a, b)$ 内に存在すれば $u^*$ は極小でない)
  - Proof (証明のスケッチ：共役点 $\bar{a} \in (a,b)$ が存在すれば $h$ を零延長した関数を変分方向に取ることで $\delta^2 J < 0$ を構成できる；共役点なしの場合は第二変分の正定値性を Sturm 比較定理で保証する)
  - Example 4.3-2 (球面測地線の共役点：半径 $R$ の球上で北極 $N$ から出る大円は南極 $S$（距離 $\pi R$）で「集まる」；$S$ が $N$ の共役点；$N$ から $S$ を超える大円弧は弧長の極小を与えない（最短でなくなる）ことの確認)
  - Remark (共役点と焦点の類比：共役点は光学の「焦点」に対応する；「焦点より遠くを進む光線は必ずしも最短光路でない」という直感と一致する；シンプレクティック幾何学（5-17）ではラグランジアン部分多様体の焦点として統一的に記述される)

## Chapter 5: ハミルトン形式
**テーマ:** ラグランジュ形式からハミルトン形式への変換と正準方程式

### §5.1 ルジャンドル変換と一般化運動量
**内容:** 速度から運動量への変数変換とハミルトニアンの定義

- **1. ルジャンドル変換の定義**
  - Definition 5.1-1 (一般化運動量：ラグランジアン $F(x, u, u')$ に対して $p = F_{u'}(x, u, u')$；$u'$ を $p$ で表す変換を考える)
  - Definition 5.1-2 (ハミルトニアン：$H(x, u, p) = pu' - F(x, u, u')$；ここで $u'$ は $p = F_{u'}$ から逆に求めた関数 $u' = \psi(x, u, p)$；$p$ と $u$ が独立変数になる)
  - Proposition 5.1-1 (ルジャンドル変換の対称性：$F$ が $u'$ について凸（$F_{u'u'} > 0$）ならば $p = F_{u'}$ が可逆で $H$ が $p$ について凸；かつ $(F_{u'})_{u'} = 1/H_{pp}$（逆関数微分）)
  - Proof ($F_{u'u'} > 0$ のもとで陰関数定理により $p = F_{u'}$ が $u' = \psi(x,u,p)$ として局所的に解ける；$H_{pp} = 1/F_{u'u'} > 0$)
  - Example 5.1-1 (古典力学のルジャンドル変換：$F = \frac{m}{2}u'^2 - V(u)$（速度を $u'$、ポテンシャルを $V$ とする）；$p = mu'$（通常の運動量）；$H = pu' - F = \frac{p^2}{2m} + V(u)$（全エネルギー）の確認)

- **2. 変数変換の逆方向**
  - Example 5.1-2 (ハミルトニアンからラグランジアンの復元：$H = p^2/(2m) + V(u)$ から $u' = H_p = p/m$、$F = pu' - H = p^2/(2m) - V = mu'^2/2 - V$（運動エネルギーからポテンシャルを引いた量）；ラグランジアンが回収されることの確認)
  - Remark (ルジャンドル変換の幾何学的意味：$F(u')$ のグラフの接線の $y$ 切片として $-H(p)$ を定義する；凸関数の接線族の包絡線が双対関数を与えるという凸解析（4-3）の基本定理と同一の構造)

### §5.2 正準方程式
**内容:** ハミルトンの正準方程式とオイラー＝ラグランジュ方程式との同値性

- **1. 正準方程式の導出**
  - Theorem 5.2-1 (ハミルトンの正準方程式：$u' = H_p$、$p' = -H_u$；この連立 1 階系は E-L 方程式 $F_u - (F_{u'})' = 0$ と同値)
  - Proof (同値性の証明：$u' = \psi$ と $p = F_{u'}$ を $x$ で微分し $p' = F_{uu'}\psi + F_{u'u'}\psi' = F_{uu} + (\frac{d}{dx}F_{u'})$（$u = u(\psi)$ の連鎖律）；E-L 方程式 $p' = F_u$ を代入すると $-H_u = F_u - pu'_u$（$H$ の定義から）が整理される)
  - Example 5.2-1 (単振子の正準方程式：$F = \frac{m}{2}u'^2 - mgl(1-\cos u)$；$p = mu'$；$H = p^2/(2m) + mgl(1-\cos u)$；正準方程式 $u' = p/m$、$p' = -mgl\sin u$；第 2 式を代入すると $mu'' = -mgl\sin u$（振子の運動方程式）に一致することの確認)
  - Remark (相空間の概念：$(u, p)$ を座標とする空間を相空間という；ODE の相平面解析（3-5 §3.1）が 2 次元相空間での議論であり、ハミルトン形式はそれを一般次元に体系化したもの；シンプレクティック構造（5-17）が自然に現れる舞台)

- **2. ポアソン括弧と保存量**
  - Definition 5.2-2 (ポアソン括弧：2 つの関数 $f(u, p)$、$g(u, p)$ に対して $\{f, g\} = \frac{\partial f}{\partial u}\frac{\partial g}{\partial p} - \frac{\partial f}{\partial p}\frac{\partial g}{\partial u}$)
  - Proposition 5.2-2 (ポアソン括弧と保存量：$H$ の時間微分 $\frac{d}{dx}f = \{f, H\} + \partial f/\partial x$（正準方程式を使用）；特に $\{f, H\} = 0$ かつ $\partial f/\partial x = 0$ ならば $f$ は保存量)
  - Proof ($\frac{d}{dx}f = f_u u' + f_p p' = f_u H_p - f_p H_u = \{f, H\}$（正準方程式を代入）)
  - Example 5.2-2 (ハミルトニアン自身の保存：$\{H, H\} = 0$（ポアソン括弧の反対称性）かつ $\partial H/\partial x = 0$（自律系）のとき $H$ は保存量；自律的なハミルトン系でエネルギーが保存されることの証明)

### §5.3 物理への応用
**内容:** 解析力学のハミルトン形式と連続媒体への拡張

- **1. 解析力学との対応**
  - Example 5.3-1 (多自由度系の正準方程式：$n$ 自由度系 $J[q_1,\ldots,q_n] = \int L(t, q_i, \dot{q}_i)\,dt$；一般化座標 $q_k$・一般化運動量 $p_k = L_{\dot{q}_k}$・ハミルトニアン $H = \sum p_k\dot{q}_k - L$；正準方程式 $\dot{q}_k = H_{p_k}$、$\dot{p}_k = -H_{q_k}$；電磁場中の荷電粒子や剛体の運動への応用)
  - Remark (ハミルトン・ヤコビ方程式：正準変換の理論を推し進めると $H + \partial S/\partial t = 0$（ハミルトン・ヤコビ方程式）が得られ、量子力学（シュレーディンガー方程式）の半古典近似や幾何光学の基礎となる；詳細はシンプレクティック幾何学（5-17）で扱う)

- **2. 場の変分原理への展望**
  - Remark (連続体・場への拡張：独立変数を時間 $t$ 一変数から $(t, x)$ 多変数に拡張すると「場のラグランジアン」が定義できる；E-L 方程式が波動方程式・ラプラス方程式・Einstein 方程式などの PDE を与える；ノーターの定理（Chapter 6）との合わせ技で場の保存則が組織的に導出される)

## Chapter 6: ノーターの定理
**テーマ:** 連続的対称性から保存則を系統的に導出する変分法の中心定理

### §6.1 対称性と 1 パラメータ変換群
**内容:** 汎関数の不変性の定式化

- **1. 1 パラメータ変換群の定義**
  - Definition 6.1-1 (1 パラメータ変換群：$s \in \mathbb{R}$ をパラメータとする変換族 $\Phi_s : (x, u) \mapsto (\tilde{x}(x,u,s), \tilde{u}(x,u,s))$ で $\Phi_0 = \mathrm{id}$、$\Phi_s \circ \Phi_t = \Phi_{s+t}$ を満たすもの；$s$ についての微分で得られるベクトル場 $(\xi, \phi) = \frac{d}{ds}(\tilde{x}, \tilde{u})\big|_{s=0}$ を無限小生成子という)
  - Example 6.1-1 (平行移動群：$\tilde{x} = x + s$、$\tilde{u} = u$；生成子 $(\xi, \phi) = (1, 0)$；時間変数の平行移動がエネルギー保存に結びつく（§6.3 参照）)
  - Example 6.1-2 (スケール変換群：$\tilde{x} = e^s x$、$\tilde{u} = e^{\alpha s}u$；生成子 $(\xi, \phi) = (x, \alpha u)$；次元解析と関連する対称性)

- **2. 汎関数の不変性**
  - Definition 6.1-2 (汎関数の不変性：$J[\tilde{u}] = J[u]$（変換後の汎関数が変換前と等しい）とき、汎関数は変換群 $\Phi_s$ に関して不変という；より正確には $\int_{\tilde{a}}^{\tilde{b}} F(\tilde{x}, \tilde{u}, \tilde{u}')\,d\tilde{x} = \int_a^b F(x, u, u')\,dx$)
  - Example 6.1-3 (弧長の平行移動不変性：$J[u] = \int_a^b \sqrt{1+u'^2}\,dx$ に $\tilde{x} = x + s$ を適用すると $\int_{a+s}^{b+s}\sqrt{1+\tilde{u}'^2}\,d\tilde{x} = J[u]$；弧長は $x$ の平行移動で変わらないことの確認)

### §6.2 ノーターの定理の証明
**内容:** 連続的対称性から保存量を構成する定理

- **1. ノーターの定理**
  - Theorem 6.2-1 (ノーターの定理：汎関数 $J[u] = \int_a^b F(x,u,u')\,dx$ が生成子 $(\xi, \phi)$ を持つ 1 パラメータ変換群に関して不変で $u$ が E-L 方程式を満たすならば、$I = F\xi + F_{u'}(\phi - u'\xi)$ は $x$ によらない定数（保存量）；すなわち $\frac{d}{dx}I = 0$)
  - Proof (不変性の $s$ 微分：$\frac{d}{ds}J[\tilde{u}]\big|_{s=0} = 0$ を計算する；E-L 方程式を使って被積分関数を全微分の形 $\frac{d}{dx}(F\xi + F_{u'}(\phi - u'\xi))$ にまとめ、これがゼロであることから $I$ = const)
  - Remark (定理の意義：「対称性 $\Leftrightarrow$ 保存則」という物理の深い原理を数学的に精密化した定理；1918 年に Emmy Noether が証明；現代物理学の理論的基礎として不可欠な定理)

- **2. 保存量の計算手順**
  - Example 6.2-1 (エネルギー保存則の導出：$F = F(u, u')$（$x$ を陽に含まない）に $\xi = 1$、$\phi = 0$（$x$ 平行移動）；$I = F - u'F_{u'}$；これが一定値を取ることが「エネルギー（ハミルトニアン）の保存」に他ならない)
  - Example 6.2-2 (古典力学の場合の確認：$F = \frac{m}{2}u'^2 - V(u)$（$x$ を時間 $t$ として）；$I = F - u'F_{u'} = \frac{m}{2}u'^2 - V - mu'^2 = -(\frac{m}{2}u'^2 + V) = -H$；ハミルトニアン $H$ が保存されることの確認)

### §6.3 例：エネルギー・運動量・角運動量保存
**内容:** 3 つの基本保存則とそれに対応する対称性

- **1. 3 つの基本保存則**
  - Example 6.3-1 (時間並進対称性とエネルギー保存：$F$ が $t$ を陽に含まない（$F_t = 0$）場合；生成子 $(1, 0)$；ノーターの定理より $H = u'F_{u'} - F$（ハミルトニアン）が保存量；時間の一様性がエネルギー保存に対応)
  - Example 6.3-2 (空間並進対称性と運動量保存：多粒子系で全ラグランジアンが空間全体の平行移動 $\tilde{u}_k = u_k + s$ に不変な場合；各粒子の $F_{u_k'}$ の総和 $\sum_k p_k$ が保存；空間の一様性が運動量保存に対応)
  - Example 6.3-3 (回転対称性と角運動量保存：2 次元での回転 $(\tilde{x}, \tilde{y}) = (x\cos s - y\sin s, x\sin s + y\cos s)$；生成子 $\xi = (-y, x)$；ノーターの保存量が角運動量 $L = xu_y' - yu_x'$ に対応；空間の等方性が角運動量保存に対応)
  - Remark (物理の対称原理との対応：時間並進・空間並進・回転の 3 種の対称性は「時空間の一様性・等方性」という物理の基本原理から来る；場の理論ではゲージ対称性が電荷保存・色荷保存などの保存則を生む)

## Chapter 7: 等周問題と制約付き変分問題
**テーマ:** 積分型制約のもとでの汎関数の極値問題

### §7.1 等周問題の設定
**内容:** 長さ一定のもとで面積を最大化する古典問題

- **1. 等周問題の定義**
  - Definition 7.1-1 (等周問題：制約 $G[u] = \int_a^b g(x,u,u')\,dx = \ell$（定数）のもとで汎関数 $J[u] = \int_a^b F(x,u,u')\,dx$ を極値化する問題；「等しい周長のもとで面積を最大化」が名前の由来)
  - Example 7.1-1 (等周問題の古典例：周長 $\ell$ が一定の閉曲線で囲まれる面積を最大にする問題；答えは円（ディド問題）；変分法の枠組みで「曲率一定 $\Rightarrow$ 円」を導く)
  - Example 7.1-2 (懸垂線問題：両端固定・長さ一定の鎖が重力で垂れ下がる形（懸垂線）；重力ポテンシャル（重心高さ）の最小化問題；等周制約付き変分問題として定式化)

- **2. 等周不等式との関係**
  - Remark (等周不等式：$\mathbb{R}^2$ で囲んだ面積 $A$ と周長 $L$ の間には $4\pi A \leq L^2$ が成立し等号は円；変分法的証明（第一変分がゼロ $\Rightarrow$ 曲率一定 $\Rightarrow$ 円）と幾何学的証明の両方が知られる；高次元版はソボレフ不等式と密接に関係する)

### §7.2 変分問題のラグランジュ乗数法
**内容:** 制約付き変分問題の E-L 方程式の導出

- **1. ラグランジュ乗数の導入**
  - Theorem 7.2-1 (制約付き変分問題の E-L 方程式：等周制約 $G[u] = \ell$ のもとで $J[u]$ の極値を与える $u^*$ は、ある定数 $\lambda$（ラグランジュ乗数）に対して補助汎関数 $J^\lambda = J - \lambda G$ の（無制約）E-L 方程式 $F_u - \frac{d}{dx}F_{u'} - \lambda(g_u - \frac{d}{dx}g_{u'}) = 0$ を満たす)
  - Proof (証明のスケッチ：極値関数 $u^*$ での $\delta J = 0$（$G$ を保つ変分に限定）と $\delta G = 0$（制約の変分）から、$\delta J = \lambda \delta G$ が任意変分に対して成立するような $\lambda$ の存在を示す；有限次元のラグランジュ乗数法と同じ構造)
  - Example 7.2-1 (等周問題の E-L 方程式：$J[y] = \frac{1}{2}\oint y\,dx$（面積）、$G[y] = \oint \sqrt{1+y'^2}\,dx = \ell$（周長）；E-L 方程式は $-y''/\sqrt{1+y'^2}^3 = \lambda$；左辺は曲率 $\kappa$；$\kappa = \lambda$（定数）$\Rightarrow$ 円)

- **2. 具体的な等周問題の解**
  - Example 7.2-2 (懸垂線の導出：重力ポテンシャルの最小化 $J[y] = \int_a^b y\sqrt{1+y'^2}\,dx$（重心高さに比例）を制約 $G[y] = \int_a^b \sqrt{1+y'^2}\,dx = \ell$ のもとで解く；$J^\lambda = \int (y-\lambda)\sqrt{1+y'^2}\,dx$；E-L 方程式を解くと $y = \lambda + c\cosh((x-x_0)/c)$（懸垂線・カテナリー）)
  - Remark (ラグランジュ乗数の意味：$\lambda$ は「制約の緩和に伴う極値の変化率」（感度分析）；経済学では限界費用、熱力学では温度などとして解釈される；同一の数学構造が多分野に現れる)

### §7.3 複数制約の場合
**内容:** 複数の積分制約と点制約の一般化

- **1. 複数積分制約**
  - Theorem 7.3-1 (複数ラグランジュ乗数：$k$ 個の積分制約 $G_i[u] = \ell_i$（$i=1,\ldots,k$）のもとでの極値関数は $J - \sum_{i=1}^k \lambda_i G_i$ の E-L 方程式を満たす；乗数 $\lambda_i$ は制約条件から決定)
  - Proof ($k = 2$ の場合を例に：$\delta J = \lambda_1 \delta G_1 + \lambda_2 \delta G_2$ が任意の「両制約を保つ変分」に対して成立するよう $\lambda_1, \lambda_2$ を選ぶ；2 個の乗数が 2 つの制約で決まる)

- **2. 点制約と障害物問題**
  - Example 7.3-1 (端点での点制約：$u(c) = \gamma$（内点 $c$ での値制約）を持つ変分問題；ラグランジュ乗数 $\lambda$ は端点での $[F_{u'}]_{c^-}^{c^+} = \lambda$ という接続条件として現れる)
  - Remark (変分不等式と障害物問題：解が「障害物」（$u \geq \psi$）の制約を満たす場合、E-L 方程式の代わりに変分不等式 $\delta J[u;\eta-u] \geq 0$（すべての許容 $\eta$）が成立する；解が障害物に接する「自由境界」の決定が問題の核心；詳細は偏微分方程式（5-2）で扱う)

## Chapter 8: 多重積分汎関数と楕円型方程式への接続
**テーマ:** 多変数の変分問題と偏微分方程式との橋渡し

### §8.1 多変数の変分問題
**内容:** 多重積分型汎関数の設定と境界条件

- **1. 多重積分汎関数の定義**
  - Definition 8.1-1 (多重積分汎関数：$J[u] = \iint_\Omega F(x, y, u, u_x, u_y)\,dA$；$\Omega \subset \mathbb{R}^2$ は有界領域；$u : \Omega \to \mathbb{R}$ が未知関数)
  - Example 8.1-1 (ディリクレ積分：$J[u] = \frac{1}{2}\iint_\Omega (u_x^2 + u_y^2)\,dA$；弾性膜の歪みエネルギー・静電場のエネルギーに対応；最小化問題がラプラス方程式を与える（§8.2 参照）)
  - Example 8.1-2 (極小曲面の面積汎関数：$J[u] = \iint_\Omega \sqrt{1 + u_x^2 + u_y^2}\,dA$；石鹸膜のモデル；E-L 方程式が極小曲面方程式（平均曲率ゼロ）を与える)

- **2. 境界条件の設定**
  - Definition 8.1-2 (多変数の許容クラス：$\mathcal{A} = \{u \in C^2(\Omega) \cap C(\bar\Omega) \mid u = g \text{ on } \partial\Omega\}$（ディリクレ型）；変分方向 $\eta$ は $\partial\Omega$ 上でゼロ)
  - Remark (関数空間の問題：滑らかな許容クラスでは存在証明が困難；ヒルベルト空間 $H^1(\Omega)$（ソボレフ空間）での弱定式化が現代的なアプローチ（§8.3 および関数解析（4-1）で扱う）)

### §8.2 多変数オイラー＝ラグランジュ方程式
**内容:** 多重積分 E-L 方程式の導出とラプラス方程式・極小曲面方程式

- **1. 多変数 E-L 方程式の導出**
  - Theorem 8.2-1 (多変数 E-L 方程式：$J[u] = \iint_\Omega F(x,y,u,u_x,u_y)\,dA$ の極値関数は $F_u - \frac{\partial}{\partial x}F_{u_x} - \frac{\partial}{\partial y}F_{u_y} = 0$（$\Omega$ 内）を満たす)
  - Proof (導出：$\delta J[u;\eta] = 0$ を計算し $\iint_\Omega (F_u\eta + F_{u_x}\eta_x + F_{u_y}\eta_y)\,dA = 0$；ガウスの発散定理で $\partial$ を $\eta$ に移すと境界項は $\eta = 0$ でゼロ；多変数の基本補題を適用)
  - Example 8.2-1 (ディリクレ積分の E-L 方程式：$F = (u_x^2+u_y^2)/2$ のとき $F_u = 0$、$F_{u_x} = u_x$、$F_{u_y} = u_y$；E-L 方程式は $-(u_{xx}+u_{yy}) = 0$、すなわちラプラス方程式 $\Delta u = 0$；「ディリクレ積分の最小化 $\Leftrightarrow$ 調和関数」の変分的導出)

- **2. 極小曲面方程式**
  - Example 8.2-2 (極小曲面方程式の導出：面積汎関数 $F = \sqrt{1+u_x^2+u_y^2}$ の E-L 方程式；$F_{u_x} = u_x/W$（$W = \sqrt{1+u_x^2+u_y^2}$）を計算して $\frac{\partial}{\partial x}\frac{u_x}{W} + \frac{\partial}{\partial y}\frac{u_y}{W} = 0$；これは平均曲率 $H = 0$ の条件（極小曲面方程式）と等価であることの確認)
  - Remark (変分と PDE の深い関係：E-L 方程式が楕円型 PDE になることは偶然でなく、汎関数の「曲率（第二変分の符号）」が正であることと楕円性が対応している；変分問題の視点から PDE を理解することは、解の存在・一意性・正則性の証明にも有効な方針を与える)

### §8.3 弱定式化の動機
**内容:** 古典解の限界とソボレフ空間での変分的定式化の入門

- **1. 弱定式化の必要性**
  - Example 8.3-1 (古典解が存在しない例：$\Omega$ が非凸領域や L 字型領域の場合、ディリクレ問題の解が境界コーナーで $C^2$ にならない；ディリクレ積分を最小化する関数（弱解）は存在するが、古典的な E-L 方程式を点ごとに満たさないことがある)
  - Definition 8.3-1 (弱定式化：$J[u] = \iint_\Omega |\nabla u|^2/2\,dA - \iint_\Omega fu\,dA$ の臨界点としての弱解；$\delta J[u;\eta] = 0$（すべての $\eta \in H^1_0(\Omega)$）を「すべてのテスト関数に対する積分方程式」として解釈する；$\iint \nabla u \cdot \nabla \eta\,dA = \iint f\eta\,dA$)

- **2. Lax-Milgram 定理への展望**
  - Remark (存在定理のアウトライン：弱定式化を $H^1_0(\Omega)$ 上の有界双線形形式 $a(u,\eta) = \iint \nabla u\cdot\nabla\eta\,dA$ と線形汎関数 $\ell(\eta) = \iint f\eta\,dA$ として書くと、$a$ のコエルシビティ（$a(u,u) \geq c\|u\|^2$）と有界性から Lax-Milgram 定理（関数解析（4-1）の主要定理）によって一意の弱解の存在が保証される；変分法は PDE の存在論と関数解析を結ぶ橋渡しとなる)
  - Remark (本コースの位置づけ：変分法（3-6）は「汎関数の最小化問題の必要条件」を中心に扱った；弱解の存在・正則性の厳密な議論は関数解析（4-1）・偏微分方程式（5-2）で完成する；また最適制御・経済学への応用はシンプレクティック幾何学（5-17）・最適化理論（A-3）でさらに発展する)
