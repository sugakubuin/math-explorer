## Chapter 1: 楕円積分とアーベルの定理
**テーマ:** 楕円積分の起源・三種の正規形・反転問題・アーベルの加法定理

### §1.1 楕円積分の起源
**内容:** 弧長積分・振り子の周期から生じる積分、代数関数の積分が初等関数で表せない理由

- **1. 弧長積分と楕円積分**
  - Example 1.1-1 (楕円の弧長：楕円 $x = a\cos\theta$, $y = b\sin\theta$ の弧長が $\int_0^\theta \sqrt{a^2\sin^2 t + b^2\cos^2 t}\,dt$ で与えられ、$a \neq b$ のとき初等関数で表せないこと)
  - Example 1.1-2 (レムニスケートの弧長：曲線 $r^2 = \cos 2\theta$ の弧長が $\int_0^r (1-t^4)^{-1/2}\,dt$ の形であること；これがガウスの研究の出発点であった)
  - Remark (「楕円積分」の名前は楕円の弧長から来ているが、対象は楕円に限らず「$\sqrt{P(x)}$（$P$ は3次または4次多項式）を含む積分」の広いクラスである)

- **2. 振り子の周期と初等関数の限界**
  - Example 1.1-3 (単振り子の周期：振幅 $\alpha$ の単振り子の周期が $T = 4\sqrt{l/g}\int_0^{\pi/2}(1-k^2\sin^2\theta)^{-1/2}\,d\theta$（$k = \sin(\alpha/2)$）で与えられること；小振幅近似 $T \approx 2\pi\sqrt{l/g}$ との比較)
  - Remark (この積分は初等関数では表せない；リウヴィルの微分拡大理論により、$\int R(x, \sqrt{P(x)})\,dx$（$P$ が3次以上で重根なし）が一般に初等関数で閉じないことが証明される)

### §1.2 第一種・第二種楕円積分
**内容:** ルジャンドルの正規形 $K(k)$, $E(k)$ の定義、パラメータ $k$ の意味

- **1. 第一種楕円積分**
  - Definition 1.2-1 (第一種不完全楕円積分：$F(\varphi, k) = \int_0^\varphi (1-k^2\sin^2\theta)^{-1/2}\,d\theta$；母数（modulus）$k$（$0 < k < 1$）)
  - Definition 1.2-2 (第一種完全楕円積分：$K(k) = F(\pi/2, k) = \int_0^{\pi/2}(1-k^2\sin^2\theta)^{-1/2}\,d\theta$)
  - Example 1.2-1 ($k = 0$ のとき $K(0) = \pi/2$；$k \to 1$ のとき $K(k) \to \infty$ であること；$k = 1/\sqrt{2}$ での数値 $K \approx 1.854$ の確認)

- **2. 第二種楕円積分**
  - Definition 1.2-3 (第二種不完全楕円積分：$E(\varphi, k) = \int_0^\varphi (1-k^2\sin^2\theta)^{1/2}\,d\theta$)
  - Definition 1.2-4 (第二種完全楕円積分：$E(k) = E(\pi/2, k)$)
  - Example 1.2-2 ($E(k)$ が楕円の弧長を直接与えること：離心率 $e$ と $k$ の関係を通じた §1.1 との接続)
  - Remark ($K(k)$ と $E(k)$ はそれぞれ $k$ の関数として実解析的であり、$k = 0$ と $k = 1$ の端でのべき級数展開と対数的発散を持つ)

### §1.3 第三種楕円積分
**内容:** $\Pi(n, k)$ の定義と他の二種との関係

- **1. 第三種楕円積分の定義**
  - Definition 1.3-1 (第三種不完全楕円積分：$\Pi(n; \varphi, k) = \int_0^\varphi (1-n\sin^2\theta)^{-1}(1-k^2\sin^2\theta)^{-1/2}\,d\theta$；パラメータ $n$（特性）)
  - Example 1.3-1 ($n = 0$ のとき第一種楕円積分に帰着すること)
  - Remark (第三種楕円積分は被積分関数に単極を持ち、第一種・第二種とは解析的性質が異なる；複素解析的には対数特異性に対応する)

- **2. 三種の楕円積分の関係**
  - Remark (ルジャンドルの分類：任意の楕円積分 $\int R(x, \sqrt{P(x)})\,dx$（$P$ は3次または4次）は第一種・第二種・第三種の線形結合と有理関数部分に分解できること)
  - Example 1.3-2 ($\int_0^x (1-t^4)^{-1/2}\,dt$（レムニスケート積分）を変数変換により第一種楕円積分 $F(\varphi, 1/\sqrt{2})$ に帰着する計算)

### §1.4 楕円積分の反転問題
**内容:** 積分の逆関数として楕円関数を定義するという発想の導入

- **1. 反転の動機**
  - Remark ($\int_0^x (1-t^2)^{-1/2}\,dt = \arcsin x$ の逆関数が $\sin$ であるように、楕円積分の逆関数として新しい関数を定義するのがガウス、アーベル、ヤコビのアイデアである)
  - Example 1.4-1 ($u = \int_0^x (1-t^2)^{-1/2}(1-k^2t^2)^{-1/2}\,dt$ の逆関数 $x = \text{sn}(u, k)$ の導入；$k = 0$ のとき $\text{sn}(u, 0) = \sin u$ であること)

- **2. 反転による二重周期性の発見**
  - Remark (楕円積分を実数上で反転すると周期関数が得られるが、複素数に拡張すると二重周期性が現れる；これがガウスとアーベルの発見であった)
  - Example 1.4-2 (レムニスケート積分の反転：$u = \int_0^x (1-t^4)^{-1/2}\,dt$ の逆関数がレムニスケート正弦関数 $\text{sl}(u)$ であり、2つの独立な周期 $\omega$ と $i\omega$（$\omega$ はレムニスケート定数）を持つことの紹介)

### §1.5 アーベルの加法定理
**内容:** 楕円積分の和が再び楕円積分で表されるという定理の主張

- **1. アーベルの加法定理の主張**
  - Theorem 1.5-1 (アーベルの加法定理：第一種楕円積分に対して $F(\varphi_1, k) + F(\varphi_2, k) = F(\varphi_3, k)$ を満たす $\varphi_3$ が $\varphi_1$ と $\varphi_2$ の代数的な関係式で定まること)
  - Proof (加法定理の証明の概略：代数曲線 $y^2 = (1-x^2)(1-k^2x^2)$ 上の有理型微分形式 $dx/y$ の積分として定式化し、曲線上の3点の共線条件から導出する)
  - Example 1.5-1 ($k = 0$ の場合：$\arcsin x_1 + \arcsin x_2 = \arcsin x_3$ が $x_3 = x_1\sqrt{1-x_2^2} + x_2\sqrt{1-x_1^2}$（三角関数の加法定理）であることの確認)

- **2. 楕円曲線の群法則との対応**
  - Remark (アーベルの加法定理は代数曲線上の点の加法構造を反映しており、第5章で℘関数を通じて楕円曲線の群法則として再定式化される)
  - Example 1.5-2 (加法公式の対称性：$F(\varphi_1, k) + F(\varphi_2, k)$ が $\varphi_1$ と $\varphi_2$ に対して対称であり、この対称性が群法則の可換性に対応すること)

## Chapter 2: 複素トーラスと二重周期関数
**テーマ:** 格子・商空間としてのトーラス・二重周期関数の基本的制約

### §2.1 格子と基本周期平行四辺形
**内容:** 格子 $\Lambda = \omega_1\mathbb{Z} + \omega_2\mathbb{Z}$ の定義、$\text{Im}(\omega_2/\omega_1) > 0$ の規格化条件、基本周期平行四辺形の構成

- **1. 格子の定義**
  - Definition 2.1-1 (格子：$\omega_1, \omega_2 \in \mathbb{C}$（$\omega_2/\omega_1 \notin \mathbb{R}$）に対して $\Lambda = \{m\omega_1 + n\omega_2 \mid m, n \in \mathbb{Z}\}$)
  - Definition 2.1-2 (規格化条件：基底の選び方の自由度を減らすため $\text{Im}(\omega_2/\omega_1) > 0$ と約束する；$\tau = \omega_2/\omega_1$ を格子のモジュラスと呼ぶ)
  - Example 2.1-1 ($\omega_1 = 1$, $\omega_2 = i$ のガウス格子 $\mathbb{Z} + i\mathbb{Z}$；$\tau = i$)
  - Example 2.1-2 ($\omega_1 = 1$, $\omega_2 = e^{2\pi i/3}$ のアイゼンシュタイン格子；$\tau = e^{2\pi i/3}$)

- **2. 基本周期平行四辺形**
  - Definition 2.1-3 (基本周期平行四辺形：$\Pi = \{s\omega_1 + t\omega_2 \mid 0 \leq s, t < 1\}$)
  - Remark (基本周期平行四辺形は $\mathbb{C}/\Lambda$ の基本領域であり、楕円関数の性質は $\Pi$ 上での振る舞いから決定される)
  - Example 2.1-3 (ガウス格子の基本周期平行四辺形は正方形であること；アイゼンシュタイン格子では菱形であること)

### §2.2 複素トーラス
**内容:** 商空間 $\mathbb{C}/\Lambda$ の定義、種数1のコンパクトリーマン面としての解釈

- **1. 商空間の構成**
  - Definition 2.2-1 (複素トーラス：商空間 $\mathbb{C}/\Lambda = \{z + \Lambda \mid z \in \mathbb{C}\}$ に自然な複素構造を入れたもの)
  - Proposition 2.2-1 ($\mathbb{C}/\Lambda$ はコンパクトなリーマン面であること)
  - Proof (コンパクト性は基本周期平行四辺形の閉包が有界閉集合であることから；リーマン面構造は射影 $\pi : \mathbb{C} \to \mathbb{C}/\Lambda$ の局所同相性から座標チャートを構成する)
  - Example 2.2-1 (基本周期平行四辺形の対辺を同一視する操作でトーラスが得られることの位相的直感)

- **2. 種数1のリーマン面**
  - Proposition 2.2-2 ($\mathbb{C}/\Lambda$ の種数は $g = 1$ であること)
  - Proof (オイラー標数 $\chi = 0$ を基本周期平行四辺形の辺の同一視から計算する)
  - Remark ($\mathbb{C}/\Lambda$ は一様化定理（4-2 §5.5）における $\mathbb{C}$-型の単連結被覆を持つ種数1リーマン面の典型例であり、逆にすべての種数1コンパクトリーマン面は複素トーラスとして実現される)

### §2.3 二重周期関数の定義
**内容:** $\Lambda$-周期的な有理型関数の定義

- **1. 楕円関数の定義**
  - Definition 2.3-1 (楕円関数：$\mathbb{C}$ 上の有理型関数 $f$ で $f(z + \omega) = f(z)$（$\forall \omega \in \Lambda$）を満たすもの；すなわち複素トーラス $\mathbb{C}/\Lambda$ 上の有理型関数)
  - Example 2.3-1 (定数関数は自明な楕円関数であること；$e^z$ は二重周期性を持たないため楕円関数でないこと)
  - Remark (楕円関数は $\mathbb{C}/\Lambda$ 上の有理型関数であり、リーマン面上の有理型関数の理論（4-2 §3.3, §3.5）が適用される)

- **2. 楕円関数の位数**
  - Definition 2.3-2 (楕円関数の位数：基本周期平行四辺形内の極の位数の合計（重複度込み）)
  - Remark (位数は極の個数の測り方であり、後のリウヴィルの定理系列（§2.4）で重要な役割を果たす)

### §2.4 リウヴィルの定理系列
**内容:** 整関数である二重周期関数は定数であること、極の個数・留数の和・零点と極の位置に関する基本制約の証明

- **1. リウヴィルの基本定理**
  - Theorem 2.4-1 (リウヴィルの第一定理：整関数（極を持たない正則関数）である楕円関数は定数に限ること)
  - Proof (リウヴィルの第一定理の証明：二重周期性により基本周期平行四辺形の閉包上で有界であり、リューヴィルの定理（3-2）から定数)
  - Example 2.4-1 (楕円関数の位数は $0$（定数）または $2$ 以上であること：位数1の楕円関数は存在しないことの確認)

- **2. 極と零点に関する制約**
  - Theorem 2.4-2 (留数の和が零：楕円関数の基本周期平行四辺形内の留数の総和は $0$ であること)
  - Proof (留数の和の証明：基本周期平行四辺形の境界に沿った積分が、周期性による対辺の打ち消しから $0$ になること)
  - Theorem 2.4-3 (零点と極の個数の一致：楕円関数の基本周期平行四辺形内の零点の個数と極の個数が等しいこと（重複度込み）)
  - Proof (個数の一致の証明：$f'/f$ を基本周期平行四辺形の境界に沿って積分し、偏角の原理（3-2）を適用する)
  - Theorem 2.4-4 (零点と極の位置の制約：零点の位置の和と極の位置の和が $\Lambda$ を法として等しいこと)
  - Proof (位置の制約の証明：$zf'(z)/f(z)$ を基本周期平行四辺形の境界に沿って積分し、擬周期的な寄与を評価する)
  - Example 2.4-2 (位数2の楕円関数は2つの零点と2つの極（または2位の極1つ）を持ち、零点の和 $\equiv$ 極の和 $\pmod{\Lambda}$ であること)

## Chapter 3: ワイエルシュトラスの $\wp$ 関数
**テーマ:** 定義と収束・微分方程式・Laurent 展開と不変量

### §3.1 $\wp$ 関数の定義と収束
**内容:** 絶対収束を保証する補正項付きの無限和の定義、一様収束の証明

- **1. $\wp$ 関数の定義**
  - Definition 3.1-1 (ワイエルシュトラスの $\wp$ 関数：$\wp(z) = 1/z^2 + \sum_{\omega \in \Lambda \setminus \{0\}} (1/(z-\omega)^2 - 1/\omega^2)$)
  - Remark (補正項 $-1/\omega^2$ は各項の $z \to \infty$ での振る舞いを打ち消し、級数の絶対収束を保証するために必要である)
  - Proposition 3.1-1 ($\wp$ 関数の定義級数は $\Lambda$ の格子点を除く $\mathbb{C}$ の任意のコンパクト集合上で絶対かつ一様に収束すること)
  - Proof (収束の証明：$\sum_{\omega \neq 0} |\omega|^{-3}$ の収束を格子の点の個数評価 $|\{\omega \mid |\omega| \leq R\}| = O(R^2)$ から示し、$|1/(z-\omega)^2 - 1/\omega^2| = O(|\omega|^{-3})$ を得る)

- **2. 収束の検証**
  - Example 3.1-1 (ガウス格子 $\Lambda = \mathbb{Z} + i\mathbb{Z}$ の場合：$\sum_{(m,n) \neq (0,0)} (m^2 + n^2)^{-3/2}$ の収束の確認)
  - Example 3.1-2 ($\sum_{\omega \neq 0} |\omega|^{-s}$ が $s > 2$ のとき収束し $s \leq 2$ のとき発散すること；$s = 2$ の発散が補正項の必要性を裏付けること)

### §3.2 $\wp$ 関数の基本性質
**内容:** 偶関数性・二重周期性・唯一の二重極の確認

- **1. 偶関数性と二重周期性**
  - Proposition 3.2-1 ($\wp(-z) = \wp(z)$（偶関数性）)
  - Proof (偶関数性の証明：定義式で $z \mapsto -z$ と置換し、$\omega \mapsto -\omega$ の再ラベルで元の和に一致する)
  - Theorem 3.2-1 ($\wp(z + \omega_i) = \wp(z)$（$i = 1, 2$）（二重周期性）)
  - Proof (二重周期性の証明：$\wp'(z) = -2\sum_\omega (z-\omega)^{-3}$ が明らかに $\Lambda$-周期的であること；$\wp(z + \omega_i) - \wp(z)$ は微分が $0$ の周期関数であるから定数であり、$z = -\omega_i/2$ を代入すると偶関数性から $0$ を得る)
  - Example 3.2-1 ($\wp$ は各格子点に2位の極を持ち、基本周期平行四辺形内に極はこの1つだけであること；位数2の楕円関数であることの確認)

- **2. $\wp'$ の基本性質**
  - Proposition 3.2-2 ($\wp'(z)$ は奇関数で位数3の楕円関数であること)
  - Proof ($\wp'(-z) = -\wp'(z)$ は偶関数性の微分から；$\wp'$ は各格子点に3位の極を持つ)
  - Example 3.2-2 ($\wp'(z) = 0$ の解が半周期点 $\omega_1/2$, $\omega_2/2$, $(\omega_1+\omega_2)/2$ であること（リウヴィルの定理系列から位数3なので3つの零点を持つ）)

### §3.3 $\wp$ 関数の Laurent 展開と不変量
**内容:** 原点近傍での展開、アイゼンシュタイン級数 $g_2$, $g_3$ の定義

- **1. Laurent 展開**
  - Theorem 3.3-1 ($\wp(z)$ の原点近傍での Laurent 展開：$\wp(z) = z^{-2} + \sum_{n=1}^\infty (2n+1)G_{2n+2}z^{2n}$；ここで $G_{2k} = \sum_{\omega \neq 0} \omega^{-2k}$（アイゼンシュタイン級数）)
  - Proof (Laurent 展開の導出：$1/(z-\omega)^2 - 1/\omega^2$ を $z/\omega$ のべき級数に展開し、格子上で和を取る)
  - Example 3.3-1 (最初の数項の計算：$\wp(z) = z^{-2} + 3G_4 z^2 + 5G_6 z^4 + \cdots$)

- **2. 不変量 $g_2$, $g_3$ とアイゼンシュタイン級数**
  - Definition 3.3-1 (不変量：$g_2 = 60G_4 = 60\sum_{\omega \neq 0} \omega^{-4}$；$g_3 = 140G_6 = 140\sum_{\omega \neq 0} \omega^{-6}$)
  - Example 3.3-2 (ガウス格子 $\Lambda = \mathbb{Z} + i\mathbb{Z}$ に対する $G_4$ の値：対称性 $\omega \mapsto i\omega$ により $G_6 = 0$（$g_3 = 0$）となること)
  - Remark (一般のアイゼンシュタイン級数 $G_{2k}$ は $k \geq 2$ のとき絶対収束するが、$G_2 = \sum \omega^{-2}$ は条件付き収束であり特別な扱いが必要である；$G_2$ の変換性質はモジュラー形式（5-7）で重要となる)

### §3.4 $\wp$ 関数の満たす微分方程式
**内容:** $(\wp')^2 = 4\wp^3 - g_2\wp - g_3$ の証明

- **1. 微分方程式の導出**
  - Theorem 3.4-1 ($\wp$ 関数は微分方程式 $(\wp'(z))^2 = 4\wp(z)^3 - g_2\wp(z) - g_3$ を満たすこと)
  - Proof (微分方程式の証明：$f(z) = (\wp')^2 - 4\wp^3 + g_2\wp + g_3$ の Laurent 展開を計算し、$z = 0$ 近傍で $f(z) = O(z^2)$ であることから $f$ は正則な楕円関数、すなわちリウヴィルの定理により定数であり、$z \to 0$ で $0$ に収束するから $f \equiv 0$)
  - Example 3.4-1 (微分方程式の検証：Laurent 展開 $\wp(z) = z^{-2} + 3G_4 z^2 + \cdots$ と $\wp'(z) = -2z^{-3} + 6G_4 z + \cdots$ を代入して $(\wp')^2$ と $4\wp^3 - g_2\wp - g_3$ の主要項が一致することの確認)

- **2. 半周期点での値**
  - Definition 3.4-1 ($e_1 = \wp(\omega_1/2)$, $e_2 = \wp(\omega_2/2)$, $e_3 = \wp((\omega_1+\omega_2)/2)$)
  - Proposition 3.4-1 ($e_1, e_2, e_3$ は相異なり、$4\wp^3 - g_2\wp - g_3 = 4(\wp - e_1)(\wp - e_2)(\wp - e_3)$ であること)
  - Proof (半周期点は $\wp'$ の零点（§3.2）であるから $4e_i^3 - g_2 e_i - g_3 = 0$；リウヴィルの定理から $e_1, e_2, e_3$ は相異なる)
  - Example 3.4-2 (ヴィエトの関係式：$e_1 + e_2 + e_3 = 0$, $e_1 e_2 + e_2 e_3 + e_3 e_1 = -g_2/4$, $e_1 e_2 e_3 = g_3/4$)

### §3.5 判別式と $j$ 不変量
**内容:** $\Delta = g_2^3 - 27g_3^2$ の定義、$j = 1728\,g_2^3/\Delta$ の定義と格子の同型類との対応

- **1. 判別式**
  - Definition 3.5-1 (判別式：$\Delta = g_2^3 - 27g_3^2$)
  - Proposition 3.5-1 ($\Delta \neq 0$ であること)
  - Proof ($\Delta = 16(e_1-e_2)^2(e_2-e_3)^2(e_3-e_1)^2$ であり、$e_1, e_2, e_3$ が相異なるから $\Delta \neq 0$)
  - Example 3.5-1 (ガウス格子：$g_3 = 0$ より $\Delta = g_2^3 \neq 0$；$g_2 \neq 0$ は直接計算で確認できる)

- **2. $j$ 不変量**
  - Definition 3.5-2 ($j$ 不変量：$j(\Lambda) = 1728\,g_2^3/\Delta$)
  - Proposition 3.5-2 ($j$ 不変量は格子のスカラー倍に関して不変であること：$j(\alpha\Lambda) = j(\Lambda)$（$\alpha \in \mathbb{C}^*$）)
  - Proof (スカラー不変性の証明：$g_2(\alpha\Lambda) = \alpha^{-4}g_2(\Lambda)$, $g_3(\alpha\Lambda) = \alpha^{-6}g_3(\Lambda)$ から $j$ のスケーリングが打ち消されること)
  - Example 3.5-2 (ガウス格子：$g_3 = 0$ より $j = 1728$；アイゼンシュタイン格子：$g_2 = 0$ より $j = 0$)
  - Remark ($j$ 不変量は格子の相似類（= 楕円曲線の同型類）を完全に分類する；この分類は §5.4 で詳しく扱う)

### §3.6 任意の楕円関数の表示
**内容:** 任意の楕円関数が $\wp$ と $\wp'$ の有理式で表されることの証明

- **1. 偶関数の場合**
  - Theorem 3.6-1 (格子 $\Lambda$ に関する任意の楕円関数は $\wp$ と $\wp'$ の $\mathbb{C}$ 上の有理式として表されること)
  - Proof (表示定理の証明の方針：まず偶楕円関数が $\wp$ の有理式であることを示し（Lemma 3.6-1）、次に一般の場合を偶・奇分解で帰着する；詳細は以下)
  - Lemma 3.6-1 (偶楕円関数は $\wp$ の有理式として表されること)
  - Proof (偶楕円関数の表示の証明：偶楕円関数 $f$ の零点・極を $\wp(z) - \wp(a)$ の形の因子で分解し、残りがリウヴィルの定理から定数であることを示す)
  - Example 3.6-1 ($\wp(2z)$ を $\wp(z)$ の有理式として表す計算の概略)

- **2. 一般の場合**
  - Proof (一般の楕円関数の表示：任意の $f$ を偶部分 $(f(z)+f(-z))/2$ と奇部分 $(f(z)-f(-z))/2$ に分解し、奇部分を $\wp'$ で割ると偶楕円関数になることを利用する)
  - Example 3.6-2 ($\wp'(z)$ 自身が $\wp$ と $\wp'$ の「有理式」（$\wp'$ そのもの）であることの自明な確認)
  - Remark (この表示定理は $\mathbb{C}/\Lambda$ 上の有理型関数体が $\mathbb{C}(\wp, \wp')$ と一致し、微分方程式 $(\wp')^2 = 4\wp^3 - g_2\wp - g_3$ による1つの代数的関係式で結ばれる2次拡大体であることを意味する)

## Chapter 4: ワイエルシュトラスの $\zeta$ 関数と $\sigma$ 関数
**テーマ:** 定義・擬周期性・加法公式・楕円関数の因子による表示

### §4.1 ワイエルシュトラスの $\zeta$ 関数
**内容:** $\wp$ の原始関数としての $\zeta$ の定義、擬周期性 $\zeta(z+\omega_i) = \zeta(z) + \eta_i$ の導出

- **1. $\zeta$ 関数の定義**
  - Definition 4.1-1 (ワイエルシュトラスの $\zeta$ 関数：$\zeta(z) = 1/z + \sum_{\omega \neq 0}(1/(z-\omega) + 1/\omega + z/\omega^2)$；$\zeta'(z) = -\wp(z)$)
  - Proposition 4.1-1 ($\zeta$ 関数は $\mathbb{C} \setminus \Lambda$ 上で正則であり、各格子点に1位の極を持つこと)
  - Proof (正則性の証明：§3.1 と同様の収束評価を $|\omega|^{-3}$ のオーダーで行う)
  - Remark ($\zeta$ は $\wp$ の原始関数であるが、周期性は失われる：$\zeta$ は楕円関数ではない)

- **2. 擬周期性**
  - Theorem 4.1-1 ($\zeta(z + \omega_i) = \zeta(z) + \eta_i$（$i = 1, 2$）；ここで $\eta_i = 2\zeta(\omega_i/2)$ は「擬周期」と呼ばれる定数)
  - Proof (擬周期性の証明：$\zeta(z+\omega_i) - \zeta(z)$ を微分すると $-\wp(z+\omega_i) + \wp(z) = 0$ であるから定数；$z = -\omega_i/2$ で評価して $\eta_i$ を決定する)
  - Example 4.1-1 (リウヴィルの第二定理（§2.4 Theorem 2.4-2）の $\zeta$ による再解釈：楕円関数の留数の和が $0$ であることが、$\zeta$ の擬周期性の帰結として得られること)

### §4.2 ルジャンドルの関係式
**内容:** $\eta_1\omega_2 - \eta_2\omega_1 = 2\pi i$ の証明

- **1. ルジャンドルの関係式**
  - Theorem 4.2-1 (ルジャンドルの関係式：$\eta_1\omega_2 - \eta_2\omega_1 = 2\pi i$)
  - Proof (ルジャンドルの関係式の証明：$\zeta(z)$ を基本周期平行四辺形の境界に沿って積分する；留数定理から $\oint \zeta(z)\,dz = 2\pi i$（1位の極の留数 $1$）であり、対辺の寄与を擬周期性で計算して等式を得る)
  - Example 4.2-1 (ガウス格子 $\omega_1 = 1$, $\omega_2 = i$：対称性から $\eta_2 = i\eta_1$ であり、ルジャンドルの関係式から $\eta_1 \cdot i - i\eta_1 \cdot 1 = 2\pi i$ は自動的には決まらないが、$\eta_1(i - i) = 0$ ではなく正しくは $\eta_1 \cdot i - \eta_2 \cdot 1 = 2\pi i$ と読むべきこと)

- **2. ルジャンドルの関係式の意味**
  - Remark (ルジャンドルの関係式は周期と擬周期の間の超越的な関係であり、楕円関数論における「周期関係式」の最も基本的なものである)
  - Example 4.2-2 (擬周期 $\eta_1, \eta_2$ が格子パラメータで一意に定まることの確認)

### §4.3 ワイエルシュトラスの $\sigma$ 関数
**内容:** 整関数 $\sigma$ の定義、擬周期因子の構造

- **1. $\sigma$ 関数の定義**
  - Definition 4.3-1 (ワイエルシュトラスの $\sigma$ 関数：$\sigma(z) = z\prod_{\omega \neq 0}(1 - z/\omega)\exp(z/\omega + z^2/(2\omega^2))$；$\sigma'/\sigma = \zeta$)
  - Proposition 4.3-1 ($\sigma$ は $\mathbb{C}$ 上の整関数であり、零点がちょうど $\Lambda$ の格子点（すべて1位の零点）であること)
  - Proof (整関数性の証明：ワイエルシュトラスの分解定理（4-2 §6.2）の応用として収束を確認する)
  - Example 4.3-1 ($\sigma$ のべき級数展開の最初の数項：$\sigma(z) = z - z^5 g_2/240 - z^7 g_3/840 + \cdots$)

- **2. 擬周期因子**
  - Theorem 4.3-1 ($\sigma(z + \omega_i) = -e^{\eta_i(z + \omega_i/2)}\sigma(z)$（$i = 1, 2$）)
  - Proof (擬周期因子の証明：$\sigma(z+\omega_i)/\sigma(z)$ の対数微分が $\zeta$ の擬周期性から $\eta_i$ であることを用い、積分して指数関数の形を得る；符号は零点の位置から決定する)
  - Remark ($\sigma$ は楕円関数ではないが、$\sigma$ の比は楕円関数を与えうる；これが次節の基礎となる)

### §4.4 楕円関数の $\sigma$ 関数による表示
**内容:** 零点・極のデータから楕円関数を $\sigma$ 関数を用いて構成する方法

- **1. $\sigma$ 関数による因数分解**
  - Theorem 4.4-1 (任意の楕円関数 $f$ は $f(z) = C \cdot \prod_{i=1}^n \sigma(z-a_i)/\prod_{j=1}^n \sigma(z-b_j)$ と表されること（$\{a_i\}$ は零点、$\{b_j\}$ は極；$\sum a_i \equiv \sum b_j \pmod{\Lambda}$）)
  - Proof (因数分解の証明：右辺の $\sigma$ 比が $\sum a_i \equiv \sum b_j$ の条件のもとで二重周期性を持つことを擬周期因子の計算で確認する)
  - Example 4.4-1 ($\wp(z) - \wp(a)$（$a \notin \Lambda$）の $\sigma$ による表示：零点 $z = a, -a$ と2位の極 $z = 0$ から $\wp(z) - \wp(a) = -\sigma(z-a)\sigma(z+a)/\sigma(z)^2$（定数倍を除く）)

- **2. $\sigma$ 関数の応用**
  - Example 4.4-2 ($\wp'(z)$ の $\sigma$ 関数による表示の導出)
  - Remark ($\sigma$ 関数による表示はテータ関数（第6章）による表示と並ぶ楕円関数の基本的な構成法であり、両者の関係は §6.5 で統合される)

### §5.1 楕円曲線 $y^2 = 4x^3 - g_2 x - g_3$
**内容:** ワイエルシュトラス形式の楕円曲線の定義、非特異性と $\Delta \neq 0$ の関係

- **1. ワイエルシュトラス形式の楕円曲線**
  - Definition 5.1-1 (ワイエルシュトラス形式の楕円曲線：$E : y^2 = 4x^3 - g_2 x - g_3$（$g_2, g_3 \in \mathbb{C}$, $\Delta = g_2^3 - 27g_3^2 \neq 0$）；射影的な定義として無限遠点 $O = [0:1:0]$ を含める)
  - Proposition 5.1-1 ($\Delta \neq 0$ は曲線 $E$ が非特異であるための必要十分条件であること)
  - Proof (非特異性の証明：$F(x,y) = y^2 - 4x^3 + g_2 x + g_3$ の偏微分 $F_x = -12x^2 + g_2$, $F_y = 2y$ が同時に零になる点が存在しない条件が $\Delta \neq 0$)
  - Example 5.1-1 ($g_2 = 4$, $g_3 = 0$ の場合：$y^2 = 4x^3 - 4x = 4x(x-1)(x+1)$；$\Delta = 64 \neq 0$ で非特異)

- **2. 特異点の例**
  - Example 5.1-2 ($\Delta = 0$ の場合：$y^2 = 4x^3$（尖点）と $y^2 = 4x^2(x-1)$（結節点）の特異性の確認)
  - Remark (楕円曲線は $\Delta \neq 0$ の条件により滑らかな種数1の代数曲線であり、§3.5 で示した $\Delta \neq 0$ と整合する)

### §5.2 パラメトライゼーション定理
**内容:** 写像 $z \mapsto (\wp(z), \wp'(z))$ による $\mathbb{C}/\Lambda$ から楕円曲線への同型の証明

- **1. パラメトライゼーション**
  - Theorem 5.2-1 (写像 $\Phi : \mathbb{C}/\Lambda \to E$, $\Phi(z) = (\wp(z), \wp'(z))$（$z \notin \Lambda$）, $\Phi(0) = O$ が複素多様体としての同型を与えること)
  - Proof (同型の証明：$\Phi$ が全射であること（$\wp$ が任意の値を取ること）、単射であること（$\wp(z_1) = \wp(z_2)$ かつ $\wp'(z_1) = \wp'(z_2)$ ならば $z_1 \equiv z_2 \pmod{\Lambda}$）、正則であることを確認する)
  - Example 5.2-1 ($z = \omega_1/2$ での像：$\Phi(\omega_1/2) = (e_1, 0)$；半周期点の像が $y = 0$ 上の3点 $(e_1, 0)$, $(e_2, 0)$, $(e_3, 0)$ であること)

- **2. リーマン-ロッホの定理との整合性**
  - Remark (パラメトライゼーション定理は、種数1のコンパクトリーマン面 $\mathbb{C}/\Lambda$ をリーマン-ロッホの定理（4-2 §8.5）により $\mathbb{CP}^2$ に埋め込んだものが楕円曲線であることの具体的な実現である)

### §5.3 群法則の解析的記述
**内容:** $\mathbb{C}/\Lambda$ の加法群構造と楕円曲線の3点共線条件による群法則の一致、$\wp$ 関数の加法公式

- **1. 3点共線条件による群法則**
  - Definition 5.3-1 (楕円曲線上の加法：$P + Q + R = O$ の条件を「$P, Q, R$ が一直線上にある」（3次曲線と直線の交点として）ことで定めること)
  - Theorem 5.3-1 (パラメトライゼーション $\Phi$ のもとで、$\mathbb{C}/\Lambda$ の加法と楕円曲線の群法則が一致すること)
  - Proof (群法則の一致の証明：直線 $y = ax + b$ と $E$ の交点 $P_1, P_2, P_3$ に対応する $z_1, z_2, z_3$ が $z_1 + z_2 + z_3 \equiv 0 \pmod{\Lambda}$ を満たすことを §2.4 Theorem 2.4-4（零点と極の位置の制約）から導く)
  - Example 5.3-1 ($P = (\wp(a), \wp'(a))$, $Q = (\wp(b), \wp'(b))$ の和の計算：$P$ と $Q$ を結ぶ直線と $E$ の第3の交点 $R$ を求め、$R$ の $x$ 軸に関する反転が $P + Q$)

- **2. $\wp$ 関数の加法公式**

  - Proof (加法公式の証明：直線の傾き $\lambda = (\wp'(z)-\wp'(w))/(\wp(z)-\wp(w))$ を用い、3次方程式の根と係数の関係から第3の交点の $x$ 座標を求める)
  - Example 5.3-2 (倍加公式：$\wp(2z) = -2\wp(z) + \frac{1}{4}\left(\frac{\wp''(z)}{\wp'(z)}\right)^2$ の導出；$\wp'' = 6\wp^2 - g_2/2$ を代入して $\wp$ のみの有理式を得ること)
  - Remark (加法公式は第1章のアーベルの加法定理（§1.5）の $\wp$ 関数を通じた精密な実現であり、楕円曲線の群法則の計算公式を与える)

### §5.4 複素トーラスの同型分類
**内容:** 格子の相似類と楕円曲線の同型類の対応、$j$ 不変量による完全分類

- **1. 格子の相似と $j$ 不変量**
  - Definition 5.4-1 (格子の相似：$\Lambda \sim \Lambda'$ とは $\Lambda' = \alpha\Lambda$（$\alpha \in \mathbb{C}^*$）を満たすスカラーが存在すること)
  - Theorem 5.4-1 ($j$ 不変量による完全分類：$\mathbb{C}/\Lambda \cong \mathbb{C}/\Lambda'$（複素多様体として同型）と $j(\Lambda) = j(\Lambda')$ が同値であること)
  - Proof (分類定理の証明の概略：同型 $\mathbb{C}/\Lambda \to \mathbb{C}/\Lambda'$ がリフトして $\mathbb{C}$ の自己同型 $z \mapsto \alpha z$ を与え、$\Lambda' = \alpha\Lambda$ となること；$j$ の相似不変性（§3.5）と $j$ の全射性を組み合わせる)
  - Example 5.4-1 (ガウス格子（$j = 1728$）とアイゼンシュタイン格子（$j = 0$）が同型でないことの確認)

### §5.5 等分点とねじれ構造
**内容:** $n$-等分点 $E[n]$ の定義、$E[n] \cong (\mathbb{Z}/n\mathbb{Z})^2$ の証明、$n$-等分体の定義と整数論への接続の予告

- **1. $n$-等分点**
  - Definition 5.5-1 ($n$-等分点：$E[n] = \{P \in E \mid nP = O\} = \{z \in \mathbb{C}/\Lambda \mid nz \in \Lambda\}$)
  - Theorem 5.5-1 ($E[n] \cong (\mathbb{Z}/n\mathbb{Z})^2$（群として同型）)

  - Example 5.5-2 ($E[3]$ の $9$ 元の計算：$\wp(z) = \wp(\omega_i/3)$ を満たす点の $\wp$ 値は $\wp$ の3-等分方程式（$\wp$ に関する4次方程式）の根として得られること)

- **2. $n$-等分体と整数論への接続**
  - Definition 5.5-2 ($n$-等分体：$E[n]$ の座標を $\mathbb{Q}$ に添加して得られる体拡大)
  - Example 5.5-3 ($n = 2$ の場合：$\mathbb{Q}(e_1, e_2, e_3)$ は $4x^3 - g_2 x - g_3 = 0$ の分解体であること)
  - Remark ($n$-等分点の座標が生成する体拡大のガロア群は $\text{GL}_2(\mathbb{Z}/n\mathbb{Z})$ の部分群であり、ガロア表現の理論を通じて整数論と深く結びつく；詳細は整数論II（4-14）で扱う)

### §7.1 ヤコビの楕円関数の定義
**内容:** 第一種楕円積分の逆関数としての sn の定義、cn, dn のテータ関数による定義

- **1. sn 関数の定義**
  - Definition 7.1-1 (ヤコビの sn 関数：$u = \int_0^x (1-t^2)^{-1/2}(1-k^2t^2)^{-1/2}\,dt$ の逆関数 $x = \text{sn}(u, k)$；これは §1.4 の反転問題の解である)

- **2. cn, dn の定義とテータ関数表示**
  - Definition 7.1-2 ($\text{cn}(u, k) = \sqrt{1 - \text{sn}^2(u,k)}$, $\text{dn}(u, k) = \sqrt{1 - k^2\text{sn}^2(u,k)}$；符号は $u = 0$ での値 $\text{cn}(0) = 1$, $\text{dn}(0) = 1$ で決定)
  - Proposition 7.1-1 (テータ関数による表示：$\text{sn}(u) = (\theta_3(0)/\theta_2(0)) \cdot \theta_1(v)/\theta_4(v)$（$v = \pi u/(2K)$）など)

  - Example 7.1-1 ($k = 1/\sqrt{2}$ でのレムニスケート関数との関係：$\text{sl}(u) = \text{sn}(u, 1/\sqrt{2})$ であること)

**内容:** 周期性・加法公式・相互関係式

- **1. 周期性**
  - Proposition 7.2-1 ($\text{sn}$ の周期：実周期 $4K$ と虚周期 $2iK'$（$K = K(k)$, $K' = K(k')$, $k' = \sqrt{1-k^2}$）)
  - Proof (周期性の証明の概略：楕円積分の定義から $\text{sn}(u+2K) = -\text{sn}(u)$, $\text{sn}(u+4K) = \text{sn}(u)$ を導出する)

### §8.1 モジュラー群 $\text{SL}_2(\mathbb{Z})$ の作用
**内容:** 格子の基底変換としての $\text{SL}_2(\mathbb{Z})$ 作用、基本領域の構成

- **1. モジュラー群の作用**
  - Definition 8.1-1 (上半平面 $\mathbb{H}$ へのモジュラー群の作用：$\begin{pmatrix} a & b \\ c & d \end{pmatrix} \cdot \tau = (a\tau+b)/(c\tau+d)$)
  - Remark (この作用は格子 $\Lambda = \mathbb{Z} + \tau\mathbb{Z}$ の基底変換に対応する；$\tau$ と $\tau'$ が同じ $\text{SL}_2(\mathbb{Z})$-軌道に属することと $\mathbb{C}/(\mathbb{Z}+\tau\mathbb{Z}) \cong \mathbb{C}/(\mathbb{Z}+\tau'\mathbb{Z})$ が同値であること)
  - Example 8.1-1 (生成元 $T : \tau \mapsto \tau+1$ と $S : \tau \mapsto -1/\tau$ の意味：$T$ は格子の基底の「ずらし」、$S$ は「反転」)

- **2. 基本領域**
  - Definition 8.1-2 (基本領域：$\mathcal{F} = \{\tau \in \mathbb{H} \mid |\text{Re}(\tau)| \leq 1/2, |\tau| \geq 1\}$)
  - Proposition 8.1-1 (任意の $\tau \in \mathbb{H}$ は $\text{SL}_2(\mathbb{Z})$ の作用で $\mathcal{F}$ 内の点に移せること)
  - Proof (証明の概略：$\text{Im}((a\tau+b)/(c\tau+d)) = \text{Im}(\tau)/|c\tau+d|^2$ の最大化)
  - Remark (基本領域はモジュラー形式（5-7）の理論の出発点であり、$\mathbb{H}/\text{SL}_2(\mathbb{Z})$ が楕円曲線のモジュライ空間を与える)

**内容:** $j$ が $\text{SL}_2(\mathbb{Z})$ に関して不変な有理型関数であること

- **1. $j$ 関数の性質**
  - Proposition 8.2-1 ($j(\tau)$ は $\text{SL}_2(\mathbb{Z})$ に関して不変であること：$j((a\tau+b)/(c\tau+d)) = j(\tau)$)

  - Proposition 8.2-2 ($j : \mathbb{H}/\text{SL}_2(\mathbb{Z}) \to \mathbb{C}$ は全単射であること)
  - Proof (全射性は §5.4 から；単射性は $j$ 不変量による分類から)
  - Example 8.2-1 ($j$ の $q$ 展開：$j(\tau) = q^{-2} + 744 + 196884q^2 + \cdots$（$q = e^{i\pi\tau}$）；係数 $196884$ はモンスター群の最小非自明表現の次元 $196883 + 1$ であること（ムーンシャイン現象の概観）)

### §8.3 楕円曲線と整数論
**内容:** 有理点の群構造、整数論II（4-14）・解析的数論（5-6）との接続の概観

- **1. 有理点の群構造**
  - Remark (楕円曲線を $\mathbb{Q}$ 上で定義すると、有理点 $E(\mathbb{Q})$ は有限生成アーベル群をなす（モーデルの定理）；$E(\mathbb{Q}) \cong \mathbb{Z}^r \oplus E(\mathbb{Q})_{\text{tors}}$ における階数 $r$ の決定が整数論の中心問題)
  - Remark (§5.5 の等分点の理論はガロア表現 $\rho_n : \text{Gal}(\bar{\mathbb{Q}}/\mathbb{Q}) \to \text{GL}_2(\mathbb{Z}/n\mathbb{Z})$ を通じて整数論II（4-14）と結びつく)

### §8.4 アーベル多様体への展望
**内容:** 高種数曲線のヤコビ多様体・アーベル関数への一般化の概観

  - Remark (アーベル関数は $\mathbb{C}^g/\Lambda$ 上の有理型関数であり、$g$ 変数のテータ関数で表される；ジーゲルのモジュラー形式が対応する理論を与える)

- **2. リーマンのテータ関数**
  - Remark ($g$ 変数のテータ関数 $\theta(z|\Omega)$（$z \in \mathbb{C}^g$, $\Omega$ は $g \times g$ の対称行列で $\text{Im}(\Omega) > 0$）は第6章のテータ関数の直接的な一般化であり、高種数のリーマン面の理論に不可欠である)
