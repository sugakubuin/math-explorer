## Chapter 1: 解析接続
**テーマ:** 一致の定理、解析接続の定義、直接接続と間接接続

### §1.1 一致の定理
**内容:** 零点の孤立性と一致の定理

- **1. 零点の孤立性**
  - Theorem 1.1-1 (零点の孤立性：領域 $D$ 上の正則関数 $f$ が恒等的に零でないならば、$f$ の零点は孤立すること)
  - Proof (零点の孤立性の証明：零点 $a$ の近傍でテイラー展開 $f(z) = (z-a)^m g(z)$（$g(a) \neq 0$）と書き、$g$ の連続性から $a$ の近傍で $g \neq 0$ を得る)
  - Example 1.1-1 ($f(z) = \sin z$ の零点 $\{n\pi \mid n \in \mathbb{Z}\}$ が孤立していることの確認；各零点は1位)
  - Remark (零点の孤立性は実解析にはない複素解析の特徴的性質であり、正則関数の「硬さ」を反映する)

- **2. 一致の定理**
  - Theorem 1.1-2 (一致の定理：領域 $D$ 上の正則関数 $f, g$ が $D$ の集積点を持つ部分集合上で一致するならば、$D$ 全体で $f = g$ であること)
  - Proof (一致の定理の証明：$h = f - g$ に零点の孤立性を適用する；$h$ の零点集合が集積点を持つなら $h$ は恒等的に零)
  - Example 1.1-2 ($\sin^2 z + \cos^2 z = 1$ が実数直線上で成り立つことから複素平面全体で成り立つことの確認)
  - Example 1.1-3 (一致の定理の反例：$D$ が連結でない場合に定理が成立しないことの具体例)

### §1.2 解析接続の定義
**内容:** 関数要素・直接接続・間接接続

- **1. 関数要素と直接接続**
  - Definition 1.2-1 (関数要素：組 $(f, D)$（$D$ は領域、$f$ は $D$ 上の正則関数）)
  - Definition 1.2-2 (直接接続：2つの関数要素 $(f_1, D_1)$ と $(f_2, D_2)$ が $D_1 \cap D_2 \neq \emptyset$ かつ $D_1 \cap D_2$ 上で $f_1 = f_2$ を満たすとき、$(f_2, D_2)$ は $(f_1, D_1)$ の直接接続であるということ)
  - Example 1.2-1 ($f_1(z) = \sum_{n=0}^\infty z^n$（$|z| < 1$）と $f_2(z) = 1/(1-z)$（$\mathbb{C} \setminus \{1\}$）が直接接続であることの確認)

- **2. 間接接続と曲線に沿った解析接続**
  - Definition 1.2-3 (曲線に沿った解析接続：曲線 $\gamma$ を被覆する有限個の円板の列に沿った関数要素の鎖による接続)
  - Example 1.2-2 ($\log z$ の解析接続：正の実軸から出発して上半平面を経由し負の実軸に達する経路と、下半平面を経由する経路で異なる値が得られることの計算)
  - Remark (曲線に沿った解析接続の結果は経路に依存しうる；この経路依存性こそが多価関数の起源であり、モノドロミー（第2章）の主題となる)

### §1.3 解析接続の一意性
**内容:** 同じ曲線に沿った接続の一意性

- **1. 一意性定理**
  - Theorem 1.3-1 (解析接続の一意性：同じ曲線 $\gamma$ に沿った解析接続は、被覆の取り方に依らず同じ結果を与えること)
  - Proof (一意性の証明：2つの被覆に共通部分を取る細分を構成し、各ステップで一致の定理を繰り返し適用する)
  - Example 1.3-1 ($|z| < 1$ での $\sum z^n$ を2通りの被覆で $|z-1/2| < 1$ まで接続し、いずれも $1/(1-z)$ を与えることの確認)

- **2. 一意性の意味と限界**
  - Remark (一意性は「同じ曲線に沿った」場合に限られる；異なる曲線に沿った解析接続は異なる結果を与えうる)
  - Example 1.3-2 ($\sqrt{z}$ を単位円に沿って一周する経路で解析接続すると値が $-\sqrt{z}$ に変わることの計算；一周すると元に戻らない)

### §1.4 多価関数と分枝
**内容:** $\log z$, $z^\alpha$, $\sqrt{z}$ の多価性、分枝点と分枝切断

- **1. 多価関数の例**
  - Definition 1.4-1 (多価関数：1つの点に対して複数の値を返す対応；§1.2 の解析接続の結果が経路に依存する場合に現れるもの)
  - Example 1.4-1 ($\log z = \ln|z| + i\arg z$ の多価性：$\arg z$ が $2\pi$ の不定性を持つため $\log z$ は可算無限個の値を取ること；$z = 1$ での値 $\{2n\pi i \mid n \in \mathbb{Z}\}$ の確認)
  - Example 1.4-2 ($z^\alpha = e^{\alpha \log z}$（$\alpha \notin \mathbb{Z}$）の多価性：$\alpha = 1/2$ のとき $\sqrt{z}$ が2価であること；$\alpha$ が無理数のとき可算無限価であること)

- **2. 分枝点と分枝切断**
  - Definition 1.4-2 (分枝点：その周りを一周すると関数値が変わる点)
  - Definition 1.4-3 (分枝切断：分枝点を結ぶ曲線を取り除くことで多価関数を一価正則関数に制限する操作)
  - Example 1.4-3 ($\log z$ の分枝：負の実軸を分枝切断とする主値 $\text{Log}\,z$（$-\pi < \text{Im}(\text{Log}\,z) \leq \pi$）の定義；分枝切断上では不連続)
  - Example 1.4-4 ($\sqrt{z^2 - 1}$ の分枝点が $z = \pm 1$ であること：区間 $[-1, 1]$ を分枝切断とする一価化の構成)
  - Remark (分枝切断による一価化は便利だが人工的な操作である；多価性を本質的に解消するにはリーマン面（第3章）の概念が必要となる)

## Chapter 2: モノドロミーとリーマン面の準備
**テーマ:** モノドロミー定理、普遍被覆、多価関数の一価化

### §2.1 モノドロミー定理
**内容:** 単連結領域における経路非依存性

- **1. モノドロミー定理の主張**
  - Theorem 2.1-1 (モノドロミー定理：領域 $D$ が単連結ならば、$D$ 内の任意の2つのホモトピックな曲線に沿った解析接続は同じ結果を与えること)
  - Proof (モノドロミー定理の証明の概略：ホモトピーの各段階で解析接続の一意性（§1.3）を適用し、中間曲線の族に沿った連続的変形で値が保存されることを示す)
  - Example 2.1-1 ($\log z$ の単連結領域 $\mathbb{C} \setminus (-\infty, 0]$ 上での一価性：この領域内の任意の2経路はホモトピックであるため、解析接続の結果が経路に依存しないこと)

- **2. 単連結でない場合のモノドロミー**
  - Example 2.1-2 ($\log z$ の $\mathbb{C} \setminus \{0\}$ 上でのモノドロミー：原点を囲むループに沿った解析接続で値が $2\pi i$ だけ変化すること；モノドロミー群が $\mathbb{Z}$ と同型)
  - Definition 2.1-1 (モノドロミー群：基点における基本群の解析接続への作用が定める群)
  - Example 2.1-3 ($\sqrt{z}$ のモノドロミー群：原点を囲むループで値が $-1$ 倍されるため $\mathbb{Z}/2\mathbb{Z}$ と同型であること)
  - Remark (モノドロミー群は多価関数の「周期構造」を代数的に捉えるものであり、被覆空間の変換群（位相空間論II（3-12）§7.1）と対応する)

### §2.2 基本群と被覆空間の復習
**内容:** $\pi_1$ と普遍被覆概念の確認

- **1. 基本群の復習**
  - Remark (本節は位相空間論II（3-12）の内容を複素解析の文脈で復習するものである)
  - Example 2.2-1 ($\pi_1(\mathbb{C} \setminus \{0\}) \cong \mathbb{Z}$：原点を $n$ 回囲むループが生成元の $n$ 乗に対応すること)
  - Example 2.2-2 ($\pi_1(\mathbb{C} \setminus \{0, 1\}) \cong F_2$（自由群）：2つの穴を囲むループが自由に生成する非可換群であること)

- **2. 被覆空間と持ち上げ**
  - Example 2.2-3 ($\exp : \mathbb{C} \to \mathbb{C} \setminus \{0\}$ が $\mathbb{C} \setminus \{0\}$ の普遍被覆であること；$\log z$ の多価性が被覆空間のファイバーの多重性として理解できること)
  - Example 2.2-4 ($z \mapsto z^2$ が $\mathbb{C} \setminus \{0\}$ の2枚被覆であること；$\sqrt{z}$ の2価性との対応)
  - Remark (解析接続は被覆空間における道の持ち上げ（3-12 §6.2）と同じ操作であり、モノドロミー群は被覆変換群（3-12 §7.1）と一致する)

### §2.3 多価関数の一価化の方針
**内容:** リーマン面を通じて多価性を解消するアプローチ

- **1. 一価化の基本的アイデア**
  - Remark (多価関数の問題点：$\mathbb{C}$ 上の1つの領域に複数の値を対応させるのではなく、定義域そのものを「広げる」ことで一価の正則関数を得る)
  - Example 2.3-1 ($\log z$ の一価化：$\mathbb{C}$ から $\mathbb{C} \setminus \{0\}$ への指数写像を用いると、$\log z$ は被覆空間 $\mathbb{C}$ 上の恒等写像として一価に実現されること)
  - Example 2.3-2 ($\sqrt{z}$ の一価化：$z \mapsto z^2$ の2枚被覆上で $\sqrt{z}$ が一価になる構造)

- **2. リーマン面の必然性**
  - Example 2.3-3 ($\sqrt{z^2-1}$ の一価化：2つの分枝点 $z = \pm 1$ を持つ場合、被覆空間は種数0のリーマン面（球面）であること)
  - Remark (一般の代数関数 $w^n = P(z)$ を一価化する定義域がリーマン面であり、その位相的・解析的構造を系統的に研究するのが第3章の主題である)

## Chapter 3: リーマン面
**テーマ:** 構成・基本例・位相的分類

### §3.1 リーマン面の定義
**内容:** 複素1次元多様体とチャートの適合条件

- **1. リーマン面の定義**
  - Definition 3.1-1 (リーマン面：連結ハウスドルフ空間 $S$ であって、座標チャート $(U_\alpha, \varphi_\alpha)$（$\varphi_\alpha : U_\alpha \to \mathbb{C}$ は開集合への同相）の族を持ち、座標変換 $\varphi_\beta \circ \varphi_\alpha^{-1}$ がすべて正則であるもの)
  - Definition 3.1-2 (正則アトラス：座標チャートの族が $S$ を被覆し、相互の座標変換がすべて正則であること；極大アトラスとしてのリーマン面構造)
  - Example 3.1-1 ($\mathbb{C}$ 自身が恒等写像 $\text{id} : \mathbb{C} \to \mathbb{C}$ をただ1つのチャートとするリーマン面であることの確認)
  - Remark (リーマン面は「複素1次元多様体」の別名であり、位相空間論II（3-12）で扱った位相空間の構造の上に正則性を加えたもの)

- **2. リーマン面上の正則関数**
  - Definition 3.1-3 (リーマン面上の正則関数：座標チャートで表示したとき通常の意味で正則であること)
  - Example 3.1-2 ($\mathbb{C}$ 上の正則関数は通常の正則関数と一致すること)
  - Remark (リーマン面上の正則関数の定義は座標チャートの選択に依存しないことが、座標変換の正則性によって保証される)

### §3.2 基本的なリーマン面の構成
**内容:** $\log z$, $\sqrt{z}$ のリーマン面の貼り合わせ構造

- **1. $\log z$ のリーマン面**
  - Example 3.2-1 ($\log z$ のリーマン面の構成：$\mathbb{C} \setminus \{0\}$ の無限枚のコピーを分枝切断に沿って貼り合わせる；結果は螺旋状の面で、$\mathbb{C}$ と同相であること；指数写像 $\exp : \mathbb{C} \to \mathbb{C} \setminus \{0\}$ が被覆写像を与えること)
  - Remark (この構成は§2.3 で述べた一価化の具体的実現であり、$\log z$ はこのリーマン面上の一価正則関数となる)

- **2. $\sqrt{z}$ と代数関数のリーマン面**
  - Example 3.2-2 ($\sqrt{z}$ のリーマン面の構成：$\mathbb{C} \setminus \{0\}$ の2枚のコピーを負の実軸に沿って交差的に貼り合わせる；結果はやはり $\mathbb{C}$ と同相であること)
  - Example 3.2-3 ($\sqrt{z(z-1)}$ のリーマン面：分枝点 $z = 0, 1$ を結ぶ切断に沿って2枚の球面を貼り合わせ、種数0のリーマン面を得ること)
  - Example 3.2-4 ($\sqrt{z(z-1)(z-\lambda)}$ のリーマン面：3つの分枝点と無限遠点を含む4つの分枝点による貼り合わせ；種数1のリーマン面（トーラス）が得られること)

### §3.3 リーマン球面とコンパクト化
**内容:** $\mathbb{C} \cup \{\infty\}$ の構造とコンパクトリーマン面

- **1. リーマン球面の構成**
  - Definition 3.3-1 (リーマン球面 $\hat{\mathbb{C}} = \mathbb{C} \cup \{\infty\}$：2つのチャート $\varphi_1(z) = z$（$z \neq \infty$）と $\varphi_2(z) = 1/z$（$z \neq 0$）による複素構造；座標変換 $z \mapsto 1/z$ が $\mathbb{C}^*$ 上で正則であること)
  - Proposition 3.3-1 (リーマン球面はコンパクトなリーマン面であること)
  - Proof (コンパクト性の証明：立体射影により $S^2$ と同相であり、$S^2$ はコンパクト)
  - Example 3.3-1 ($f(z) = 1/z$ がリーマン球面上の正則自己同型であること：$\varphi_2$ のチャートで $\infty$ の近傍を調べると $z \mapsto z$ で正則)

- **2. 有理型関数とリーマン球面**
  - Proposition 3.3-2 (リーマン球面上の有理型関数は有理関数に限ること)
  - Proof (証明の概略：コンパクト性から極は有限個であり、極の主要部を引くと全有界正則関数が残るためリューヴィルの定理から定数)
  - Example 3.3-2 ($f(z) = (z^2+1)/(z-1)$ がリーマン球面上の有理型関数であることの確認；$z = 1$ に1位の極、$z = \infty$ に1位の極)
  - Remark (リーマン球面上の正則関数は定数に限られるため、非自明な関数を得るには有理型関数（極を許容する関数）を考える必要がある)

### §3.4 リーマン面の位相的分類
**内容:** 種数 $g$ による分類とオイラー標数

- **1. 種数の定義**
  - Definition 3.4-1 (コンパクトリーマン面の種数 $g$：「穴の数」としての直感的定義；向き付け可能な閉曲面の分類定理（3-12 §5.3）との対応)
  - Example 3.4-1 (リーマン球面の種数は $g = 0$；トーラスの種数は $g = 1$)
  - Definition 3.4-2 (オイラー標数 $\chi = 2 - 2g$)
  - Example 3.4-2 (種数 $g$ のコンパクトリーマン面のオイラー標数：$g = 0$ で $\chi = 2$、$g = 1$ で $\chi = 0$、$g = 2$ で $\chi = -2$)

- **2. 位相的分類の意味**
  - Theorem 3.4-1 (コンパクトリーマン面の位相的分類：コンパクトな向き付け可能閉曲面は種数 $g$ で完全に分類される（位相空間論II（3-12）§5.3 の結果）)
  - Proof (分類定理の証明の参照：位相空間論IIで証明済みであり、ここでは結果を引用する)
  - Example 3.4-3 (代数曲線のリーマン面の種数：$y^2 = P(z)$ で $P$ が $2g+1$ 次または $2g+2$ 次の多項式のとき種数 $g$ であることの確認)
  - Remark (同じ種数のコンパクトリーマン面は位相的には同じだが、複素構造は一般に異なる；複素構造のパラメータ空間が「モジュライ空間」であり、楕円関数論（4-3）で $g = 1$ の場合を扱う)

### §3.5 リーマン面上の正則写像と有理型関数
**内容:** 分枝点・分枝次数とリーマン-ハーヴィッツの公式

- **1. 正則写像と分枝**
  - Definition 3.5-1 (リーマン面の間の正則写像：座標チャートで表示したとき正則である写像)
  - Definition 3.5-2 (分枝点と分枝次数：正則写像 $f : S \to T$ の点 $p$ における局所的な振る舞いが $z \mapsto z^{e_p}$ と同型であるとき、$e_p$ を $p$ における分枝次数といい、$e_p \geq 2$ のとき $p$ を分枝点と呼ぶこと)
  - Example 3.5-1 ($f(z) = z^n : \hat{\mathbb{C}} \to \hat{\mathbb{C}}$ の分枝点：$z = 0$ と $z = \infty$ が分枝次数 $n$ の分枝点であること)

- **2. リーマン-ハーヴィッツの公式**
  - Theorem 3.5-1 (リーマン-ハーヴィッツの公式：$f : S \to T$ が $n$ 次の正則写像で $S$, $T$ のそれぞれの種数が $g_S$, $g_T$ のとき、$2g_S - 2 = n(2g_T - 2) + \sum_p (e_p - 1)$ が成り立つこと)
  - Proof (リーマン-ハーヴィッツの公式の証明の概略：三角形分割のオイラー標数の比較による)
  - Example 3.5-2 ($y^2 = (z - a_1)(z - a_2)(z - a_3)(z - a_4)$ の射影 $(y, z) \mapsto z$ への適用：$n = 2$, $g_T = 0$, 分枝点4つで $2g_S - 2 = 2(0-2) + 4 = 0$ より $g_S = 1$；種数1（トーラス）であることの確認)
  - Example 3.5-3 (一般の超楕円曲線 $y^2 = \prod_{i=1}^{2g+2}(z - a_i)$ の種数が $g$ であることの計算)
  - Remark (リーマン-ハーヴィッツの公式は「分枝の総量」と「位相型の変化」を結ぶ基本公式であり、第8章のリーマン-ロッホの定理とも関連する)

## Chapter 4: 等角写像とリーマンの写像定理
**テーマ:** 等角写像の性質、メビウス変換、リーマンの写像定理

### §4.1 等角写像の基本性質
**内容:** 正則写像の角度保存性と幾何的意味

- **1. 等角性の定義**
  - Definition 4.1-1 (等角写像：$f'(z_0) \neq 0$ である正則関数 $f$ は $z_0$ の近傍で角度を保存すること)
  - Proposition 4.1-1 ($f'(z_0) \neq 0$ のとき $f$ は $z_0$ の近傍で局所的に双正則であること)
  - Proof (局所双正則性の証明：逆関数定理の複素版を適用する)
  - Example 4.1-1 ($f(z) = z^2$ が $z_0 = 1$ で等角であることの確認；$z_0 = 0$ では $f'(0) = 0$ であり角度が2倍になる)

- **2. 等角写像の幾何学的意味**
  - Example 4.1-2 ($f(z) = e^z$ による直交格子 $\{x + iy \mid x = \text{const}, y = \text{const}\}$ の像：同心円と放射線の直交族に写ること；角度の保存を目視で確認)
  - Example 4.1-3 (ジューコフスキー変換 $f(z) = z + 1/z$ が $|z| > 1$ で等角であること；翼型断面の生成への応用の概観)
  - Remark (等角写像は2次元の流体力学、静電場、熱伝導の問題で基本的な道具であり、複雑な領域の問題を単純な領域に変換する)

### §4.2 メビウス変換
**内容:** 変換群の構造と円・直線の保存性、三点一意性

- **1. メビウス変換の定義と性質**
  - Definition 4.2-1 (メビウス変換：$f(z) = (az + b)/(cz + d)$（$ad - bc \neq 0$）；リーマン球面の自己同型)
  - Proposition 4.2-1 (メビウス変換は円と直線（リーマン球面上の「円」）を円と直線に写すこと)
  - Proof (円の保存の証明：メビウス変換を平行移動・回転・拡大・反転に分解し、各操作が円を保存することを確認する)
  - Example 4.2-1 ($f(z) = (z - i)/(z + i)$ が上半平面を単位円板に写すことの確認；実軸が単位円に写ること)

- **2. 三点一意性と変換群**
  - Theorem 4.2-1 (三点一意性：リーマン球面上の任意の3点を他の任意の3点に写すメビウス変換が一意に存在すること)
  - Proof (三点一意性の証明：$z_1 \mapsto 0$, $z_2 \mapsto 1$, $z_3 \mapsto \infty$ に写す変換を構成し、合成で一般の場合に帰着する)
  - Example 4.2-2 ($0, 1, \infty$ を $1, i, -1$ に写すメビウス変換の具体的計算)
  - Remark (メビウス変換全体は群 $\text{PSL}(2, \mathbb{C})$ をなし、リーマン球面の自己同型群 $\text{Aut}(\hat{\mathbb{C}})$ と一致する)

### §4.3 正規族とモンテルの定理
**内容:** 一様有界な正則関数族の正規性とコンパクト収束

  - Remark (正規族の概念はアスコリ-アルツェラの定理の正則関数版であり、リーマンの写像定理の証明で中心的な役割を果たす)

  - Proof (モンテルの定理の証明の概略：コーシーの積分公式から導関数の一様評価を得て同程度連続性を示し、アスコリ-アルツェラの定理を適用する)

  - Example 4.3-2 ($D$ 上で $f(z) \neq 0$ かつ $f(z) \neq 1$ を満たす正則関数族が正規であることの確認)

### §4.4 リーマンの写像定理
**内容:** 単連結領域から単位円板への双正則写像の存在・一意性

## Chapter 5: 双曲幾何とシュワルツ-ピック補題
**テーマ:** 双曲計量、等長写像、一様化定理

### §5.1 シュワルツの補題
**内容:** 単位円板上の正則写像に関するシュワルツの補題

- **1. シュワルツの補題**
  - Theorem 5.1-1 (シュワルツの補題：$f : \mathbb{D} \to \mathbb{D}$ が正則で $f(0) = 0$ ならば $|f(z)| \leq |z|$ かつ $|f'(0)| \leq 1$；等号成立は $f$ が回転 $f(z) = e^{i\theta}z$ のときに限ること)
  - Proof (シュワルツの補題の証明：$g(z) = f(z)/z$（$z \neq 0$）は除去可能特異点を持ち $\mathbb{D}$ 上正則；最大値原理を $|z| \leq r < 1$ に適用して $|g(z)| \leq 1/r$ を得、$r \to 1$ で $|g(z)| \leq 1$)
  - Example 5.1-1 ($f(z) = z^2$ が $\mathbb{D} \to \mathbb{D}$, $f(0) = 0$ を満たし、$|f(z)| = |z|^2 \leq |z|$ が成り立つことの確認；等号は $|z| = 0$ でのみ成立し、$f$ は回転でないため厳密な不等式)

- **2. シュワルツの補題の系**
  - Corollary 5.1-1 ($f : \mathbb{D} \to \mathbb{D}$ が正則で $f(a) = 0$ ならば $|f(z)| \leq |(z-a)/(1-\bar{a}z)|$；メビウス変換による一般化)
  - Proof (メビウス変換 $\varphi_a(z) = (z-a)/(1-\bar{a}z)$ で $a$ を原点に移してシュワルツの補題を適用する)
  - Example 5.1-2 ($f : \mathbb{D} \to \mathbb{D}$ で $f(1/2) = 0$ のとき $|f(0)| \leq 1/2$ であることの確認)
  - Remark (シュワルツの補題は「単位円板から単位円板への正則写像は縮小的である」という直感を精密化したものであり、双曲幾何の出発点となる)

### §5.2 双曲計量の定義
**内容:** ポアンカレ計量 $ds^2$ の定義と等長群

- **1. ポアンカレ計量**
  - Definition 5.2-1 (ポアンカレ計量：$\mathbb{D}$ 上の計量 $ds = 2|dz|/(1 - |z|^2)$)
  - Remark (ポアンカレ計量は境界 $|z| = 1$ に近づくほど「距離が引き伸ばされる」計量であり、ユークリッド計量とは本質的に異なる)
  - Example 5.2-1 (原点から点 $r$（$0 < r < 1$）までの双曲的長さ：$\int_0^r 2/(1-t^2)\,dt = \log((1+r)/(1-r))$；$r \to 1$ で双曲的長さが $\infty$ に発散すること)

- **2. 等長群**
  - Proposition 5.2-1 ($\mathbb{D}$ のメビウス自己同型 $\varphi_a(z) = e^{i\theta}(z-a)/(1-\bar{a}z)$ がポアンカレ計量の等長変換であること)
  - Proof (等長性の証明：$|\varphi_a'(z)|/(1 - |\varphi_a(z)|^2) = 1/(1 - |z|^2)$ の直接計算)
  - Theorem 5.2-1 ($\text{Aut}(\mathbb{D}) = \text{PSU}(1,1)$ がポアンカレ計量の等長群全体と一致すること)
  - Proof (証明の概略：等長変換は正則または反正則であること、反正則の場合は向きを逆転するため除外されること)
  - Example 5.2-2 (上半平面モデル $\mathbb{H}$ でのポアンカレ計量：$ds = |dz|/\text{Im}(z)$；等長群が $\text{PSL}(2, \mathbb{R})$ であること)

### §5.3 シュワルツ-ピック補題
**内容:** 双曲計量を縮小しない正則写像の性質

- **1. シュワルツ-ピック補題**
  - Theorem 5.3-1 (シュワルツ-ピックの補題：$f : \mathbb{D} \to \mathbb{D}$ が正則ならば、双曲距離について $d_{\mathbb{D}}(f(z), f(w)) \leq d_{\mathbb{D}}(z, w)$；等号成立は $f$ がメビウス自己同型のときに限ること)
  - Proof (シュワルツ-ピックの証明：メビウス変換 $\varphi_{f(z)}$ と $\varphi_z$ で正規化してシュワルツの補題に帰着する)
  - Example 5.3-1 ($f(z) = z^2$ に対する双曲距離の縮小の計算：$z = 1/2$ と $w = -1/2$ の双曲距離が $f$ によって厳密に減少することの確認)

- **2. 微分形式での表現**
  - Corollary 5.3-1 (微分形式：$|f'(z)|/(1 - |f(z)|^2) \leq 1/(1 - |z|^2)$；等号成立は自己同型のときに限る)
  - Proof (Theorem 5.3-1 の微分版として直ちに従う)
  - Example 5.3-2 ($f(z) = z/2$ に対する微分的評価の確認：$|f'(z)|/(1 - |f(z)|^2) = (1/2)/(1 - |z|^2/4) < 1/(1 - |z|^2)$)
  - Remark (シュワルツ-ピック補題は「正則写像は双曲距離を縮小する」という幾何学的原理を表し、アールフォルスの曲率条件への拡張（一般のリーマン面上の正則写像の縮小性）へとつながる)

### §5.4 双曲距離と測地線
**内容:** 双曲距離の明示公式と測地線の計算

- **1. 双曲距離の公式**
  - Definition 5.4-1 (双曲距離：$d_{\mathbb{D}}(z, w) = \text{arctanh}\,|(z - w)/(1 - \bar{w}z)|$)
  - Proposition 5.4-1 (双曲距離が距離の公理を満たすこと：非退化性、対称性、三角不等式)
  - Proof (三角不等式の証明の概略：シュワルツ-ピック補題から双曲距離のメビウス変換不変性を用い、原点を通る場合に帰着する)
  - Example 5.4-1 (原点と点 $r$（$0 < r < 1$）の双曲距離：$d_{\mathbb{D}}(0, r) = \text{arctanh}(r) = \frac{1}{2}\log\frac{1+r}{1-r}$；$r = 1/2$ で $d \approx 0.549$, $r = 0.9$ で $d \approx 1.472$, $r = 0.99$ で $d \approx 2.646$ の計算)

- **2. 測地線の計算**
  - Proposition 5.4-2 ($\mathbb{D}$ 上の測地線は直径と、端点が $\partial\mathbb{D}$ 上にある円弧（$\partial\mathbb{D}$ と直交する円弧）であること)
  - Proof (測地線の証明の概略：メビウス変換の等長性により原点を通る場合に帰着すると直径であり、一般の場合は変換で移す)
  - Example 5.4-2 (上半平面モデルでの測地線：実軸に直交する半円と虚軸に平行な半直線；$i$ と $2i$ を結ぶ測地線の長さ $\log 2$ の計算)
  - Remark (双曲平面は定曲率 $-1$ のリーマン多様体であるが、曲率の精密な議論は微分幾何II（4-6）に譲る)

### §5.5 一様化定理
**内容:** 単連結リーマン面はリーマン球面・複素平面・単位円板のいずれかに双正則同値であること

- **1. 一様化定理の主張**
  - Theorem 5.5-1 (一様化定理：すべての単連結リーマン面は、$\hat{\mathbb{C}}$（リーマン球面）、$\mathbb{C}$（複素平面）、$\mathbb{D}$（単位円板）のいずれかに双正則同値であること)
  - Proof (一様化定理の証明は長大であり本コースの範囲を超える；非コンパクトの場合はリーマンの写像定理の一般化として、グリーン関数の存在・非存在で $\mathbb{C}$ 型と $\mathbb{D}$ 型を判別する)
  - Remark (一様化定理はリーマンの写像定理（§4.4）の一般化であり、リーマン面論の頂点の一つである；証明は長大であり詳細は省略するが、定理の主張と帰結を理解することが重要)
  - Example 5.5-1 (3つの場合の代表例：$\hat{\mathbb{C}}$ 自身（コンパクト、種数0）；$\mathbb{C}$（非コンパクト、放物型）；$\mathbb{D}$（非コンパクト、双曲型）)

- **2. 各場合の特徴づけと帰結**
  - Proposition 5.5-1 (単連結コンパクトリーマン面は $\hat{\mathbb{C}}$ に双正則同値であること)
  - Proof (証明の概略：コンパクトリーマン面上の有理型関数が有限個の極を持つことから、射影直線との同型を得る)
  - Proposition 5.5-2 ($\mathbb{C}$ と $\mathbb{D}$ の区別：リューヴィルの定理により $\mathbb{C}$ から $\mathbb{D}$ への正則写像は定数であるため双正則同値でないこと)
  - Proof ($f : \mathbb{C} \to \mathbb{D}$ が正則ならば有界整関数であり、リューヴィルの定理から定数)
  - Example 5.5-2 (一般のリーマン面への応用：コンパクトリーマン面 $S$ の普遍被覆が一様化定理の3つの場合のいずれかであること；種数 $g = 0$ で $\hat{\mathbb{C}}$、$g = 1$ で $\mathbb{C}$、$g \geq 2$ で $\mathbb{D}$ が対応すること)
  - Remark (一様化定理は複素解析・位相幾何学・双曲幾何を結ぶ深い定理である；種数 $g = 1$ の場合は楕円関数論（4-3）で複素トーラスとして、$g \geq 2$ の場合は双曲幾何のフックス群による商空間として具体化される)

## Chapter 6: 整関数・有理型関数の分解理論
**テーマ:** ワイエルシュトラス分解定理、ミッタク-レフラー定理、ハダマール因数分解

### §6.1 無限積の収束
**内容:** 絶対収束と対数の収束の同値性

- **1. 無限積の定義と収束**
  - Definition 6.1-1 (無限積 $\prod_{n=1}^\infty (1 + a_n)$ の収束：部分積の極限が0でない有限値に収束すること)
  - Proposition 6.1-1 (無限積の絶対収束と級数 $\sum |a_n|$ の収束が同値であること)
  - Proof (同値性の証明：$\log(1 + a_n) \sim a_n$（$|a_n|$ が小さいとき）を用いて無限積の対数と級数を比較する)
  - Example 6.1-1 ($\prod_{n=1}^\infty (1 + 1/n^2)$ の絶対収束の確認：$\sum 1/n^2 < \infty$ であるため収束)

- **2. 正則関数の無限積**
  - Proposition 6.1-2 (正則関数の無限積：$\sum |f_n(z)|$ が広義一様収束するとき $\prod (1 + f_n(z))$ は広義一様収束し零点の位置が保存されること)
  - Proof (広義一様収束の証明：対数の部分和が広義一様収束する正則関数列をなすことを確認する)
  - Example 6.1-2 ($\sin(\pi z)/(\pi z) = \prod_{n=1}^\infty (1 - z^2/n^2)$ のオイラー積表示の紹介；収束の確認は §6.2 で行う)

### §6.2 ワイエルシュトラスの分解定理
**内容:** 任意の零点列に基づく整関数の構成論

- **1. ワイエルシュトラスの初等因子**
  - Definition 6.2-1 (ワイエルシュトラスの初等因子：$E_p(z) = (1-z)\exp(z + z^2/2 + \cdots + z^p/p)$；$E_0(z) = 1 - z$)
  - Proposition 6.2-1 ($|z| \leq 1/2$ のとき $|1 - E_p(z)| \leq |z|^{p+1}$ であること)
  - Proof (評価の証明：$\log E_p(z)$ のテイラー展開で $z, z^2, \ldots, z^p$ の項が打ち消されることを確認する)
  - Example 6.2-1 ($E_0(z) = 1 - z$, $E_1(z) = (1-z)e^z$, $E_2(z) = (1-z)e^{z+z^2/2}$ の各場合の零点と漸近的振る舞い)

- **2. ワイエルシュトラスの分解定理**
  - Theorem 6.2-1 (ワイエルシュトラスの分解定理：$\{a_n\}$（$0 < |a_1| \leq |a_2| \leq \cdots$, $|a_n| \to \infty$）を零点列とする整関数が存在すること；$f(z) = z^m e^{g(z)} \prod_{n=1}^\infty E_{p_n}(z/a_n)$ と表されること)
  - Proof (分解定理の証明の概略：$p_n = n - 1$ と取れば $\sum |z/a_n|^{p_n+1}$ の収束がコンパクト集合上で保証されること；無限積の広義一様収束を示す)
  - Example 6.2-2 ($\sin(\pi z) = \pi z \prod_{n=1}^\infty (1 - z^2/n^2)$ がワイエルシュトラスの分解定理の適用例であること：零点列 $\{n \mid n \in \mathbb{Z} \setminus \{0\}\}$, $p_n = 0$ で $E_0$ が使えることの確認)
  - Remark (ワイエルシュトラスの分解定理は「指定された零点を持つ整関数が必ず構成できる」ことを保証する存在定理である)

### §6.3 ミッタク-レフラーの定理
**内容:** 指定された極と主要部を持つ有理型関数の構成

- **1. ミッタク-レフラーの定理**
  - Theorem 6.3-1 (ミッタク-レフラーの定理：$\mathbb{C}$ 上の指定された極の列 $\{a_n\}$ と各極での主要部 $P_n(1/(z - a_n))$（$P_n$ は多項式）が与えられたとき、これらを主要部とする有理型関数が存在すること)
  - Proof (ミッタク-レフラーの定理の証明の概略：各主要部から $a_n = 0$ 近傍でのテイラー多項式を引いて補正し、補正後の級数が広義一様収束することを示す)
  - Example 6.3-1 ($\pi\cot(\pi z) = 1/z + \sum_{n=1}^\infty (1/(z-n) + 1/(z+n)) = 1/z + \sum_{n=1}^\infty 2z/(z^2 - n^2)$ が各整数点に1位の極を持つ有理型関数であることの確認)

- **2. 部分分数分解との対比**
  - Example 6.3-2 ($1/(\sin(\pi z))^2$ の部分分数展開の導出)
  - Remark (有理関数の部分分数分解は有限和で表されるが、ミッタク-レフラーの定理はこれを無限和（可算個の極を持つ有理型関数）に拡張したものである)

### §6.4 ハダマールの因数分解定理
**内容:** 有限階数の整関数の積表示と階数・型の関係

- **1. 整関数の位数と階数**
  - Definition 6.4-1 (整関数の位数 $\rho$：$\rho = \limsup_{r \to \infty} \log\log M(r) / \log r$；$M(r) = \max_{|z|=r} |f(z)|$)
  - Definition 6.4-2 (整関数の階数：ワイエルシュトラスの分解定理における初等因子の最小の次数)
  - Example 6.4-1 ($e^z$ の位数は $\rho = 1$（$M(r) = e^r$ より）；$\sin z$ の位数も $\rho = 1$)
  - Example 6.4-2 ($e^{z^2}$ の位数は $\rho = 2$；多項式の位数は $\rho = 0$)

- **2. ハダマールの因数分解定理**
  - Theorem 6.4-1 (ハダマールの因数分解定理：位数 $\rho$ の整関数は $f(z) = z^m e^{Q(z)} \prod E_p(z/a_n)$ と表され、$Q$ は次数 $\leq \rho$ の多項式、$p \leq \rho$ であること)
  - Proof (ハダマールの定理の証明の概略：位数 $\rho$ の条件から零点の個数の増大度評価 $n(r) = O(r^{\rho+\varepsilon})$ を導き、初等因子の次数を制御する)
  - Example 6.4-3 ($\sin(\pi z)$ への適用：$\rho = 1$ であり、$\sin(\pi z) = \pi z \prod (1 - z^2/n^2)$（$Q(z) = 0$, $p = 0$）が実際にハダマールの形式であることの確認)
  - Remark (ハダマールの因数分解定理はワイエルシュトラスの分解定理を精密化し、位数と積表示の構造を結びつけるものである)

### §6.5 ガンマ関数への応用
**内容:** 積表示を用いた $\Gamma$ 関数の解析的性質の整理

- **1. ガンマ関数の積表示**
  - Definition 6.5-1 (ガンマ関数：$\Gamma(z) = \int_0^\infty t^{z-1}e^{-t}\,dt$（$\text{Re}(z) > 0$）；解析接続による $\mathbb{C} \setminus \{0, -1, -2, \ldots\}$ 上の有理型関数への拡張)
  - Theorem 6.5-1 (ワイエルシュトラスの積表示：$1/\Gamma(z) = ze^{\gamma z}\prod_{n=1}^\infty (1 + z/n)e^{-z/n}$（$\gamma$ はオイラー-マスケローニ定数）)
  - Proof (積表示の証明の概略：$1/\Gamma(z)$ が位数1の整関数であること（$0, -1, -2, \ldots$ に1位の零点を持つ）を確認し、ハダマールの因数分解定理を適用する)
  - Example 6.5-1 ($\Gamma(1) = 1$ と $\Gamma(z+1) = z\Gamma(z)$（関数方程式）を積表示から導出する計算)

- **2. ガンマ関数の補充公式**
  - Theorem 6.5-2 (オイラーの補充公式：$\Gamma(z)\Gamma(1-z) = \pi/\sin(\pi z)$)
  - Proof (補充公式の証明：$\Gamma(z)$ と $\sin(\pi z)$ の積表示を掛け合わせ、ワイエルシュトラス積と比較する)
  - Example 6.5-2 ($z = 1/2$ を代入して $\Gamma(1/2)^2 = \pi$、すなわち $\Gamma(1/2) = \sqrt{\pi}$ を導出)
  - Remark (ガンマ関数は解析的数論（5-6）でゼータ関数の関数方程式の証明に不可欠であり、ここで整理した解析的性質が基礎となる)

## Chapter 7: ピカールの定理と整関数の値分布
**テーマ:** 小ピカール・大ピカール定理、モンテルの正規族との関連

### §7.1 小ピカールの定理
**内容:** 非定値整関数の値の存在性

- **1. 小ピカールの定理**
  - Theorem 7.1-1 (小ピカールの定理：非定値整関数はたかだか1つの例外値を除いてすべての複素数値を取ること)
  - Proof (小ピカールの定理の証明の概略：$f$ が値 $a, b$（$a \neq b$）をともに取らないと仮定する；$(f - a)/(b - a)$ は $0$ と $1$ を取らない整関数であり、被覆空間の議論を通じて有界性を示しリューヴィルの定理から矛盾を得る)
  - Example 7.1-1 ($e^z$ がすべての値を取ること（$e^z = w \neq 0$ は $z = \log w$ で解ける）；唯一の例外値は $0$（$e^z \neq 0$）)
  - Example 7.1-2 (多項式は代数学の基本定理（複素解析I（3-2））により例外値を持たないこと)

- **2. 小ピカールの定理の帰結**
  - Corollary 7.1-1 ($f$ と $g$ が非定値整関数で $e^f = e^g$ ならば $f - g$ は $2\pi i\mathbb{Z}$ 値の整関数であり、リューヴィルの定理から定数であること)
  - Proof ($e^{f-g} = 1$ であるから $f - g$ は $2\pi in$（$n \in \mathbb{Z}$）の値のみを取り、連続性と連結性から定数)
  - Example 7.1-3 ($\sin z = 0 \implies z = n\pi$ が全ての解であることの確認)
  - Remark (小ピカールの定理はリューヴィルの定理の驚くべき精密化であり、「有界な整関数は定数」を「1つの値を省く整関数は定数」にまで強めたものである)

### §7.2 大ピカールの定理
**内容:** 真性特異点近傍での値分布の精密化

- **1. 大ピカールの定理**
  - Theorem 7.2-1 (大ピカールの定理：$f$ が $z_0$ に真性特異点を持つならば、$z_0$ の任意の穴あき近傍で $f$ はたかだか1つの例外値を除いてすべての値を無限回取ること)
  - Proof (大ピカールの定理の証明の概略：モンテルの定理の強化形（§4.3）を反復的に適用し、真性特異点の近傍の環状領域列で正規族の議論を行う)
  - Example 7.2-1 ($e^{1/z}$ の $z = 0$（真性特異点）近傍での値分布：$e^{1/z} = w \neq 0$ は $z = 1/\log w$ で無限個の解を持つこと；唯一の例外値は $0$)

- **2. カソラティ-ワイエルシュトラスの定理との比較**
  - Theorem 7.2-2 (カソラティ-ワイエルシュトラスの定理：$f$ が $z_0$ に真性特異点を持つならば、$z_0$ の任意の穴あき近傍で $f$ の値は $\mathbb{C}$ で稠密であること)
  - Proof (カソラティ-ワイエルシュトラスの証明：$f(z) - w$ が $z_0$ の近傍で有界下限を持てば極か除去可能特異点であることに矛盾)
  - Example 7.2-2 ($\sin(1/z)$ の $z = 0$ 近傍での値：任意の $w \in [-1, 1]$ に対して $\sin(1/z_n) = w$ を満たす $z_n \to 0$ が存在すること)
  - Remark (カソラティ-ワイエルシュトラスは「稠密」、大ピカールは「たかだか1つを除いてすべて」と結論が飛躍的に強められている)

### §7.3 ピカール定理とモンテル定理の関係
**内容:** 正規性を用いた定理の証明の流れ

- **1. 正規族による証明の枠組み**
  - Remark (ピカールの両定理は、モンテルの定理（§4.3）を通じた正規族の議論で統一的に理解できる)
  - Proposition 7.3-1 (ブロッホの定理：$f(0) = 0$, $f'(0) = 1$ を満たす正則関数の像は半径 $B$（ブロッホ定数）以上の円板を含むこと)
  - Proof (ブロッホの定理の証明の概略：面積的議論と正規族の議論を組み合わせる)
  - Example 7.3-1 (ブロッホ定数 $B$ の評価：$B \geq \sqrt{3}/4$ であり、正確な値は未知であることの指摘)

- **2. 証明の論理的構造**
  - Remark (小ピカールの定理の別証明の流れ：2つの値を省く整関数から正規族を構成し、リューヴィルの定理に帰着する；大ピカールの定理では穴あき円板の列に対して同じ議論を行う)
  - Example 7.3-2 (正規族の非正規性からピカールの定理を示す対偶的な証明の概観)

### §7.4 ネヴァンリンナ理論への展望
**内容:** 有理型関数の定量的解析の導入

- **1. ネヴァンリンナの特性関数**
  - Definition 7.4-1 (ネヴァンリンナの特性関数 $T(r, f)$：近接関数 $m(r, a)$ と計数関数 $N(r, a)$ の和)
  - Remark (ピカールの定理は「$f$ がほとんどすべての値を取る」という定性的結果であるが、ネヴァンリンナ理論はこれを「各値をどれだけの頻度で取るか」の定量的解析に発展させたものである)

- **2. ネヴァンリンナの基本定理**
  - Theorem 7.4-1 (ネヴァンリンナの第一基本定理の主張：$T(r, f) = m(r, a) + N(r, a) + O(1)$；「各値が同じ頻度で取られる」ことの精密化)
  - Proof (第一基本定理の証明は本コースの範囲を超えるため主張の紹介にとどめる；ジェンセンの公式を出発点とする)
  - Theorem 7.4-2 (ネヴァンリンナの第二基本定理の主張：例外値の個数を定量的に制限する不等式；小ピカールの定理を含む一般化)
  - Proof (第二基本定理の証明も本コースの範囲を超える；対数微分の補題が中心的な道具である)
  - Example 7.4-1 ($e^z$ に対する特性関数 $T(r, e^z) = r/\pi$ の計算)
  - Remark (ネヴァンリンナ理論は20世紀の複素解析の中心的理論であり、ディオファントス近似との類似（ヴォイタの辞書）を通じて整数論とも深く関わる)

## Chapter 8: リーマン-ロッホの定理
**テーマ:** 除数、線形系、定理の主張と応用

### §8.1 コンパクトリーマン面上の有理型関数と除数
**内容:** 除数群・次数と主除数の概念

- **1. 除数の定義**
  - Definition 8.1-1 (除数：コンパクトリーマン面 $S$ 上の点の形式的な整数係数の有限和 $D = \sum n_p \cdot p$)
  - Definition 8.1-2 (除数の次数：$\deg D = \sum n_p$；除数群 $\text{Div}(S)$ と加法構造)
  - Example 8.1-1 (リーマン球面上の有理関数 $f(z) = z^2(z-1)/(z+1)^3$ の除数 $(f) = 2 \cdot 0 + 1 \cdot 1 - 3 \cdot (-1) + 0 \cdot \infty$ の計算；$\infty$ での零点・極の次数の確認)

- **2. 主除数と線形同値**
  - Definition 8.1-3 (主除数：有理型関数 $f$ の零点と極から定まる除数 $(f) = \sum \text{ord}_p(f) \cdot p$)
  - Theorem 8.1-1 (コンパクトリーマン面上の主除数の次数は0であること：$\deg(f) = 0$)
  - Proof (次数0の証明：$f$ の零点と極の個数が等しいことを留数定理（複素解析I（3-2））の一般化から導く)
  - Example 8.1-2 (上の例の確認：$\deg(f) = 2 + 1 - 3 + 0 = 0$)
  - Definition 8.1-4 (線形同値：$D_1 \sim D_2$ とは $D_1 - D_2 = (f)$ となる有理型関数 $f$ が存在すること)

### §8.2 線形系と $l(D)$
**内容:** 除数に付随する有理型空間 $L(D)$ とその次元

- **1. 有理型関数の空間 $L(D)$**
  - Definition 8.2-1 (除数に付随する空間：$L(D) = \{f : \text{有理型} \mid (f) + D \geq 0\} \cup \{0\}$；すなわち $D$ で指定された極以下の極しか持たない有理型関数の全体)
  - Definition 8.2-2 ($l(D) = \dim_{\mathbb{C}} L(D)$)
  - Example 8.2-1 (リーマン球面上の $D = n \cdot \infty$ に対する $L(D)$：$\infty$ に $n$ 位以下の極を持つ有理型関数は次数 $n$ 以下の多項式であるため $l(D) = n + 1$)

- **2. $l(D)$ の基本性質**
  - Proposition 8.2-1 ($l(D) < \infty$ であること（コンパクトリーマン面上）)
  - Proof (有限次元性の証明の概略：$\deg D < 0$ ならば $L(D) = \{0\}$；$\deg D \geq 0$ の場合はコンパクト性による評価)
  - Proposition 8.2-2 ($D_1 \sim D_2$ ならば $l(D_1) = l(D_2)$ であること)
  - Proof (線形同値性の証明：$(f)$ による掛け算が $L(D_1)$ と $L(D_2)$ の間の同型を与える)
  - Example 8.2-2 (種数1のリーマン面上で $D = 1 \cdot p$ のとき $l(D) = 1$ であること：定数関数のみ)

### §8.3 微分形式と標準除数
**内容:** 正則 1-形式の空間 $K$ の定義と次元

- **1. リーマン面上の微分形式**
  - Definition 8.3-1 (有理型1-形式：局所座標 $z$ で $\omega = f(z)\,dz$ と書ける形式的対象；座標変換則 $f(z)\,dz = f(z(w)) z'(w)\,dw$)
  - Example 8.3-1 (リーマン球面上の $\omega = dz$：$w = 1/z$ のチャートでは $\omega = -dw/w^2$ となり $w = 0$（すなわち $z = \infty$）に2位の極を持つこと)
  - Definition 8.3-2 (微分形式の除数：$(\omega) = \sum \text{ord}_p(\omega) \cdot p$)

- **2. 標準除数と正則1-形式**
  - Definition 8.3-3 (標準除数 $K$：任意の有理型1-形式 $\omega$ の除数 $(\omega)$ の線形同値類)
  - Theorem 8.3-1 (標準除数の次数：$\deg K = 2g - 2$（$g$ は種数）)
  - Proof (次数の証明の概略：リーマン-ハーヴィッツの公式（§3.5）を適用する)
  - Example 8.3-2 (リーマン球面（$g = 0$）の場合：$\deg K = -2$；$dz$ の除数は $-2 \cdot \infty$ であり確かに次数 $-2$)
  - Example 8.3-3 (種数1のトーラスの場合：$\deg K = 0$；$dz$ が正則1-形式（極を持たない）であり $l(K) = 1$)

### §8.4 リーマン-ロッホの定理の主張と証明の概略
**内容:** 次元公式の幾何学的な解釈

- **1. リーマン-ロッホの定理の主張**
  - Theorem 8.4-1 (リーマン-ロッホの定理：種数 $g$ のコンパクトリーマン面 $S$ 上の除数 $D$ に対して、$l(D) - l(K - D) = \deg D - g + 1$)
  - Proof (リーマン-ロッホの定理の証明の概略：$D$ の次数に関する帰納法で示す；$D$ を1点分変化させたときの $l(D)$ の変化を追跡し、基底ケースとして $D = 0$ の場合（$l(0) = 1$, $l(K) = g$）を確認する)
  - Remark (公式の各項の意味：$l(D)$ は $D$ で許容される有理型関数の「自由度」、$l(K-D)$ は双対的な量（セール双対性）、$\deg D - g + 1$ は位相的な情報)
  - Example 8.4-1 (リーマン球面（$g = 0$）での検証：$D = n \cdot \infty$ に対して $l(D) = n + 1$, $K - D = -2\cdot\infty - n\cdot\infty$ で $\deg(K-D) = -n - 2 < 0$ より $l(K-D) = 0$；$l(D) - l(K-D) = n + 1 = n - 0 + 1$)

- **2. 証明の補足と現代的視点**
  - Remark (証明の帰納法のステップ：$D$ に1点 $p$ を加えて $D' = D + p$ とすると $L(D) \subset L(D')$ であり、次元の変化が $l(D') - l(D) \in \{0, 1\}$ であることを利用する)
  - Remark (より現代的な証明はセール双対定理 $l(K-D) = \dim H^1(S, \mathcal{O}(D))$ を経由するが、ここでは古典的なアプローチに従う)

### §8.5 応用
**内容:** 種数と有理型関数の関係性、射影的埋め込みの概観

- **1. 種数と有理型関数の存在**
  - Corollary 8.5-1 ($\deg D \geq 2g$ ならば $l(D) = \deg D - g + 1$ であること)
  - Proof ($\deg(K - D) = 2g - 2 - \deg D < 0$ より $l(K-D) = 0$)
  - Example 8.5-1 (種数2の曲線上の次数4の除数 $D$ に対して $l(D) = 3$；すなわち3つの独立な有理型関数が存在すること)
  - Corollary 8.5-2 (種数 $g \geq 1$ のコンパクトリーマン面上には非定値有理型関数が存在すること)
  - Proof ($\deg D$ を十分大きく取れば $l(D) \geq 2$ となり、定数でない関数が得られる)

- **2. 射影的埋め込み**
  - Theorem 8.5-1 ($\deg D \geq 2g + 1$ ならば $L(D)$ の基底による写像 $S \to \mathbb{CP}^{l(D)-1}$ が埋め込みを与えること)
  - Proof (埋め込みの証明の概略：$D$ の次数が十分大きいとき $L(D)$ が点の分離と接線方向の分離を達成することをリーマン-ロッホの定理から確認する)
  - Example 8.5-2 (種数1の楕円曲線：$D = 3 \cdot p$ で $l(D) = 3$；$\mathbb{CP}^2$ への埋め込みがワイエルシュトラスの楕円曲線 $y^2 = 4x^3 - g_2 x - g_3$ を与えること；楕円関数論（4-3）への接続)
  - Remark (すべてのコンパクトリーマン面は射影代数曲線として実現できる；この事実は代数幾何I（4-12）で扱うGAGA原理の出発点となる)
