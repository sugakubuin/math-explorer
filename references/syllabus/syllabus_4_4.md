## Chapter 1: 凸集合
**テーマ:** 定義・性質・射影・分離定理・サポート関数

### §1.1 凸集合の定義と例
**内容:** 線分条件による定義、半空間・球・多面体・錐などの例

- **1. 凸集合の定義**
  - Definition 1.1-1 (凸集合：$\mathbb{R}^n$ の部分集合 $C$ が凸であるとは、任意の $x, y \in C$ と $\lambda \in [0,1]$ に対して $\lambda x + (1-\lambda)y \in C$ が成り立つこと)
  - Example 1.1-1 (半空間の凸性：$C = \{x \in \mathbb{R}^n \mid \langle a, x \rangle \leq b\}$ が凸であることの確認；$\lambda x + (1-\lambda)y$ を代入し不等式の保存を検証する)
  - Example 1.1-2 (球の凸性：$B(0, r) = \{x \mid \|x\| \leq r\}$ が凸であることの三角不等式による確認)
  - Example 1.1-3 (凸でない集合の例：$\mathbb{R}^2$ における単位円周 $\{x \mid \|x\| = 1\}$；$(1,0)$ と $(-1,0)$ の中点 $(0,0)$ が円周上にないことの確認)

- **2. 多面体と錐**
  - Definition 1.1-2 (多面体：有限個の半空間の共通部分として表される集合)
  - Example 1.1-4 ($\mathbb{R}^2$ の三角形の多面体表現：3本の不等式 $x \geq 0$、$y \geq 0$、$x+y \leq 1$ の共通部分として構成し、凸性が半空間の凸性と共通部分の性質から従うことの確認)
  - Definition 1.1-3 (錐：$x \in C$ かつ $\lambda \geq 0$ ならば $\lambda x \in C$ を満たす集合)
  - Example 1.1-5 ($\mathbb{R}^2$ における半正定値錐の例：$C = \{(x,y) \mid x \geq 0, y \geq 0\}$ が錐であり凸でもあることの確認)
  - Remark (凸錐は錐かつ凸な集合であり、「半空間」「正定値行列の全体」など最適化で頻出する)

### §1.2 凸包と凸錐
**内容:** 凸包の定義と最小性、凸錐の定義と極錐の概念

- **1. 凸包**
  - Definition 1.2-1 (凸包：集合 $S$ を含む最小の凸集合 $\mathrm{conv}(S)$；同値的に、$S$ の有限個の点の凸結合全体)
  - Proposition 1.2-1 ($\mathrm{conv}(S)$ は $S$ を含むすべての凸集合の共通部分に等しい)
  - Proof (二つの集合の包含関係を示す；凸結合全体が凸集合であること、および凸集合が凸結合を含むことをそれぞれ確認する)
  - Example 1.2-1 ($\mathbb{R}^2$ での凸包の計算：$S = \{(0,0), (1,0), (0,1), (1/3, 1/3)\}$ の凸包は三角形 $\mathrm{conv}\{(0,0),(1,0),(0,1)\}$；第4点は既に三角形の内部にあることの確認)

- **2. 凸錐と極錐**
  - Definition 1.2-2 (凸錐：錐であり凸である集合；同値的に、非負結合に閉じた集合)
  - Example 1.2-2 (非負直交象限 $\mathbb{R}^n_+ = \{x \mid x_i \geq 0\}$ が凸錐であることの確認)
  - Definition 1.2-3 (極錐：凸錐 $K$ に対して $K^\circ = \{y \mid \langle y, x \rangle \leq 0 \ \forall x \in K\}$)
  - Example 1.2-3 (非負直交象限の極錐：$(\mathbb{R}^n_+)^\circ = \mathbb{R}^n_- = \{y \mid y_i \leq 0\}$ であることの確認；各基底ベクトル $e_i$ との内積条件から $y_i \leq 0$ を導出する)
  - Proposition 1.2-2 (極錐は閉凸錐である)
  - Proof (極錐が半空間の共通部分として書けることから、閉性と凸性が従う)
  - Remark (極錐は双対性の幾何的表現であり、フェンシェル双対性（第5章）への伏線となる)

### §1.3 凸集合の位相的性質
**内容:** 内部・閉包・相対的内部の関係、凸集合の連結性

- **1. 内部と閉包**
  - Proposition 1.3-1 (凸集合の閉包と内部はともに凸である)
  - Proof (閉包の凸性は極限操作と凸結合の交換により示す；内部の凸性は $\mathrm{int}(C)$ の点の近傍と凸結合の組み合わせにより示す)
  - Example 1.3-1 (開半空間 $\{x \mid \langle a, x \rangle < b\}$ は凸であり、その閉包 $\{x \mid \langle a, x \rangle \leq b\}$ も凸；両者の内部と閉包の一致を確認する)

- **2. 相対的内部**
  - Definition 1.3-1 (アフィン包 $\mathrm{aff}(C)$：$C$ を含む最小のアフィン部分空間)
  - Definition 1.3-2 (相対的内部 $\mathrm{ri}(C)$：アフィン包の位相に関する $C$ の内部)
  - Example 1.3-2 ($\mathbb{R}^3$ 内の線分 $C = \{(t, 0, 0) \mid 0 \leq t \leq 1\}$；$\mathrm{int}(C) = \emptyset$ だが $\mathrm{aff}(C)$ は $x$-軸であり $\mathrm{ri}(C) = \{(t, 0, 0) \mid 0 < t < 1\}$ となることの確認)
  - Proposition 1.3-2 (空でない凸集合 $C$ に対して $\mathrm{ri}(C) \neq \emptyset$)
  - Proof (凸集合のアフィン包の次元に関する帰納法による)
  - Remark (相対的内部は劣微分の和の公式（§3.3）やフェンシェル双対の強双対定理（§5.2）で不可欠な概念である)

### §1.4 分離定理
**内容:** 超平面分離定理・支持超平面定理の証明と幾何的意味

- **1. 超平面分離定理**
  - Definition 1.4-1 (超平面と半空間：超平面 $H = \{x \mid \langle a, x \rangle = b\}$ による空間の二分割)
  - Theorem 1.4-1 (分離定理：交わらない2つの凸集合のうち少なくとも一方が開であるとき、それらを分離する超平面が存在する)
  - Proof (一方の集合から他方への最短距離を実現するベクトルを法線として用い、内積の不等式を導く；閉凸集合の場合は射影定理（§1.5）を利用する)
  - Example 1.4-1 ($\mathbb{R}^2$ での分離：$C_1 = \{(x,y) \mid x^2 + y^2 \leq 1\}$ と $C_2 = \{(x,y) \mid x \geq 3\}$ を分離する超平面 $x = 2$ の構成)

- **2. 支持超平面定理**
  - Theorem 1.4-2 (支持超平面定理：凸集合の境界点を通る支持超平面が存在する)
  - Proof (境界点と集合の内部との分離定理を適用し、極限操作を行う)
  - Example 1.4-2 ($\mathbb{R}^2$ の単位球 $B(0,1)$ の境界点 $(1,0)$ における支持超平面 $x = 1$ の確認；すべての $\|p\| \leq 1$ に対して $p_1 \leq 1$ が成立する)
  - Remark (分離定理はフェンシェル双対性の基礎であり、最適化理論（A-3）においても制約条件の幾何的理解に不可欠である)

### §1.5 射影定理（凸集合版）
**内容:** 閉凸集合への最近傍点の存在・一意性、射影の非拡大性

- **1. 射影の存在と一意性**
  - Definition 1.5-1 (凸射影：閉凸集合 $C$ に対して $\mathrm{proj}_C(x) = \arg\min_{y \in C} \|x - y\|$)
  - Theorem 1.5-1 (閉凸集合への射影の存在と一意性：$\mathbb{R}^n$ における閉凸集合 $C$ に対して、任意の $x$ に最近傍点が一意に存在する)
  - Proof (存在は下限列のコーシー性と完備性から示す；一意性は中点の議論と凸性の厳密不等式から導く)
  - Example 1.5-1 (半空間への射影の計算：$C = \{x \mid \langle a, x \rangle \leq b\}$ に対する $\mathrm{proj}_C(x)$ の明示公式を導出し、$x$ が $C$ の内部にある場合と外部にある場合を比較する)

- **2. 射影の特徴づけと非拡大性**
  - Proposition 1.5-1 (射影の特徴づけ：$p = \mathrm{proj}_C(x)$ であることと $\langle x - p, y - p \rangle \leq 0$ が任意の $y \in C$ で成り立つことが同値)
  - Proof (一次の最適性条件として、方向微分の非負性から内積条件を導く)
  - Theorem 1.5-2 (非拡大性：$\|\mathrm{proj}_C(x) - \mathrm{proj}_C(y)\| \leq \|x - y\|$)
  - Proof (特徴づけの不等式を二点 $x, y$ に適用し、内積の評価から非拡大性を導く)
  - Example 1.5-2 (非拡大性の数値的確認：$\mathbb{R}^2$ で $C = \{(x,y) \mid x+y \leq 1, x \geq 0, y \geq 0\}$ に対して2点の射影を計算し、距離が縮むことを検証する)
  - Remark (射影の非拡大性は近接作用素の理論（第7章）の出発点となる；関数解析（4-1）ではヒルベルト空間に一般化される)

### §1.6 サポート関数
**内容:** サポート関数の定義、凸集合の双対的記述としての役割、分離定理との接続

- **1. サポート関数の定義と性質**
  - Definition 1.6-1 (サポート関数：凸集合 $C$ に対して $\sigma_C(y) = \sup_{x \in C} \langle y, x \rangle$)
  - Proposition 1.6-1 (サポート関数は正斉次かつ劣加法的であり、したがって凸関数である)
  - Proof (正斉次性は定義の $\sup$ のスカラー倍から直ちに従う；劣加法性は $\sup$ の和に対する不等式から示す)
  - Example 1.6-1 (ノルム球のサポート関数：$C = \{x \mid \|x\|_2 \leq 1\}$ のとき $\sigma_C(y) = \|y\|_2$ となることの計算；コーシー・シュワルツの不等式を用いた上界と達成の確認)
  - Example 1.6-2 (多面体のサポート関数：$C = \{x \mid Ax \leq b\}$ の頂点集合を用いた $\sigma_C$ の計算；有限個の頂点上の最大値として表現されることの確認)

- **2. サポート関数による凸集合の復元**
  - Theorem 1.6-1 (閉凸集合はそのサポート関数から一意に復元できる：$C = \{x \mid \langle y, x \rangle \leq \sigma_C(y) \ \forall y\}$)
  - Proof (包含関係の一方は定義から直ちに成立する；逆は分離定理（Theorem 1.4-1）を適用し、$C$ の外の点 $x_0$ を分離する超平面の法線 $y_0$ が $\langle y_0, x_0 \rangle > \sigma_C(y_0)$ を満たすことを示す)
  - Example 1.6-3 ($\ell_1$-球 $\{x \mid \|x\|_1 \leq 1\}$ のサポート関数が $\sigma_C(y) = \|y\|_\infty$ であることの確認；$\langle y, x \rangle \leq \|y\|_\infty \|x\|_1$ から上界を得て、基底ベクトルの符号調整で達成を示す)
  - Remark (サポート関数は指示関数の共役関数（§4.2）であり、凸解析における集合と関数の双対性を結ぶ中心的な概念である)

## Chapter 2: 凸関数
**テーマ:** 定義・特徴づけ・下半連続性・真の凸関数・指示関数

### §2.1 凸関数の定義と例
**内容:** ジェンセンの不等式、指数関数・ノルム・二次形式などの例

- **1. 凸関数の定義**
  - Definition 2.1-1 (凸関数：$f: \mathbb{R}^n \to \mathbb{R} \cup \{+\infty\}$ が凸であるとは、任意の $x, y$ と $\lambda \in [0,1]$ に対して $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$ が成り立つこと)
  - Definition 2.1-2 (狭義凸関数：上の不等式が $x \neq y$ かつ $0 < \lambda < 1$ のとき厳密な不等式となる関数)
  - Example 2.1-1 (二次関数の凸性：$f(x) = x^2$ が凸であることの直接的検証；$\lambda x^2 + (1-\lambda)y^2 - (\lambda x + (1-\lambda)y)^2 = \lambda(1-\lambda)(x-y)^2 \geq 0$ の計算)
  - Example 2.1-2 (ノルムの凸性：任意のノルム $\|\cdot\|$ が凸関数であることの三角不等式と斉次性による確認)

- **2. ジェンセンの不等式**
  - Proposition 2.1-1 (ジェンセンの不等式の有限版：凸関数 $f$ に対して $f(\sum \lambda_i x_i) \leq \sum \lambda_i f(x_i)$)
  - Proof ($n$ に関する帰納法で示す)
  - Example 2.1-3 (指数関数のジェンセンの不等式：$f(x) = e^x$ に対して相加相乗平均の不等式が系として得られることの確認)
  - Remark (ジェンセンの不等式は確率論（3-7）でも $E[f(X)] \geq f(E[X])$ の形で本質的に用いられる)

### §2.2 エピグラフによる特徴づけ
**内容:** 凸関数とエピグラフの凸性の同値、下半連続性との関係

- **1. エピグラフの定義と凸性**
  - Definition 2.2-1 (エピグラフ：$\mathrm{epi}(f) = \{(x, \alpha) \in \mathbb{R}^n \times \mathbb{R} \mid f(x) \leq \alpha\}$)
  - Proposition 2.2-1 ($f$ が凸関数であることと $\mathrm{epi}(f)$ が $\mathbb{R}^{n+1}$ の凸集合であることは同値)
  - Proof (凸関数の不等式をエピグラフの凸結合に翻訳し、両方向を確認する)
  - Example 2.2-1 ($f(x) = |x|$ のエピグラフが $\mathbb{R}^2$ における V 字型の領域であることの図示的確認；$(−1, 1)$ と $(1, 1)$ の中点 $(0, 1)$ がエピグラフに含まれることの検証)

- **2. 下半連続性**
  - Definition 2.2-2 (下半連続関数：すべての $\alpha \in \mathbb{R}$ に対してレベル集合 $\{x \mid f(x) \leq \alpha\}$ が閉集合であること)
  - Proposition 2.2-2 ($f$ が下半連続であることと $\mathrm{epi}(f)$ が閉集合であることは同値)
  - Proof (レベル集合の閉性とエピグラフの閉性を点列の収束で翻訳する)
  - Example 2.2-2 ($f(x) = 0$（$x \neq 0$）、$f(0) = 1$ は下半連続でない；$f(x) = 0$（$x \neq 0$）、$f(0) = -1$ は下半連続；エピグラフの閉性との対応を確認する)
  - Remark (閉凸関数、すなわち下半連続かつ凸な関数はフェンシェル-モロの定理（§4.4）の主役となる)

### §2.3 真の凸関数と拡張実数値関数
**内容:** 有効定義域、真の凸関数の定義と取り扱い上の注意

- **1. 拡張実数値関数**
  - Definition 2.3-1 (拡張実数値関数：$f: \mathbb{R}^n \to \mathbb{R} \cup \{+\infty\}$ または $f: \mathbb{R}^n \to [-\infty, +\infty]$)
  - Definition 2.3-2 (有効定義域：$\mathrm{dom}(f) = \{x \mid f(x) < +\infty\}$)
  - Example 2.3-1 ($f(x) = \sqrt{x}$（$x \geq 0$）、$f(x) = +\infty$（$x < 0$）の有効定義域は $[0, \infty)$ であることの確認)

- **2. 真の凸関数**
  - Definition 2.3-3 (真の凸関数（proper convex function）：$f$ が凸であり、$f(x) > -\infty$ がすべての $x$ で成り立ち、かつ $\mathrm{dom}(f) \neq \emptyset$)
  - Example 2.3-2 ($f \equiv +\infty$ は凸だが真の凸関数ではない；$f(x) = x^2$ は真の凸関数であることの確認)
  - Example 2.3-3 ($f(x) = -\sqrt{x}$（$x \geq 0$）は凹関数であり、凸関数ではないことの検証；$x = 0, y = 4, \lambda = 1/2$ で不等式が成り立たないことを計算する)
  - Remark (共役関数やフェンシェル双対性の理論では「真の閉凸関数」の範疇で議論を行う；$-\infty$ を値域に許すと双共役の理論が破綻する)

### §2.4 指示関数と制約の凸解析的表現
**内容:** 凸集合の指示関数の定義、制約付き最適化の無制約問題への帰着、サポート関数が指示関数の共役であることの確認

- **1. 指示関数の定義**
  - Definition 2.4-1 (指示関数：凸集合 $C$ に対して $\delta_C(x) = 0$（$x \in C$）、$\delta_C(x) = +\infty$（$x \notin C$）)
  - Proposition 2.4-1 ($C$ が凸集合であることと $\delta_C$ が凸関数であることは同値)
  - Proof ($\delta_C$ の凸性を凸結合の条件で展開し、$C$ の凸性と同一であることを確認する)
  - Example 2.4-1 (非負直交象限 $\mathbb{R}^n_+$ の指示関数：$\delta_{\mathbb{R}^n_+}(x) = 0$（$x \geq 0$）、$+\infty$（それ以外）の確認)

- **2. 制約付き最適化への応用**
  - Proposition 2.4-2 (制約付き問題 $\min_{x \in C} f(x)$ は無制約問題 $\min_x f(x) + \delta_C(x)$ と同値)
  - Proof (最適解の条件が同一であることを有効定義域の制限により確認する)
  - Example 2.4-2 ($\min_{x \geq 0} x^2 - 2x$ を $\min_x (x^2 - 2x) + \delta_{[0,\infty)}(x)$ に書き換え、最適性条件 $0 \in \partial(f + \delta_C)(x^*)$ の形に帰着することの確認)
  - Remark (指示関数の共役関数がサポート関数（§1.6）に一致することは §4.2 で確認する；この対応は凸解析の中心的な双対構造である)

### §2.5 凸関数の連続性と微分可能性
**内容:** 内点での連続性定理、局所リプシッツ性、方向微分の存在と凸性の関係、ラデマッハーの定理による概微分可能性

- **1. 内点での連続性と局所リプシッツ性**
  - Theorem 2.5-1 (凸関数は有効定義域の内点で連続である)
  - Proof (凸結合による上界評価と下界評価を組み合わせ、凸性から局所有界性を導き、そこから連続性を示す)
  - Corollary 2.5-1 (凸関数は有効定義域の内部の任意のコンパクト集合上で局所リプシッツ連続)
  - Proof (内点の近傍で上界と下界が有界であることから、差分の評価によりリプシッツ定数を構成する)
  - Example 2.5-1 ($f(x) = |x|$ は $\mathbb{R}$ 上でリプシッツ定数 $L = 1$ のリプシッツ連続；$|f(x) - f(y)| \leq |x - y|$ の確認)
  - Example 2.5-2 (境界点での連続性の崩壊：$f(x) = 0$（$0 \leq x < 1$）、$f(1) = 1$ は $[0,1]$ 上で凸だが $x = 1$ で連続でないことの確認)

- **2. 方向微分と概微分可能性**
  - Definition 2.5-1 (方向微分：$f'(x; d) = \lim_{t \downarrow 0} (f(x+td) - f(x))/t$)
  - Proposition 2.5-1 (凸関数は $\mathrm{int}(\mathrm{dom}(f))$ の各点で任意の方向に方向微分が存在し、$d \mapsto f'(x; d)$ は劣線形関数である)
  - Proof (差分商 $(f(x+td)-f(x))/t$ が $t$ について単調非減少であることを凸性から示す；有界下から単調な関数の極限が存在する)
  - Theorem 2.5-2 (ラデマッハーの定理との接続：凸関数は有効定義域の内部でほとんどすべての点で微分可能；局所リプシッツ性とラデマッハーの定理（測度論（3-3）の結果）を適用)
  - Proof (Corollary 2.5-1 により凸関数が局所リプシッツ連続であることを確認し、ラデマッハーの定理を適用する)
  - Example 2.5-3 ($f(x) = |x|$ は $x = 0$ で微分不可能だが、$x \neq 0$ で微分可能；微分不可能点の集合は $\{0\}$ でルベーグ測度 0 である)
  - Remark (方向微分の存在は劣微分（第3章）の定義への橋渡しであり、劣線形関数としての方向微分は劣勾配の支持超平面的役割を反映する)

### §2.6 強凸関数
**内容:** 強凸性の定義、最小値の存在・一意性への応用

- **1. 強凸関数の定義**
  - Definition 2.6-1 (強凸関数：パラメータ $m > 0$ に対して $f(x) - \frac{m}{2}\|x\|^2$ が凸であること；同値的に $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y) - \frac{m}{2}\lambda(1-\lambda)\|x-y\|^2$)
  - Example 2.6-1 ($f(x) = \frac{1}{2}x^T A x$ が強凸であるための条件：$A$ の最小固有値 $\lambda_{\min}(A) > 0$；$A = \begin{pmatrix} 2 & 0 \\ 0 & 1 \end{pmatrix}$ のとき $m = 1$ で強凸であることの確認)
  - Example 2.6-2 ($f(x) = |x|$ は凸だが強凸ではない；$x = 1, y = -1, \lambda = 1/2$ で強凸性の不等式が任意の $m > 0$ に対して成立しないことの検証)

- **2. 強凸関数の最小化**
  - Theorem 2.6-1 (真の閉強凸関数は一意の最小値を持つ)
  - Proof (強凸性から最小化列がコーシー列となることを示し、完備性から収束を得る；一意性は狭義凸性から従う)
  - Example 2.6-3 ($f(x) = x^2 + e^{-x}$ は $\mathbb{R}$ 上で強凸であり、数値的に $x^* \approx 0.3517$ に唯一の最小値を持つ)
  - Remark (強凸性は勾配法の線形収束率（最適化理論（A-3）§7.2）の鍵であり、正則化項 $\frac{m}{2}\|x\|^2$ を加えることで強凸化する手法は機械学習で広く用いられる)

## Chapter 3: 劣微分
**テーマ:** 劣勾配・劣微分の定義・計算規則・制約想定・最適性条件

### §3.1 劣勾配と劣微分の定義
**内容:** 微分不可能点での接超平面の一般化、劣微分集合の凸・コンパクト性

- **1. 劣勾配の定義**
  - Definition 3.1-1 (劣勾配：凸関数 $f$ に対して $g \in \mathbb{R}^n$ が $x$ における劣勾配であるとは、すべての $y$ に対して $f(y) \geq f(x) + \langle g, y - x \rangle$ が成り立つこと)
  - Definition 3.1-2 (劣微分：$x$ における劣勾配全体の集合 $\partial f(x)$)
  - Example 3.1-1 ($f(x) = |x|$ の劣微分の計算：$x > 0$ のとき $\partial f(x) = \{1\}$、$x < 0$ のとき $\partial f(x) = \{-1\}$、$x = 0$ のとき $\partial f(0) = [-1, 1]$；$x = 0$ の場合は $|y| \geq gy$ が $g \in [-1,1]$ と同値であることの確認)

- **2. 劣微分の幾何的意味と基本性質**
  - Proposition 3.1-1 (劣微分 $\partial f(x)$ は閉凸集合であり、$x \in \mathrm{int}(\mathrm{dom}(f))$ のとき空でなくコンパクトである)
  - Proof (閉凸性は半空間の共通部分として書けることから従う；空でないことは方向微分の劣線形性とハーン-バナッハの分離論法により示す；コンパクト性は劣勾配の大きさが局所リプシッツ定数で抑えられることから従う)
  - Proposition 3.1-2 ($f$ が $x$ で微分可能ならば $\partial f(x) = \{\nabla f(x)\}$)
  - Proof (微分可能性から一階近似の等式が成り立ち、劣勾配が勾配に一致することを示す)
  - Example 3.1-2 ($f(x) = x^2$ の劣微分：すべての点で $\partial f(x) = \{2x\}$ であることの確認；微分可能な凸関数では劣微分が通常の微分に退化する)
  - Remark (劣微分は「微分の凸解析的一般化」であり、非平滑最適化の理論的基盤となる)

### §3.2 劣微分の計算例
**内容:** 絶対値関数・ノルム・最大値関数・指示関数の劣微分の計算

- **1. ノルムの劣微分**
  - Example 3.2-1 ($f(x) = \|x\|_2$ の劣微分：$x \neq 0$ のとき $\partial f(x) = \{x/\|x\|_2\}$；$x = 0$ のとき $\partial f(0) = \{g \mid \|g\|_2 \leq 1\}$（単位球）；劣勾配条件 $\|y\| \geq \langle g, y \rangle$ がコーシー・シュワルツの不等式から $\|g\| \leq 1$ と同値であることの確認)
  - Example 3.2-2 ($f(x) = \|x\|_1$ の劣微分：各成分 $g_i$ が $x_i > 0$ なら $g_i = 1$、$x_i < 0$ なら $g_i = -1$、$x_i = 0$ なら $g_i \in [-1,1]$ となることの確認)

- **2. 最大値関数と指示関数の劣微分**
  - Example 3.2-3 (最大値関数 $f(x) = \max(f_1(x), f_2(x))$ の劣微分：活性な関数の劣微分の凸包として表されること；$f(x) = \max(x, -x) = |x|$ の $x = 0$ での計算で $\partial f(0) = \mathrm{conv}\{1, -1\} = [-1, 1]$ となることの検証)
  - Example 3.2-4 (指示関数の劣微分：$\partial \delta_C(x)$ は $x \in C$ のとき正規錐 $N_C(x) = \{g \mid \langle g, y - x \rangle \leq 0 \ \forall y \in C\}$ に等しいことの確認；$C = [0, \infty)$ で $x = 0$ のとき $N_C(0) = (-\infty, 0]$ の計算)
  - Remark (指示関数の劣微分が正規錐であるという事実は、制約付き最適化の最適性条件（§3.5）で中心的な役割を果たす)

### §3.3 劣微分の和の公式と制約想定
**内容:** 和の公式の一般的な包含関係、相対的内部の交差条件による等号成立の証明

- **1. 和の公式の包含関係**
  - Proposition 3.3-1 (包含関係：任意の凸関数 $f, g$ に対して $\partial f(x) + \partial g(x) \subseteq \partial(f+g)(x)$)
  - Proof (劣勾配の定義の不等式を加え合わせる)
  - Example 3.3-1 (等号が成り立つ場合：$f(x) = x^2$、$g(x) = |x|$ のとき $\partial(f+g)(0) = \{0\} + [-1,1] = [-1,1]$；直接計算でも $\partial(x^2 + |x|)(0) = [-1,1]$ となることの確認)

- **2. 和の公式の等号条件**
  - Theorem 3.3-1 (モロー-ロカフェラーの定理：$\mathrm{ri}(\mathrm{dom}(f)) \cap \mathrm{ri}(\mathrm{dom}(g)) \neq \emptyset$ ならば $\partial(f+g)(x) = \partial f(x) + \partial g(x)$)
  - Proof (分離定理を $\mathrm{epi}(f)$ と $\mathrm{epi}(g)$ に適用し、相対的内部条件から等号を保証する；エピグラフの和とインフィマル畳み込みの関係を利用する)
  - Example 3.3-2 (制約想定が崩れる場合：$f(x) = \delta_{\{0\}}(x)$、$g(x) = \delta_{\{1\}}(x)$ のとき $\mathrm{dom}(f) \cap \mathrm{dom}(g) = \emptyset$ であり、$(f+g)(x) \equiv +\infty$ なので劣微分は常に空集合；包含は意味をなすが等号の前提が欠けている)
  - Remark (この制約想定は $\mathrm{dom}(f)$ と $\mathrm{dom}(g)$ が「十分に重なっている」ことを要求しており、フェンシェル双対の強双対定理（§5.2）でも同型の条件が現れる)

### §3.4 劣微分の合成則
**内容:** アフィン写像との合成の劣微分、スカラー倍の公式、一般の合成に対する連鎖律の制約想定

- **1. アフィン写像との合成**
  - Proposition 3.4-1 ($h(x) = f(Ax + b)$ のとき $\partial h(x) = A^T \partial f(Ax + b)$；ただし $A$ の値域と $\mathrm{ri}(\mathrm{dom}(f))$ が交わる制約想定の下)
  - Proof (劣勾配の定義に $y = Ax + b$ の代入を行い、変数変換により示す)
  - Example 3.4-1 ($h(x) = |2x - 1|$ の劣微分：$f(t) = |t|$、$A = 2$、$b = -1$ として $\partial h(x) = 2 \cdot \partial f(2x-1)$；$x = 1/2$ で $\partial h(1/2) = 2 \cdot [-1,1] = [-2, 2]$ となることの計算)

- **2. スカラー倍と正の同次性**
  - Proposition 3.4-2 ($\alpha > 0$ のとき $\partial(\alpha f)(x) = \alpha \, \partial f(x)$)
  - Proof (劣勾配の定義の不等式両辺に $\alpha$ を乗じる)
  - Example 3.4-2 ($f(x) = 3|x|$ の劣微分：$\partial(3|x|)(0) = 3 \cdot [-1,1] = [-3, 3]$ の確認)
  - Remark (一般の非線形合成 $f \circ g$ に対する連鎖律は、$g$ が微分可能で $f$ が単調非減少凸関数のような特殊な場合にしか単純な形では成立しない；凸解析では合成を避け、和やアフィン変換に帰着する手法が基本となる)

### §3.5 最適性条件
**内容:** 無制約最小化問題における $0 \in \partial f$ の条件、指示関数を用いた制約付き問題への応用

- **1. 無制約問題の最適性**
  - Theorem 3.5-1 (フェルマの規則：$x^*$ が凸関数 $f$ の最小値を達成するための必要十分条件は $0 \in \partial f(x^*)$)
  - Proof (劣勾配の定義に $g = 0$ を代入すると $f(y) \geq f(x^*)$ が全 $y$ で成立し、最小性が従う；逆に最小性から劣勾配条件を導く)
  - Example 3.5-1 ($f(x) = |x - 3| + 2|x|$ の最小値を求める：$\partial f(x) = \partial|x-3| + \partial(2|x|)$；$x = 0$ で $\partial f(0) = \{-1\} + [-2, 2] = [-3, 1] \ni 0$ なので $x^* = 0$ は最小値ではない計算を実行し、$0 < x < 3$ の範囲で $\partial f(x) = \{-1\} + \{2\} = \{1\} \not\ni 0$ を確認、正しい最適解の探索を行う)

- **2. 制約付き問題の最適性**
  - Corollary 3.5-1 ($\min_{x \in C} f(x)$ の最適性条件は $0 \in \partial f(x^*) + N_C(x^*)$)
  - Proof ($\min_x f(x) + \delta_C(x)$ に Theorem 3.5-1 を適用し、和の公式（Theorem 3.3-1）を用いて $\partial(f + \delta_C) = \partial f + \partial \delta_C = \partial f + N_C$ と展開する)
  - Example 3.5-2 ($\min_{x \geq 0} (x - 1)^2$ の最適性条件：$\partial f(x) = \{2(x-1)\}$、$N_{[0,\infty)}(0) = (-\infty, 0]$；$x = 0$ で $0 \in \{-2\} + (-\infty, 0]$ は $-2 \leq 0$ より成立しないので $x^* = 0$ は最適でない；$x = 1$ で $0 \in \{0\} + \{0\}$ が成立し $x^* = 1$ が最適解であることの確認)
  - Remark (この条件は KKT 条件（§5.4）の凸解析的な一般化であり、非平滑最適化における最適性の基本判定手段である)

## Chapter 4: 共役関数（フェンシェル共役）
**テーマ:** 定義・例・双共役・フェンシェル-モロの定理・インフィマル畳み込み

### §4.1 共役関数の定義
**内容:** ルジャンドル-フェンシェル変換による定義と幾何的意味

- **1. 共役関数の定義**
  - Definition 4.1-1 (共役関数（フェンシェル共役）：$f^*(y) = \sup_{x} \{\langle y, x \rangle - f(x)\}$)
  - Proposition 4.1-1 ($f^*$ は常に閉凸関数である（$f$ の凸性を問わない）)
  - Proof ($f^*$ はアフィン関数の上限として表されるため、凸かつ下半連続である)
  - Example 4.1-1 (幾何的意味：$y$ を固定して $\langle y, x \rangle - f(x)$ のグラフを考えると、$f^*(y)$ は傾き $y$ の直線で $f$ のグラフの下を通るもののうち最も高い切片に等しいことの確認)

- **2. ヤング-フェンシェルの不等式**
  - Proposition 4.1-2 (ヤング-フェンシェルの不等式：すべての $x, y$ に対して $f(x) + f^*(y) \geq \langle x, y \rangle$)
  - Proof (共役関数の定義の $\sup$ が $\langle y, x \rangle - f(x)$ 以上であることから直ちに従う)
  - Proposition 4.1-3 (等号条件：$f$ が閉真凸関数のとき $f(x) + f^*(y) = \langle x, y \rangle$ と $y \in \partial f(x)$ は同値)
  - Proof ($y \in \partial f(x)$ の定義式を変形して $\sup$ の達成に帰着する)
  - Example 4.1-2 ($f(x) = \frac{1}{2}x^2$ のとき $f^*(y) = \frac{1}{2}y^2$；$x = 3, y = 3$ で $f(3) + f^*(3) = 9/2 + 9/2 = 9 = \langle 3, 3 \rangle$；$\nabla f(3) = 3 = y$ で等号条件の確認)

### §4.2 共役関数の計算例
**内容:** 二次関数・指数関数・ノルム・指示関数の共役関数の計算、指示関数とサポート関数の双対関係の確認

- **1. 基本的な共役関数の計算**
  - Example 4.2-1 (二次関数：$f(x) = \frac{1}{2}x^T A x$（$A$ 正定値）のとき $f^*(y) = \frac{1}{2}y^T A^{-1} y$；最大化の一階条件 $Ax = y$ から $x = A^{-1}y$ を代入して計算する)
  - Example 4.2-2 (指数関数：$f(x) = e^x$ のとき $f^*(y) = y \log y - y$（$y > 0$）、$f^*(0) = 0$、$f^*(y) = +\infty$（$y < 0$）；$\sup_x (yx - e^x)$ の一階条件 $y = e^x$ から $x = \log y$ を代入して計算する)
  - Example 4.2-3 (ノルムの共役：$f(x) = \|x\|$ のとき $f^*(y) = \delta_{B^*}(y)$（双対ノルム単位球の指示関数）；$\|y\|_* \leq 1$ のとき $\sup_x (\langle y, x \rangle - \|x\|) = 0$、$\|y\|_* > 1$ のとき $\sup = +\infty$ であることの確認)

- **2. 指示関数とサポート関数の双対関係**
  - Proposition 4.2-1 (凸集合 $C$ の指示関数の共役はサポート関数：$\delta_C^*(y) = \sigma_C(y)$)
  - Proof ($\delta_C^*(y) = \sup_x \{\langle y, x \rangle - \delta_C(x)\} = \sup_{x \in C} \langle y, x \rangle = \sigma_C(y)$)
  - Example 4.2-4 (ユークリッド単位球 $B = \{x \mid \|x\|_2 \leq 1\}$ の指示関数の共役が $\delta_B^*(y) = \|y\|_2$ であることの確認；§1.6 の Example 1.6-1 との整合性)
  - Remark (この双対関係により、凸集合に関する議論を凸関数の共役の枠組みで統一的に扱えるようになる)

### §4.3 双共役と下半連続凸包
**内容:** 双共役が閉じた凸関数の最大の下界となることの証明

- **1. 双共役の定義と性質**
  - Definition 4.3-1 (双共役関数：$f^{**}(x) = (f^*)^*(x) = \sup_y \{\langle x, y \rangle - f^*(y)\}$)
  - Proposition 4.3-1 (任意の関数 $f$ に対して $f^{**} \leq f$)
  - Proof (ヤング-フェンシェルの不等式 $f^*(y) \geq \langle y, x \rangle - f(x)$ において $y$ の $\sup$ を取ることで $f^{**}(x) \leq f(x)$ を導く)
  - Example 4.3-1 ($f(x) = x^2$ は閉凸関数なので $f^{**} = f$；実際に $f^*(y) = y^2/4$ を計算し、$f^{**}(x) = \sup_y (xy - y^2/4) = x^2$ を確認する)

- **2. 双共役の幾何的意味**
  - Theorem 4.3-1 ($f^{**}$ は $f$ 以下のすべての閉凸関数の上限に等しい（すなわち $f$ の閉凸包）)
  - Proof (エピグラフの閉凸包と双共役のエピグラフの一致を分離定理により示す)
  - Example 4.3-2 (非凸関数の双共役：$f(x) = x^2(x-1)^2$ の双共役は $[0,1]$ の間で $f$ の下側凸包となることの確認；$f^{**}(x) \leq f(x)$ だが $f^{**}(1/2) < f(1/2)$ となる)
  - Remark (双共役は「凸化」の操作であり、非凸最適化問題の凸緩和の理論的基盤となる)

### §4.4 フェンシェル-モロの定理
**内容:** 閉真凸関数が双共役に一致することの証明

- **1. フェンシェル-モロの定理**
  - Theorem 4.4-1 (フェンシェル-モロの定理：$f$ が閉真凸関数ならば $f^{**} = f$)
  - Proof ($f^{**} \leq f$ は Proposition 4.3-1 で示済み；逆向きは $f$ が閉凸であるとき $\mathrm{epi}(f)$ が閉凸集合であり、分離定理を適用して $f^{**}(x) \geq f(x)$ を示す)
  - Example 4.4-1 ($f(x) = |x|$ は閉真凸関数であり $f^{**} = f$；$f^*(y) = \delta_{[-1,1]}(y)$ を計算し、$f^{**}(x) = \sup_{|y| \leq 1} xy = |x|$ を確認する)
  - Example 4.4-2 (閉でない凸関数の反例：$f(x) = -\sqrt{1-x^2}$（$|x| < 1$）、$f(\pm 1) = +\infty$ は凸だが閉でなく、$f^{**}$ は端点を含む閉包に修正されることの確認)
  - Remark (フェンシェル-モロの定理は閉真凸関数の圏における「完全な双対性」を保証し、共役変換が対合であること $f \mapsto f^* \mapsto f^{**} = f$ を意味する)

### §4.5 インフィマル畳み込み
**内容:** インフィマル畳み込みの定義、共役関数の和との双対関係、モロー包絡への接続

- **1. インフィマル畳み込みの定義**
  - Definition 4.5-1 (インフィマル畳み込み：$(f \Box g)(x) = \inf_y \{f(y) + g(x - y)\}$)
  - Proposition 4.5-1 ($f, g$ が凸関数ならば $f \Box g$ も凸関数である)
  - Proof ($f \Box g$ の凸性を凸結合に関する $\inf$ の評価により示す)
  - Example 4.5-1 ($f(x) = \frac{1}{2}x^2$、$g(x) = |x|$ のインフィマル畳み込み：$(f \Box g)(x) = \inf_y (\frac{1}{2}y^2 + |x - y|)$ を $y$ について最適化し、モロー包絡の形に帰着する計算)

- **2. 共役関数との双対関係**
  - Theorem 4.5-1 ($(f \Box g)^* = f^* + g^*$；ただし $f, g$ は真凸関数)
  - Proof ($\sup_{x} \inf_{y} \{\langle p, x \rangle - f(y) - g(x-y)\}$ を変数変換 $z = x - y$ として分離し、$\sup$ の独立性から $f^*(p) + g^*(p)$ を導く)
  - Corollary 4.5-1 ($f, g$ が閉真凸関数で適切な制約想定が満たされるとき $(f + g)^* = f^* \Box g^*$)
  - Proof (フェンシェル-モロの定理を適用して Theorem 4.5-1 の双対を取る)
  - Example 4.5-2 (モロー包絡としての解釈：$M_{\lambda f}(x) = \inf_y \{f(y) + \frac{1}{2\lambda}\|x-y\|^2\} = (f \Box \frac{1}{2\lambda}\|\cdot\|^2)(x)$；$f(x) = |x|$、$\lambda = 1$ のとき $M_f(x) = \frac{1}{2}x^2$（$|x| \leq 1$）、$|x| - \frac{1}{2}$（$|x| > 1$）となることの計算)
  - Remark (インフィマル畳み込みは凸関数の「平滑化」の操作と見なせる；モロー包絡は近接作用素（第7章）の理論的基盤であり、非平滑関数を滑らかな関数に置き換える役割を果たす)

## Chapter 5: フェンシェル双対性
**テーマ:** 主問題と双対問題・双対ギャップ・強双対定理・KKT条件・ミニマックス定理

### §5.1 主問題と双対問題の設定
**内容:** フェンシェル双対問題の定式化と弱双対定理

- **1. フェンシェル双対問題の定式化**
  - Definition 5.1-1 (フェンシェル双対問題：主問題 $\inf_x \{f(x) + g(Ax)\}$ に対する双対問題 $\sup_y \{-f^*(-A^T y) - g^*(y)\}$)
  - Remark (ラグランジュ乗数法が等式・不等式制約を扱うのに対し、フェンシェル双対は共役関数を通じてより一般的な構造を持つ)

- **2. 弱双対定理**
  - Theorem 5.1-1 (弱双対定理：主問題の最適値は双対問題の最適値以上である)
  - Proof (任意の実行可能な $x$ と $y$ に対してヤング-フェンシェルの不等式 $f(x) \geq \langle x, z \rangle - f^*(z)$ と $g(Ax) \geq \langle Ax, y \rangle - g^*(y)$ を適用し、$z = -A^T y$ と置いて加え合わせる)
  - Example 5.1-1 (具体的な弱双対の確認：$\min_x (x^2 + |x-1|)$ の主問題に対するフェンシェル双対を $f(x) = x^2$、$g(x) = |x-1|$ として定式化し、双対最適値が主最適値以下であることを数値的に確認する)
  - Definition 5.1-2 (双対ギャップ：主問題の最適値と双対問題の最適値の差)

### §5.2 強双対定理とスレーターの条件
**内容:** 双対ギャップが零になるための制約想定の条件、相対的内部条件による証明

- **1. 強双対定理**
  - Theorem 5.2-1 (フェンシェルの強双対定理：$f, g$ が閉真凸関数で $\mathrm{ri}(\mathrm{dom}(f)) \cap A^{-1}(\mathrm{ri}(\mathrm{dom}(g))) \neq \emptyset$ ならば双対ギャップは零であり、双対問題の最大値が達成される)
  - Proof (エピグラフの分離定理を適用し、制約想定により分離超平面が定数でないことを保証する；相対的内部の交差条件がエピグラフの重なりを保証する役割を示す)
  - Example 5.2-1 ($\min_{x} \|x\|_1$ subject to $Ax = b$ の問題にフェンシェル双対を適用し、$f = \|\cdot\|_1$、$g = \delta_{\{b\}}$ として双対問題が $\max_y \langle b, y \rangle$ subject to $\|A^T y\|_\infty \leq 1$ となることの導出)

- **2. 制約想定が崩れる場合**
  - Example 5.2-2 (双対ギャップが正となる例：$\mathbb{R}^2$ 上で $f(x_1, x_2) = \delta_{\{x_1 = 0\}}$ と $g(x_1, x_2) = \delta_{\{x_2 = 0\}}$ に不適切な $A$ を設定し、$\mathrm{dom}(f) \cap A^{-1}(\mathrm{dom}(g))$ が相対的内部で交わらない状況を構成する)
  - Remark (スレーターの条件は凸最適化における最も基本的な制約想定であり、最適化理論（A-3）での凸計画法とも共通する；最適輸送理論（A-4）の双対問題にも同型の条件が現れる)

### §5.3 ラグランジュ双対性との関係
**内容:** ラグランジュ乗数法とフェンシェル双対の統一的な見方、最適化理論（A-3）との棲み分け

- **1. ラグランジュ双対問題の導出**
  - Definition 5.3-1 (ラグランジュ関数：$\mathcal{L}(x, \lambda, \mu) = f(x) + \lambda^T h(x) + \mu^T g(x)$；等式制約 $h(x) = 0$、不等式制約 $g(x) \leq 0$)
  - Proposition 5.3-1 (ラグランジュ双対問題 $\max_{\lambda, \mu \geq 0} \inf_x \mathcal{L}(x, \lambda, \mu)$ がフェンシェル双対の特殊場合として得られること)
  - Proof (制約を指示関数で表現し、共役関数の計算を通じてフェンシェル双対と一致することを示す)
  - Example 5.3-1 (二次計画問題 $\min \frac{1}{2}x^T Q x + c^T x$ subject to $Ax \leq b$ のラグランジュ双対を計算し、フェンシェル双対と一致することの確認)

- **2. 最適化理論（A-3）との棲み分け**
  - Remark (最適化理論（A-3）では凸最適化を応用的観点からアルゴリズムの設計と解析に重点を置いて扱い、本コースでは共役関数やフェンシェル双対性の理論的構造を厳密に展開する；両者は KKT 条件で合流する)

### §5.4 KKT条件
**内容:** 鞍点条件・相補スラック性・最適性の一階条件の導出

- **1. KKT条件の導出**
  - Theorem 5.4-1 (KKT条件：凸最適化問題 $\min f(x)$ subject to $g_i(x) \leq 0$、$h_j(x) = 0$ において、スレーターの条件の下、$x^*$ が最適解であるための必要十分条件は、$\mu_i^* \geq 0$、$\mu_i^* g_i(x^*) = 0$（相補スラック性）かつ $0 \in \partial f(x^*) + \sum_i \mu_i^* \partial g_i(x^*) + \sum_j \lambda_j^* \nabla h_j(x^*)$ を満たす乗数 $\mu^*, \lambda^*$ が存在すること)
  - Proof (強双対定理から鞍点の存在を示し、鞍点条件から劣微分を用いた最適性条件を導出する；相補スラック性は鞍点における $\mu$ の最大化から従う)
  - Example 5.4-1 ($\min x^2 + y^2$ subject to $x + y \geq 1$ の KKT 条件：$\nabla f = (2x, 2y)$、$\mu(-(x+y-1)) = 0$ から $\mu = 1$、$x = y = 1/2$ を導出する)

- **2. 相補スラック性の意味**
  - Proposition 5.4-1 (相補スラック性 $\mu_i g_i(x^*) = 0$ は「制約が有効でなければ乗数は零」を意味する)
  - Proof (鞍点条件においてラグランジュ関数を $\mu_i$ について最大化する際、$g_i(x^*) < 0$ ならば $\mu_i = 0$ が最大を与えることから従う)
  - Example 5.4-2 ($\min (x-2)^2$ subject to $x \geq 0$、$x \leq 3$ の解析：$x^* = 2$ で両制約とも非有効であり $\mu_1 = \mu_2 = 0$；$x^* = 0$ は KKT 条件を満たさないことの確認)
  - Remark (KKT 条件は微分可能な場合のラグランジュ乗数法の一般化であり、微分積分学II（2-2）で扱ったラグランジュ乗数法の精密化に位置づけられる)

### §5.5 ミニマックス定理と鞍点
**内容:** 鞍点と max-min = min-max の一致条件、フォン・ノイマンのミニマックス定理のフェンシェル双対性からの導出、シオンのミニマックス定理への一般化

- **1. 鞍点の定義と双対性**
  - Definition 5.5-1 (鞍点：$(x^*, y^*)$ が $L(x, y)$ の鞍点であるとは $L(x^*, y) \leq L(x^*, y^*) \leq L(x, y^*)$ がすべての $x, y$ で成り立つこと)
  - Proposition 5.5-1 ($(x^*, y^*)$ が鞍点ならば $\max_y \min_x L(x, y) = \min_x \max_y L(x, y) = L(x^*, y^*)$)
  - Proof ($\min_x L(x, y^*) \leq L(x^*, y^*) \leq \max_y L(x^*, y)$ と一般の不等式 $\max\min \leq \min\max$ を合わせて等号を導く)
  - Example 5.5-1 ($L(x, y) = x^2 - y^2 + 2xy$ について $(0, 0)$ が鞍点であるかの検証；$\partial L / \partial x = 2x + 2y = 0$、$\partial L / \partial y = -2y + 2x = 0$ から $(x, y) = (0, 0)$；$L(0, y) = -y^2 \leq 0 = L(0, 0) \leq x^2 = L(x, 0)$ の確認)

- **2. フォン・ノイマンのミニマックス定理**
  - Theorem 5.5-1 (フォン・ノイマンのミニマックス定理：$X, Y$ がコンパクト凸集合、$L(x, y)$ が $x$ に関して凸、$y$ に関して凹で連続ならば $\max_y \min_x L(x, y) = \min_x \max_y L(x, y)$)
  - Proof (フェンシェル双対の強双対定理を適用する方法の概略：$L$ を適切に凸関数と凹関数に分解し、共役関数の枠組みで双対ギャップが零であることを示す)
  - Example 5.5-2 (ゼロ和行列ゲーム：$A = \begin{pmatrix} 3 & 1 \\ 0 & 4 \end{pmatrix}$ に対して $L(x, y) = x^T A y$、$X, Y$ は確率単体；$\max_y \min_x x^T A y$ と $\min_x \max_y x^T A y$ の一致を具体計算で確認する)
  - Theorem 5.5-2 (シオンのミニマックス定理：$Y$ のコンパクト性のみを仮定し、$X$ は凸集合、$L$ は $x$ に凸・$y$ に準凹で適切な連続性条件を満たせば結論が成り立つ)
  - Proof (証明の概略：$Y$ のコンパクト性と $x$ の凸性から有限交差性を用いた論法で示す)
  - Remark (ミニマックス定理はゲーム理論（A-10）の基礎であり、混合戦略ナッシュ均衡の存在証明に直接用いられる)

## Chapter 6: 単調作用素と凸解析
**テーマ:** 単調作用素・劣微分の単調性・極大単調性・ミンティの定理

### §6.1 単調作用素の定義と例
**内容:** 単調作用素・狭義単調作用素の定義、勾配写像・劣微分の例

- **1. 単調作用素の定義**
  - Definition 6.1-1 (単調作用素：集合値写像 $T: \mathbb{R}^n \rightrightarrows \mathbb{R}^n$ が単調であるとは、$u \in T(x)$、$v \in T(y)$ ならば $\langle u - v, x - y \rangle \geq 0$ が成り立つこと)
  - Definition 6.1-2 (狭義単調作用素：$x \neq y$ のとき上の不等式が厳密に成り立つ作用素)
  - Example 6.1-1 ($T(x) = Ax$ で $A$ が正定値対称行列のとき $T$ は狭義単調；$\langle Ax - Ay, x - y \rangle = (x-y)^T A (x-y) > 0$ の確認)
  - Example 6.1-2 ($\mathbb{R}$ 上の増加関数 $T(x) = x^3$ は単調作用素；$(x^3 - y^3)(x - y) \geq 0$ が因数分解により確認できる)

- **2. 劣微分は単調作用素**
  - Proposition 6.1-1 (凸関数 $f$ の劣微分 $\partial f$ は単調作用素である)
  - Proof ($u \in \partial f(x)$、$v \in \partial f(y)$ のとき劣勾配の定義から $f(y) \geq f(x) + \langle u, y - x \rangle$ と $f(x) \geq f(y) + \langle v, x - y \rangle$ を加えて $\langle u - v, x - y \rangle \geq 0$ を導く)
  - Example 6.1-3 ($f(x) = |x|$ の劣微分 $\partial f$ の単調性の検証：$x = 1$ で $u = 1$、$y = -1$ で $v = -1$ のとき $(1 - (-1))(1 - (-1)) = 4 \geq 0$；$x = 0$ で $u = 1/2$、$y = 2$ で $v = 1$ のとき $(1/2 - 1)(0 - 2) = 1 \geq 0$)
  - Remark (単調作用素の理論は凸最適化と変分不等式を統一的に扱う枠組みであり、関数解析（4-1）で無限次元に一般化される)

### §6.2 劣微分の極大単調性
**内容:** 閉真凸関数の劣微分が極大単調作用素であることの証明

- **1. 極大単調作用素の定義**
  - Definition 6.2-1 (極大単調作用素：$T$ が単調であり、$T$ のグラフを真に含む単調作用素が存在しないこと)
  - Example 6.2-1 ($T(x) = \{x\}$（恒等作用素）は極大単調；$T$ のグラフに $(x_0, u_0)$ で $u_0 \neq x_0$ を追加すると単調性が崩れることの確認)

- **2. ロカフェラーの定理**
  - Theorem 6.2-1 (ロカフェラーの定理：$f$ が閉真凸関数ならば $\partial f$ は極大単調作用素である)
  - Proof (証明の核心：$(x_0, u_0) \notin \mathrm{graph}(\partial f)$ のとき $u_0 \notin \partial f(x_0)$ であり、劣勾配条件を満たさない方向が存在する；フェンシェル-モロの定理と共役関数の性質を用いて、$\langle u_0 - v, x_0 - y \rangle < 0$ となる $(y, v) \in \mathrm{graph}(\partial f)$ を構成する)
  - Example 6.2-2 ($f(x) = \delta_{[0,1]}(x)$ の劣微分 $\partial f(x) = N_{[0,1]}(x)$（正規錐）が極大単調であることの確認：$x \in (0,1)$ で $N = \{0\}$、$x = 0$ で $N = (-\infty, 0]$、$x = 1$ で $N = [0, +\infty)$；グラフに点を追加すると単調性が崩れることの確認)
  - Remark (逆に、有限次元では任意の極大単調作用素が劣微分で表せるわけではない；微分方程式の弱解の理論では極大単調性が解の存在を保証する鍵となる)

### §6.3 ミンティの定理
**内容:** 極大単調作用素の値域条件と解の存在定理

- **1. ミンティの定理**
  - Theorem 6.3-1 (ミンティの定理（有限次元版）：$T$ が極大単調作用素で強制的（coercive）ならば $T$ は全射である；すなわち任意の $b$ に対して $b \in T(x)$ を満たす $x$ が存在する)
  - Proof (証明の核心：$\mathbb{R}^n$ の有限次元性を用いて Brouwer の不動点定理を適用する；ヨシダ近似 $T_\lambda = (J_\lambda - I)/\lambda$（$J_\lambda$ は $T$ のレゾルベント）を導入し、$\lambda \to 0$ で極限を取る)
  - Example 6.3-1 ($T = \partial f$ で $f(x) = \frac{1}{2}\|x\|^2 + \|x\|_1$ のとき $0 \in T(x)$ の解が $x = 0$ であることの確認；$\partial f(0) = \{0\} + [-1,1]^n \ni 0$)

- **2. 凸最適化への応用**
  - Corollary 6.3-1 ($f$ が閉真凸関数で $\|x\| \to \infty$ のとき $f(x) \to \infty$（強制的）ならば $f$ は最小値を達成する)
  - Proof ($0 \in \partial f(x^*)$ は $\partial f$ が全射であることから存在が保証される；これはフェルマの規則（Theorem 3.5-1）による最小性と同値)
  - Example 6.3-2 ($f(x) = \|x\|_1 + \frac{1}{2}\|x - a\|^2$ は強制的な閉真凸関数であり、一意の最小値 $x^* = \mathrm{prox}_{\|\cdot\|_1}(a)$（軟閾値作用素）を持つ)
  - Remark (ミンティの定理の無限次元版は関数解析（4-1）で扱うバナッハ空間上の非線形作用素の理論と接続する；変分不等式の存在定理の基礎でもある)

### §6.4 単調作用素の零点問題と最適化
**内容:** $0 \in T(x)$ の形の問題と凸最適化の接続、関数解析（4-1）への展望

- **1. 零点問題の定式化**
  - Definition 6.4-1 (零点問題：極大単調作用素 $T$ に対して $0 \in T(x)$ を満たす $x$ を求める問題)
  - Proposition 6.4-1 ($T = \partial f$ のとき零点問題は $f$ の最小化問題に等価)
  - Proof (フェルマの規則（Theorem 3.5-1）から直ちに従う)
  - Example 6.4-1 (和の構造：$T = \partial f + \partial g$ の零点を求める問題が $\min (f + g)$ に対応することの確認；分割法（§7.5）への接続)

- **2. レゾルベントと反映**
  - Definition 6.4-2 (レゾルベント：$J_T = (I + T)^{-1}$)
  - Proposition 6.4-2 ($T$ が極大単調のとき $J_T$ は単値で $\mathbb{R}^n$ 上で定義され、非拡大写像である)
  - Proof (極大単調性から $(I + T)$ の全射性を示し、非拡大性は単調性の条件を展開して導く)
  - Example 6.4-2 ($T = \partial \delta_C$（正規錐作用素）のレゾルベントが射影 $\mathrm{proj}_C$ に一致することの確認：$(I + \partial \delta_C)^{-1}(x) = \arg\min_y (\frac{1}{2}\|y - x\|^2 + \delta_C(y))$)
  - Remark (レゾルベントは近接作用素（§7.1）の一般化であり、第7章のアルゴリズムの理論的基盤となる；関数解析（4-1）では非有界作用素の半群理論に接続する)

## Chapter 7: 近接作用素と凸最適化アルゴリズム
**テーマ:** 近接作用素・モロー包絡・劣勾配法・近接点法・分割法・ADMM

### §7.1 近接作用素の定義と性質
**内容:** モロー包絡とインフィマル畳み込みの関係、近接作用素の定義、非拡大性の証明

- **1. 近接作用素の定義**
  - Definition 7.1-1 (近接作用素：$\mathrm{prox}_f(x) = \arg\min_y \{f(y) + \frac{1}{2}\|y - x\|^2\}$)
  - Definition 7.1-2 (モロー包絡：$M_f(x) = \min_y \{f(y) + \frac{1}{2}\|y - x\|^2\}$；インフィマル畳み込み $f \Box \frac{1}{2}\|\cdot\|^2$ に等しい)
  - Proposition 7.1-1 ($f$ が閉真凸関数ならば $\mathrm{prox}_f(x)$ は各 $x$ に対して一意に存在する)
  - Proof ($f(y) + \frac{1}{2}\|y-x\|^2$ は強凸な閉真凸関数であり、Theorem 2.6-1 により一意の最小値を持つ)
  - Example 7.1-1 ($f(x) = \lambda|x|$ の近接作用素：$\mathrm{prox}_{\lambda|\cdot|}(x) = \mathrm{sign}(x) \max(|x| - \lambda, 0)$（軟閾値作用素）；$x = 3, \lambda = 1$ のとき $\mathrm{prox}(3) = 2$ の計算)

- **2. 近接作用素の基本性質**
  - Proposition 7.1-2 ($p = \mathrm{prox}_f(x)$ であることと $x - p \in \partial f(p)$ は同値)
  - Proof (最適性条件 $0 \in \partial f(p) + (p - x)$ を整理する)
  - Theorem 7.1-1 (近接作用素は非拡大写像である：$\|\mathrm{prox}_f(x) - \mathrm{prox}_f(y)\| \leq \|x - y\|$)
  - Proof (Proposition 7.1-2 の特徴づけを二点に適用し、$\partial f$ の単調性から内積評価を行い、コーシー・シュワルツの不等式により結論を導く)
  - Example 7.1-2 (非拡大性の数値確認：$f(x) = |x|$ に対して $\mathrm{prox}_f(3) = 2$、$\mathrm{prox}_f(0.5) = 0$；$|2 - 0| = 2 \leq |3 - 0.5| = 2.5$ の確認)
  - Remark (近接作用素は $\partial f$ のレゾルベント $J_{\partial f} = (I + \partial f)^{-1}$ に等しい（§6.4 参照）；この対応が第6章と第7章を接続する)

### §7.2 モローの分解定理
**内容:** 近接作用素と共役関数の近接作用素の和による分解、射影との関係

- **1. モローの分解定理**
  - Theorem 7.2-1 (モローの分解：閉真凸関数 $f$ に対して $x = \mathrm{prox}_f(x) + \mathrm{prox}_{f^*}(x)$)
  - Proof (フェンシェル-モロの定理 $f^{**} = f$ とヤング-フェンシェルの等号条件 $y \in \partial f(p) \iff p \in \partial f^*(y)$ を用いて、$p = \mathrm{prox}_f(x)$ のとき $x - p \in \partial f(p)$ より $p \in \partial f^*(x - p)$ を導き、$x - p = \mathrm{prox}_{f^*}(x)$ を示す)
  - Example 7.2-1 ($f = \|\cdot\|_1$ のとき $f^* = \delta_{B_\infty}$（$\ell_\infty$ 単位球の指示関数）；$\mathrm{prox}_f(x)$（軟閾値）と $\mathrm{prox}_{f^*}(x) = \mathrm{proj}_{B_\infty}(x)$（$[-1,1]^n$ への射影）の和が $x$ に等しいことの数値確認；$x = 3$ のとき $\mathrm{prox}_f(3) = 2$、$\mathrm{proj}_{B_\infty}(3) = 1$、$2 + 1 = 3$)

- **2. 指示関数への特殊化**
  - Corollary 7.2-1 ($f = \delta_C$ のとき $\mathrm{prox}_f = \mathrm{proj}_C$、$f^* = \sigma_C$ であり、$x = \mathrm{proj}_C(x) + \mathrm{prox}_{\sigma_C}(x)$)
  - Proof (モローの分解定理に $f = \delta_C$ を代入する)
  - Example 7.2-2 ($C = \mathbb{R}^n_+$ のとき $\mathrm{proj}_C(x)_i = \max(x_i, 0)$、$\sigma_C(y) = \delta_{\mathbb{R}^n_-}(y)$ なので $\mathrm{prox}_{\sigma_C}(x)_i = \min(x_i, 0)$；$\max(x_i, 0) + \min(x_i, 0) = x_i$ の確認)
  - Remark (モローの分解は凸集合の射影を共役空間の射影に関連づけ、アルゴリズム設計において「主空間と双対空間の同時計算」を可能にする)

### §7.3 近接作用素の計算例
**内容:** L1ノルム・指示関数（= 射影）・二次関数の近接作用素の具体計算

- **1. L1ノルムと二次関数**
  - Example 7.3-1 ($f(x) = \lambda\|x\|_1$ の近接作用素が成分ごとの軟閾値作用素であることの導出：各成分 $p_i = \mathrm{sign}(x_i)\max(|x_i| - \lambda, 0)$ を最適性条件 $0 \in \lambda \partial|\cdot|(p_i) + p_i - x_i$ から場合分けで導く)
  - Example 7.3-2 ($f(x) = \frac{1}{2}x^T A x + b^T x$（$A$ 正定値）の近接作用素：$\mathrm{prox}_f(x) = (I + A)^{-1}(x - b)$；最適性条件 $Ap + b + p - x = 0$ を解く)

- **2. 指示関数と射影**
  - Example 7.3-3 ($f = \delta_C$ のとき $\mathrm{prox}_f = \mathrm{proj}_C$ であることの再確認と、$C = \{x \mid \|x\|_2 \leq r\}$ への射影の明示公式 $\mathrm{proj}_C(x) = r \cdot x/\|x\|_2$（$\|x\| > r$ のとき）の導出)
  - Example 7.3-4 (アフィン集合 $C = \{x \mid Ax = b\}$ への射影の計算：$\mathrm{proj}_C(x) = x - A^T(AA^T)^{-1}(Ax - b)$；線形代数学II（2-1）の正規方程式との関係)
  - Remark (近接作用素の効率的な計算がアルゴリズムの実用性を決定する；閉形式が得られない場合には内部の最適化問題を反復法で近似的に解く)

### §7.4 劣勾配法
**内容:** 非平滑凸最適化への勾配法の拡張、ステップサイズの選択と収束速度の評価

- **1. 劣勾配法のアルゴリズム**
  - Definition 7.4-1 (劣勾配法：$x_{k+1} = x_k - \alpha_k g_k$（$g_k \in \partial f(x_k)$）)
  - Remark (勾配法とは異なり、劣勾配方向は必ずしも降下方向ではない；ステップサイズの選択が収束の鍵となる)
  - Proposition 7.4-1 (ステップサイズ条件：$\alpha_k > 0$、$\sum \alpha_k = \infty$、$\sum \alpha_k^2 < \infty$ の下で $\min_{j \leq k} f(x_j) \to f^*$)
  - Proof (距離の再帰不等式 $\|x_{k+1} - x^*\|^2 \leq \|x_k - x^*\|^2 - 2\alpha_k(f(x_k) - f^*) + \alpha_k^2\|g_k\|^2$ を導出し、テレスコープ和による評価を行う)

- **2. 収束速度の評価**
  - Theorem 7.4-1 (固定ステップサイズ $\alpha_k = \alpha$ のとき $\min_{j \leq k}f(x_j) - f^* \leq O(1/\sqrt{k})$)
  - Proof (Proposition 7.4-1 の再帰不等式を加算し、$k$ で割って評価する)
  - Example 7.4-1 ($f(x) = |x|$ に対して $x_0 = 5$、$\alpha_k = 1/\sqrt{k+1}$ で劣勾配法を実行し、最初の5ステップを手で追う：$g_0 = 1$、$x_1 = 4$、$g_1 = 1$、$x_2 \approx 3.29$ などの計算)
  - Remark ($O(1/\sqrt{k})$ の収束速度は勾配法の $O(1/k)$ より遅いが、これは非平滑凸最適化の情報理論的下界と一致する最善の速度である)

### §7.5 近接点法と前向き後向き分割
**内容:** 和の構造を持つ凸問題への分割アルゴリズム、ダグラス-ラクフォード分割法

- **1. 近接点法**
  - Definition 7.5-1 (近接点法：$x_{k+1} = \mathrm{prox}_{\alpha_k f}(x_k)$)
  - Theorem 7.5-1 ($f$ が閉真凸で $\arg\min f \neq \emptyset$ ならば $x_k$ は最小化点に収束する)
  - Proof ($\mathrm{prox}$ の非拡大性により $\|x_{k+1} - x^*\|$ が単調非増加であることを示し、フェルマの規則から収束を導く)
  - Example 7.5-1 ($f(x) = |x - 2|$ に対して $x_0 = 0$、$\alpha = 1$ で近接点法を実行：$x_1 = \mathrm{prox}_{|\cdot - 2|}(0)$；軟閾値作用素を平行移動して $x_1 = 1$ と計算し、$x_2 = \mathrm{prox}(1) = 2 - \mathrm{sign}(1-2)\max(|1-2|-1, 0) = 2$ に収束することの確認)

- **2. 前向き後向き分割法**
  - Definition 7.5-2 (前向き後向き分割法：$\min f(x) + g(x)$（$f$ は微分可能、$g$ は非平滑）に対して $x_{k+1} = \mathrm{prox}_{\alpha g}(x_k - \alpha \nabla f(x_k))$)
  - Theorem 7.5-2 ($f$ が $L$-リプシッツ連続な勾配を持つ凸関数、$g$ が閉真凸関数のとき、$0 < \alpha < 2/L$ ならば収束する)
  - Proof ($x_{k+1}$ の定義を $T = \mathrm{prox}_{\alpha g} \circ (I - \alpha \nabla f)$ の不動点反復と見なし、$T$ の非拡大性を $f$ の共単調性と $g$ の近接作用素の非拡大性から導く)
  - Example 7.5-2 (LASSO 問題：$\min \frac{1}{2}\|Ax - b\|^2 + \lambda\|x\|_1$ に前向き後向き分割を適用；$\nabla f(x) = A^T(Ax - b)$、$\mathrm{prox}_{\alpha\lambda\|\cdot\|_1}$ は成分ごとの軟閾値；ISTA（反復軟閾値アルゴリズム）として知られることの確認)
  - Remark (加速版（FISTA）は $O(1/k^2)$ の収束速度を達成し、最適化理論（A-3）のネステロフの加速法に対応する)

### §7.6 ADMM（交互方向乗数法）
**内容:** 制約付き問題への分散的アルゴリズムの構成と収束性、ダグラス-ラクフォード法との関係

- **1. ADMM のアルゴリズム**
  - Definition 7.6-1 (ADMM：$\min f(x) + g(z)$ subject to $Ax + Bz = c$ に対して、拡大ラグランジュ関数の $x$-最小化、$z$-最小化、双対変数の更新を交互に行う)
  - Example 7.6-1 ($f(x) = \frac{1}{2}\|x - a\|^2$、$g(z) = \lambda\|z\|_1$、$x = z$ の場合の ADMM ステップの明示的計算：$x$-更新は線形系、$z$-更新は軟閾値、$u$-更新はスケーリングされた双対上昇)

- **2. 収束性とダグラス-ラクフォード法との関係**
  - Theorem 7.6-1 ($f, g$ が閉真凸関数で $A, B$ が適切な条件を満たすとき ADMM は収束する)
  - Proof (証明の核心：ADMM が双対空間上のダグラス-ラクフォード分割法と等価であることを示し、非拡大性から収束を導く)
  - Proposition 7.6-1 ($A = I$、$B = -I$、$c = 0$ の場合、ADMM は $\mathrm{prox}_f$ と $\mathrm{prox}_g$ のダグラス-ラクフォード反復に帰着する)
  - Proof (ADMM の3ステップを双対変数 $u$ に関するダグラス-ラクフォード反復 $u_{k+1} = T_{DR}(u_k)$ として書き直す；$T_{DR} = (2\mathrm{prox}_g - I) \circ (2\mathrm{prox}_f - I)$ の反映作用素の合成として等価性を確認する)
  - Example 7.6-2 (LASSO への ADMM 適用：$f(x) = \frac{1}{2}\|Ax-b\|^2$、$g(z) = \lambda\|z\|_1$、$x = z$ と設定し、$x$-更新が $(A^TA + \rho I)^{-1}(A^Tb + \rho(z_k - u_k))$、$z$-更新が軟閾値となることの導出)
  - Remark (ADMM は大規模分散最適化の標準的手法であり、機械学習・画像処理・統計推定で広く応用される；最適輸送理論（A-4）のシンクホーンアルゴリズムとも双対的な関係を持つ)
