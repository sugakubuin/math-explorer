## Chapter 1: 基本概念
**テーマ:** グラフの定義・頂点と辺・次数・同型・補グラフ・道と閉路・連結性

### §1.1 グラフの定義と種類
**内容:** 無向グラフ・有向グラフ・多重グラフ・単純グラフの定義、隣接行列・接続行列による表現

- **1. グラフの定義**
  - Definition 1.1-1 (グラフ：頂点の集合 $V$ と辺の集合 $E$ の組 $G = (V, E)$；無向グラフでは各辺は2頂点の非順序対 $\{u, v\}$、有向グラフでは順序対 $(u, v)$)
  - Definition 1.1-2 (単純グラフ：自己ループ（端点が同じ辺）と多重辺（同じ端点の組を結ぶ複数の辺）を持たないグラフ；以降、特に断らない限りグラフは有限単純無向グラフを指す)
  - Example 1.1-1 (頂点3個のグラフ：$V = \{a, b, c\}$、$E = \{\{a,b\}, \{b,c\}\}$（辺2本の道グラフ $P_3$）と $E = \{\{a,b\}, \{b,c\}, \{a,c\}\}$（辺3本の完全グラフ $K_3$）の比較)
  - Example 1.1-2 (多重グラフの例：ケーニヒスベルクの橋の問題は7本の辺（多重辺を含む）と4頂点の多重グラフとして定式化される)

- **2. 行列による表現**
  - Definition 1.1-3 (隣接行列：$n$ 頂点のグラフ $G$ に対して $A = (a_{ij})$（$a_{ij} = 1$ iff $\{v_i, v_j\} \in E$）；対称行列であること)
  - Definition 1.1-4 (接続行列（接続行列）：頂点と辺の関係を表す $n \times m$ 行列 $B = (b_{ij})$（$b_{ij} = 1$ iff 頂点 $v_i$ が辺 $e_j$ の端点）)
  - Example 1.1-3 ($K_3$ の隣接行列の構成：$A = \begin{pmatrix} 0 & 1 & 1 \\ 1 & 0 & 1 \\ 1 & 1 & 0 \end{pmatrix}$；各行の和が2（= 次数）であることの確認)
  - Remark (隣接行列は頂点の番号付けに依存するが、グラフの構造を代数的に扱うための基本的な道具であり、第9章で固有値の議論に活用する)

### §1.2 グラフの同型と自己同型
**内容:** グラフの同型の定義、同型判定の基本的な不変量（頂点数・辺数・次数列）、自己同型群の定義と例

- **1. グラフの同型**
  - Definition 1.2-1 (グラフの同型：$G_1 = (V_1, E_1)$ と $G_2 = (V_2, E_2)$ が同型であるとは、全単射 $f : V_1 \to V_2$ で $\{u, v\} \in E_1 \iff \{f(u), f(v)\} \in E_2$ を満たすものが存在すること；$G_1 \cong G_2$ と書く)
  - Example 1.2-1 (同型なグラフの確認：$V_1 = \{1,2,3,4\}$、$E_1 = \{\{1,2\},\{2,3\},\{3,4\},\{4,1\}\}$（4-閉路 $C_4$）と $V_2 = \{a,b,c,d\}$、$E_2 = \{\{a,c\},\{c,b\},\{b,d\},\{d,a\}\}$；$f(1)=a, f(2)=c, f(3)=b, f(4)=d$ が同型を与えることの確認)
  - Proposition 1.2-1 (同型不変量：$G_1 \cong G_2$ ならば頂点数・辺数・次数列（次数を昇順に並べたもの）が一致する)
  - Proof (全単射が辺を保存するので辺数が一致する；各頂点の隣接関係が保存されるので次数も保存される)
  - Example 1.2-2 (同型でないことの判定：$C_4$（4-閉路）と $K_{1,3}$（星型グラフ）はどちらも頂点4個・辺4本だが、次数列が異なる（$C_4$ は $(2,2,2,2)$、$K_{1,3}$ は $(1,1,1,3)$）ので非同型)

- **2. 自己同型群**
  - Definition 1.2-2 (自己同型群 $\mathrm{Aut}(G)$：$G$ から $G$ 自身への同型写像全体がなす群)
  - Example 1.2-3 ($K_3$ の自己同型群：3頂点の任意の置換が同型写像を与えるので $\mathrm{Aut}(K_3) \cong S_3$（位数6）；$P_3$（道グラフ）の自己同型群は端点の入れ替えのみで位数2であることの確認)
  - Remark (グラフ同型判定問題は一般には計算量が大きい問題であり、効率的なアルゴリズムの存在はグラフ理論の重要な未解決問題の一つである)

### §1.3 補グラフと基本的なグラフ演算
**内容:** 補グラフ・部分グラフ・誘導部分グラフの定義、グラフの和・直積の定義と例

- **1. 補グラフと部分グラフ**
  - Definition 1.3-1 (補グラフ：$\bar{G} = (V, \binom{V}{2} \setminus E)$；$G$ で隣接する頂点対が $\bar{G}$ では非隣接、逆も同様)
  - Example 1.3-1 ($C_5$（5-閉路）の補グラフ $\bar{C_5}$：$C_5$ の5本の辺以外の $\binom{5}{2} - 5 = 5$ 本の辺を持ち、$\bar{C_5} \cong C_5$ であることの確認（自己補グラフ）)
  - Definition 1.3-2 (部分グラフ：$H = (V', E')$ が $G = (V, E)$ の部分グラフとは $V' \subseteq V$、$E' \subseteq E$ であること；誘導部分グラフ：$V' \subseteq V$ に対して $G[V']$ は $V'$ 内の辺をすべて含む部分グラフ)
  - Example 1.3-2 ($K_4$ から頂点 $v_4$ を除いた誘導部分グラフ $K_4[\{v_1,v_2,v_3\}] = K_3$ であることの確認)

- **2. グラフの演算**
  - Definition 1.3-3 (グラフの和：$G_1 \cup G_2 = (V_1 \cup V_2, E_1 \cup E_2)$（$V_1 \cap V_2 = \emptyset$ と仮定）；直積：$G_1 \times G_2$ は頂点集合 $V_1 \times V_2$ で $(u_1,u_2)$ と $(v_1,v_2)$ が隣接するのは $u_1 = v_1$ かつ $\{u_2,v_2\} \in E_2$、または $u_2 = v_2$ かつ $\{u_1,v_1\} \in E_1$ のとき（カルテシアン積とも呼ぶ）)
  - Example 1.3-3 ($K_2 \times K_2$ の構成：4頂点 $(1,1),(1,2),(2,1),(2,2)$ と4辺からなり $C_4$ と同型であることの確認)
  - Remark (補グラフは彩色理論（第6章）やラムゼー理論（組合せ論（A-2））で自然に現れる概念である)

### §1.4 次数と握手補題
**内容:** 頂点の次数の定義、全頂点の次数の和が辺数の2倍であることの証明

- **1. 次数の定義**
  - Definition 1.4-1 (次数：頂点 $v$ に接続する辺の本数 $\deg(v)$；最大次数 $\Delta(G) = \max_v \deg(v)$、最小次数 $\delta(G) = \min_v \deg(v)$)
  - Definition 1.4-2 ($k$-正則グラフ：すべての頂点の次数が $k$ であるグラフ)
  - Example 1.4-1 ($K_n$ は $(n-1)$-正則；$C_n$ は2-正則；ペテルセングラフは3-正則であることの確認)

- **2. 握手補題**
  - Theorem 1.4-1 (握手補題：$\sum_{v \in V} \deg(v) = 2|E|$)
  - Proof (各辺は丁度2つの頂点に寄与するので、次数の総和は辺数の2倍)
  - Corollary 1.4-1 (奇数次の頂点の個数は偶数)
  - Proof (次数の和が偶数なので、奇数次の頂点が奇数個あると和が奇数になり矛盾)
  - Example 1.4-2 (すべての頂点の次数が3のグラフは辺数が $3n/2$（$n$ は偶数でなければならない）；頂点5個で3-正則グラフが存在しないことの確認)

### §1.5 道・歩道・閉路
**内容:** 歩道・道・閉路の定義の区別、長さの定義、最短路の概念

- **1. 歩道と道**
  - Definition 1.5-1 (歩道（walk）：頂点と辺の交互列 $v_0, e_1, v_1, \ldots, e_k, v_k$（辺の重複を許す）；道（path）：頂点の重複がない歩道；閉じた歩道は始点と終点が一致するもの)
  - Definition 1.5-2 (閉路（cycle）：長さ3以上で始点と終点以外の頂点が重複しない閉じた歩道；長さ $k$ の閉路を $C_k$ と書く)
  - Example 1.5-1 ($K_4$ において $v_1, v_2, v_3, v_1, v_4$ は歩道だが道でない（$v_1$ が重複）；$v_1, v_2, v_3, v_4$ は道；$v_1, v_2, v_3, v_1$ は $C_3$（三角形）)

- **2. 距離と最短路**
  - Definition 1.5-3 (距離 $d(u,v)$：$u$ と $v$ を結ぶ最短の道の長さ；$u$ と $v$ が連結でないとき $d(u,v) = \infty$)
  - Proposition 1.5-1 ($d$ は距離の公理（非負性、対称性、三角不等式）を満たす)
  - Proof (三角不等式は最短路の連結により示す)
  - Example 1.5-2 ($C_6$ の頂点 $v_1$ と $v_4$ の距離：時計回り・反時計回りの道の長さはそれぞれ3；$d(v_1, v_4) = 3$ の確認)
  - Definition 1.5-4 (直径：$\mathrm{diam}(G) = \max_{u,v \in V} d(u,v)$)

### §1.6 連結性と連結成分
**内容:** 連結グラフの定義、連結成分の定義と個数、橋と切断点の定義

- **1. 連結性**
  - Definition 1.6-1 (連結グラフ：任意の2頂点間に道が存在するグラフ)
  - Definition 1.6-2 (連結成分：頂点間の到達可能性で定まる同値類の各クラスが誘導する部分グラフ)
  - Example 1.6-1 ($V = \{1,2,3,4,5\}$、$E = \{\{1,2\},\{2,3\},\{4,5\}\}$ のグラフの連結成分は $\{1,2,3\}$ と $\{4,5\}$ の2個であることの確認)

- **2. 橋と切断点**
  - Definition 1.6-3 (橋：除去すると連結成分の数が増える辺；切断点：除去すると連結成分の数が増える頂点)
  - Example 1.6-2 (道グラフ $P_4 = v_1 - v_2 - v_3 - v_4$ のすべての辺は橋；$v_2, v_3$ は切断点、$v_1, v_4$ は切断点でないことの確認)
  - Proposition 1.6-1 (辺 $e$ が橋であることと $e$ がどの閉路にも含まれないことは同値)
  - Proof (一方は閉路内の辺を除去しても代替路が残ることから示し、他方は橋でない辺に対して閉路を構成する)
  - Remark (橋と切断点の概念はネットワークの脆弱性の解析に応用され、第8章のメンガーの定理（§8.5）と接続する)

### §1.7 二部グラフの特徴づけ
**内容:** 二部グラフの定義、奇閉路を含まないことへの同値性の証明

- **1. 二部グラフの定義**
  - Definition 1.7-1 (二部グラフ：$V = X \sqcup Y$（$X, Y$ は非空で $X \cap Y = \emptyset$）に分割でき、すべての辺が $X$ の頂点と $Y$ の頂点を結ぶグラフ；$G = (X, Y, E)$ と書く；完全二部グラフ $K_{m,n}$)
  - Example 1.7-1 ($K_{2,3}$ の構成と隣接行列：$|X| = 2$、$|Y| = 3$ で $X$ の各頂点が $Y$ のすべての頂点に隣接；辺数は $2 \times 3 = 6$；$C_4$ は $K_{2,2}$ に同型)

- **2. 奇閉路による特徴づけ**
  - Theorem 1.7-1 (グラフ $G$ が二部グラフであることと $G$ が奇数長の閉路を含まないことは同値)
  - Proof (二部グラフが奇閉路を含まないことは各辺が $X$-$Y$ 間を往復するので閉路の長さが偶数になることから従う；逆に奇閉路がなければ BFS の距離の偶奇で $V$ を $X$ と $Y$ に分割できることを示す)
  - Example 1.7-2 ($C_5$ が二部グラフでないことの確認：$C_5$ は長さ5の奇閉路を含む；$C_6$ は二部グラフであり $X = \{v_1, v_3, v_5\}$、$Y = \{v_2, v_4, v_6\}$ と分割できることの確認)
  - Remark (二部グラフは第7章のマッチング理論（ホール婚姻定理・ケーニヒの定理）の舞台であり、第9章では隣接行列のスペクトルが原点に関して対称になることと対応する)

## Chapter 2: 有向グラフの基礎
**テーマ:** 有向グラフの基本性質・強連結性・DAG・位相ソート

### §2.1 有向グラフの基本概念
**内容:** 有向辺・入次数・出次数の定義、有向グラフの握手補題

- **1. 有向グラフの定義**
  - Definition 2.1-1 (有向グラフ：$D = (V, A)$（$A \subseteq V \times V$）；各有向辺（弧）$(u,v)$ は $u$（尾）から $v$（頭）への向き付けをもつ)
  - Definition 2.1-2 (入次数 $\deg^-(v)$：$v$ を頭とする弧の本数；出次数 $\deg^+(v)$：$v$ を尾とする弧の本数)
  - Example 2.1-1 ($V = \{a,b,c\}$、$A = \{(a,b),(b,c),(c,a)\}$（有向3-閉路）の入次数と出次数：すべての頂点で $\deg^+ = \deg^- = 1$ であることの確認)

- **2. 有向グラフの握手補題**
  - Proposition 2.1-1 ($\sum_{v \in V} \deg^+(v) = \sum_{v \in V} \deg^-(v) = |A|$)
  - Proof (各弧は丁度1つの頂点の出次数と丁度1つの頂点の入次数に寄与する)
  - Example 2.1-2 (ウェブページのリンク構造：各ページが頂点、ハイパーリンクが有向辺；あるページの出次数はリンク数、入次数は被リンク数に対応する)

### §2.2 有向道と強連結性
**内容:** 有向道・有向閉路の定義、強連結グラフの定義、強連結成分への分解

- **1. 有向道と強連結性**
  - Definition 2.2-1 (有向道：弧の向きに沿って頂点を辿る道；有向閉路：始点と終点が一致する有向道)
  - Definition 2.2-2 (強連結：有向グラフ $D$ が強連結であるとは、任意の2頂点 $u, v$ に対して $u$ から $v$ への有向道と $v$ から $u$ への有向道がともに存在すること)
  - Example 2.2-1 (有向3-閉路 $(a \to b \to c \to a)$ は強連結：任意の2頂点間を有向道で結べることの確認)
  - Example 2.2-2 ($V = \{1,2,3\}$、$A = \{(1,2),(2,3)\}$ は強連結でない：$3$ から $1$ への有向道が存在しない)

- **2. 強連結成分**
  - Definition 2.2-3 (強連結成分：強連結性で定まる同値関係の各クラスが誘導する部分有向グラフ)
  - Proposition 2.2-1 (強連結成分を1頂点に縮約すると DAG（有向非巡回グラフ）が得られる)
  - Proof (縮約グラフに有向閉路があれば、その閉路上の成分をすべて含む強連結成分が存在して矛盾する)
  - Example 2.2-3 ($V = \{1,2,3,4,5\}$、$A = \{(1,2),(2,1),(2,3),(3,4),(4,5),(5,3)\}$ の強連結成分：$\{1,2\}$ と $\{3,4,5\}$；縮約グラフは $\{1,2\} \to \{3,4,5\}$ の2頂点 DAG)

### §2.3 有向非巡回グラフ（DAG）と位相ソート
**内容:** DAG の定義、位相ソートの存在と一意性の条件、DFS に基づく位相ソートアルゴリズム

- **1. DAG の特徴づけ**
  - Definition 2.3-1 (DAG：有向閉路を含まない有向グラフ)
  - Proposition 2.3-1 ($D$ が DAG であることと $D$ に入次数0の頂点（ソース）が存在することは同値（有限グラフの場合）)
  - Proof (DAG ならば最長有向道の始点が入次数0；逆に入次数0の頂点がなければ後退的に辿って有限性から有向閉路が見つかる)
  - Example 2.3-1 (前提科目の履修順序グラフ：各科目を頂点、前提関係を有向辺とすると DAG をなす；入次数0の頂点が「前提なしで履修可能な科目」に対応)

- **2. 位相ソート**
  - Definition 2.3-2 (位相ソート：DAG の頂点の全順序 $v_1, v_2, \ldots, v_n$ で、$(v_i, v_j) \in A$ ならば $i < j$ を満たすもの)
  - Theorem 2.3-1 (位相ソートの存在：$D$ が DAG であることと位相ソートが存在することは同値)
  - Proof (ソースを繰り返し除去する帰納法；各ステップで入次数0の頂点を選び番号を付ける)
  - Example 2.3-2 (5科目の履修順序の具体例：複数の位相ソートが存在しうることの確認；位相ソートが一意になるのは全順序の場合（ハミルトン有向道が存在する場合）のみ)
  - Remark (位相ソートはプロジェクト管理（タスクの依存関係）やコンパイル順序の決定に応用される)

### §2.4 有向木と根付き木
**内容:** 根付き木の定義、二分木・完全木の基本性質、探索木への応用の概観

- **1. 根付き木の定義**
  - Definition 2.4-1 (根付き木：1つの頂点 $r$（根）を指定し、$r$ から他のすべての頂点への一意の有向道が存在する有向グラフ；親・子・葉・深さ・高さの用語)
  - Example 2.4-1 (ファイルシステムのディレクトリ構造：ルートディレクトリを根とする根付き木；深さ2の木の具体例を構成し、葉の個数を数える)

- **2. 二分木と完全木**
  - Definition 2.4-2 (二分木：各頂点の子の数が最大2の根付き木；完全二分木：すべての内部頂点が丁度2つの子を持ち、すべての葉が同じ深さにある二分木)
  - Proposition 2.4-1 (高さ $h$ の完全二分木は $2^{h+1} - 1$ 個の頂点と $2^h$ 枚の葉を持つ)
  - Proof (深さ $k$ の頂点数が $2^k$ であることの帰納法；$\sum_{k=0}^{h} 2^k = 2^{h+1} - 1$)
  - Example 2.4-2 (高さ3の完全二分木：$2^4 - 1 = 15$ 頂点、$2^3 = 8$ 枚の葉の確認)
  - Remark (二分探索木は計算機科学の基本的なデータ構造であり、探索・挿入・削除の計算量が木の高さに依存する)

## Chapter 3: 木
**テーマ:** 木の定義と特徴づけ・全域木・最小全域木アルゴリズム

### §3.1 木の定義と同値条件
**内容:** 閉路を含まない連結グラフとしての定義、辺数・頂点数・連結性・非閉路性による複数の同値条件

- **1. 木の定義**
  - Definition 3.1-1 (木：閉路を含まない連結グラフ；森：閉路を含まないグラフ（連結とは限らない）)
  - Example 3.1-1 ($P_n$（$n$ 頂点の道グラフ）は木；$K_3$ は閉路を含むので木ではないことの確認)

- **2. 木の同値条件**
  - Theorem 3.1-1 ($n$ 頂点のグラフ $G$ について以下は同値：(i) $G$ は木、(ii) $G$ は連結で $|E| = n-1$、(iii) $G$ は閉路を含まず $|E| = n-1$、(iv) 任意の2頂点間に一意の道が存在する)
  - Proof ((i)→(ii)：葉の存在を示し帰納法；(ii)→(iii)：閉路があれば辺を除去しても連結であり $|E| < n-1$ の矛盾；(iii)→(iv)：$n-1$ 辺の閉路なしグラフが連結であることの確認；(iv)→(i)：一意性から閉路がないことが従う)
  - Example 3.1-2 (4頂点の木の列挙：星型 $K_{1,3}$ と道 $P_4$ の2種類（同型を除く）；いずれも $|E| = 3 = 4 - 1$ の確認)
  - Proposition 3.1-1 (葉の存在：$n \geq 2$ の木は少なくとも2つの次数1の頂点（葉）を持つ)
  - Proof (握手補題 $\sum \deg(v) = 2(n-1)$ から平均次数が $2 - 2/n < 2$；次数0の頂点は連結性により存在しないので、次数1の頂点が少なくとも2つ必要)

### §3.2 全域木の存在と構成
**内容:** 連結グラフの全域木の存在定理、BFS・DFS による構成

- **1. 全域木の存在**
  - Definition 3.2-1 (全域木（spanning tree）：$G$ の部分グラフで $G$ のすべての頂点を含む木)
  - Theorem 3.2-1 (連結グラフは全域木を持つ)
  - Proof (連結グラフから閉路内の辺を1つずつ除去する操作を繰り返し、閉路がなくなった時点で得られるグラフが全域木)
  - Example 3.2-1 ($K_4$ から全域木を構成する：6辺のうち3辺を除去して $K_4$ の全域木を3つ具体的に構成する)

- **2. BFS木とDFS木**
  - Example 3.2-2 (BFS による全域木の構成：$K_4$ の頂点 $v_1$ から BFS を開始し、発見順に辺を追加して全域木を得る手順の実行)
  - Example 3.2-3 (DFS による全域木の構成：同じ $K_4$ で DFS を実行し、BFS 木とは異なる全域木が得られることの確認)
  - Remark (全域木の個数はキルヒホッフの行列木定理（§9.2 の Remark で言及）により隣接行列から計算できる)

### §3.3 プリューファー符号と標識木の個数
**内容:** 標識木の数え上げ、ケイリーの公式 $n^{n-2}$ の証明

- **1. プリューファー符号**
  - Definition 3.3-1 (プリューファー符号：$n$ 頂点の標識木から $\{1, \ldots, n\}^{n-2}$ への符号化；最小ラベルの葉を除去し、その隣接頂点のラベルを記録する操作の繰り返し)
  - Example 3.3-1 ($n = 5$の標識木 $E = \{\{1,2\},\{2,3\},\{3,4\},\{3,5\}\}$ のプリューファー符号の計算：葉の除去順序 $1 \to 4 \to 5$ に対応して符号 $(2, 3, 3)$ を得る)

- **2. ケイリーの公式**
  - Theorem 3.3-1 (ケイリーの公式：$n$ 頂点の標識木の総数は $n^{n-2}$)
  - Proof (プリューファー符号が標識木と $\{1, \ldots, n\}^{n-2}$ の間の全単射を与えることを示す；符号の各成分は $n$ 通りの選択肢を持つので総数は $n^{n-2}$)
  - Example 3.3-2 ($n = 3$ のとき $3^{3-2} = 3$ 本の標識木が存在することの確認：辺の組 $\{1,2\}+\{2,3\}$、$\{1,3\}+\{2,3\}$、$\{1,2\}+\{1,3\}$ の列挙)
  - Example 3.3-3 ($n = 4$ のとき $4^2 = 16$ 本の標識木；$K_4$ の全域木の数と一致する)

### §3.4 最小全域木とクラスカル法
**内容:** 重み付きグラフの最小全域木の定義、クラスカル法の正当性の証明

- **1. 最小全域木の定義**
  - Definition 3.4-1 (重み付きグラフ：各辺 $e$ に実数の重み $w(e)$ が付与されたグラフ；最小全域木（MST）：全域木のうち辺の重みの和が最小のもの)
  - Example 3.4-1 (5頂点の重み付きグラフの具体例を構成し、すべての全域木を列挙して重みの和が最小のものを特定する)

- **2. クラスカル法**
  - Theorem 3.4-1 (クラスカル法の正当性：辺を重みの昇順にソートし、閉路を作らない辺を順に選択すると MST が得られる)
  - Proof (カットの性質：任意のカット（頂点集合の分割）を横切る辺のうち最小重みの辺は MST に含まれることを示し、クラスカル法の各ステップがこの性質を満たすことを確認する)
  - Example 3.4-2 (6頂点10辺の重み付きグラフにクラスカル法を適用する手順の実行：各ステップで選択される辺と棄却される辺（閉路形成）を追跡する)

### §3.5 プリム法とその計算量
**内容:** プリム法の手順と正当性、クラスカル法との比較

- **1. プリム法**
  - Theorem 3.5-1 (プリム法の正当性：開始頂点から成長する木に対して、木と残りの頂点を結ぶ最小重みの辺を順に追加すると MST が得られる)
  - Proof (各ステップでカットを横切る最小重み辺を選択しているので、カットの性質から MST の辺が選ばれることを帰納的に示す)
  - Example 3.5-1 (§3.4 と同じグラフにプリム法を適用し、クラスカル法と同じ MST が得られることの確認；辺の選択順序が異なることの比較)

- **2. 計算量の比較**
  - Proposition 3.5-1 (クラスカル法の計算量は辺のソートに支配され $O(|E| \log |E|)$；プリム法は二分ヒープを用いると $O(|E| \log |V|)$)
  - Proof (各ステップの操作回数を数え上げる概略)
  - Remark (辺が少ない疎グラフではクラスカル法、辺が多い密グラフではプリム法が有利な傾向がある)

## Chapter 4: オイラー路とハミルトン路
**テーマ:** オイラーグラフの判定・ハミルトン閉路・計算複雑性の違い

### §4.1 オイラー路の定義と判定条件
**内容:** オイラー路・オイラー閉路の定義、奇数次頂点の個数による判定定理の証明

- **1. オイラー路の定義**
  - Definition 4.1-1 (オイラー路：グラフのすべての辺を丁度1回ずつ通る歩道；オイラー閉路：始点と終点が一致するオイラー路)
  - Example 4.1-1 (ケーニヒスベルクの橋：4頂点7辺のグラフでオイラー路が存在しないことの確認；すべての頂点が奇数次（次数3, 3, 3, 5）)

- **2. オイラーの定理**
  - Theorem 4.1-1 (連結グラフ $G$ がオイラー閉路を持つことと $G$ のすべての頂点が偶数次であることは同値；$G$ がオイラー路（閉路でない）を持つことと奇数次の頂点が丁度2個であることは同値)
  - Proof (必要性：オイラー路が各頂点を通過するたびに次数を2ずつ消費するので偶数次が必要；十分性：帰納法で閉路を取り出し残りのグラフに再帰的に適用する)
  - Example 4.1-2 ($K_4$ はすべての頂点が次数3（奇数）なのでオイラー閉路を持たないが、$K_4$ に1辺を追加して2頂点を偶数次にするとオイラー路が存在することの確認)
  - Example 4.1-3 ($K_5$ はすべての頂点が次数4（偶数）なのでオイラー閉路を持つ；具体的なオイラー閉路を1つ構成する)

### §4.2 フルーリーのアルゴリズム
**内容:** オイラー路の構成的アルゴリズム、橋を避ける選択の役割

- **1. フルーリーのアルゴリズム**
  - Definition 4.2-1 (フルーリーのアルゴリズム：任意の頂点から出発し、「橋でない辺」を優先的に選んで辿るアルゴリズム)
  - Proposition 4.2-1 (フルーリーのアルゴリズムはオイラー閉路（路）を正しく構成する)
  - Proof (橋を避けることで残りのグラフの連結性が保たれ、行き詰まりが生じないことを示す)
  - Example 4.2-1 ($K_5$ にフルーリーのアルゴリズムを適用し、各ステップで選択される辺を追跡する)

- **2. 橋判定の役割**
  - Remark (橋判定は DFS を用いて $O(|E|)$ で行えるため、フルーリーのアルゴリズム全体の計算量は $O(|E|^2)$；ヒエルホルツァーのアルゴリズムは $O(|E|)$ でオイラー閉路を構成する改良版である)

### §4.3 有向グラフのオイラー路
**内容:** 有向オイラー閉路の条件、各頂点の入次数と出次数の一致

- **1. 有向オイラー閉路の判定**
  - Theorem 4.3-1 (連結な有向グラフ $D$ が有向オイラー閉路を持つことと、すべての頂点で $\deg^+(v) = \deg^-(v)$ であることは同値)
  - Proof (無向の場合と同様に、弧を通過するたびに出次数と入次数が1ずつ減少することを利用する)
  - Example 4.3-1 (3頂点の有向完全グラフ（各2頂点間に両方向の弧）で有向オイラー閉路を構成する)

- **2. 有向オイラー路の判定**
  - Corollary 4.3-1 (有向オイラー路（閉路でない）の存在条件：1頂点で $\deg^+ = \deg^- + 1$（始点）、1頂点で $\deg^- = \deg^+ + 1$（終点）、残りは $\deg^+ = \deg^-$)
  - Proof (始点と終点を追加辺で結んで有向オイラー閉路の条件に帰着する)
  - Example 4.3-2 (ド・ブラインのグラフ：長さ $k$ の2進列をすべて含む巡回列の構成問題がオイラー閉路の問題に帰着することの確認)

### §4.4 ハミルトン閉路の定義と例
**内容:** ハミルトン路・閉路の定義、正十二面体グラフなどの例

- **1. ハミルトン路と閉路**
  - Definition 4.4-1 (ハミルトン路：グラフのすべての頂点を丁度1回ずつ通る道；ハミルトン閉路：始点に戻るハミルトン路)
  - Example 4.4-1 ($K_n$（$n \geq 3$）はハミルトン閉路を持つ：$v_1, v_2, \ldots, v_n, v_1$ が1つの例；ハミルトン閉路の総数は $(n-1)!/2$（巡回置換と逆回りの重複を除く）；$K_5$ で $(5-1)!/2 = 12$ の確認)
  - Example 4.4-2 (ペテルセングラフはハミルトン閉路を持たないことの確認：3-正則・10頂点であり、場合分けにより構成できないことを示す)

- **2. オイラー路との対比**
  - Remark (オイラー路の存在判定は次数条件のみで完全に決定できる（多項式時間）のに対し、ハミルトン閉路の存在判定は NP 完全問題であり、効率的な判定法は知られていない；この対照はグラフ理論における計算複雑性の典型例である)

### §4.5 ディラックの定理とオーレの定理
**内容:** ハミルトン閉路の存在の十分条件の証明、オイラー路との計算複雑性の対比

- **1. ディラックの定理**
  - Theorem 4.5-1 (ディラックの定理：$n \geq 3$ で $\delta(G) \geq n/2$ ならば $G$ はハミルトン閉路を持つ)
  - Proof (背理法：ハミルトン閉路を持たない $G$ に辺を最大限追加した飽和グラフを考え、ハミルトン道 $v_1, \ldots, v_n$ の端点間の辺がないことを利用する；$v_1$ の隣接頂点と $v_n$ の隣接頂点の「ずれ」を鳩の巣論法で解析し矛盾を導く)
  - Example 4.5-1 ($C_n$ は $\delta(C_n) = 2 < n/2$（$n \geq 5$）だがハミルトン閉路を持つ；ディラックの条件は十分条件であり必要条件ではないことの確認)

- **2. オーレの定理**
  - Theorem 4.5-2 (オーレの定理：$n \geq 3$ で、隣接しない任意の2頂点 $u, v$ が $\deg(u) + \deg(v) \geq n$ を満たすならば $G$ はハミルトン閉路を持つ)
  - Proof (ディラックの定理と同様の飽和グラフの論法；ハミルトン道の端点の次数和条件から閉路への閉じ込めを導く)
  - Corollary 4.5-1 (ディラックの定理はオーレの定理の系：$\delta(G) \geq n/2$ ならば任意の2頂点の次数和は $n$ 以上)
  - Proof ($\deg(u) + \deg(v) \geq 2\delta(G) \geq n$)
  - Example 4.5-2 ($K_{2,3}$ にオーレの条件を適用：$\deg(u) + \deg(v) = 2 + 2 = 4 < 5 = n$（$X$ 側の2頂点）なので条件を満たさない；実際 $K_{2,3}$ はハミルトン閉路を持たない（頂点数が奇数で二部グラフ）)

## Chapter 5: 平面グラフ

### §5.1 平面グラフの定義と平面埋め込み
**内容:** 辺の交差なしの埋め込みの定義、面の定義

- **1. 平面グラフの定義**
  - Definition 5.1-1 (平面グラフ：辺の交差なしに平面 $\mathbb{R}^2$ に描画できるグラフ；このような描画を平面埋め込みという)
  - Definition 5.1-2 (面：平面埋め込みにおいて辺と頂点を除去したときの連結領域；外部の無限領域も1つの面に数える)
  - Example 5.1-1 ($K_4$ の平面埋め込みの構成：三角形の内部に1頂点を配置し、4面（外部面を含む）であることの確認)

- **2. 非平面グラフの例**
  - Example 5.1-2 ($K_5$ が平面グラフでないことの直感的説明：5頂点をどう配置しても辺の交差が避けられないこと（厳密な証明は §5.3）)
  - Example 5.1-3 ($K_{3,3}$ が平面グラフでないことの直感的説明（3軒と3つの公共設備の問題）)
  - Remark (平面グラフの理論は地図の彩色（第6章）やグラフの描画問題と深く結びつく)

### §5.2 オイラーの多面体公式
**内容:** 連結平面グラフの $V - E + F = 2$ の証明、球面との位相的対応

- **1. オイラーの多面体公式**
  - Theorem 5.2-1 (オイラーの多面体公式：連結平面グラフに対して $|V| - |E| + |F| = 2$（$|F|$ は面の個数）)
  - Proof (辺の除去に関する帰納法：木のとき $|E| = |V| - 1$、$|F| = 1$ で成立；閉路に属する辺を除去すると面が1つ減り辺が1つ減るので等式が保存される)
  - Example 5.2-1 ($K_4$ の平面埋め込みで $|V| = 4$、$|E| = 6$、$|F| = 4$（三角形3つ + 外部面）；$4 - 6 + 4 = 2$ の確認)
  - Example 5.2-2 (正二十面体のグラフ：$|V| = 12$、$|E| = 30$、$|F| = 20$；$12 - 30 + 20 = 2$ の確認)

- **2. 球面との対応**
  - Remark (立体射影により平面と球面は位相的に同値であり、オイラーの公式は球面上のグラフにも成立する；右辺の2はオイラー標数 $\chi(S^2) = 2$ に対応する)

### §5.3 平面グラフの辺数の上界
**内容:** $E \leq 3V - 6$ の証明、$K_5$・$K_{3,3}$ の非平面性の導出

- **1. 辺数の上界**
  - Theorem 5.3-1 ($|V| \geq 3$ の単純連結平面グラフに対して $|E| \leq 3|V| - 6$)
  - Proof (各面は少なくとも3辺で囲まれるので $2|E| \geq 3|F|$（各辺は最大2面に接するので $2|E| = \sum_f \deg(f)$）；オイラーの公式に代入して $|E| \leq 3|V| - 6$)
  - Corollary 5.3-1 ($K_5$ は平面グラフでない：$|V| = 5$、$|E| = 10 > 3 \cdot 5 - 6 = 9$)
  - Proof (Theorem 5.3-1 の不等式に代入する)

- **2. 二部グラフの辺数の上界**
  - Proposition 5.3-1 (二部平面グラフでは $|E| \leq 2|V| - 4$)
  - Proof (二部グラフには三角形がないので各面は4辺以上で囲まれる；$2|E| \geq 4|F|$ をオイラーの公式に代入する)
  - Corollary 5.3-2 ($K_{3,3}$ は平面グラフでない：$|V| = 6$、$|E| = 9 > 2 \cdot 6 - 4 = 8$)
  - Proof (Proposition 5.3-1 の不等式に代入する)

### §5.4 クラトフスキーの定理
**内容:** グラフが平面的であるための完全な特徴づけ

- **1. 細分の定義**
  - Definition 5.4-1 (細分（subdivision）：辺を次数2の新頂点の挿入で置き換える操作；$G$ の細分 $G'$ は $G$ と「位相的に同じ」構造を持つ)
  - Example 5.4-1 ($K_4$ の辺 $\{1,2\}$ に頂点 $5$ を挿入して細分を作る；得られるグラフは $K_4$ と位相的に同じ)

- **2. クラトフスキーの定理**
  - Theorem 5.4-1 (クラトフスキーの定理：$G$ が平面グラフであることと $G$ が $K_5$ または $K_{3,3}$ の細分を部分グラフとして含まないことは同値)
  - Proof (証明の概略：必要性は §5.3 の系から従う；十分性は3-連結グラフの場合の帰納的議論による（完全な証明は省略する）)
  - Example 5.4-2 (ペテルセングラフが非平面であることの確認：$K_{3,3}$ の細分を部分グラフとして含むことを具体的に構成する)

### §5.5 ワグナーの定理とマイナー
**内容:** マイナーの定義、グラフが平面的であるための完全な特徴づけのマイナー版

- **1. グラフのマイナー**
  - Definition 5.5-1 (マイナー：$H$ が $G$ のマイナーであるとは、$G$ の部分グラフから辺の縮約（辺の両端を同一視）を繰り返して $H$ が得られること)
  - Example 5.5-1 ($K_4$ は $K_5$ のマイナー：$K_5$ から1辺を縮約すると4頂点のグラフが得られ、多重辺を除くと $K_4$ になることの確認)

- **2. ワグナーの定理**

## Chapter 6: グラフの彩色
**テーマ:** 頂点彩色・彩色数・ブルックスの定理・辺彩色・四色定理

### §6.1 頂点彩色と彩色数
**内容:** $k$-彩色の定義、彩色数 $\chi(G)$ の定義、二部グラフの彩色数が最大2であることの証明

- **1. 頂点彩色の定義**
  - Definition 6.1-1 ($k$-彩色：$V \to \{1, 2, \ldots, k\}$ の関数で、隣接する頂点に異なる色を割り当てるもの；$k$-彩色可能であるとき $G$ は $k$-彩色可能という)
  - Definition 6.1-2 (彩色数 $\chi(G)$：$G$ が $k$-彩色可能であるような最小の $k$)
  - Example 6.1-1 ($\chi(K_n) = n$：各頂点が他のすべてと隣接するので $n$ 色が必要かつ十分；$\chi(C_5) = 3$：2色では塗れない（奇閉路）が3色で塗れることの確認)

- **2. 二部グラフの彩色数**
  - Proposition 6.1-1 (辺を持つグラフ $G$ に対して $\chi(G) = 2$ と $G$ が二部グラフであることは同値)
  - Proof (2色の割り当ては $V$ の二分割に対応し、隣接頂点が異なる色であることは全辺が2クラス間を結ぶことと同値)
  - Example 6.1-2 ($\chi(C_6) = 2$（偶閉路は二部グラフ）；$\chi(C_5) = 3$（奇閉路は二部でない）；Theorem 1.7-1 との整合性の確認)

### §6.2 貪欲彩色とブルックスの定理
**内容:** 貪欲彩色アルゴリズムによる上界、ブルックスの定理の証明、クリーク数による下界

- **1. 貪欲彩色の上界**
  - Proposition 6.2-1 (貪欲彩色：頂点に順序を付け、各頂点に隣接頂点で未使用の最小の色を割り当てると $\chi(G) \leq \Delta(G) + 1$)
  - Proof (各頂点は最大 $\Delta(G)$ 個の隣接頂点を持つので、少なくとも1色は使用可能)
  - Example 6.2-1 ($C_5$ で頂点を $v_1, \ldots, v_5$ の順に貪欲彩色：$v_1 \to 1$、$v_2 \to 2$、$v_3 \to 1$、$v_4 \to 2$、$v_5 \to 3$；$\Delta(C_5) + 1 = 3$ と一致)

- **2. ブルックスの定理とクリーク数**
  - Theorem 6.2-1 (ブルックスの定理：$G$ が連結で完全グラフでも奇閉路でもなければ $\chi(G) \leq \Delta(G)$)
  - Proof (証明の核心：$G$ が $\Delta$-正則でない場合は次数が $\Delta$ 未満の頂点を最後に塗ることで $\Delta$ 色で足りる；$\Delta$-正則の場合は2つの非隣接頂点を先頭に配置する巧妙な順序付けで同様に示す)
  - Example 6.2-2 ($C_6$ は2-正則で完全グラフでも奇閉路でもないので $\chi(C_6) \leq \Delta = 2$；ペテルセングラフは3-正則で $\chi = 3 = \Delta$ で等号成立)
  - Proposition 6.2-2 (クリーク数による下界：$\chi(G) \geq \omega(G)$（$\omega(G)$ は最大クリーク（完全部分グラフ）のサイズ）)
  - Proof (クリーク内の頂点はすべて異なる色を必要とするので)
  - Example 6.2-3 ($\chi(G) > \omega(G)$ となる例：マイセルスキーグラフは $\omega = 2$（三角形フリー）だが $\chi = 4$)

### §6.3 四色定理の主張
**内容:** 平面グラフの彩色数が4以下であるという主張、五色定理の証明、コンピュータ証明の意義

- **1. 五色定理**
  - Theorem 6.3-1 (五色定理：任意の平面グラフは5-彩色可能)
  - Proof (頂点数に関する帰納法：平面グラフには次数5以下の頂点が存在する（$|E| \leq 3|V| - 6$ から）；次数5以下の頂点 $v$ を除去し帰納法を適用；$v$ の隣接頂点に5色すべてが使われている場合、ケンペ鎖の議論で色の入れ替えにより $v$ に使える色を確保する)
  - Example 6.3-1 (正八面体のグラフ（$K_{2,2,2}$）の5-彩色の具体的構成：6頂点に1から5の色を割り当てる手順)

- **2. 四色定理**
  - Theorem 6.3-2 (四色定理：任意の平面グラフは4-彩色可能)
  - Proof (アッペルとハーケンの証明の概略：(1) すべての平面グラフは有限個の「不可避な配位」のいずれかを含むことを示す（放電法）；(2) 各配位が「可約」であること（その配位を含むグラフが4-彩色可能ならば配位の外側も4-彩色可能であること）をコンピュータで検証する)
  - Remark (四色定理はアッペルとハーケンにより1976年に証明されたが、その証明は約1500個の場合分けをコンピュータで確認するものであり、「人間の手では検証できない証明」の最初の重要例となった；後にロバートソンらにより場合分けの数は約600個に縮小された)

### §6.4 辺彩色とヴィジングの定理
**内容:** 辺彩色数 $\chi'(G)$ の定義、ヴィジングの定理、クラス1・クラス2の分類

- **1. 辺彩色の定義**
  - Definition 6.4-1 (辺彩色：各辺に色を割り当て、共通の端点を持つ辺が異なる色になるようにすること；辺彩色数 $\chi'(G)$ はそのための最小色数)
  - Proposition 6.4-1 ($\chi'(G) \geq \Delta(G)$：最大次数の頂点に接続する辺はすべて異なる色を必要とする)
  - Proof (定義から直ちに従う)
  - Example 6.4-1 ($\chi'(K_3) = 3 = \Delta(K_3)$（3辺がすべて端点を共有）；$\chi'(K_4) = 3 = \Delta(K_4)$ であることの具体的な3-辺彩色の構成)

- **2. ヴィジングの定理**
  - Theorem 6.4-1 (ヴィジングの定理：単純グラフに対して $\chi'(G) = \Delta(G)$ または $\chi'(G) = \Delta(G) + 1$)
  - Proof (証明の概略：増加路の手法により、$\Delta + 1$ 色の辺彩色が常に可能であることを帰納的に示す)
  - Definition 6.4-2 (クラス1：$\chi'(G) = \Delta(G)$ のグラフ；クラス2：$\chi'(G) = \Delta(G) + 1$ のグラフ)
  - Example 6.4-2 ($K_n$ は $n$ が偶数のときクラス1（$\chi' = n - 1 = \Delta$）、$n$ が奇数のときクラス2（$\chi' = n = \Delta + 1$）；$K_3$ がクラス2、$K_4$ がクラス1であることの確認)
  - Remark (あるグラフがクラス1かクラス2かの判定は NP 完全問題であり、彩色数の判定も NP 完全である)

### §6.5 彩色多項式
**内容:** グラフの彩色多項式の定義、削除-縮約公式による計算、彩色多項式と彩色数の関係

- **1. 彩色多項式の定義**
  - Definition 6.5-1 (彩色多項式 $P(G, k)$：$G$ の頂点を $k$ 色で（正しく）彩色する方法の数を $k$ の多項式として表したもの)
  - Example 6.5-1 ($P(K_n, k) = k(k-1)(k-2)\cdots(k-n+1)$（降下階乗）；$P(K_3, 3) = 3 \cdot 2 \cdot 1 = 6$ の確認)
  - Example 6.5-2 (木 $T_n$ の彩色多項式：$P(T_n, k) = k(k-1)^{n-1}$；根から順に塗ると各頂点は親と異なる $k-1$ 通りの選択肢を持つ)

- **2. 削除-縮約公式**
  - Theorem 6.5-1 (削除-縮約公式：辺 $e = \{u,v\}$ に対して $P(G, k) = P(G - e, k) - P(G / e, k)$；$G - e$ は辺の除去、$G / e$ は辺の縮約)
  - Proof ($G - e$ の彩色のうち $u$ と $v$ が異なる色のものが $P(G, k)$、同じ色のものが $P(G/e, k)$ に対応する)
  - Example 6.5-3 ($C_4$ の彩色多項式の計算：辺 $e$ について $P(C_4, k) = P(P_4, k) - P(C_3, k)$；$P(P_4, k) = k(k-1)^3$、$P(C_3, k) = k(k-1)(k-2)$；$P(C_4, k) = k(k-1)^3 - k(k-1)(k-2) = k(k-1)(k^2-3k+3)$ の計算)
  - Remark ($\chi(G)$ は $P(G, k) > 0$ となる最小の正整数 $k$ として特徴づけられる)

## Chapter 7: マッチングとカバー
**テーマ:** マッチング・ホール婚姻定理・チュッテの定理・最大マッチング・ケーニヒの定理

### §7.1 マッチングの定義と完全マッチング
**内容:** マッチング・極大マッチング・最大マッチング・完全マッチングの定義

- **1. マッチングの定義**
  - Definition 7.1-1 (マッチング：共通の端点を持たない辺の集合 $M \subseteq E$)
  - Definition 7.1-2 (極大マッチング：辺を追加するとマッチングでなくなるマッチング；最大マッチング：辺数が最大のマッチング；完全マッチング：すべての頂点がマッチングの辺の端点であるマッチング)
  - Example 7.1-1 ($C_6$ の最大マッチング：$\{\{v_1,v_2\}, \{v_3,v_4\}, \{v_5,v_6\}\}$ は辺数3の完全マッチング；$\{\{v_1,v_2\}, \{v_4,v_5\}\}$ は極大だが最大でないことの確認)

- **2. 完全マッチングの存在条件（導入）**
  - Example 7.1-2 ($K_{2,3}$ は完全マッチングを持たない：$|X| = 2 \neq 3 = |Y|$ なので $X$ 側のすべての頂点を覆えない)
  - Remark (二部グラフの完全マッチングの存在条件（ホールの定理、§7.2）と一般グラフの完全マッチングの存在条件（チュッテの定理、§7.5）は本章の2つの主要定理である)

### §7.2 ホール婚姻定理
**内容:** 二部グラフに完全マッチングが存在するための必要十分条件（ホール条件）の証明

- **1. ホール条件の定義**
  - Definition 7.2-1 (ホール条件：二部グラフ $G = (X, Y, E)$ において、$X$ のすべての部分集合 $S$ に対して $|N(S)| \geq |S|$（$N(S)$ は $S$ の隣接頂点の集合）が成り立つこと)
  - Theorem 7.2-1 (ホール婚姻定理：$G = (X, Y, E)$ が $X$ の完全マッチングを持つことと $G$ がホール条件を満たすことは同値)
  - Proof (必要性は完全マッチングが $S$ の各頂点を $N(S)$ の異なる頂点に対応させることから従う；十分性は $|X|$ に関する帰納法；すべての $S \subsetneq X$ で $|N(S)| > |S|$ の場合と等号が成立する $S$ がある場合に分けて議論する)

- **2. ホール婚姻定理の応用**
  - Example 7.2-1 ($K_{3,3}$ はホール条件を満たす（すべての $S \subseteq X$ で $|N(S)| = 3 \geq |S|$）ので $X$ 側の完全マッチングが存在する)
  - Example 7.2-2 (ホール条件が崩れる例：$X = \{x_1, x_2, x_3\}$、$Y = \{y_1, y_2, y_3\}$ で $x_1, x_2$ がともに $y_1$ のみに隣接する場合、$S = \{x_1, x_2\}$ に対して $|N(S)| = 1 < 2 = |S|$)
  - Corollary 7.2-1 ($k$-正則二部グラフ（$k \geq 1$）は完全マッチングを持つ)
  - Proof ($|S|$ 頂点から出る辺は $k|S|$ 本、$N(S)$ に入る辺は高々 $k|N(S)|$ 本であり、$k|S| \leq k|N(S)|$ よりホール条件が成立)

### §7.3 増加路とベルゲの定理
**内容:** 増加路の定義、マッチングが最大であることの特徴づけ

- **1. 増加路の定義**
  - Definition 7.3-1 (増加路（augmenting path）：マッチング $M$ に関して、$M$ に含まれない辺と $M$ に含まれる辺を交互に通り、両端点が $M$ に覆われていない道)
  - Example 7.3-1 ($P_6$ でマッチング $M = \{\{v_2,v_3\}, \{v_4,v_5\}\}$ に対する増加路 $v_1, v_2, v_3, v_4, v_5, v_6$ の確認；$M$ の辺と非 $M$ 辺を入れ替えるとマッチング $\{\{v_1,v_2\},\{v_3,v_4\},\{v_5,v_6\}\}$ が得られ辺数が1増える)

- **2. ベルゲの定理**
  - Theorem 7.3-1 (ベルゲの定理：$M$ が最大マッチングであることと $M$ に関する増加路が存在しないことは同値)
  - Proof (増加路があれば辺の入れ替えで $|M|$ が増えるので最大でない；逆に $M$ が最大でなければ $|M'| > |M|$ なるマッチング $M'$ を取り $M \triangle M'$（対称差）を解析して増加路を見つける)
  - Example 7.3-2 (ペテルセングラフにおけるマッチングの増加路の探索の実行例)
  - Remark (ベルゲの定理はマッチングの最大性を判定する理論的基盤であり、§7.6 のアルゴリズムの正当性の根拠となる)

### §7.4 ケーニヒの定理
**内容:** 二部グラフの最大マッチングのサイズと最小頂点被覆のサイズの一致の証明

- **1. 頂点被覆の定義**
  - Definition 7.4-1 (頂点被覆：すべての辺の少なくとも一方の端点を含む頂点集合 $C \subseteq V$)
  - Proposition 7.4-1 (任意のグラフで最大マッチングのサイズ $\nu(G)$ は最小頂点被覆のサイズ $\tau(G)$ 以下)
  - Proof (マッチングの各辺は少なくとも1頂点が被覆に含まれるので $\nu(G) \leq \tau(G)$)

- **2. ケーニヒの定理**
  - Theorem 7.4-1 (ケーニヒの定理：二部グラフでは $\nu(G) = \tau(G)$)
  - Proof (最大マッチング $M$ から最小頂点被覆 $C$（$|C| = |M|$）を構成する；ホール婚姻定理またはネットワークフロー（第8章）への帰着により示す)
  - Example 7.4-1 ($K_{3,3}$ で最大マッチングのサイズは3、最小頂点被覆のサイズも3（例えば $X$ 全体）であることの確認)
  - Example 7.4-2 (一般グラフでは $\nu \neq \tau$ となりうる：$K_3$ で $\nu = 1$、$\tau = 2$)
  - Remark (ケーニヒの定理は LP 双対性の整数版であり、最適化理論（A-3）の線形計画法との接続がある)

### §7.5 チュッテの定理
**内容:** 一般グラフの完全マッチングの存在条件の主張と証明の概略

- **1. チュッテの定理**
  - Definition 7.5-1 (奇成分 $o(G - S)$：頂点集合 $S$ を除去したときの奇数頂点の連結成分の個数)
  - Theorem 7.5-1 (チュッテの定理：$G$ が完全マッチングを持つことと、$V$ のすべての部分集合 $S$ に対して $o(G - S) \leq |S|$ であることは同値)
  - Proof (必要性：$S$ を除去して得られる各奇成分は少なくとも1つのマッチング辺で $S$ の頂点と結ばれる必要があり $o(G-S) \leq |S|$；十分性の証明は背理法と極大性の議論による（概略のみ）)
  - Example 7.5-1 ($K_4$ は完全マッチングを持つ：$S = \emptyset$ で $o(K_4) = 0 \leq 0$；$S = \{v_1\}$ で $o(K_4 - v_1) = 1 \leq 1$；すべての $S$ でチュッテ条件を確認)
  - Example 7.5-2 (ペテルセングラフは完全マッチングを持つことのチュッテ条件による確認)

- **2. ホール婚姻定理との関係**
  - Corollary 7.5-1 (二部グラフでチュッテの定理はホール婚姻定理に帰着する)
  - Proof (二部グラフの構造を利用して $o(G-S)$ の条件をホール条件に翻訳する)
  - Remark (チュッテの定理は一般グラフのマッチング理論の出発点であり、チュッテ-ベルゲの公式として最大マッチングの大きさの公式に拡張される)

### §7.6 一般グラフの最大マッチングアルゴリズム
**内容:** エドモンズの開花アルゴリズムの概観、奇数サイクルの縮約操作

- **1. 開花アルゴリズムの概観**
  - Definition 7.6-1 (開花（blossom）：増加路の探索において発見される奇数長の交互閉路)
  - Proposition 7.6-1 (開花の縮約：奇閉路を1頂点に縮約しても最大マッチングのサイズは変化しない)
  - Proof (奇数長の閉路上でマッチング辺と非マッチング辺が交互に並ぶ構造を利用し、縮約と復元の対応を示す（概略）)
  - Example 7.6-1 (7頂点のグラフに開花アルゴリズムを適用する手順の実行：奇閉路の発見、縮約、増加路の探索、復元を追跡する)

- **2. 計算量と意義**
  - Remark (エドモンズの開花アルゴリズムは $O(|V|^2|E|)$ で最大マッチングを求める多項式時間アルゴリズムであり、一般グラフのマッチング問題が効率的に解けることを示した最初の成果である；ハミルトン閉路問題（NP 完全）との計算量の対照は注目に値する)

## Chapter 8: ネットワークフロー
**テーマ:** フローネットワーク・最大流・最小カット・最大流最小カット定理・メンガーの定理

### §8.1 フローネットワークの定義
**内容:** 容量・フロー・源泉・吸収の定義、フローの保存則

- **1. フローネットワークの定義**
  - Definition 8.1-1 (フローネットワーク：有向グラフ $D = (V, A)$ に容量関数 $c : A \to \mathbb{R}_{\geq 0}$ と源泉 $s$・吸収 $t$（$s \neq t$）を指定したもの)
  - Definition 8.1-2 (フロー：関数 $f : A \to \mathbb{R}_{\geq 0}$ で (i) 容量制約 $f(e) \leq c(e)$、(ii) フロー保存則（$s, t$ 以外の各頂点で流入量 = 流出量）を満たすもの；フローの値 $|f|$ は $s$ からの純流出量)
  - Example 8.1-1 (4頂点のフローネットワークの具体例を構成し、容量制約と保存則を満たすフローを1つ求め、フローの値を計算する)

- **2. フローの基本性質**
  - Proposition 8.1-1 (フローの値は $t$ への純流入量に等しい)
  - Proof (フロー保存則を $s, t$ 以外のすべての頂点について加えて消去する)
  - Remark (フローネットワークは輸送問題・通信ネットワーク・サプライチェーンなど広範な応用を持つ)

### §8.2 増加路とフォード-ファルカーソン法
**内容:** 残余グラフ・増加路の定義、アルゴリズムの手順と終了条件

- **1. 残余グラフと増加路**
  - Definition 8.2-1 (残余グラフ $D_f$：フロー $f$ に対して各弧に残余容量 $c_f(e) = c(e) - f(e)$（順方向）と $f(e)$（逆方向）を持たせた有向グラフ)
  - Definition 8.2-2 ($s$-$t$ 増加路：残余グラフ $D_f$ 上の $s$ から $t$ への有向道)
  - Example 8.2-1 (§8.1 の具体例で残余グラフを構成し、増加路を1つ見つけ、フローを更新する手順の実行)

- **2. フォード-ファルカーソン法**
  - Definition 8.2-3 (フォード-ファルカーソン法：残余グラフ上で増加路を見つけ、その上のボトルネック容量だけフローを増加させる操作を繰り返すアルゴリズム)
  - Proposition 8.2-1 (増加路が存在しなくなったときフローは最大)
  - Proof (最大流最小カット定理（§8.3）の系として示す)
  - Remark (容量が無理数の場合はフォード-ファルカーソン法が収束しない例が存在する；エドモンズ-カープ法（§8.4）はこの問題を解消する)

### §8.3 最大流最小カット定理
**内容:** $s$-$t$ カットの定義、最大フローの値と最小カットの容量の一致の証明

- **1. カットの定義**
  - Definition 8.3-1 ($s$-$t$ カット：$V$ の分割 $(S, T)$（$s \in S$、$t \in T$）；カットの容量 $c(S, T) = \sum_{(u,v) \in A, u \in S, v \in T} c(u,v)$)
  - Proposition 8.3-1 (弱双対性：任意のフロー $f$ と任意の $s$-$t$ カット $(S,T)$ に対して $|f| \leq c(S,T)$)
  - Proof ($|f| = \sum_{S \to T} f - \sum_{T \to S} f \leq \sum_{S \to T} c = c(S,T)$)

- **2. 最大流最小カット定理**
  - Theorem 8.3-1 (最大流最小カット定理：最大フローの値と最小カットの容量は等しい)
  - Proof (フォード-ファルカーソン法が増加路なしで終了したとき、$s$ から残余グラフで到達可能な頂点集合を $S$ とすると $(S, V \setminus S)$ がカットであり $|f| = c(S, V \setminus S)$；弱双対性と合わせて最大性と最小性が同時に示される)
  - Example 8.3-1 (具体的なネットワークで最大フローと最小カットを求め、値の一致を確認する)
  - Remark (最大流最小カット定理は線形計画法の強双対定理の特殊な場合であり、最適化理論（A-3）と接続する)

### §8.4 エドモンズ-カープ法の計算量
**内容:** BFS による増加路探索、$O(VE^2)$ の計算量解析

- **1. エドモンズ-カープ法**
  - Definition 8.4-1 (エドモンズ-カープ法：フォード-ファルカーソン法において増加路を BFS（最短路）で選択するアルゴリズム)
  - Theorem 8.4-1 (エドモンズ-カープ法の計算量は $O(|V| \cdot |E|^2)$)
  - Proof (増加路の探索は $O(|E|)$；BFS による最短路を選ぶことで増加の反復回数が $O(|V| \cdot |E|)$ に抑えられることを示す（ボトルネック辺の距離が単調増加することを利用する）)
  - Example 8.4-1 (6頂点8弧のネットワークにエドモンズ-カープ法を適用し、BFS による増加路の選択順序と各ステップのフロー増加量を追跡する)

- **2. より高速なアルゴリズム**
  - Remark (プッシュ-再ラベル法は $O(|V|^2|E|)$、ディニックのアルゴリズムは $O(|V|^2|E|)$ で最大流を求める；容量が整数の場合はフォード-ファルカーソン法が有限ステップで終了する)

### §8.5 メンガーの定理
**内容:** 辺素な道・頂点素な道の最大本数とカットの最小サイズの一致、最大流最小カット定理の系としての導出

- **1. メンガーの定理（辺素版）**
  - Theorem 8.5-1 (メンガーの定理（辺素版）：$s$ と $t$ を結ぶ辺素な道の最大本数は $s$-$t$ 辺カットの最小サイズに等しい)
  - Proof (各辺の容量を1としたフローネットワークに最大流最小カット定理を適用する；フローの整数性により辺素な道に分解できる)
  - Example 8.5-1 (ペテルセングラフの2頂点間の辺素な道の最大本数を最大流で計算する)

- **2. メンガーの定理（頂点素版）**
  - Theorem 8.5-2 (メンガーの定理（頂点素版）：$s$ と $t$ を結ぶ内部頂点素な道の最大本数は $s$-$t$ 頂点カットの最小サイズに等しい)
  - Proof (各頂点を「入力頂点」と「出力頂点」に分割し容量1の辺で結ぶことで辺素版に帰着する)
  - Corollary 8.5-1 ($k$-連結グラフの任意の2頂点間に $k$ 本の内部頂点素な道が存在する)
  - Proof (メンガーの頂点素版から直ちに従う)
  - Remark (メンガーの定理はグラフの連結度の定量的な特徴づけであり、ネットワークの信頼性解析や通信路の冗長性設計に応用される)

### §8.6 最大流の応用
**内容:** 二部マッチングの最大流への帰着、連結度の計算、プロジェクトスケジューリングへの応用

- **1. 二部マッチングへの帰着**
  - Proposition 8.6-1 (二部グラフの最大マッチングは最大流問題に帰着できる：$s$ から $X$ の各頂点に容量1、$Y$ の各頂点から $t$ に容量1の辺を追加し、$X$-$Y$ 間の辺に容量1を設定する)
  - Proof (整数最大流がマッチングに対応し、フロー値が $|M|$ に一致する)
  - Example 8.6-1 ($K_{3,3}$ の最大マッチングを最大流で求める手順の実行)

- **2. 連結度の計算とスケジューリング**
  - Example 8.6-2 (辺連結度 $\lambda(G)$ の計算：任意の頂点 $s$ を固定し、他のすべての頂点 $t$ との最大流を求め最小値を取る)
  - Remark (プロジェクトスケジューリングでは各タスクの依存関係を DAG（第2章）として表し、最大流・最小カットの技法で最長経路（クリティカルパス）を求めることができる)

## Chapter 9: 代数的グラフ理論入門
**テーマ:** 隣接行列のスペクトル・ラプラシアン行列・スペクトルによるグラフの性質の判定

### §9.1 隣接行列の固有値
**内容:** 隣接行列のスペクトルの定義、正則グラフのスペクトルの基本性質、スペクトルの計算例

- **1. スペクトルの定義**
  - Definition 9.1-1 (グラフのスペクトル：隣接行列 $A$ の固有値 $\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_n$ の多重集合；ここで固有値とは $Av = \lambda v$（$v \neq 0$）を満たす実数 $\lambda$ のこと；$A$ が対称行列なのですべての固有値は実数)
  - Example 9.1-1 ($K_3$ の隣接行列 $A = \begin{pmatrix} 0&1&1 \\ 1&0&1 \\ 1&1&0 \end{pmatrix}$ の特性多項式 $\det(A - \lambda I) = -\lambda^3 + 3\lambda + 2 = -(\lambda-2)(\lambda+1)^2$；固有値は $2, -1, -1$ であることの確認)
  - Example 9.1-2 ($P_3$（3頂点の道）の固有値：$A = \begin{pmatrix} 0&1&0 \\ 1&0&1 \\ 0&1&0 \end{pmatrix}$；特性多項式 $-\lambda^3 + 2\lambda = -\lambda(\lambda-\sqrt{2})(\lambda+\sqrt{2})$；固有値 $\sqrt{2}, 0, -\sqrt{2}$ の計算)

- **2. 正則グラフのスペクトル**
  - Proposition 9.1-1 ($k$-正則グラフの最大固有値は $k$ であり、対応する固有ベクトルは全成分1のベクトル $(1, 1, \ldots, 1)^T$)
  - Proof ($A \cdot (1,\ldots,1)^T = (k, \ldots, k)^T = k \cdot (1,\ldots,1)^T$；最大性はペロン-フロベニウスの定理の系として従う（正行列の最大固有値に対応する固有ベクトルが正であることの適用）)
  - Example 9.1-3 (ペテルセングラフは3-正則であり最大固有値は3；全スペクトルは $\{3, 1, 1, 1, 1, -2, -2, -2, -2, -2\}$ であることの確認（対称性からの導出）)
  - Remark (グラフのスペクトルは同型でないグラフを区別する道具になりうるが、スペクトルが同じだが同型でないグラフ（共スペクトルグラフ）も存在する)

### §9.2 ラプラシアン行列
**内容:** グラフラプラシアン $L = D - A$ の定義、$L$ の固有値の非負性、第2最小固有値と連結性

- **1. ラプラシアン行列の定義**
  - Definition 9.2-1 (次数行列 $D$：$D_{ii} = \deg(v_i)$（対角行列）；グラフラプラシアン $L = D - A$)
  - Proposition 9.2-1 ($L$ は正半定値対称行列であり、すべての固有値は非負；最小固有値は常に0で対応する固有ベクトルは $(1,\ldots,1)^T$)
  - Proof ($x^T L x = \sum_{\{i,j\} \in E} (x_i - x_j)^2 \geq 0$ であることを直接計算で示す；$L \cdot (1,\ldots,1)^T = 0$ は各行の和が0であることから従う)
  - Example 9.2-1 ($C_4$ のラプラシアン行列を構成し、固有値 $0, 2, 2, 4$ を計算する)

- **2. 代数的連結度**
  - Definition 9.2-2 (代数的連結度（フィードラー値）：$L$ の第2最小固有値 $\lambda_2(L)$)
  - Theorem 9.2-1 (フィードラーの定理：$\lambda_2(L) > 0$ であることと $G$ が連結であることは同値)
  - Proof ($\lambda_2 = 0$ ならば固有値0の固有空間が2次元以上であり、指示関数による分離で非連結を示す；逆に非連結なら成分の指示関数が固有値0の2つ目の固有ベクトルを与える)
  - Example 9.2-2 ($K_4$ のラプラシアン固有値 $0, 4, 4, 4$（$\lambda_2 = 4$）と、2つの連結成分を持つ $K_2 \cup K_2$ のラプラシアン固有値 $0, 0, 2, 2$（$\lambda_2 = 0$）の比較)
  - Remark (キルヒホッフの行列木定理により、$G$ の全域木の個数は $\frac{1}{n}\lambda_2 \lambda_3 \cdots \lambda_n$（$L$ の非零固有値の積を $n$ で割ったもの）に等しい)

### §9.3 スペクトルによるグラフの性質
**内容:** 二部グラフのスペクトルの対称性、独立数・彩色数のスペクトルによる評価（ホフマンの不等式）

- **1. 二部グラフのスペクトルの対称性**
  - Proposition 9.3-1 ($G$ が二部グラフであることと $A$ のスペクトルが原点に関して対称（$\lambda$ が固有値ならば $-\lambda$ も固有値）であることは同値)
  - Proof (二部グラフの分割 $V = X \sqcup Y$ に対応する符号行列 $S$（$X$ 上で $+1$、$Y$ 上で $-1$）で $SAS = -A$ となることを利用する)
  - Example 9.3-1 ($K_{3,3}$ のスペクトル $\{3, 0, 0, 0, 0, -3\}$ が原点対称であることの確認)

- **2. ホフマンの不等式**
  - Theorem 9.3-1 (ホフマンの不等式：$\alpha(G) \leq n \cdot \frac{-\lambda_n}{\lambda_1 - \lambda_n}$（$\alpha(G)$ は最大独立集合のサイズ、$\lambda_1$ は最大固有値、$\lambda_n$ は最小固有値）)
  - Proof (独立集合の指示ベクトルとレイリー商の関係を利用する（証明の核心のみ）)
  - Example 9.3-2 (ペテルセングラフ：$\lambda_1 = 3$、$\lambda_n = -2$、$n = 10$ でホフマンの不等式は $\alpha \leq 10 \cdot 2/5 = 4$；実際に $\alpha = 4$ で等号が成立)
  - Corollary 9.3-1 (彩色数のスペクトルによる下界：$\chi(G) \geq 1 - \lambda_1/\lambda_n$)
  - Proof ($\alpha(G) \leq n/\chi(G)$ とホフマンの不等式を組み合わせる)
  - Remark (ホフマンの不等式は独立数の上界としてしばしば最良の結果を与え、ラムゼー理論や符号理論（A-7）にも応用される)

### §9.4 展望
**内容:** スペクトルクラスタリング・ランダムウォークとの接続、エクスパンダーグラフの概観、組合せ論（A-2）との接続

- **1. スペクトルクラスタリングとランダムウォーク**
  - Remark (ラプラシアンのフィードラーベクトル（$\lambda_2$ に対応する固有ベクトル）はグラフの「最良の二分割」を近似的に与え、データのクラスタリングに広く応用される（スペクトルクラスタリング）)
  - Example 9.4-1 (グラフ上のランダムウォークの推移確率行列 $P = D^{-1}A$ とラプラシアンの関係：$L = D(I - P)$；定常分布が次数に比例することの確認)

- **2. エクスパンダーグラフ**
  - Definition 9.4-1 (エクスパンダーグラフ：疎グラフ（辺数が $O(n)$）であるにもかかわらず高い連結性を持つグラフ；フィードラー値 $\lambda_2$ が $n$ に依存しない正の定数で下から抑えられるもの)
  - Remark (エクスパンダーグラフは乱数生成・暗号理論（A-8）・通信ネットワーク設計に応用され、組合せ論（A-2）の確率論的手法ともつながる深い研究対象である)
