## Chapter 1: 測度論の基礎
**テーマ:** 集合族と測度の定義

### §1.1 集合族と $\sigma$-加法族
**内容:** 集合代数と $\sigma$-加法族の定義・生成、ボレル集合族

- **1. 集合代数と $\sigma$-加法族の定義**
  - Definition 1.1-1 (集合代数：集合族 $\mathcal{A} \subset 2^X$ が集合代数（algebra）であるとは、(i) $X \in \mathcal{A}$；(ii) $A \in \mathcal{A}$ ならば $A^c \in \mathcal{A}$；(iii) $A, B \in \mathcal{A}$ ならば $A \cup B \in \mathcal{A}$；を満たすこと)
  - Example 1.1-1 (集合代数の例と限界：左開右閉区間の有限和集合全体の族；有限回の演算では閉じるが、可算和 $(0, 1) = \bigcup (0, 1-1/n]$ は含まない例)
  - Definition 1.1-2 ($\sigma$-加法族（$\sigma$-algebra）：集合代数の条件 (iii) を「可算和について閉じる」に強めたもの)
  - Remark (極限操作の正当化：「可算和で閉じる」という条件が解析学の収束定理に不可欠であること)
  - Example 1.1-2 (基本的な $\sigma$-加法族：最小の $\{\emptyset, X\}$、最大の $2^X$、有限集合上の例)

- **2. $\sigma$-加法族の生成**
  - Definition 1.1-3 (生成された $\sigma$-加法族：集合族 $\mathcal{E}$ を含む最小の $\sigma$-加法族 $\sigma(\mathcal{E})$)
  - Example 1.1-3 (生成の具体例：$X=\{1,2,3\}, E=\{1\}$ から生成される $\sigma$-加法族の構成)
  - Definition 1.1-4 (ボレル集合族：位相空間の開集合系から生成された $\sigma$-加法族 $\mathcal{B}(X)$)
  - Example 1.1-4 (実数直線上のボレル集合族：開区間、閉区間、半開区間などの異なる生成系から同じ族が導かれること)

### §1.2 測度の定義と基本的性質
**内容:** 測度の有限加法性・可算加法性と連続性

- **1. 測度の定義**
  - Definition 1.2-1（測度：可測空間 $(X, \mathcal{A})$ 上の関数 $\mu : \mathcal{A} \to [0, +\infty]$ が測度であるとは、(i) $\mu(\emptyset) = 0$；(ii) 互いに素な可算族 $\{A_n\}$ に対して $\mu\!\left(\displaystyle\bigcup_{n=1}^\infty A_n\right) = \displaystyle\sum_{n=1}^\infty \mu(A_n)$（$\sigma$-加法性）；を満たすこと；$(X, \mathcal{A}, \mu)$ を測度空間という）
  - Example 1.2-1（数え上げ測度：$\mu(A) = |A|$（$A$ の元の個数）；ディラック測度：$\delta_{x_0}(A) = 1$（$x_0 \in A$のとき）$= 0$（$x_0 \notin A$ のとき）；どちらも $\sigma$-加法性を満たすことの確認）

- **2. 基本的性質**
  - Proposition 1.2-1 (単調性：$A \subset B \implies \mu(A) \leq \mu(B)$)
  - Proof (単調性の証明：$B = A \cup (B \setminus A)$ の分解を利用)
  - Proposition 1.2-2 (劣 $\sigma$-加法性：$\mu(\bigcup A_n) \leq \sum \mu(A_n)$)
  - Proof (劣 $\sigma$-加法性の証明：互いに素な和への書き換えを利用)
  - Theorem 1.2-1 (測度の連続性：下からの連続性と上からの連続性)
  - Proof (連続性の証明：単調な和・共通部分を $\sigma$-加法性で扱う)
  - Remark ($\mu(B_1) < \infty$ の仮定が必要な例：$B_n = [n, \infty)$ の反例)

### §1.3 外測度と可測性
**内容:** 外測度の定義とカラテオドリの可測条件
