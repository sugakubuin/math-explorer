## Chapter 1: べき級数解法とフロベニウス法
**テーマ:** 通常点・確定特異点でのべき級数解の構成と特殊関数への入門

### §1.1 通常点まわりのべき級数解
**内容:** 級数解の存在とその係数を逐次決定するアルゴリズム

- **1. べき級数解の定義と収束**
  - Definition 1.1-1 (べき級数解：方程式 $y'' + p(x)y' + q(x)y = 0$ の解として、$x = x_0$ のまわりにべき級数 $y = \sum_{n=0}^\infty a_n (x-x_0)^n$ の形のものを求めること；$x_0$ を展開の中心という)
  - Definition 1.1-2 (通常点：係数 $p(x), q(x)$ がともに $x_0$ の近傍で解析的（収束べき級数に展開できる）なとき、$x_0$ を方程式の通常点という)
  - Theorem 1.1-1 (通常点でのべき級数解の存在：$x_0$ が通常点ならば、方程式には $x_0$ のまわりで収束する 2 つの線形独立なべき級数解が存在し、その収束半径は $p(x), q(x)$ の収束半径のうち小さい方以上であること)
  - Proof (存在の証明：未定係数を $y = \sum a_n (x-x_0)^n$ とおいて方程式に代入し、$x^n$ の係数比較で $a_n$ の漸化式が唯一的に定まること；収束はコーシー・アダマールの定理で保証される)
  - Remark (収束半径の意味：べき級数解は複素平面上で展開点 $x_0$ から最近の特異点までの距離以上の収束半径を持つ；実軸上の問題でも複素平面での特異点が解の有効範囲を決める)

- **2. 係数の逐次決定と具体例**
  - Example 1.1-1 (エルミート方程式 $y'' - 2xy' + 2ny = 0$（$n \in \mathbb{N}$）の通常点解：$x_0 = 0$ として $a_{k+2} = \frac{2(k-n)}{(k+2)(k+1)} a_k$ の漸化式を導出；$n = 2$ のとき $y_1(x) = 1 - 2x^2$ が多項式解となることの確認)
  - Example 1.1-2 (エアリー方程式 $y'' - xy = 0$ の通常点解：$x_0 = 0$ として $a_{n+3} = a_n / ((n+3)(n+2))$ の漸化式；$a_0 = 1, a_1 = 0$ の解と $a_0 = 0, a_1 = 1$ の解が 2 つの線形独立解をなすことの確認)
  - Remark (多項式解の重要性：漸化式が有限項で途切れるとき多項式解（例：ルジャンドル多項式・エルミート多項式）が生まれる；これらは量子力学などで固有関数として現れる)

### §1.2 確定特異点と指標方程式
**内容:** 特異点の分類とフックス型方程式の基礎

- **1. 特異点の分類**
  - Definition 1.2-1 (特異点：$p(x)$ または $q(x)$ が $x = x_0$ で解析的でないとき、$x_0$ を方程式の特異点という)
  - Definition 1.2-2 (確定特異点（正則特異点）：特異点 $x_0$ において $(x-x_0)p(x)$ と $(x-x_0)^2 q(x)$ がともに $x_0$ で解析的なとき、$x_0$ を確定特異点という；そうでない特異点を非確定特異点（不確定特異点）という)
  - Example 1.2-1 (オイラー方程式 $x^2 y'' + \alpha x y' + \beta y = 0$ の分類：$x = 0$ は確定特異点；$xp(x) = \alpha$（定数）、$x^2 q(x) = \beta$（定数）であることを確認；非確定特異点の例：$y'' + \frac{1}{x^2}y = 0$ では $x=0$ で $x^2 q = 1$ は解析的だが $xp = 0$ より確定特異点でないことの確認)
  - Remark (確定特異点の重要性：物理的に重要な方程式（ベッセル方程式・ルジャンドル方程式など）は確定特異点を持つフックス型方程式であり、フロベニウス法が適用できる)

- **2. 指標方程式の導出**
  - Definition 1.2-3 (指標方程式：確定特異点 $x_0 = 0$ のまわりで $y = x^\rho \sum_{n=0}^\infty a_n x^n$（$a_0 \neq 0$）とおいて方程式に代入したとき、最低次の係数から得られる $\rho$ についての 2 次方程式 $\rho(\rho-1) + p_0 \rho + q_0 = 0$ を指標方程式（決定方程式）という；$p_0 = \lim_{x\to 0}xp(x)$、$q_0 = \lim_{x\to 0}x^2 q(x)$)
  - Example 1.2-2 (オイラー方程式 $x^2 y'' + \alpha x y' + \beta y = 0$ の指標方程式：$p_0 = \alpha$、$q_0 = \beta$ から $\rho(\rho-1) + \alpha\rho + \beta = 0$ が導かれること；$\alpha = 1/2, \beta = 0$ のとき $\rho = 0, 1/2$ の 2 解を持つことの確認)
  - Remark (指標と解の冪乗の意味：指標 $\rho$ は解 $x^\rho (\cdots)$ の「原点での振る舞いの指数」を決める；指標が整数でない場合、解は実軸上で分枝を持ち得る)

### §1.3 フロベニウス法の解法手順
**内容:** 指標の差による 3 つのケース分類と完全解の構成

- **1. 指標の差によるケース分類**
  - Theorem 1.3-1 (フロベニウスの定理：確定特異点 $x_0 = 0$ を持つ方程式について、指標方程式の 2 解を $\rho_1 \geq \rho_2$（実部で順序付け）とするとき、常に少なくとも 1 つのフロベニウス解 $y_1 = x^{\rho_1} \sum a_n x^n$ が存在する；$\rho_1 - \rho_2 \notin \mathbb{Z}_{\geq 0}$ のとき $y_2 = x^{\rho_2}\sum b_n x^n$ も独立に存在する；差が非負整数のときは対数項が現れることがある)
  - Proof (定理の証明のスケッチ：$y = x^\rho \sum a_n x^n$ を代入して $a_n$ の漸化式を導出する；$\rho_1$ の場合は常に解が構成できる；$\rho_1 - \rho_2 \in \mathbb{Z}_{>0}$ のとき $\rho_2$ での漸化式の分母がゼロになり、対数項 $y_2 = cy_1 \ln x + x^{\rho_2}\sum b_n x^n$ を仮定することで解決する)
  - Example 1.3-1 (ケースI：指標の差が整数でない場合；方程式 $2x y'' + y' + xy = 0$ の指標方程式から $\rho_1 = 1/2, \rho_2 = 0$（差 $1/2$）；両方のフロベニウス解を構成してみると、それぞれ独立した収束べき級数になることの確認)

- **2. 対数項が現れるケース**
  - Example 1.3-2 (ケースII：指標が等しい場合；オイラー方程式 $x^2 y'' + xy' - y = 0$ の指標 $\rho_1 = \rho_2 = 1$ の場合：$y_1 = x$ と $y_2 = x \ln x$ が線形独立解となること；定数変化法による $y_2$ の導出を追う)
  - Example 1.3-3 (ケースIII：指標の差が正整数の場合；$x y'' + 2y' - xy = 0$（差が 1）：$y_1 = (\sinh x)/x$、$y_2 = (\cosh x)/x$ の構成；分母がゼロになる箇所の処理と対数項の有無の判定)
  - Remark (対数項と物理的解釈：対数項を含む解はある種の境界条件を満たさず「発散する解」として除外されることが多い；しかし数学的には完全な一般解を構成するために必要な要素である)

### §1.4 ベッセル方程式とベッセル関数
**内容:** 次数 $\nu$ のベッセル方程式のフロベニウス解・漸化式・直交性

- **1. ベッセル方程式の解の構成**
  - Definition 1.4-1 (ベッセル方程式：$x^2 y'' + xy' + (x^2 - \nu^2)y = 0$；$\nu \geq 0$ を次数という；$x = 0$ は確定特異点で指標方程式は $\rho^2 - \nu^2 = 0$、すなわち $\rho = \pm\nu$)
  - Definition 1.4-2 (第一種ベッセル関数：指標 $\rho = \nu$ のフロベニウス解として $J_\nu(x) = \sum_{m=0}^\infty \frac{(-1)^m}{m!\,\Gamma(m+\nu+1)}\left(\frac{x}{2}\right)^{2m+\nu}$；$\nu$ が非整数のとき $J_{-\nu}$ も線形独立解をなす)
  - Proof (係数の決定：漸化式 $a_{m+2} = -a_m / ((m+2+\nu)(m+2))$ から $a_{2m} = (-1)^m a_0 / (m!\,\Gamma(m+\nu+1))4^m$；$a_0 = 1/(2^\nu \Gamma(\nu+1))$ と正規化)
  - Example 1.4-1 ($J_0(x)$ の計算：$\nu = 0$ のとき $J_0(x) = \sum_{m=0}^\infty \frac{(-1)^m}{(m!)^2}\left(\frac{x}{2}\right)^{2m} = 1 - \frac{x^2}{4} + \frac{x^4}{64} - \cdots$；$J_0(0) = 1$、$J_0$ が偶関数であることの確認)

- **2. 漸化式・直交性と応用**
  - Proposition 1.4-1 (ベッセル関数の漸化式：$(x^\nu J_\nu)' = x^\nu J_{\nu-1}$ および $(x^{-\nu}J_\nu)' = -x^{-\nu}J_{\nu+1}$；これより $J_{\nu-1}(x) + J_{\nu+1}(x) = \frac{2\nu}{x}J_\nu(x)$)
  - Proof (漸化式の証明：定義の級数を項別微分し、添字のシフトによって式を整理する)
  - Example 1.4-2 ($J_1(x)$ の漸化式による計算：$\nu = 0$ の漸化式 $J_1 = -J_0'$ を使い $J_1(x) = -J_0'(x) = \frac{x}{2} - \frac{x^3}{16} + \cdots$ を求める；$J_1(0) = 0$ であることの確認)
  - Theorem 1.4-2 (ベッセル関数の直交性：$[0,1]$ 上で $j_{\nu,m}, j_{\nu,n}$（$J_\nu$ の正零点）を用いて $\int_0^1 x J_\nu(j_{\nu,m}x)J_\nu(j_{\nu,n}x)\,dx = 0$（$m \neq n$）；円柱座標系における固有関数展開の根拠となる)
  - Proof (直交性の証明：$J_\nu(j_{\nu,m}x)$ と $J_\nu(j_{\nu,n}x)$ がそれぞれ重み $x$ つきシュトゥルム・リウヴィル問題の異なる固有値に対応する固有関数であることから（§5.3 参照）、一般理論によって直交性が従う)
  - Remark (ベッセル関数の応用：円柱の熱伝導・ドラムの振動・電磁波の導波管など、円柱・球対称を持つ物理問題の固有関数として中心的役割を果たす)

### §1.5 ルジャンドル方程式とルジャンドル多項式
**内容:** ルジャンドル多項式の定義とロドリゲスの公式・直交性

- **1. ルジャンドル方程式の解**
  - Definition 1.5-1 (ルジャンドル方程式：$(1-x^2)y'' - 2xy' + n(n+1)y = 0$；$n \in \mathbb{N}_0$ のとき $x = 0$ は通常点；$x = \pm 1$ が確定特異点)
  - Definition 1.5-2 (ルジャンドル多項式：ルジャンドル方程式の多項式解（通常点解が有限項で途切れる場合）を $P_n(x)$ とし、$P_n(1) = 1$ によって正規化する；$P_0(x) = 1$、$P_1(x) = x$、$P_2(x) = (3x^2 - 1)/2$)
  - Example 1.5-1 ($n = 2$ の場合の逐次決定：$a_{k+2} = a_k(k(k+1) - n(n+1))/((k+2)(k+1))$ に $n = 2$ を代入すると $a_4 = a_2 \cdot (-2)/12$；偶数次のみ残り $y_1 = a_0(1 - 3x^2)$；$P_2(1) = 1$ の条件から $a_0 = -1/2$ を得て $P_2(x) = (3x^2-1)/2$)

- **2. ロドリゲスの公式と直交性**
  - Theorem 1.5-1 (ロドリゲスの公式：$P_n(x) = \frac{1}{2^n n!}\frac{d^n}{dx^n}(x^2 - 1)^n$)
  - Proof (ロドリゲスの公式の証明：$w = (x^2-1)^n$ とおくと $w' = 2nx(x^2-1)^{n-1}$ から $(1-x^2)w' + 2nxw = 0$；両辺を $n+1$ 回微分してライプニッツの公式を適用すると $w^{(n)}$ がルジャンドル方程式を満たすことが分かる)
  - Example 1.5-2 ($n = 3$ のロドリゲスの公式：$(x^2-1)^3 = x^6 - 3x^4 + 3x^2 - 1$ を 3 回微分すると $6(5x^3 - 3x)$；$1/(8 \cdot 6) = 1/48$ をかけると $P_3(x) = (5x^3 - 3x)/2$；$P_3(1) = 1$ の確認)
  - Theorem 1.5-2 (ルジャンドル多項式の直交性：$\int_{-1}^1 P_m(x)P_n(x)\,dx = \frac{2}{2n+1}\delta_{mn}$)
  - Proof (直交性の証明：$m \neq n$ のとき $P_m, P_n$ がそれぞれ異なる固有値 $m(m+1), n(n+1)$ に対応するシュトゥルム・リウヴィル問題の固有関数であることから直交性が従う；ノルムの計算はロドリゲスの公式と部分積分を繰り返す)
  - Remark (ルジャンドル多項式の応用：球座標系でのラプラス方程式の解が $P_n(\cos\theta)$ を含むため、重力ポテンシャル・電磁気学・量子力学（球面調和関数）に広く現れる)

## Chapter 2: 行列指数関数と連立 ODE
**テーマ:** 行列の指数関数を軸にした連立 1 階常微分方程式の完全解法

### §2.1 行列指数関数の定義と基本性質
**内容:** べき級数による定義と積の公式・微分公式

- **1. 行列指数関数の定義**
  - Definition 2.1-1 (行列指数関数：正方行列 $A$ に対して $e^A = \sum_{n=0}^\infty \frac{A^n}{n!}$；この級数は任意の行列ノルムで絶対収束する)
  - Proposition 2.1-1 (基本性質：(i) $e^0 = I$；(ii) $e^A$ は常に正則で $(e^A)^{-1} = e^{-A}$；(iii) $AB = BA$ ならば $e^{A+B} = e^A e^B$；(iv) $(e^A)^T = e^{A^T}$)
  - Proof (性質の証明：(i)(iii) はコーシー積を用いた級数の計算；(ii) は (iii) において $B = -A$ とおく；$AB \neq BA$ のとき $e^{A+B} \neq e^A e^B$ になる反例として $A = \begin{pmatrix}0&1\\0&0\end{pmatrix}, B = \begin{pmatrix}0&0\\1&0\end{pmatrix}$ がある)
  - Example 2.1-1 (対角行列の指数関数：$A = \mathrm{diag}(\lambda_1, \ldots, \lambda_n)$ のとき $e^{tA} = \mathrm{diag}(e^{\lambda_1 t}, \ldots, e^{\lambda_n t})$；$e^{tA}$ の $(i,j)$ 成分を直接べき級数で確認)

- **2. 微分公式と初期値問題との接続**
  - Theorem 2.1-2 (行列指数関数の微分：$\frac{d}{dt}e^{tA} = A e^{tA} = e^{tA} A$)
  - Proof (微分公式の証明：べき級数を項別微分し $\sum_{n=1}^\infty A^n t^{n-1}/(n-1)! = A e^{tA}$ を確認する；収束は一様なので項別微分が許される)
  - Example 2.1-2 (初期値問題の解：$\dot{x} = Ax$、$x(0) = x_0$ の解が $x(t) = e^{tA}x_0$ であることの確認；$A$ が $1\times 1$（スカラー）の場合に退化すると $x(t) = e^{\lambda t}x_0$ に一致する)
  - Remark (行列指数関数と ODE の完全な対応：スカラーの場合 $\dot{x} = \lambda x$ の解 $e^{\lambda t}x_0$ と全く同じ形で一般解が書けることが $e^{tA}$ の定義の動機である)

### §2.2 ジョルダン標準形による $e^{tA}$ の計算
**内容:** ジョルダン細胞を利用した具体的な行列指数の計算法

- **1. ジョルダン細胞の指数関数**
  - Theorem 2.2-1 (ジョルダン細胞の指数関数：次数 $k$ のジョルダン細胞 $J = \lambda I + N$（$N$ はべき零行列）に対して $e^{tJ} = e^{\lambda t}\sum_{m=0}^{k-1} \frac{t^m}{m!} N^m$；成分で書くと上三角の多項式行列となる)
  - Proof (ジョルダン細胞の計算：$J = \lambda I + N$ と $\lambda I$、$N$ が可換なので $e^{tJ} = e^{t\lambda I}e^{tN} = e^{\lambda t}e^{tN}$；$N^k = 0$ より $e^{tN} = I + tN + \cdots + t^{k-1}N^{k-1}/(k-1)!$ が有限和になる)
  - Example 2.2-1 ($2 \times 2$ ジョルダン細胞の例：$J = \begin{pmatrix}\lambda & 1\\0&\lambda\end{pmatrix}$ のとき $e^{tJ} = e^{\lambda t}\begin{pmatrix}1&t\\0&1\end{pmatrix}$；$t = 0$ で単位行列になること・$t$ で微分すると $J e^{tJ}$ になることを直接確認)

- **2. 一般行列への拡張と計算手順**
  - Proposition 2.2-2 (ジョルダン分解による $e^{tA}$ の計算：$A = PJP^{-1}$（$J$ がジョルダン標準形）ならば $e^{tA} = P e^{tJ} P^{-1}$；$e^{tJ}$ は各ジョルダン細胞の指数関数をブロック対角に並べたもの)
  - Proof (分解の証明：$A^n = P J^n P^{-1}$ を利用して $e^{tA} = \sum t^n A^n/n! = P(\sum t^n J^n/n!) P^{-1} = Pe^{tJ}P^{-1}$)
  - Example 2.2-2 ($A = \begin{pmatrix}3&1\\0&3\end{pmatrix}$ の場合：既にジョルダン形なので $e^{tA} = e^{3t}\begin{pmatrix}1&t\\0&1\end{pmatrix}$；初期値問題 $\dot{x} = Ax$、$x(0) = (1, 0)^T$ の解は $x(t) = e^{3t}(1, 0)^T$ となることの確認)
  - Remark (重複固有値と多項式増大：ジョルダン細胞が現れる場合、解は $e^{\lambda t}$ に多項式 $t^m$ を掛けた項を含む；これは「代数的・幾何的重複度の差」が生じたときの本質的現象である)

### §2.3 基本行列と一般解
**内容:** 基本行列の定義と初期値問題の解の公式

- **1. 基本行列の定義**
  - Definition 2.3-1 (基本行列：$\dot{X} = AX$ を満たし $\det X(t) \neq 0$（任意の $t$）となる行列値関数 $X(t)$ を基本行列という；各列 $x_j(t)$ は線形独立な解になる)
  - Theorem 2.3-1 (基本行列の一般形：$e^{tA}$ は初期条件 $X(0) = I$ を満たす基本行列であり、一般の基本行列は $X(t) = e^{tA}C$（$C$ は正則定数行列）と書ける)
  - Proof (基本行列の記述：$e^{tA}$ が $\dot{X} = AX$ と $X(0) = I$ を満たすことは §2.1 の微分公式から従う；他の基本行列との差は右から定数行列を掛けることで吸収できる)
  - Example 2.3-1 ($2\times 2$ 系 $\dot{x} = \begin{pmatrix}0&1\\-1&0\end{pmatrix}x$ の基本行列：固有値 $\pm i$、固有ベクトルから $e^{tA} = \begin{pmatrix}\cos t & \sin t\\-\sin t & \cos t\end{pmatrix}$；$t = 0$ で単位行列になることの確認)

- **2. 一般解と初期値問題の解**
  - Theorem 2.3-2 (同次系の一般解：$\dot{x} = Ax$ の一般解は $x(t) = e^{tA}c$（$c \in \mathbb{R}^n$ は任意定数ベクトル）であり、初期値 $x(0) = x_0$ のとき $c = x_0$ となる)
  - Example 2.3-2 (初期値問題の解：$\dot{x} = \begin{pmatrix}1&2\\0&3\end{pmatrix}x$、$x(0) = (1,-1)^T$ の解を $e^{tA}$ を計算して求める；固有値 $1, 3$ のジョルダン分解から $e^{tA}$ を構成し $x(t)$ を計算)
  - Remark (ウロンスキー行列式の正則性：ラブの公式 $\det X(t) = \det X(0) \cdot e^{\int_0^t \mathrm{tr}A\,ds}$ により、$\det X(0) \neq 0$ ならば $\det X(t) \neq 0$（常に正則）が保証される；これが基本行列の定義と整合する)

### §2.4 非同次連立 ODE の解法
**内容:** 定数変化法による一般解の公式と具体的な計算手順

- **1. 定数変化法と一般解の公式**
  - Theorem 2.4-1 (非同次系の一般解：$\dot{x} = Ax + f(t)$ の解は $x(t) = e^{tA}c + \int_0^t e^{(t-s)A}f(s)\,ds$（初期値 $x(0) = x_0$ のとき $c = x_0$）)
  - Proof (定数変化法の適用：同次系の解 $e^{tA}c$ の定数 $c$ を $c(t)$ に替え、代入すると $e^{tA}\dot{c}(t) = f(t)$；$e^{-tA}$ をかけて積分すれば $c(t) = x_0 + \int_0^t e^{-sA}f(s)\,ds$ が得られる)
  - Example 2.4-1 ($\dot{x} = \begin{pmatrix}0&1\\-1&0\end{pmatrix}x + \begin{pmatrix}\cos t\\0\end{pmatrix}$、$x(0) = 0$ の解：§2.3 の例の $e^{tA}$ を用いて畳み込み積分を計算し $x(t)$ を求める；共鳴（強制振動の周波数が固有振動数と一致）の場合に $t\sin t$ の項が現れることの確認)

- **2. 応用：共鳴現象と安定性**
  - Example 2.4-2 (共鳴の具体例：$\ddot{y} + \omega^2 y = \cos\omega t$（強制振動と固有振動が同周波数）の連立 1 階形式での解；$x(t)$ の成分が $t\sin\omega t$ の項を含み時間とともに発散すること；$\omega$ を変えた場合（非共鳴）との比較)
  - Remark (工学的意味：共鳴は橋や建物の崩壊原因（タコマナローズ橋）、楽器の倍音増幅など工学・物理で中心的役割を担う；非同次方程式の解法は周波数応答関数（伝達関数）の概念と直結する)

## Chapter 3: 安定性理論
**テーマ:** 平衡点の安定性のリャプノフ的解析と 2 次元系の大域的挙動

### §3.1 自律系の平衡点と相平面の分類
**内容:** 2 次元線形系の相平面の完全分類と判定基準

- **1. 平衡点と線形化**
  - Definition 3.1-1 (自律系：右辺が時間 $t$ を陽に含まない ODE 系 $\dot{x} = f(x)$；$f(x^*) = 0$ を満たす $x^*$ を平衡点（均衡点）という)
  - Definition 3.1-2 (非線形系の線形化：平衡点 $x^*$ のまわりで $f(x) \approx Df(x^*)(x - x^*)$ と近似した線形系 $\dot{y} = Ay$（$A = Df(x^*)$、$y = x - x^*$）を $x^*$ まわりの線形化系という)
  - Remark (線形化の有効性：ハルトマン・グロブマン定理（§3.2 の Remark 参照）により、双曲型平衡点（固有値に虚部ゼロがない）においては線形化系と元の系が局所的に位相的同値になる；退化ケースでは線形化の情報だけでは不十分なことがある)

- **2. $2\times 2$ 線形系の相平面分類**
  - Theorem 3.1-1 (2 次元線形系の分類：$A$ の固有値の実部・虚部・符号に応じて平衡点は「安定ノード・不安定ノード・鞍点・安定スパイラル・不安定スパイラル・中心」に分類される；$\mathrm{tr}A$ と $\det A$ の符号が分類の完全な基準を与える)
  - Proof (分類の証明のスケッチ：ジョルダン標準形に帰着し各ケースの $e^{tA}$ の長時間挙動を調べる；実固有値で同符号 → ノード、異符号 → 鞍点、複素固有値 → スパイラル・中心)
  - Example 3.1-1 (鞍点の例：$A = \begin{pmatrix}1&0\\0&-1\end{pmatrix}$；固有値 $1, -1$ から $x_1$ 方向は発散・$x_2$ 方向は収束；相平面の安定・不安定多様体（座標軸）の描き方)
  - Example 3.1-2 (安定スパイラルの例：$A = \begin{pmatrix}-1&2\\-2&-1\end{pmatrix}$；固有値 $-1 \pm 2i$ から軌道は原点に螺旋を描きながら収束することの確認；$e^{tA}$ の成分を計算して軌道を追う)

### §3.2 リャプノフの安定性の定義
**内容:** 安定性・漸近安定性・不安定性の厳密な定義と基本例

- **1. リャプノフ意味での安定性**
  - Definition 3.2-1 (安定性（リャプノフ意味）：平衡点 $x^*$ が安定であるとは、任意の $\varepsilon > 0$ に対しある $\delta > 0$ が存在して $|x(0) - x^*| < \delta$ ならば $|x(t) - x^*| < \varepsilon$（すべての $t \geq 0$）が成立すること)
  - Definition 3.2-2 (漸近安定性：安定かつ $|x(0) - x^*| < \delta$ ならば $x(t) \to x^*$（$t \to \infty$）が成立するとき、$x^*$ を漸近安定という)
  - Definition 3.2-3 (不安定性：安定でない平衡点を不安定という)
  - Example 3.2-1 (安定だが漸近安定でない例：$\dot{x}_1 = -x_2$、$\dot{x}_2 = x_1$ の中心（純虚数固有値）；軌道は原点を取り囲む閉曲線で、$x(t)$ は $x^*=0$ に収束しない；Definition 3.2-1 の $\varepsilon$-$\delta$ を満たすことを確認)
  - Example 3.2-2 (不安定の例：$\dot{x} = x$ の平衡点 $x^* = 0$；$\delta$ をどう小さく取っても $x(0) = \delta/2$ なら $x(t) = \frac{\delta}{2}e^t$ が $t$ とともに $\varepsilon$ を超えることの確認)

- **2. 線形系の安定性判定**
  - Theorem 3.2-1 (線形系の安定性：$\dot{x} = Ax$ の平衡点 $x^* = 0$ について、(i) すべての固有値の実部が負 $\Rightarrow$ 漸近安定；(ii) ある固有値の実部が正 $\Rightarrow$ 不安定；(iii) 固有値の実部が最大で 0 の場合は固有値の幾何・代数的重複度による)
  - Proof (線形系の判定：$e^{tA}$ の各成分が $e^{\mathrm{Re}\lambda_j t}$ の多項式倍として書けることを §2.2 の計算から読み取る；実部がすべて負なら $e^{tA}x_0 \to 0$)
  - Remark (ハルトマン・グロブマン定理との関係：平衡点の固有値に実部ゼロのものがなければ（双曲型の場合）、非線形系の局所的な位相的性質は線形化で決まる；実部がゼロの固有値が存在する場合は高次の非線形項が重要になる)

### §3.3 リャプノフ関数法
**内容:** 正定値関数を用いた安定性の十分条件

- **1. リャプノフ関数の定義**
  - Definition 3.3-1 (正定値関数・負定値関数：連続関数 $V : U \to \mathbb{R}$（$U$ は $x^*$ の近傍）が正定値であるとは $V(x^*) = 0$ かつ $x \neq x^*$ で $V(x) > 0$；$-V$ が正定値のとき $V$ を負定値という)
  - Definition 3.3-2 (リャプノフ関数：自律系 $\dot{x} = f(x)$ に対して、正定値関数 $V$ で $\dot{V}(x) = \nabla V(x) \cdot f(x) \leq 0$（$U$ 上）を満たすものをリャプノフ関数という；$\dot{V} < 0$（負定値）のとき強いリャプノフ関数という)
  - Example 3.3-1 (2 次形式のリャプノフ関数：$V(x) = x^TPx$（$P$ は正定値対称行列）；$\dot{V} = x^T(PA + A^TP)x$；$A = \begin{pmatrix}-1&0\\0&-2\end{pmatrix}$、$P = I$ のとき $\dot{V} = -2x_1^2 - 4x_2^2 < 0$（負定値）であることの確認)

- **2. リャプノフ安定性定理**
  - Theorem 3.3-1 (リャプノフの安定性定理：リャプノフ関数 $V$（$\dot{V} \leq 0$）が存在すれば $x^*$ は安定；強いリャプノフ関数（$\dot{V} < 0$）が存在すれば漸近安定)
  - Proof (安定性定理の証明：等位面 $\{V = c\}$ が $f$ の向きと整合的であることを示す；$V$ が一定以下のレベル集合に軌道が留まることを $\dot{V} \leq 0$ から導く；漸近安定は LaSalle の不変原理（§3.5）で補足)
  - Example 3.3-2 (非線形系への適用：$\dot{x}_1 = -x_1 + x_1 x_2^2$、$\dot{x}_2 = -x_2$ に $V = x_1^2 + x_2^2$ を試す；$\dot{V} = -2(x_1^2 + x_2^2) + 2x_1^2 x_2^2 \leq -2V + V^2/2$；$V$ が小さい領域で $\dot{V} < 0$ となり局所的漸近安定が従うことの確認)
  - Theorem 3.3-2 (不安定性定理（チェタエフの定理）：$V(x^*)=0$ だが $x^*$ の任意の近傍に $V > 0$ の点が存在し、その集合で $\dot{V} > 0$ ならば $x^*$ は不安定)
  - Proof (不安定性の証明：$V > 0$ の領域で軌道が「坂を登る」ことを示し、有限時間でその領域の境界に到達することを $\dot{V} > 0$ から導く)
  - Remark (リャプノフ関数の構成の困難さ：適切なリャプノフ関数を見つけることは一般に難しい；線形系では $PA + A^TP = -Q$ のリャプノフ方程式（$Q > 0$）を解くことで二次形式のリャプノフ関数を系統的に得られる（詳細は制御理論（A-6）で扱う）)

### §3.4 ポアンカレ・ベンディクソンの定理
**内容:** 2 次元系の閉軌道の存在と相平面上の大域的挙動の決定

- **1. 定理の主張と適用条件**
  - Theorem 3.4-1 (ポアンカレ・ベンディクソンの定理：$\mathbb{R}^2$ の有界閉領域 $D$ に平衡点を含まず、かつ $D$ に入った軌道が $D$ から出ないとき、$D$ の中に少なくとも 1 つの閉軌道（極限サイクル）が存在する)
  - Proof (証明のスケッチ：$D$ 内の正の $\omega$-極限集合 $\omega(x)$ が空でなく有界なこと（コンパクト性）；$\omega$-極限集合が平衡点を含まないとき、ポアンカレ断面に対して帰着写像がある点に収束し、その軌道が閉曲線となることを示す)
  - Example 3.4-1 (ファン・デル・ポール方程式 $\ddot{x} - \mu(1-x^2)\dot{x} + x = 0$（$\mu > 0$）への適用：環状領域 $r_1 \leq \sqrt{x^2 + \dot{x}^2} \leq r_2$ を構成し内外で $\dot{r} < 0$（境界を越えない）を確認；定理より極限サイクルが存在することの確認)

- **2. 閉軌道の非存在条件**
  - Theorem 3.4-2 (デュラックの基準：$\mathbb{R}^2$ の単連結領域 $D$ で $\mathrm{div}(gf) = \partial(gf_1)/\partial x_1 + \partial(gf_2)/\partial x_2$ が定符号（または恒等的にゼロでない）となる $C^1$ 関数 $g$ が存在すれば、$\dot{x} = f(x)$ は $D$ 内に閉軌道を持たない)
  - Proof (デュラックの基準の証明：閉軌道が存在すれば囲む領域にグリーンの定理を適用して矛盾を導く；$\iint_R \mathrm{div}(gf)\,dA = \oint_\Gamma gf \cdot n\,ds = 0$（軌道上で $gf$ は接線方向）との矛盾)
  - Example 3.4-2 (デュラックの基準の適用：$\dot{x}_1 = x_2$、$\dot{x}_2 = -x_1 - x_2^3$ に $g = 1$ を試す；$\mathrm{div}f = \partial x_2/\partial x_1 + \partial(-x_1 - x_2^3)/\partial x_2 = -3x_2^2 \leq 0$（等号は $x_2 = 0$ の直線）；単連結領域 $D$ 全体でほぼ定符号だが直線上のみゼロ；閉軌道が存在しないことの確認)
  - Remark (2 次元の特殊性：ポアンカレ・ベンディクソン型の定理は 3 次元以上では成立しない；3 次元系ではローレンツアトラクターのようなカオス的挙動が可能である（§3.6 参照）)

### §3.5 大域的安定性と応用
**内容:** ラサールの不変原理と感染症モデルへの応用

- **1. ラサールの不変原理**
  - Theorem 3.5-1 (ラサールの不変原理：コンパクト集合 $K$ 内に収まる軌道に対して、$\dot{V} \leq 0$ の関数 $V$ が存在するとき、軌道は $E = \{x \in K \mid \dot{V}(x) = 0\}$ の最大不変部分集合 $M$ に収束する)
  - Proof (ラサールの不変原理の証明：$V$ が軌道上で単調減少かつ有界なので $\lim_{t\to\infty}V(x(t)) = c$ が存在する；$\omega$-極限集合 $\omega(x) \subset E$；$\omega(x)$ の不変性から $\omega(x) \subset M$)
  - Example 3.5-1 (ラサールの原理の応用：$\dot{x}_1 = -x_1 + x_2^2$、$\dot{x}_2 = -x_1 x_2$ に $V = x_1^2 + x_2^2$ を使う；$\dot{V} = -2x_1^2 \leq 0$；$E = \{x_1 = 0\}$ 上の不変集合は原点のみであることを確認；よって原点が漸近安定)

- **2. 感染症モデル（SIR モデル）への応用**
  - Example 3.5-2 (SIR モデルの安定性：$\dot{S} = -\beta SI$、$\dot{I} = \beta SI - \gamma I$、$\dot{R} = \gamma I$ において $R_0 = \beta N/\gamma$（基本再生産数）を定義；$R_0 < 1$ のとき感染消失の平衡点が漸近安定になることをリャプノフ関数法で示す；$V = I + (S - \bar{S} - \bar{S}\ln(S/\bar{S}))$ を構成して $\dot{V} \leq 0$ を確認)
  - Remark (SIR モデルの意義：COVID-19 をはじめとする感染症の流行予測に用いられる基本モデル；$R_0$ の大小による流行の有無の判定が安定性理論の直接的応用となっている)

### §3.6 分岐理論・ポアンカレ写像・カオス入門
**内容:** パラメータ変化による定性的変化とカオスの初歩

- **1. 分岐理論の基礎**
  - Definition 3.6-1 (分岐：パラメータ $\mu$ を含む系 $\dot{x} = f(x, \mu)$ において、$\mu$ が変化したとき平衡点の数・安定性・極限サイクルの有無などが定性的に変化する現象；変化が起きる $\mu$ の値を分岐値という)
  - Example 3.6-1 (鞍点-ノード分岐（サドル-ノード分岐）：$\dot{x} = \mu - x^2$ の平衡点は $\mu > 0$ で 2 個・$\mu = 0$ で 1 個・$\mu < 0$ で 0 個；$\mu = 0$ が分岐値；$\mu$ の増減に伴う相図の変化を追う)
  - Example 3.6-2 (アンドロノフ・ホップ分岐：$\dot{x}_1 = \mu x_1 - x_2 - x_1(x_1^2+x_2^2)$、$\dot{x}_2 = x_1 + \mu x_2 - x_2(x_1^2+x_2^2)$；$\mu < 0$ で安定焦点・$\mu = 0$ で中心・$\mu > 0$ で不安定焦点と半径 $\sqrt{\mu}$ の安定極限サイクルの出現)

- **2. ポアンカレ写像とカオス入門**
  - Definition 3.6-2 (ポアンカレ写像：軌道が横断的に横切る超平面 $\Sigma$ を設定し、$\Sigma$ 上の点 $x_0$ から出発した軌道が次に $\Sigma$ を横切る点 $P(x_0)$ を対応させる写像；周期軌道は $P$ の不動点に対応する)
  - Example 3.6-3 (ポアンカレ写像の不動点と安定性：1 次元ポアンカレ写像 $P(x) = ax(1-x)$（$a = 3.8$；ロジスティック写像）の数値例；$P^n(x_0)$ の軌道が不規則に見えること；初期値の差が指数的に拡大するリャプノフ指数の計算例）
  - Remark (カオスの定義と力学系理論：決定論的な方程式から生じる初期値鋭敏依存性を持つ挙動をカオスと呼ぶ；詳細な理論は力学系理論（4-6）で厳密に展開する)

## Chapter 4: 偏微分方程式入門
**テーマ:** PDE の基本概念と 1 階 PDE の完全解法・2 階 PDE の分類

### §4.1 偏微分方程式の基本概念
**内容:** 線形性・強解と弱解・初期条件と境界条件の分類

- **1. PDE の分類と線形性**
  - Definition 4.1-1 (偏微分方程式（PDE）：複数の独立変数に関する偏導関数を含む方程式；最高階の偏導関数の次数を方程式の階数という)
  - Definition 4.1-2 (線形 PDE：解 $u$ について線形な方程式、すなわち $L[u] = f$（$L$ は線形微分作用素、$f$ は既知関数）の形をしたもの；$f = 0$ のとき同次、$f \neq 0$ のとき非同次という；解の重ね合わせが成立する)
  - Definition 4.1-3 (準線形 PDE：最高次の偏導関数の係数が $u$ と低次の偏導関数に依存してよいが、最高次の偏導関数自体には線形なもの；非線形 PDE：それ以外)
  - Example 4.1-1 (PDE の例と分類：熱方程式 $u_t = u_{xx}$（線形・2 階・放物型）；波動方程式 $u_{tt} = c^2 u_{xx}$（線形・2 階・双曲型）；ラプラス方程式 $u_{xx} + u_{yy} = 0$（線形・2 階・楕円型）；バーガーズ方程式 $u_t + uu_x = 0$（準線形）の分類)

- **2. 境界条件と初期条件の分類**
  - Definition 4.1-4 (ディリクレ境界条件：境界 $\partial \Omega$ 上で $u = g$ を指定する条件；ノイマン境界条件：$\partial u/\partial n = g$（法線微分）を指定；ロバン境界条件：$\partial u/\partial n + \alpha u = g$ の混合形)
  - Example 4.1-2 (境界条件の意味：弦の振動 $u_{tt} = c^2 u_{xx}$（$0 < x < L$）の場合，ディリクレ条件 $u(0,t) = u(L,t) = 0$ は両端固定、ノイマン条件 $u_x(0,t) = u_x(L,t) = 0$ は両端自由に対応することの物理的解釈)
  - Remark (適切問題の概念：PDE の解が存在し一意で、かつデータに連続的に依存するとき適切問題（ウェルポーズド）という；詳しくは §9.4 で扱う)

### §4.2 1 階線形 PDE と特性曲線法
**内容:** 特性曲線に沿った ODE への帰着と一般解の構成

- **1. 特性曲線の定義と方程式**
  - Definition 4.2-1 (特性曲線：1 階線形 PDE $a(x,t)u_x + b(x,t)u_t = c(x,t)u + d(x,t)$ の特性曲線とは、常微分方程式系 $\frac{dx}{ds} = a$、$\frac{dt}{ds} = b$ の解曲線のこと；PDE は特性曲線上で ODE $\frac{du}{ds} = cu + d$ に帰着する)
  - Theorem 4.2-1 (1 階線形 PDE の解法：特性曲線と特性曲線上の ODE の解から一般解が構成できる；初期条件 $u(x, 0) = \phi(x)$ が与えられたとき、各初期点を通る特性曲線上の値が一意的に定まる)
  - Proof (定理の証明：特性曲線のパラメータ $s$ に沿った方向微分を計算し、PDE が常微分方程式に還元されることを代入で確認する)
  - Example 4.2-1 (輸送方程式 $u_t + cu_x = 0$、$u(x,0) = \phi(x)$ の解法：特性曲線は直線 $x - ct = \text{const}$；解は $u(x,t) = \phi(x - ct)$（右向き進行波）；特性曲線が「情報を運ぶ」ことの直感的説明)
  - Example 4.2-2 (減衰付き輸送方程式 $u_t + cu_x = -\alpha u$（$\alpha > 0$）の解：特性曲線は同じ直線 $x - ct = k$；ODE は $\dot{u} = -\alpha u$；解は $u(x,t) = \phi(x-ct)e^{-\alpha t}$；振幅が指数的に減衰することの確認)

- **2. 境界条件の処理**
  - Example 4.2-3 (領域内での境界値問題：$u_t + u_x = 0$（$x > 0, t > 0$）、$u(x,0) = e^{-x}$（初期条件）、$u(0,t) = \cos t$（境界条件）；$t < x$ の領域では初期条件から $u = e^{-(x-t)}$；$t > x$ の領域では境界条件から $u = \cos(t-x)$；2 領域の接続と特性曲線の役割を相図で確認)
  - Remark (特性曲線と情報伝播：1 階双曲型 PDE において特性曲線は「情報が伝播する経路」を表す；境界条件の情報は特性曲線に沿って領域内に運ばれる；この直感は 2 階双曲型（波動方程式）にも引き継がれる)

### §4.3 1 階準線形 PDE と衝撃波
**内容:** 特性曲線の交差と衝撃波の形成・ランキン・ユゴニオ条件

- **1. 準線形方程式と特性曲線の交差**
  - Definition 4.3-1 (1 階準線形 PDE の特性曲線：$u_t + c(u)u_x = 0$ の特性曲線は $dx/dt = c(u)$、$du/dt = 0$；すなわち特性曲線上で $u$ は一定値を保ちながら速度 $c(u)$ で $x$ 方向に移動する)
  - Example 4.3-1 (バーガーズ方程式 $u_t + uu_x = 0$、$u(x,0) = \phi(x)$ の特性曲線：各初期点 $x_0$ から速度 $\phi(x_0)$ で移動する直線 $x = x_0 + \phi(x_0)t$；$\phi$ が右下がりの場所では速い特性曲線に遅い特性曲線が追いつき有限時間 $t^* = -1/\min \phi'$ に交差する（衝撃波の形成））

- **2. 衝撃波とランキン・ユゴニオ条件**
  - Definition 4.3-2 (衝撃波：特性曲線が交差した後に古典解（$C^1$ 解）が存在しなくなり、解が不連続性（ジャンプ）を持つ曲面を衝撃波という；物理的には流体の圧縮衝撃などに対応する)
  - Theorem 4.3-1 (ランキン・ユゴニオ条件：保存則 $u_t + (F(u))_x = 0$ の弱解において、衝撃波の速度 $s$ は $s(u_R - u_L) = F(u_R) - F(u_L)$（$u_L, u_R$ は衝撃波左右の値）を満たさなければならない)
  - Proof (ランキン・ユゴニオ条件の証明：弱解の定義（テスト関数に対して積分で方程式が成立）を衝撃波面で分割し、ストークスの定理を適用することで境界積分から条件を導く)
  - Example 4.3-2 (バーガーズ方程式の衝撃波速度：初期値 $u(x,0) = 1$（$x < 0$）、$u(x,0) = 0$（$x > 0$）；ランキン・ユゴニオ条件 $s = (F(1) - F(0))/(1 - 0) = 1/2$（$F(u) = u^2/2$）から衝撃波が速度 $1/2$ で右進することの確認)
  - Remark (エントロピー条件：ランキン・ユゴニオ条件は複数の衝撃波解を許す場合がある；物理的に適切な解を選ぶエントロピー条件（レイエフ条件）が必要で、これは希薄波（膨張扇）を正しく選択する基準を与える)

### §4.4 2 階線形 PDE の分類
**内容:** 判別式による楕円型・放物型・双曲型の分類と標準形への変換

- **1. 2 変数 2 階線形 PDE の分類**
  - Definition 4.4-1 (2 階線形 PDE の分類：$Au_{xx} + 2Bu_{xy} + Cu_{yy} + \cdots = 0$ の主部の判別式 $\Delta = B^2 - AC$ の符号により、$\Delta < 0$（楕円型）、$\Delta = 0$（放物型）、$\Delta > 0$（双曲型）に分類される)
  - Theorem 4.4-1 (標準形の存在：適切な座標変換によって楕円型は $u_{\xi\xi} + u_{\eta\eta} + \cdots = 0$、双曲型は $u_{\xi\eta} + \cdots = 0$ または $u_{\xi\xi} - u_{\eta\eta} + \cdots = 0$、放物型は $u_{\eta\eta} + \cdots = 0$ の標準形に変換できる)
  - Proof (標準形変換の証明：変換 $\xi = \xi(x,y)$、$\eta = \eta(x,y)$ の連鎖律で各係数を書き直し、クロス項（$u_{\xi\eta}$）や主部の係数をゼロにする条件を求める；双曲型では特性曲線方向への座標変換が標準形を与える)
  - Example 4.4-1 (分類の確認：波動方程式 $u_{tt} - c^2 u_{xx} = 0$ は $\Delta = c^2 > 0$（双曲型）；熱方程式 $u_t - u_{xx} = 0$ は $A = -1, B = 0, C = 0$ なので $\Delta = 0$（放物型）；ラプラス方程式 $\Delta = -1 < 0$（楕円型）の確認)

- **2. 分類の物理的意味**
  - Example 4.4-2 (双曲型の標準形変換：$u_{tt} - c^2 u_{xx} = 0$ の特性曲線 $x \pm ct = \text{const}$ を新座標 $\xi = x + ct$、$\eta = x - ct$ とすると標準形 $u_{\xi\eta} = 0$；一般解 $u = F(\xi) + G(\eta) = F(x+ct) + G(x-ct)$（ダランベールの解）の導出)
  - Remark (型の物理的意味：双曲型は「波の伝播」（有限速度での情報伝達）；放物型は「熱の拡散」（無限伝播速度・時間の向きの固定）；楕円型は「定常状態」（時間方向がなく全領域が結合）を表す；これが各型の境界条件・初期条件の設定の違いを生む)

## Chapter 5: 分離変数法とシュトゥルム・リウヴィル問題
**テーマ:** 偏微分方程式を常微分方程式に帰着させる中心的技法とその理論的基盤

### §5.1 分離変数法の手順と適用条件
**内容:** 積形の試解とその正当化、境界条件の処理

- **1. 分離変数法の基本手順**
  - Definition 5.1-1 (分離変数法：PDE の解を $u(x,t) = X(x)T(t)$ の積の形に仮定し、代入によって $X$ と $T$ の独立な ODE に分解する手法；各 ODE の比が定数（分離定数 $-\lambda$）になることが鍵)
  - Example 5.1-1 (熱方程式 $u_t = \kappa u_{xx}$（$0 < x < L$）の変数分離：$u = X(x)T(t)$ を代入すると $T'/\kappa T = X''/X = -\lambda$；境界条件 $X(0) = X(L) = 0$ から $\lambda = (n\pi/L)^2$、$X_n(x) = \sin(n\pi x/L)$；$T_n(t) = e^{-\kappa(n\pi/L)^2 t}$；一般解は線形結合 $u = \sum a_n \sin(n\pi x/L)e^{-\kappa(n\pi/L)^2 t}$)
  - Remark (分離変数法の適用条件：線形・同次・係数が変数分離形（$a(x)$ と $b(t)$ の積）であることが必要；境界条件も空間変数のみに依存する場合に直接適用できる)

- **2. 初期条件の処理とフーリエ係数**
  - Example 5.1-2 (初期条件の処理：$u(x,0) = f(x)$ が与えられたとき $f(x) = \sum a_n \sin(n\pi x/L)$（フーリエ正弦級数展開）；係数は $a_n = \frac{2}{L}\int_0^L f(x)\sin(n\pi x/L)\,dx$；$f(x) = x(L-x)$ の場合の $a_n$ を直接計算し $a_n = 8L^2/(n^3\pi^3)$（$n$ 奇数）・$0$（$n$ 偶数）を得る)
  - Remark (分離変数法とフーリエ解析の接続：固有関数 $\sin(n\pi x/L)$ への展開は本質的にフーリエ級数（3-4 で学んだ）と同一の構造を持つ；分離変数法はフーリエ解析の PDE 版と捉えることができる)

### §5.2 シュトゥルム・リウヴィル問題の設定
**内容:** 正則・特異 SL 問題の設定と固有値・固有関数の一般論

- **1. シュトゥルム・リウヴィル方程式**
  - Definition 5.2-1 (シュトゥルム・リウヴィル方程式：$\frac{d}{dx}\!\left[p(x)\frac{dy}{dx}\right] + \left[q(x) + \lambda w(x)\right]y = 0$（$a < x < b$）の形をした固有値問題；$p, p', q, w$ が連続かつ $p > 0$、$w > 0$ のとき正則と呼ぶ；境界条件は通常 $\alpha_1 y(a) + \alpha_2 y'(a) = 0$、$\beta_1 y(b) + \beta_2 y'(b) = 0$ の形)
  - Example 5.2-1 (シュトゥルム・リウヴィル形式への変換：$y'' + \lambda y = 0$（$0 < x < L$）、$y(0) = y(L) = 0$ は $p = 1$、$q = 0$、$w = 1$ の最も基本的な SL 問題；ベッセル方程式も $p = x$、$w = x$ と置くことで SL 形式に書ける（特異 SL 問題）)

- **2. 固有値の実数性と固有関数の直交性の見通し**
  - Theorem 5.2-1 (自己共役性：正則 SL 問題の微分作用素 $L = -\frac{d}{dx}\!\left[p\frac{d}{dx}\right] - q(x)$ に境界条件を込めると自己共役作用素になる；すなわち $\langle Lu, v\rangle_w = \langle u, Lv\rangle_w$（$\langle \cdot, \cdot \rangle_w$ は重み $w$ 付き $L^2$ 内積）)
  - Proof (自己共役性の証明：$\langle Lu, v\rangle_w - \langle u, Lv\rangle_w$ を計算すると境界項 $[p(uv' - u'v)]_a^b$ が残り、境界条件から境界項がゼロになることを確認する)
  - Remark (自己共役と固有値の実数性の関係：自己共役作用素の固有値は実数であり、異なる固有値に対応する固有関数は直交する；これが次節の定理を導く（関数解析（4-1）でより一般的に扱う）)

### §5.3 固有値と固有関数の性質
**内容:** 実数性・単調性・直交性の証明と固有値の定性的性質

- **1. 固有値の実数性と直交性**
  - Theorem 5.3-1 (固有値の実数性：正則 SL 問題のすべての固有値は実数である)
  - Proof (実数性の証明：固有値 $\lambda$ と対応する固有関数 $\phi$ に対して $\langle L\phi, \phi\rangle_w = \lambda\|\phi\|_w^2$；自己共役性から $\lambda\|\phi\|_w^2 = \overline{\lambda}\|\phi\|_w^2$ を得て $\lambda = \overline{\lambda}$)
  - Theorem 5.3-2 (固有関数の直交性：正則 SL 問題の異なる固有値 $\lambda_m \neq \lambda_n$ に対応する固有関数 $\phi_m$、$\phi_n$ は重み $w$ 付き $L^2$ 内積で直交する：$\int_a^b \phi_m(x)\phi_n(x)w(x)\,dx = 0$)
  - Proof (直交性の証明：$L\phi_m = \lambda_m w \phi_m$ と $L\phi_n = \lambda_n w\phi_n$ から $(\lambda_m - \lambda_n)\langle\phi_m,\phi_n\rangle_w = \langle L\phi_m,\phi_n\rangle_w - \langle\phi_m,L\phi_n\rangle_w = 0$（自己共役性より）；$\lambda_m \neq \lambda_n$ で割る)
  - Example 5.3-1 ($\phi_n(x) = \sin(n\pi x/L)$ の直交性の直接確認：$\int_0^L \sin(m\pi x/L)\sin(n\pi x/L)\,dx = 0$（$m \neq n$）を積和公式で計算して確認；これは Theorem 5.3-2 の特殊ケース)

- **2. 固有値の定性的性質**
  - Theorem 5.3-3 (固有値の単調性と無限性：正則 SL 問題の固有値は番号をつけると $\lambda_1 < \lambda_2 < \lambda_3 < \cdots$ と単調増大し $\lambda_n \to +\infty$（$n \to \infty$）；各固有値の重複度は 1（単純）)
  - Proof (証明のスケッチ：固有値の下限が有限であることを変分的評価から示す；隣接する固有関数 $\phi_n$ と $\phi_{n+1}$ の節点定理（$\phi_n$ がちょうど $n-1$ 個の零点を持つ）から固有値の単純性と単調性が従う)
  - Remark (節点定理の直感：固有関数 $\phi_n$ が $[a,b]$ 内でちょうど $n-1$ 回ゼロになるという性質は、$n$ 番目の「振動モード」が $n-1$ 個の節点を持つという物理的直感と一致する)

### §5.4 固有関数展開と完全性
**内容:** 固有関数による展開の正当化と解の構成

- **1. 固有関数展開の収束**
  - Theorem 5.4-1 (固有関数展開の完全性：正則 SL 問題の固有関数全体 $\{\phi_n\}_{n=1}^\infty$ は重み $w$ 付き $L^2(a,b)$ の完全正規直交系をなす；すなわち任意の $f \in L^2_w(a,b)$ は $f = \sum_{n=1}^\infty c_n \phi_n$（$c_n = \langle f, \phi_n\rangle_w / \|\phi_n\|_w^2$）と展開でき、等式は $L^2_w$ の意味で成立する)
  - Proof (完全性の証明のスケッチ：自己共役コンパクト作用素（$L^{-1}$ のコンパクト性を利用）のスペクトル定理から固有関数の完全性が従う；詳細は関数解析（4-1）で扱う)
  - Example 5.4-1 ($f(x) = x$（$0 < x < L$）のフーリエ正弦展開：固有関数 $\sin(n\pi x/L)$ に対して $c_n = \frac{2}{L}\int_0^L x\sin(n\pi x/L)\,dx = \frac{2L}{\pi n}(-1)^{n+1}$；$N = 5$ での部分和と元の関数を比較し収束を確認)

- **2. PDE の解の構成への応用**
  - Example 5.4-2 (非同次境界値問題の解：$u_{xx} + \lambda u = f(x)$（$0 < x < 1$）、$u(0) = u(1) = 0$；固有関数展開 $u = \sum c_n \sin(n\pi x)$ と $f = \sum f_n \sin(n\pi x)$ を代入すると $c_n = f_n/(n^2\pi^2 - \lambda)$；$\lambda = n^2\pi^2$ のとき分母がゼロになる（共鳴）ことの意味と対処法)
  - Remark (完全性と PDE の解の一意性：固有関数展開の完全性は PDE の解の展開が「十分一般的」であることを保証する；一方、完全性の証明には関数解析の道具（コンパクト作用素のスペクトル定理）が不可欠であり、これは Stage 4 の関数解析（4-1）への重要な橋頭堡となる)

## Chapter 6: 波動方程式
**テーマ:** 双曲型方程式の理論と多様な解法

### §6.1 1 次元波動方程式の導出
**内容:** 弦の振動モデルからの物理的導出と波速の意味

- **1. 弦の運動方程式の導出**
  - Example 6.1-1 (弦の振動のモデル化：長さ $L$ の弦を微小区間 $[x, x+\Delta x]$ に分割し張力 $T$ と線密度 $\rho$ のニュートンの運動方程式 $\rho \Delta x \cdot u_{tt} \approx T\sin\theta(x+\Delta x) - T\sin\theta(x)$；小振幅仮定 $\sin\theta \approx \tan\theta = u_x$ のもとで $\Delta x \to 0$ とすると $u_{tt} = c^2 u_{xx}$（$c = \sqrt{T/\rho}$）の導出)
  - Remark (波速の意味：$c$ の次元は速度（m/s）；弦の張力が強いほど・線密度が小さいほど波は速く伝わる；ギターの弦の高音弦（細い・張りが強い）は速く振動するという日常経験と対応する)

- **2. 方程式の基本性質**
  - Proposition 6.1-1 (波動方程式の線形性と重ね合わせ原理：$u_1, u_2$ が波動方程式を満たせば $\alpha u_1 + \beta u_2$ も満たす；これにより分離変数解の無限和が解を与える正当性の根拠)
  - Proof (線形性の確認：$u_{tt} = c^2 u_{xx}$ の左辺・右辺に $\alpha u_1 + \beta u_2$ を代入し、微分の線形性から直接確認する)
  - Example 6.1-2 (初期条件と境界条件の設定：有限弦の場合 $0 < x < L$、$t > 0$ に対して初期変位 $u(x,0) = f(x)$・初期速度 $u_t(x,0) = g(x)$・境界条件 $u(0,t) = u(L,t) = 0$ を組み合わせた初期境界値問題の設定)

### §6.2 ダランベールの解（初期値問題）
**内容:** 一般解の導出と双曲型 PDE の特性曲線の役割

- **1. ダランベールの解の導出**
  - Theorem 6.2-1 (ダランベールの解：波動方程式 $u_{tt} = c^2 u_{xx}$（$-\infty < x < \infty$）、$u(x,0) = f(x)$、$u_t(x,0) = g(x)$ の解は $u(x,t) = \frac{f(x+ct)+f(x-ct)}{2} + \frac{1}{2c}\int_{x-ct}^{x+ct} g(s)\,ds$)
  - Proof (ダランベールの解の導出：標準形 $u_{\xi\eta} = 0$（$\xi = x+ct$、$\eta = x-ct$）の一般解 $u = F(\xi) + G(\eta)$ を初期条件に代入して $F + G = f$ と $c(F' - G') = g$ を解く；$F' = (f' + g/c)/2$ を積分して $F$ を求め、$G = f - F$ とする)
  - Example 6.2-1 ($g(x) = 0$、$f(x) = \sin x$ の場合：$u(x,t) = (\sin(x+ct) + \sin(x-ct))/2 = \sin x \cos ct$；これは「定在波」の形をしており進行波の重ね合わせで生じることの確認)
  - Example 6.2-2 ($f(x) = 0$、$g(x) = \delta_0(x)$（点源速度）の場合：$u(x,t) = 1/(2c)$（$|x| < ct$）、$= 0$（$|x| > ct$）；波が速度 $c$ の前線の内側のみに存在する（1 次元ホイヘンスの原理の類似）)

- **2. 依存域・影響域と有限伝播速度**
  - Definition 6.2-1 (依存域：点 $(x_0, t_0)$ の解の値が依存する初期データの区間 $[x_0 - ct_0, x_0 + ct_0]$；影響域：点 $(x_0, 0)$ の初期データが影響を与える時空間領域 $\{(x,t): |x - x_0| < ct\}$)
  - Example 6.2-3 (有限伝播速度の確認：初期データが $x \in [1, 2]$ のみサポートを持つ場合（例：$f(x) = e^{-1/(x-1)(2-x)}$（$1 < x < 2$）・外では 0）；時刻 $t$ での $u(x,t)$ のサポートは $[1 - ct, 2 + ct]$ の外でゼロになること；波が速度 $c$ より速く伝わらないことの確認)
  - Remark (波動方程式の双曲型としての特徴：有限伝播速度と依存域・影響域の存在は双曲型方程式に特有の性質；楕円型（ラプラス方程式）では無限速度で情報が伝わり依存域が全域になる（§8.2 の平均値定理と対比）)

### §6.3 分離変数による境界値問題
**内容:** 定在波の構成とフーリエ展開による初期条件の処理

- **1. 定在波と固有モード**
  - Example 6.3-1 (固有モードの計算：$u_{tt} = c^2 u_{xx}$（$0 < x < L$）、$u(0,t) = u(L,t) = 0$ に $u = X(x)T(t)$ を代入すると $X_n(x) = \sin(n\pi x/L)$、$T_n(t) = A_n\cos(\omega_n t) + B_n\sin(\omega_n t)$（$\omega_n = cn\pi/L$）；解の各モード $u_n = X_n T_n$ が「波長 $2L/n$・角振動数 $\omega_n$」の定在波であることの確認)
  - Example 6.3-2 (フーリエ展開による初期条件の充足：初期変位 $u(x,0) = f(x) = x(L-x)$・初期速度 $u_t(x,0) = 0$；$A_n = \frac{2}{L}\int_0^L f(x)\sin(n\pi x/L)\,dx$・$B_n = 0$；$A_n$ を計算して一般解を具体的に構成)

- **2. エネルギー解釈と倍音**
  - Remark (倍音と音楽の関係：基本振動（$n=1$）の上に $n=2, 3, \ldots$ の倍音が重なることが楽器の音色（timbre）を決める；弦の素材・長さ・張り方によって各 $A_n$ の大きさが変わり、音色の多様性が生まれる)
  - Proposition 6.3-1 (エネルギーの保存：波動方程式の解のエネルギー $E(t) = \frac{1}{2}\int_0^L (u_t^2 + c^2 u_x^2)\,dx$ は時間不変である)
  - Proof (エネルギー保存の証明：$\dot{E} = \int_0^L (u_t u_{tt} + c^2 u_x u_{xt})\,dx$；$u_{tt} = c^2 u_{xx}$ を代入し部分積分すると $\dot{E} = c^2[u_t u_x]_0^L = 0$（境界条件から）)

### §6.4 エネルギー保存則と一意性
**内容:** エネルギー法による解の一意性の証明

- **1. 一意性定理**
  - Theorem 6.4-1 (波動方程式の解の一意性：初期境界値問題（$u_{tt} = c^2 u_{xx}$、$0 < x < L$、$t > 0$；$u(0,t) = u(L,t) = 0$；$u(x,0) = f(x)$、$u_t(x,0) = g(x)$）の古典解は一意である)
  - Proof (一意性の証明：2 つの解の差 $w = u_1 - u_2$ は $w_{tt} = c^2 w_{xx}$・ゼロ初期条件を満たす；$E(t) = \frac{1}{2}\int_0^L(w_t^2 + c^2 w_x^2)\,dx$ はエネルギー保存（Proposition 6.3-1）と $E(0) = 0$ から $E(t) = 0$；$w_t = w_x = 0$ かつ $w(x,0) = 0$ より $w \equiv 0$)
  - Remark (エネルギー法の汎用性：エネルギー量の定義と「差が満たす方程式」への適用という証明の枠組みは、熱方程式・ラプラス方程式にも同様に応用できる（§7.5, §8.2 参照）)

- **2. 連続依存性と適切問題**
  - Proposition 6.4-1 (初期データへの連続依存性：初期データ $f, g$ のエネルギーノルムが小さければ解のエネルギーも小さい；すなわち $E(t) = E(0)$ から $\|u\|_{H^1} \leq C\|(f,g)\|$ の評価が従う)
  - Proof (連続依存性の証明：エネルギー保存 $E(t) = E(0) = \frac{1}{2}\int(g^2 + c^2 (f')^2)\,dx$ を初期データのノルムで評価する)
  - Remark (適切問題の 3 条件の充足：存在（§6.3 の構成）・一意性（Theorem 6.4-1）・連続依存性（Proposition 6.4-1）が揃って初期境界値問題が「適切（ウェルポーズド）」になる)

### §6.5 高次元への拡張
**内容:** ホイヘンスの原理と 3 次元での解の概略

- **1. 高次元波動方程式**
  - Proposition 6.5-1 (球面平均による 3 次元波動方程式の解：3 次元波動方程式 $u_{tt} = c^2 \Delta u$ の球対称初期データに対する解はキルヒホッフの公式で与えられる；解の値は半径 $ct$ の球面上のデータの積分として表せる)

## Chapter 7: 熱方程式
**テーマ:** 放物型方程式の理論と解析的性質

### §7.1 熱方程式の導出と基本的性質
**内容:** フーリエの熱伝導則からの導出と不可逆性

- **1. 熱方程式の物理的導出**
  - Example 7.1-1 (熱伝導則から PDE の導出：一次元棒の微小区間 $[x, x+\Delta x]$ での熱収支；フーリエの熱伝導則 $q = -\kappa u_x$（熱流束は温度勾配に逆比例）と保存則からエネルギー保存を書くと $\rho c_p u_t = \kappa u_{xx}$；$\alpha = \kappa/(\rho c_p)$（熱拡散率）とおくと $u_t = \alpha u_{xx}$)
  - Definition 7.1-1 (熱方程式（拡散方程式）：$u_t = \kappa \Delta u$（$\kappa > 0$）；$\kappa$ を熱拡散率（diffusivity）という；同じ形の方程式がブラウン運動・物質拡散・オプション価格（ブラック・ショールズ方程式）にも現れる)
  - Remark (波動方程式との対比：波動方程式 $u_{tt} = c^2 u_{xx}$ は時間方向で対称（$t \to -t$ の置換で同じ形）；熱方程式 $u_t = \kappa u_{xx}$ は時間方向で非対称（$t \to -t$ でILL-posed）；不可逆性が「熱力学第二法則」の数学的反映)

- **2. 平滑化効果**
  - Theorem 7.1-1 (即時平滑化：$t > 0$ で $u$ は無限回微分可能（$C^\infty$）；初期値 $u(x,0) = f(x)$ がどれだけ不規則でも正の時刻では $u(\cdot, t)$ は滑らかになる)
  - Proof (平滑化の証明スケッチ：§7.2 の熱核による解の表示 $u(x,t) = \int K(x-y,t)f(y)\,dy$ において、ガウス関数 $K$ の任意の $x$ 微分が $L^1$ に属することから $x$ に関する無限回微分可能性が従う；積分を微分の内側に入れる操作は $t > 0$ でのみ正当化される)
  - Remark (逆問題の不良設定性：平滑化効果から、$t > 0$ での温度分布から $t = 0$ の初期値を復元する逆問題は本質的に不安定（小さいデータ誤差が初期値の大きな誤差につながる）；過去の温度推定が難しい数学的理由)

### §7.2 基本解（熱核）
**内容:** ガウス関数による基本解の導出と性質

- **1. 熱核の導出**
  - Definition 7.2-1 (熱核（基本解）：$K(x,t) = \frac{1}{\sqrt{4\pi\kappa t}}e^{-x^2/(4\kappa t)}$（$t > 0$）；$t \to 0^+$ のとき $K(\cdot, t) \to \delta_0$（ディラックのデルタ関数）の意味で収束する；$K$ は熱方程式を満たす)
  - Proof (熱核の導出：$u_t = \kappa u_{xx}$ にスケーリング $u(x,t) = t^{-\alpha}v(xt^{-\beta})$ を仮定し $\alpha = 1/2$、$\beta = 1/2$ で自己相似解を求める；$v$ が満たす ODE を解くとガウス関数が得られる；積分が 1 になるよう正規化)
  - Example 7.2-1 (熱核の具体的計算：$\kappa = 1$ のとき $K(x, t) = (4\pi t)^{-1/2}e^{-x^2/(4t)}$；$t = 0.01, 0.1, 1.0$ での形状比較；$t$ が増えるほど「低く広がる」ことの確認；$\int_{-\infty}^\infty K(x,t)\,dx = 1$（ガウス積分から）の計算)

- **2. 熱核の性質**
  - Proposition 7.2-1 (熱核の積の性質：$K(x,t) * K(x,s) = K(x,t+s)$（たたみ込み）；すなわち「$t$ 時間後と $s$ 時間後」の拡散を順番に適用することと「$t+s$ 時間後」の拡散が等価であること)
  - Proof (積の性質の証明：ガウス積分 $\int e^{-a(x-y)^2}e^{-b y^2}\,dy = \sqrt{\pi/(a+b)} \cdot e^{-abx^2/(a+b)}$ を計算することで直接確認する)
  - Remark (半群性：$K$ のたたみ込みの積の性質は「熱方程式の解の時間発展が半群をなす」という代数的構造の表れ；$T(t)f = K(\cdot,t)*f$ と書けばマルコフ半群の例となる（確率論・関数解析への接続）)

### §7.3 初期値問題とたたみ込みによる解
**内容:** 畳み込みによる一般解の構成と非有界域での解析

- **1. たたみ込み解の構成**
  - Theorem 7.3-1 (熱方程式の初期値問題の解：$u_t = \kappa u_{xx}$（$x \in \mathbb{R}$、$t > 0$）、$u(x,0) = f(x)$ の解は $u(x,t) = \int_{-\infty}^\infty K(x-y,t)f(y)\,dy = (K(\cdot,t)*f)(x)$；$f$ が有界かつ連続ならば $u$ は $C^\infty$ で熱方程式を満たし初期条件を達成する)
  - Proof (証明のスケッチ：$K$ が熱方程式を満たし $y$ に関する積分は定数なので $u_t = \kappa u_{xx}$ が従う；初期条件は $K(\cdot,t) \to \delta_0$（近似恒等式の性質）から $u(x,t) \to f(x)$)
  - Example 7.3-1 (ガウス型初期値の解：$f(x) = e^{-ax^2}$（$a > 0$）のとき $u(x,t) = K(\cdot,t)*f = \frac{1}{\sqrt{1+4\kappa at}}e^{-ax^2/(1+4\kappa at)}$；初期の幅 $1/\sqrt{a}$ のガウス分布が時刻 $t$ に幅 $\sqrt{(1+4\kappa at)/a}$ に広がることの確認)

- **2. 非同次問題とデュアメルの原理**
  - Theorem 7.3-2 (デュアメルの原理：非同次熱方程式 $u_t = \kappa u_{xx} + h(x,t)$、$u(x,0) = 0$ の解は $u(x,t) = \int_0^t \int_{-\infty}^\infty K(x-y,t-s)h(y,s)\,dy\,ds$)
  - Proof (デュアメルの原理の証明：$v(x,t;s) = \int K(x-y,t-s)h(y,s)\,dy$（固定した $s$ での解）を定義し $u = \int_0^t v\,ds$ の微分を Leibniz 則で計算する；$v(x,s;s) = h(x,s)$ を利用)
  - Remark (デュアメルの原理の意味：非同次項 $h$ を「時刻 $s$ で打ち込まれた熱源」として扱い、各時刻の寄与を重ね合わせる直感；ODE の定数変化法の PDE 版)

### §7.4 境界値問題と分離変数
**内容:** ディリクレ・ノイマン境界での解の構成と長時間挙動

- **1. ディリクレ境界条件での解**
  - Example 7.4-1 (ディリクレ境界値問題：$u_t = \kappa u_{xx}$（$0 < x < L$）、$u(0,t) = u(L,t) = 0$、$u(x,0) = f(x)$；分離変数により $u(x,t) = \sum_{n=1}^\infty a_n \sin(n\pi x/L) e^{-\kappa (n\pi/L)^2 t}$；$f(x) = 100$（一様初期温度）のとき $a_n = 200/(n\pi)(1 - (-1)^n)$ を計算して奇数次のみ残ることを確認)
  - Remark (熱の「平滑化と均一化」：ディリクレ条件（端点 0 度固定）のもとで $u(x,t) \to 0$（$t \to \infty$）；高周波成分（大きな $n$）ほど指数 $e^{-\kappa(n\pi/L)^2 t}$ の減衰が速く、長時間では基本モード（$n=1$）が支配的になる)

- **2. ノイマン境界条件での解**
  - Example 7.4-2 (ノイマン境界値問題：$u_t = \kappa u_{xx}$、$u_x(0,t) = u_x(L,t) = 0$、$u(x,0) = f(x)$；固有関数は $\cos(n\pi x/L)$（$n = 0, 1, 2, \ldots$）；$n=0$ の成分が $a_0 = \frac{1}{L}\int_0^L f(x)\,dx$（$f$ の平均値）で一定；$u(x,t) \to a_0$（$t \to \infty$）として熱平衡に達することの確認)
  - Remark (ディリクレとノイマンの違い：ディリクレ条件は「端点の温度を固定する」（熱浴に接触）；ノイマン条件は「端点での熱流束をゼロにする」（断熱壁）；後者では熱が外に逃げないので棒全体が平均温度に収束する)

### §7.5 最大値原理と一意性
**内容:** 最大値原理の証明と解の一意性・安定性への応用

- **1. 最大値原理**
  - Theorem 7.5-1 (弱最大値原理：$u_t - \kappa u_{xx} \leq 0$（有界な領域 $Q_T = (0,L) \times (0,T)$ で）ならば $u$ の最大値は $Q_T$ の放物的境界（$t=0$ のボトムと $x=0,L$ のサイド）で達成される；内部や上辺 $t = T$ では最大値を取らない)
  - Proof (弱最大値原理の証明：内部の最大点 $(x_0, t_0)$（$t_0 < T$）では $u_t(x_0,t_0) \geq 0$ かつ $u_{xx}(x_0,t_0) \leq 0$；方程式 $u_t = \kappa u_{xx}$ から $0 \leq u_t = \kappa u_{xx} \leq 0$ で矛盾；$t_0 = T$ の場合は $\varepsilon t$ の補正項を加えた関数で処理する)
  - Example 7.5-1 (最大値原理の応用：$u_t = u_{xx}$（$0 < x < 1, 0 < t < 1$）、$u(x,0) = \sin(\pi x)$（$\leq 1$）、$u(0,t) = u(1,t) = 0$；最大値原理から $0 \leq u(x,t) \leq 1$（すべての $(x,t)$ で）が従う；境界・初期値の最大 $1$ が内部で超えられないことの確認)

- **2. 一意性と安定性**
  - Theorem 7.5-2 (熱方程式の解の一意性：$u_t = \kappa u_{xx}$（$0 < x < L$、$0 < t \leq T$）の有界な古典解は初期・境界条件により一意に定まる)
  - Proof (一意性の証明：2 解の差 $w = u_1 - u_2$ は同次方程式を満たす；最大値原理を $w$ と $-w$ に適用すると $w \leq 0$ かつ $w \geq 0$、よって $w = 0$)
  - Proposition 7.5-1 (安定性：境界・初期データの $\sup$ ノルムの差が $\varepsilon$ ならば解の差も $\varepsilon$ 以下；最大値原理の直接適用)
  - Proof (安定性の証明：差 $w = u_1 - u_2$ に最大値原理を適用すると $\|w\|_\infty \leq \max(\|f_1 - f_2\|_\infty, \|g_1 - g_2\|_\infty) \leq \varepsilon$)
  - Remark (最大値原理の応用範囲：熱方程式だけでなく、一般的な放物型方程式・楕円型方程式（§8.2）にも最大値原理の類似が成り立つ；「解が境界データの範囲内に収まる」という直感は PDE 全体を通じた重要な性質である)

## Chapter 8: ラプラス方程式とポアソン方程式
**テーマ:** 楕円型方程式の理論と調和関数の深い解析的性質

### §8.1 調和関数の定義と基本的性質
**内容:** ラプラス方程式を満たす関数の定義と代表例

- **1. 調和関数の定義**
  - Definition 8.1-1 (調和関数：開集合 $\Omega \subset \mathbb{R}^n$ 上で $\Delta u = \sum_{i=1}^n \partial^2 u/\partial x_i^2 = 0$ を満たす $C^2$ 関数 $u$ を調和関数という)
  - Example 8.1-1 (調和関数の例：1 変数では $u(x) = ax + b$；2 変数では $u = x^2 - y^2$（$\Delta u = 2 - 2 = 0$ の確認）、$u = e^x\cos y$（コーシー・リーマン方程式から）；$n$ 変数では $u = |x|^{2-n}$（$n \geq 3$；ニュートンポテンシャル）)
  - Remark (複素解析との接続：正則関数 $f(z) = u(x,y) + iv(x,y)$ の実部・虚部がともに調和関数になる（3-2 §2.3 の結果）；これにより複素解析の道具が 2 次元調和関数の研究に使える)

- **2. 解析性と無限回微分可能性**
  - Theorem 8.1-1 (調和関数の解析性：調和関数は $C^\infty$ かつ実解析的（実べき級数に展開できる）)
  - Proof (解析性の証明スケッチ：§8.2 の平均値定理と§8.4 のポアソン積分公式を用いると、解が定数係数の積分核との畳み込みで表せる；この表示から $x$ に関する任意の偏微分が有界であることが従い、テイラー展開の収束が示せる)
  - Remark (楕円型方程式の平滑化効果：調和関数の解析性は「楕円型方程式は無限大の速度で情報を広める」という直感と一致する；波動方程式の有限伝播速度（§6.2）との鋭い対比)

### §8.2 平均値定理と最大値原理
**内容:** 調和関数の「平均値が中心値に一致する」という特徴的性質とその帰結

- **1. 平均値定理**
  - Theorem 8.2-1 (平均値定理：$u$ が球 $B_r(x_0) \subset \Omega$ で調和ならば $u(x_0) = \frac{1}{|\partial B_r|}\int_{\partial B_r(x_0)} u\,dS$（球面平均と一致）かつ $u(x_0) = \frac{1}{|B_r|}\int_{B_r(x_0)} u\,dx$（球体平均とも一致）)
  - Proof (平均値定理の証明：$\phi(r) = \frac{1}{|\partial B_r|}\int_{\partial B_r} u\,dS$ とおき $\phi'(r) = 0$ を確認する；法線微分の公式と $\Delta u = 0$ からグリーンの公式で $\phi'(r) = \frac{1}{|\partial B_r|}\int_{B_r}\Delta u\,dx = 0$)
  - Example 8.2-1 (平均値定理の確認：$u = x^2 - y^2$ が $\mathbb{R}^2$ で調和；原点中心半径 $r$ の円周上での平均 $\frac{1}{2\pi}\int_0^{2\pi}(r^2\cos^2\theta - r^2\sin^2\theta)\,d\theta = 0 = u(0,0)$ の計算)

- **2. 最大値原理と一意性**
  - Theorem 8.2-2 (強最大値原理：連結開領域 $\Omega$ の内部で調和かつ内部で最大値を取るならば、$u$ は定数である；特に $\Omega$ が有界なら最大値は境界で達成される)
  - Proof (強最大値原理の証明：$u(x_0) = \max_{\bar\Omega} u$ が内部点で達成されたとする；$u(x_0)$ と近傍の球面平均が等しいとき（平均値定理）、球面上に $u < u(x_0)$ の点があれば矛盾が生じる；接続論法で内部全体が最大値定数になることを示す)
  - Theorem 8.2-3 (ディリクレ問題の解の一意性：有界領域 $\Omega$ でのディリクレ問題 $\Delta u = 0$（$\Omega$）、$u = g$（$\partial \Omega$）の古典解は高々一つ)
  - Proof (一意性の証明：2 解の差 $w$ は調和で $w = 0$（$\partial \Omega$ 上）；最大値原理から $\max_{\bar\Omega} w = \max_{\partial\Omega} w = 0$ かつ $\min_{\bar\Omega} w \geq 0$；ゆえに $w \equiv 0$)
  - Remark (平均値定理の逆：$\Omega$ 上の連続関数で「すべての球での平均値が中心値に一致する」ならば $u$ は調和である（ガウスの定理の逆）；すなわち平均値性質が調和性と同値になる)

### §8.3 ディリクレ問題とノイマン問題
**内容:** 境界条件の違いと解の存在・一意性・不定性

- **1. ディリクレ問題の設定**
  - Definition 8.3-1 (ディリクレ問題：領域 $\Omega$ 上の $\Delta u = f$（$\Omega$）、$u = g$（$\partial\Omega$）；$f = 0$ をラプラス方程式のディリクレ問題、$f \neq 0$ をポアソン方程式のディリクレ問題という)
  - Example 8.3-1 (1 次元の類比：$u'' = 0$（$0 < x < 1$）、$u(0) = a$、$u(1) = b$ の解は唯一 $u(x) = a + (b-a)x$；2 次元の場合は同様の一意性が最大値原理から従うが、存在証明には追加の議論が必要)

- **2. ノイマン問題の設定と必要条件**
  - Definition 8.3-2 (ノイマン問題：$\Delta u = f$（$\Omega$）、$\partial u/\partial n = g$（$\partial\Omega$）；$n$ は外向き単位法線ベクトル)
  - Theorem 8.3-1 (ノイマン問題の適合条件：ノイマン問題が解を持つ必要条件として $\int_\Omega f\,dx + \int_{\partial\Omega} g\,dS = 0$（発散定理から）が成立しなければならない；適合条件が満たされても解は定数の不定性を持つ)
  - Proof (適合条件の証明：$u$ が解ならばグリーンの公式 $\int_\Omega \Delta u\,dx = \int_{\partial\Omega}\partial u/\partial n\,dS$ を適用すると $\int f\,dx = \int g\,dS$；この等式が成立しなければ解が存在しない)
  - Example 8.3-2 (1 次元の類比：$u'' = f$（$0 < x < 1$）、$u'(0) = a$、$u'(1) = b$；積分すると $b - a = \int_0^1 f\,dx$ が必要条件；さらに $u(0) = c$（任意定数）の自由度があり解は定数不定)
  - Remark (ノイマン問題の物理的意味：ディリクレ条件は境界での「温度固定」；ノイマン条件は「熱流束固定」；適合条件は「入ってくる熱と出ていく熱の収支が合う」という物理的エネルギー保存に対応する)

### §8.4 ポアソン方程式と基本解
**内容:** $n$ 次元の基本解の導出とポアソン積分公式

- **1. 基本解の構成**
  - Definition 8.4-1 ($n$ 次元ラプラス作用素の基本解：$\Delta \Phi = \delta_0$（ディラックのデルタ関数）を満たす球対称解；$\Phi(x) = -\frac{1}{(n-2)\omega_n}|x|^{2-n}$（$n \geq 3$）、$\Phi(x) = \frac{1}{2\pi}\ln|x|$（$n = 2$）；$\omega_n$ は $n$ 次元単位球の表面積)
  - Proof (基本解の導出：$|x| > 0$ で球対称な調和関数 $\Phi(r)$（$r = |x|$）の ODE $\Phi'' + \frac{n-1}{r}\Phi' = 0$ を解くと $\Phi = cr^{2-n}$（$n \neq 2$）；定数 $c$ は $\int_{B_\epsilon}\Delta\Phi\,dx = 1$ の正規化条件から決定)
  - Example 8.4-1 ($n = 3$ の基本解：$\Phi(x) = -1/(4\pi|x|)$；これはニュートンの重力ポテンシャルおよびクーロン電位に対応；点電荷・点質量の「場」が基本解として実現されていることの物理的意味)

- **2. ポアソン方程式の解の表示**
  - Theorem 8.4-1 (ポアソン方程式の解の積分表示：$\mathbb{R}^n$ での $\Delta u = f$（$f$ はコンパクトサポート）の解は $u(x) = \int \Phi(x-y)f(y)\,dy = (\Phi * f)(x)$；「ソース $f$ が各点に基本解 $\Phi$ を置いた重ね合わせ」として解を表示できる)
  - Proof (積分表示の証明：$\Delta u = \Delta(\Phi * f) = (\Delta\Phi) * f = \delta_0 * f = f$；厳密には$\Phi$ の特異性を $\varepsilon$ 除外領域で処理してから $\varepsilon \to 0$ の極限を取る)
  - Example 8.4-2 ($n = 3$、$f(x) = e^{-|x|^2}$ のポアソン方程式：$u(x) = \int \frac{-1}{4\pi|x-y|}e^{-|y|^2}\,dy$；球対称性から $u$ も球対称で $u(r) = -\frac{1}{4\pi r}\int_0^r 4\pi s^2 e^{-s^2}\,ds$ + 同様の項；数値的に計算して $r \to 0$ での挙動を確認)
  - Remark (ポテンシャル理論との接続：ニュートン重力・クーロン静電気・ストークス流体などで現れる「遠隔力のポテンシャル」は数学的にはポアソン方程式 $\Delta u = f$（$f$：ソース密度）の問題であり、基本解による積分表示がそれらを統一的に記述する)

### §8.5 グリーン関数
**内容:** 対称性を持つ領域でのグリーン関数の構成と応用

- **1. グリーン関数の定義**
  - Definition 8.5-1 (グリーン関数：有界領域 $\Omega$ と基本解 $\Phi$ に対して $G(x,y) = \Phi(x-y) - H^x(y)$；ここで $H^x(\cdot)$ は $\Delta_y H^x = 0$（$y \in \Omega$）かつ $H^x = \Phi(x-\cdot)$（$y \in \partial\Omega$）を満たす補正項；$G$ は $y \in \partial\Omega$ でゼロになりディリクレ問題のグリーン関数という)
  - Theorem 8.5-1 (グリーン関数による解の表示：ディリクレ問題 $\Delta u = 0$（$\Omega$）、$u = g$（$\partial\Omega$）の解は $u(x) = -\int_{\partial\Omega} g(y)\frac{\partial G}{\partial n_y}(x,y)\,dS_y$（ポアソン積分公式の一般形）)
  - Proof (解の表示の証明：$u$ と $G(x,\cdot)$ にグリーンの第二恒等式を適用し、内部特異点 $x$ を $\varepsilon$ 球で除外して $\varepsilon \to 0$ の極限を取る)

- **2. 半空間と球でのグリーン関数の構成**
  - Example 8.5-1 (上半空間のグリーン関数（鏡像法）：$\Omega = \{y_n > 0\}$ のグリーン関数は $G(x,y) = \Phi(x-y) - \Phi(x^* - y)$（$x^*$ は $x$ の $y_n = 0$ 面に関する鏡像）；$y_n = 0$ で $G = 0$ になることの確認；ポアソン核 $P(x,y') = -\partial G/\partial y_n|_{y_n=0}$ の具体的な計算)
  - Example 8.5-2 (球のグリーン関数（ケルビン変換）：単位球 $\Omega = B_1(0)$ のグリーン関数は $G(x,y) = \Phi(x-y) - \Phi(|x| (y - x/|x|^2))$（$x$ の反転点 $x^* = x/|x|^2$ を使う）；球面でゼロになることの確認)
  - Remark (グリーン関数の普遍的重要性：グリーン関数は「点ソース $\delta_x$ に対する解」であり、任意のソースへの解が $G$ との畳み込みで表せる；量子力学のグリーン関数（伝播関数）はこの概念の拡張である（詳細は作用素環論（5-8）・数理物理学（A-11）で扱う）)

## Chapter 9: 解の正則性と一般解論
**テーマ:** PDE の解の滑らかさ・存在・一意性・連続依存性の一般的理論

### §9.1 弱解の概念と動機
**内容:** 古典解が存在しない例と弱解の定式化

- **1. 古典解の限界**
  - Example 9.1-1 (古典解が存在しない例：バーガーズ方程式（§4.3）で衝撃波が形成された後は $C^1$ 解が存在しない；弱解として不連続な衝撃波解が意味を持つことの再確認；もう一つの例として $u = |x|$ の場合：$u'' = 2\delta_0$（ディラックのデルタ関数）が現れ「古典的な 2 階微分」が存在しない)
  - Definition 9.1-1 (弱解：$L u = f$ の弱解とは、「すべてのテスト関数 $\phi \in C_0^\infty$ に対して $\int u L^*\phi\,dx = \int f\phi\,dx$（$L^*$ は $L$ の $L^2$ 共役）」が成立する局所可積分関数 $u$；部分積分によって微分をテスト関数に移すことで定義する)
  - Example 9.1-2 ($u'' = 2\delta_0$ の弱解：$\int u \phi''\,dx = 2\phi(0)$（テスト関数 $\phi$ に対して）を満たす関数は $u(x) = |x|$；$\int |x|\phi''\,dx = 2\phi(0)$ を部分積分 2 回で確認)

- **2. 弱解と古典解の関係**
  - Proposition 9.1-1 (古典解は弱解：$u$ が古典的な意味での解ならば弱解でもある；逆は一般に成立しない)
  - Proof (確認：$Lu = f$ が点ごとに成立するとき $\phi$ をかけて積分し部分積分すれば弱解の定式化が得られる)
  - Remark (ソボレフ空間との接続：弱解の正確な関数空間は $L^2$ ベースのソボレフ空間 $H^k(\Omega) = W^{k,2}(\Omega)$（$k$ 階偏導関数が $L^2$ に属する関数の空間）；弱解の存在は Lax-Milgram 定理（関数解析（4-1）で扱う）によって楕円型方程式について系統的に保証される)

### §9.2 楕円型方程式の正則性
**内容:** 内部での正則性の概論と解析性

- **1. 内部正則性の主張**
  - Theorem 9.2-1 (楕円型方程式の内部正則性：$\Omega$ 上で $\Delta u = f$ かつ $f \in C^k(\Omega)$ ならば $u \in C^{k+2}(\Omega)$；ソボレフ空間での版：$f \in H^k(\Omega)$ ならば任意のコンパクト $K \subset\subset \Omega$ で $u \in H^{k+2}(K)$)
  - Proof (証明のスケッチ：差分商を用いて弱解の $H^1$ 推定から $H^2$ 推定へと「ブートストラップ」させる；各段階で楕円型作用素のエネルギー評価を使う；詳細は関数解析（4-1）で扱う)
  - Example 9.2-1 (正則性の向上の確認：$\Delta u = f$ で $f = 0$（ラプラス方程式）のとき $u$ は $C^\infty$ かつ解析的（§8.1 Theorem 8.1-1 の再確認）；$f$ が不連続なら $u$ は $f$ より 2 階分だけ滑らか（例：$f \in L^2$ なら $u \in H^2$）)

- **2. 境界正則性**
  - Remark (境界近くでの正則性：滑らかな境界と滑らかな境界データがあれば、解は境界まで滑らか（$u \in C^{k+2}(\bar\Omega)$）になる；境界のコーナーや切れ込みでは正則性が落ちることがある（「角の特異性」）；これは工学的な亀裂進展解析（破壊力学）に重要な意味を持つ)

### §9.3 コーシー・コワレフスカヤの定理
**内容:** 解析的局所解の存在と一意性

- **1. 定理の主張**
  - Theorem 9.3-1 (コーシー・コワレフスカヤの定理：主部において $t$ の最高次微分が陽に解ける形 $\partial_t^m u = F(x, t, \partial^\alpha u)$（$|\alpha| \leq m$、$\alpha_0 < m$）の PDE で $F$ が解析的・初期データも解析的ならば、局所的に一意の解析的解が存在する)
  - Proof (証明のスケッチ：解をべき級数 $u = \sum a_\alpha (x-x_0)^\alpha (t-t_0)^k$ とおき係数の逐次決定を行う；優級数（majorant series）を構成して収束を示すコワレフスカヤのオリジナルの証明の概略)
  - Example 9.3-1 (定理の適用例：$u_t = u_x^2$（コーシー問題）の解析的初期値 $u(x,0) = 0$ への適用；局所解が $t$ のべき級数として構成されること；ただし大域的な解析解は存在しない場合があることの注意)

- **2. 定理の限界：ルレイの反例**
  - Example 9.3-2 (非解析的初期データへの拡張不可：$u_t = u_{xx}$（熱方程式）の「後退問題」$u_{-t} = u_{xx}$（すなわち $u_t = -u_{xx}$）は一般に不適切；$u(x,0) = \sum e^{-\sqrt{n}}\cos(nx)$（$C^\infty$ だが非解析的）に対して解が存在しないルレイの例の概観)
  - Remark (コーシー・コワレフスカヤ定理の位置づけ：この定理は「解析的カテゴリーでは PDE が常に解ける」という一般性を持つが、$C^\infty$ や $L^2$ レベルでは双曲型・楕円型・放物型が本質的に異なる挙動を示す；適切問題の概念（§9.4）が各型の正しい問題設定を規定する)

### §9.4 適切問題の概念
**内容:** ウェルポーズドの 3 条件と不適切な問題の例

- **1. 適切問題の定義**
  - Definition 9.4-1 (ハダマールの意味での適切問題（ウェルポーズド）：次の 3 条件がすべて満たされるとき、PDE の境界値問題または初期値問題を適切という；(i) 解の存在；(ii) 解の一意性；(iii) データへの連続依存性（小さなデータの変化が小さな解の変化を生む）)
  - Example 9.4-1 (各型の適切な問題設定：波動方程式 + 初期値問題（存在・一意・安定、§6.4）；熱方程式 + 初期境界値問題（存在・一意・安定、§7.5）；ラプラス方程式 + ディリクレ境界値問題（存在・一意・安定、§8.2）)

- **2. 不適切な問題の例**
  - Example 9.4-2 (ラプラス方程式の不適切なコーシー問題：$\Delta u = 0$（$y > 0$）、$u(x,0) = 0$、$u_y(x,0) = \varepsilon \sin(nx)/n$；厳密解は $u(x,y) = \varepsilon\sinh(ny)\sin(nx)/n^2$；$n \to \infty$ で初期データ $\to 0$ だが解は $y > 0$ で無限大に発散；連続依存性（条件 (iii)）の破綻；これがハダマールの不良設定問題の古典例)
  - Remark (不良設定問題の正則化：地球物理や医療画像（CT・MRI）では「データから内部の場を復元する」逆問題が本質的に不良設定問題になる；ティホノフ正則化などの方法で安定な近似解を求める手法は応用数学・機械学習の基盤にもなっている（詳細は偏微分方程式（5-2）で扱う）)
  - Example 9.4-3 (後退熱方程式の不適切性：$u_t = -u_{xx}$（後退熱方程式）の初期値問題；$u(x,0) = \sin(nx)$ に対して解は $u(x,t) = e^{n^2 t}\sin(nx)$；$t > 0$ で指数的に発散し、小さな初期値が無限大に成長する；楕円型方程式の不適切なコーシー問題と同じ構造)
